Atributos de dominio

Utilice la página Atributos del dominio para establecer atributos de dominio avanzados de Servidor empresarial.

La página Atributos del dominio contiene las siguientes opciones.

Raíz de aplicaciones

Ruta completa hacia el directorio donde residen las aplicaciones implementadas. El valor predeterminado es domain-dir/applications.

Raíz de registro

Identifica el lugar en el que se guardan los archivos de registro del servidor. El valor predeterminado es domain-dir/logs.

Configuración regional

Si se deja este campo en blanco, se utilizará la configuración regional predeterminada del host.

La configuración regional es un identificador que especifica una combinación concreta de idioma y región. Por ejemplo, la configuración regional para inglés de Estados Unidos es en_US y para japonés es ja_JP. Para utilizar una configuración regional que no sea del idioma inglés, Servidor empresarial tiene que estar localizado, un proceso que incluye traducir el inglés a otro idioma.
