Cómo establecer el módulo de auditoría activo

  1. En el árbol de navegación, expanda el nodo Configuración.
  2. En el nodo Configuración, seleccione el nodo Seguridad.

    Se abrirá la página Seguridad.

  3. En la lista desplegable Módulos de auditoría, seleccione el nombre del módulo de auditoría que utilizará el servidor.

    De manera predeterminada, hay un único módulo de auditoría disponible llamado default.

  4. Haga clic en Guardar.
