Registro del producto

Para registrar su instalación de Sun GlassFishTMServidor empresarial con Sun Connection, debe disponer de una cuenta en línea de Sun. Si no tiene una cuenta Sun Online, Servidor empresarial le permite crear una cuenta al registrar este producto.

La página Registro del producto contiene las siguientes opciones.

Ya tengo una cuenta Sun Online o una cuenta de Sun Developer Network (SDN)

Seleccione si ya tiene una cuenta. Si selecciona esta opción, se verán los siguientes campos.

Nombre de usuario

El nombre de usuario especificado al registrar su cuenta Sun Online.

Contraseña

La contraseña especificada al registrar su cuenta Sun Online.

Host del proxy

El nombre de host del servidor proxy del equipo en el que se está ejecutando Servidor empresarial.

Este campo sólo es aplicable si el equipo en el que se está ejecutando Servidor empresarial se conecta a Internet a través de un servidor proxy.

Puerto del proxy

El número de puerto del servidor proxy del equipo en el que se está ejecutando Servidor empresarial.

Este campo sólo es aplicable si el equipo en el que se está ejecutando Servidor empresarial se conecta a Internet a través de un servidor proxy.

No tengo ninguna cuenta en línea de Sun. Registrarme.

Seleccione esta opción si no tiene una cuenta y desea registrar su instalación de Servidor empresarial con Sun Connection. Si selecciona esta opción, se verán los siguientes campos.

Dirección de correo electrónico

La dirección de correo electrónico, que será su nombre de usuario de la cuenta en línea de Sun.

Contraseña

La contraseña que se utiliza con su Cuenta en línea de Sun. La contraseña distingue entre mayúsculas y minúsculas y debe tener 6–10 caracteres.

Volver a escribir la contraseña

Debe volver a escribir la contraseña que especificó en el campo Contraseña.

Nombre

Su nombre.

Apellido

Su apellido.

Empresa

La empresa para la que trabaja, si procede.

País/territorio

El país o territorio en el que reside.

Host del proxy

El nombre de host del servidor proxy del equipo en el que se está ejecutando Servidor empresarial.

Este campo sólo es aplicable si el equipo en el que se está ejecutando Servidor empresarial se conecta a Internet a través de un servidor proxy.

Puerto del proxy

El número de puerto del servidor proxy del equipo en el que se está ejecutando Servidor empresarial.

Este campo sólo es aplicable si el equipo en el que se está ejecutando Servidor empresarial se conecta a Internet a través de un servidor proxy.
