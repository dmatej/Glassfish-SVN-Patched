Propiedades del sistema

Consulte la página Propiedades del sistema para ver y administrar las propiedades del servidor de la aplicación.

Una propiedad del sistema define un valor común para una configuración a nivel del servidor. Puede consultar una propiedad del sistema en un campo de texto de la Consola de administración colocándola entre llaves y con un signo de dólar. Por ejemplo, para consultar una propiedad llamada prop-name, debería utilizar la sintaxis ${prop-name}.

La página Propiedades del sistema contiene la siguiente información para cada propiedad.

Nombre

Nombre de la propiedad.

Valor

Valor de la propiedad.

Descripción

Descripción adicional de la propiedad.

Existe un número de propiedades del sistema predefinidas pero no se muestran en esta página. Para obtener más información, consulte Propiedades del sistema predefinidas.
