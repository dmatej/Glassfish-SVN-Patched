Establecimiento del proveedor JACC activo

  1. En el árbol de navegación, expanda el nodo Configuración.
  2. En el nodo Configuración, seleccione el nodo Seguridad.

    Se abrirá la página Seguridad.

  3. En la lista desplegable JACC, seleccione el proveedor JACC que utilizará el servidor.

    De manera predeterminada, hay dos proveedores JACC disponibles, llamados default y simple. El proveedor predeterminado es default.

  4. Haga clic en Guardar.
