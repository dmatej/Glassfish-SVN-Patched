So ändern Sie das Administrator-Passwort

  1. Wählen Sie im Navigationsbaum den Knoten Enterprise Server aus.

    Die Seite "Allgemeine Informationen" wird geöffnet.

  2. Klicken Sie auf die Registerkarte "Administratorpasswort".

    Die Seite zeigt Benutzer-ID und Gruppenname für das aktuelle Administratorkonto an. Bei der Benutzer-ID handelt es sich standardmäßig um admin und beim Gruppennamen um asadmin.

  3. Geben Sie im Feld "Neues Passwort" das neue Passwort für das aktuelle Administratorkonto ein.
  4. Geben Sie in das Feld "Neues Passwort bestätigen" das Passwort des Benutzers erneut ein.
  5. Klicken Sie auf "Speichern".
