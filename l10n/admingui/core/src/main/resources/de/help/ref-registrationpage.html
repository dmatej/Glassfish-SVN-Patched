Produktregistrierung

Um Ihre Installation von Sun GlassFishTMEnterprise Server mit Sun Connection zu registrieren, benötigen Sie einSun-Onlinekonto. Wenn Sie nicht über ein Sun-Onlinekonto verfügen, ermöglicht Enterprise Server die Erstellung eines Kontos bei der Registrierung.

Die Seite Product Registration (Produktregistrierung) umfasst die folgenden Optionen.

Ich habe bereits ein Sun-Onlinekonto oder ein Sun Delevoper Network(SDN)-Konto.

Wählen Sie diese Option aus, wenn Sie bereits über ein Konto verfügen. Wenn Sie diese Option wählen, werden folgende Felder angezeigt.

Benutzername

Der Benutzername, den Sie beim Registrieren für Ihr Sun-Onlinekonto angegeben haben.

Passwort

Das Passwort, das Sie beim Registrieren für Ihr Sun-Onlinekonto angegeben haben.

Proxy-Host

Der Hostname des Proxy-Servers für den Computer, auf dem Enterprise Server ausgeführt wird.

Dieses Feld gilt nur, wenn der Computer, auf dem Enterprise Server ausgeführt wird, über einen Proxy-Server mit dem Internet verbunden wird.

Proxy-Port

Die Port-Nummer des Proxy-Servers für den Computer, auf dem Enterprise Server ausgeführt wird.

Dieses Feld gilt nur, wenn der Computer, auf dem Enterprise Server ausgeführt wird, über einen Proxy-Server mit dem Internet verbunden wird.

Ich besitze kein Sun-Onlinekonto. Anmelden.

Wählen Sie diese Option aus, wenn Sie nicht über ein Konto verfügen und Ihre Installation von Enterprise Server bei Sun Connection registrieren möchten. Wenn Sie diese Option wählen, werden folgende Felder angezeigt.

E-Mail-Adresse

Ihre E-Mail-Adresse, die als Benutzername für Ihr Sun-Onlinekonto verwendet wird.

Passwort

Das für Ihr Sun-Onlinekonto verwendete Passwort. Für das Passwort wird die Groß- und Kleinschreibung beachtet, und es muss 6–10 Zeichen umfassen.

Passwort wiederholen

Sie müssen das im Passwortfeld eingegebene Passwort wiederholen.

Vorname

Ihr Vorname.

Familienname

Ihr Nachname.

Firma

Die Firma, für die Sie arbeiten (sofern zutreffend).

Land

Das Land, in dem sich Ihr Standort befindet.

Proxy-Host

Der Hostname des Proxy-Servers für den Computer, auf dem Enterprise Server ausgeführt wird.

Dieses Feld gilt nur, wenn der Computer, auf dem Enterprise Server ausgeführt wird, über einen Proxy-Server mit dem Internet verbunden wird.

Proxy-Port

Die Port-Nummer des Proxy-Servers für den Computer, auf dem Enterprise Server ausgeführt wird.

Dieses Feld gilt nur, wenn der Computer, auf dem Enterprise Server ausgeführt wird, über einen Proxy-Server mit dem Internet verbunden wird.
