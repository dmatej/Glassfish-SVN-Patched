So stellen Sie den aktiven JACC-Anbieter ein

  1. Erweitern Sie in der Navigationsstruktur den Konfigurationsknoten.
  2. Wählen Sie unter dem Konfigurationsknoten den Sicherheitsknoten aus.

    Die Seite "Sicherheit" wird geöffnet.

  3. Wählen Sie aus der Dropdown-Liste JACC den JACC-Anbieter aus, der vom Server verwendet werden soll.

    Standardmäßig stehen zwei JACC-Anbieter zur Verfügung, die mit Standard und Einfach benannt sind. Der Standardwert ist Standard.

  4. Klicken Sie auf "Speichern".
