So aktivieren oder deaktivieren Sie die Überwachungsprotokollierung

  1. Erweitern Sie in der Navigationsstruktur den Konfigurationsknoten.
  2. Wählen Sie unter dem Konfigurationsknoten den Sicherheitsknoten aus.

    Die Seite "Sicherheit" wird geöffnet.

  3. Markieren Sie auf der Seite "Sicherheit" das Kontrollkästchen "Überwachungsprotokollierung aktiviert", um die Überwachungsprotokollierung zu aktivieren.

    Zum Aktivieren der Überwachungsprotokollierung müssen Sie auch die Eigenschaft auditOn für das aktive Überwachungsmodul auf wahr einstellen.

  4. Klicken Sie auf "Speichern".
