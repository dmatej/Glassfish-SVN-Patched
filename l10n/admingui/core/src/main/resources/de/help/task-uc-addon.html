So installieren Sie Add-On-Komponenten

Das Installieren einer Add-On-Komponente umfasst folgende Schritte:

  • Übertragung des Inhalts der Komponente und der Komponentenabhängigkeiten aus einem Netzwerk-Repository

  • Installation der Komponente unter dem Enterprise Server Installations-Image auf Ihrem System

Falls eine Komponente andere Komponenten erfordert, die noch nicht installiert sind, werden die erforderlichen Komponenten ebenfalls installiert.

  1. Wählen Sie in der Navigationsstruktur den Knoten "Update-Tool" aus.

    Die Seite "Verfügbare Add-Ons" wird geöffnet.

  2. Wählen Sie auf der Seite "Verfügbare Add-Ons" in der Tabelle mit den verfügbaren Add-Ons die zu instalierenden Komponenten aus.

    Sind keine Add-On-Komponenten verfügbar, ist die Tabelle leer.

    • Um eine einzelne Komponente auszuwählen, aktivieren Sie das Kontrollkästchen neben dem Namen der Komponente.
    • Um alle Komponenten auszuwählen, klicken Sie in der Kopfzeile der Tabelle auf das Symbol "Alle auswählen".
    • Um die Auswahl aller Komponenten aufzuheben, klicken Sie in der Kopfzeile der Tabelle auf das Symbol "Auswahl für alle aufheben".
  3. Klicken Sie auf „Installieren“.

    Sie werden in einem Dialogfeld aufgefordert zu bestätigen, dass die ausgewählten Komponenten und alle dafür erforderlichen Komponenten installiert werden sollen.

  4. Klicken Sie in diesem Dialogfeld auf "OK".

    Sie werden gefragt, ob Sie die Lizenzvereinbarungen für die zu installierenden Komponenten akzeptieren.

  5. Akzeptieren Sie die Lizenzvereinbarungen für die zu installierenden Komponenten.

    Enterprise Server bestätigt den Abschluss der Installation. Die Komponenten werden aus der Tabelle der Add-On-Komponenten gelöscht und zur Tabelle mit den installierten Komponenten auf der Seite "Installierte Komponenten" hinzugefügt.

  6. Starten Sie den Server neu.

    Weitere Informationen finden Sie in der folgenden Dokumentation:

    • To Stop a Domain in Sun GlassFish Enterprise Server v3 Administration Guide

    • To Start a Domain in Sun GlassFish Enterprise Server v3 Administration Guide
