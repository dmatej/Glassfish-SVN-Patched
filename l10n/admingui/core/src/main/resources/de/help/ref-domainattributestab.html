Domänen-Attribute

Verwenden Sie die Seite Domain Attributes (Domänen-Attribute), um die erweiterten Domänenattribute für den Enterprise Server festzulegen.

Die Seite Domain Attributes (Domänen-Attribute) umfasst die folgenden Optionen.

Application Root (Anwendungs-Root)

Der vollständige Pfad des Verzeichnisses, in dem sich bereitgestellte Anwendungen befinden. Der Standardwert ist domain-dir/applications.

Log Root (Protokoll-Root)

Gibt an, wo die Server-Protokolldateien gespeichert sind. Der Standardwert ist domain-dir/logs.

Locale (Ländereinstellung)

Wenn Sie dieses Feld leer lassen, wird die standardmäßige Ländereinstellung des Hosts verwendet.

Eine Ländereinstellung ist ein Bezeichner, der eine bestimmte Kombination aus Sprache und Region angibt. Beispielsweise lautet die Ländereinstellung für Englisch (USA) en_US, die Ländereinstellung für Japanisch lautet ja_JP. Um eine nicht englische Ländereinstellung zu verwenden, muss Enterprise Server lokalisiert, d. h. in eine andere Sprache übersetzt werden.

Verwandte Aufgaben

  • So führen Sie die "Erweiterte Anwendungskonfiguration" durch

  • So legen Sie die erweiterten Domänenattribute fest
