Vordefinierte Systemeigenschaften

Die folgenden Systemeigenschaften sind im Voraus für Enterprise Server festgelegt.

com.sun.aas.installRoot

Das Verzeichnis, in dem Enterprise Server installiert ist.

com.sun.aas.instanceRoot

Das obere Verzeichnis für eine Enterprise Server Domäne.

com.sun.aas.hostName

Der Name des Hosts (Rechner).

com.sun.aas.javaRoot

Das Installationsverzeichnis für die Java-Laufzeitumgebung.

com.sun.aas.imqLib

Das Bibliothekenverzeichnis für die Software Sun GlassFish Message Queue.

com.sun.aas.domainName

Der Name der Domäne. Diese Eigenschaft wird nicht in der Standard-Konfiguration verwendet, kann jedoch verwendet werden, um die Konfiguration anzupassen. Der Standardwert ist domain1.
