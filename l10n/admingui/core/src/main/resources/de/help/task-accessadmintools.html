So gewähren Sie Zugriff auf Administrationstools

Nur Benutzer in der Gruppe asadmin können auf die Administration Console und das Befehlszeilendienstprogramm asadmin zugreifen. Um Benutzern den Zugriff auf diese Verwaltungstools zu gewähren, fügen Sie sie der Gruppe asadmin im admin-realm hinzu.

  1. Erweitern Sie in der Navigationsstruktur den Konfigurationsknoten.
  2. Erweitern Sie den Knoten "Sicherheit".
  3. Erweitern Sie den Knoten "Bereiche".
  4. Wählen Sie den Knoten admin-realm.

    Die Seite "Bereich bearbeiten" wird geöffnet.

  5. Klicken Sie auf der Seite "Bereich bearbeiten" auf die Schaltfläche "Benutzer verwalten".

    Die Seite "Dateibenutzer" wird geöffnet.

    Nach der Installation werden die während der Installation eingegebenen Werte für Benutzername und Passwort des Administrators in einer Datei mit dem Namen admin-keyfile aufgelistet. Standardmäßig gehört dieser Benutzer zur Gruppe asadmin, welche über Berechtigungen zum Ändern von Enterprise Server verfügt. Fügen Sie zu dieser Gruppe nur Benutzer hinzu, denen Sie Administratorrechte für Enterprise Server zuweisen möchten.

    Wenn Sie einen Benutzer dem Bereich admin-realm, jedoch nicht der Gruppe asadmin zuweisen, werden die Benutzerinformationen zwar in die Datei admin-keyfile geschrieben, der Benutzer ist jedoch nicht für den Zugriff auf die Verwaltungstools oder Anwendungen im Bereich file berechtigt.

  6. Klicken Sie auf der Seite "Dateibenutzer" auf "Neu", um dem admin-realm-Bereich einen neuen Benutzer hinzuzufügen.

    Die Seite "Neuer Dateibereichsbenutzer" wird geöffnet.

  7. Geben Sie in das Feld "Benutzer-ID" den neuen Benutzernamen ein.
  8. Geben Sie in das Feld "Gruppenliste" die Namen der Gruppen ein, zu denen der Benutzer gehört.

    Um einem Benutzer Berechtigungen zum Durchführen von Änderungen an Enterprise Server zuzuweisen, fügen Sie die Gruppe asadmin im Feld "Gruppenliste" hinzu.

  9. Geben Sie in die Felder "Neues Passwort" und "Neues Passwort bestätigen" das Passwort für den neuen Benutzer ein.
  10. Klicken Sie auf "OK", um diesen Benutzer dem Bereich admin-realm hinzuzufügen, oder klicken Sie auf "Abbrechen", um den Vorgang ohne Speichern zu beenden.
