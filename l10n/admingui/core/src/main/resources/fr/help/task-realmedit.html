Modifier un domaine

  1. Dans l'arborescence de navigation, développez le nœud Configuration.
  2. Développez le nœud Sécurité.
  3. Développez le nœud Domaines.
  4. Sélectionnez le nom d'un domaine existant.

    La page Modification du domaine s'ouvre.

  5. Pour modifier le nom de la classe du domaine, procédez de l'une des manières suivantes :
    • Dans la liste déroulante Nom de la classe, sélectionnez le nom de classe du domaine à créer.
    • Sélectionnez le second bouton radio et saisissez le nom de classe du domaine personnalisé à créer.
  6. (Facultatif) Dans la section Autres propriétés, spécifiez d'autres propriétés.

    Les propriétés valides dépendent du type de domaine.

    • Pour une description des propriétés de domaine de fichier, consultez Propriétés spécifiques de la classe FileRealm.

    • Pour une description des propriétés de domaine de certificat, consultez Propriétés spécifiques de la classe CertificateRealm.

    • Pour une description des propriétés de domaine JDBC, consultez Propriétés spécifiques de la classe JDBCRealm.

    • Pour une description des propriétés de domaine LDAP, consultez Propriétés spécifiques à la classe LDAPRealm .

    • Pour une description des propriétés de domaine Solaris, consultez Propriétés spécifiques à la classe SolarisRealm .

    • Pour ajouter une propriété, cliquez sur le bouton Ajouter propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur.
    • Pour modifier une propriété, modifiez le champ Valeur de cette propriété.
    • Pour supprimer une propriété, cochez la case située à gauche du champ Nom de la propriété que vous supprimez, puis cliquez sur le bouton Supprimer propriétés.
  7. Cliquez sur Enregistrer.
