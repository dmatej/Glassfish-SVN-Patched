Configurer les options JVM

La page Options JVM vous permet de spécifier les options du lanceur de l'application Java (outil java) exécutant Enterprise Server. Les options -D désignent les propriétés spécifiques à Enterprise Server.

  1. Dans l'arborescence de navigation, sélectionnez le noeud Configuration.
  2. Sous le nœud Configuration, sélectionnez le nœud Paramètres JVM.

    La page Paramètres JVM s'ouvre.

  3. Dans la page Paramètres généraux JVM, cliquez sur l'onglet Options JVM.

    La page Options JVM s'ouvre.

  4. Pour modifier une option, modifiez le contenu du champ Valeur.
  5. Pour ajouter une option :
    1. Cliquez sur le bouton Ajouter une option JVM.
    2. Sur la ligne vide qui apparaît, saisissez les informations d'option dans le champ Valeur.
  6. Pour supprimer une option :
    1. Cochez la case située à gauche du champ Valeur de l'option à supprimer.
    2. Cliquez sur Delete.
  7. Cliquez sur Enregistrer.
  8. Redémarrez le serveur.
