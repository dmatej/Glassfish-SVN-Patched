Détails des composants

La page Détails des composants contient des informations détaillées sur un composant.

La page Détails des composants contient les options suivantes :

Précédente

Ce bouton permet de revenir à la page sur laquelle vous avez sélectionné le composant dont vous consultez les détails.

Nom du package

Nom du package comprenant le composant.

Identificateur unique

Identificateur complet du composant tel qu'il existe dans le référentiel.

Catégorie

Catégorie du logiciel à laquelle le composant est associé, comme par exemple serveurs d'application, outils système et bases de données. La catégorie est affectée au composant par le fabricant du composant au moment de sa création.

Version

Numéro de version du package comprenant le composant.

Taille de l'installation

Espace disque utilisé pour l'installation du composant.

Date de publication

Date et heure auxquelles cette version du composant a été publiée dans le référentiel. Cette date ne correspond pas à la date et à l'heure auxquelles le composant a été installé dans l'image d'installation.

Source

Pour un composant installé, nom du Référentiel réseau à partir duquel le composant a été obtenu.

Pour un composant add-on ou une mise à jour, nom du référentiel réseau à partir duquel le composant a été obtenu.

Description

Présentation du composant.

Tâches connexes

Afficher les informations détaillées d'un composant

aucune.
