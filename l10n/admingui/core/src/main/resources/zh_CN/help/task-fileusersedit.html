编辑 File 领域用户信息

  1. 在导航树中，展开“配置”节点。
  2. 展开“安全性”节点。
  3. 展开“领域”节点。
  4. 选择 "File" 节点。

    将打开“编辑领域”页面。

  5. 在“编辑领域”页面上，单击“管理用户”按钮。

    将打开“文件用户”页面。

  6. 在“文件用户”页面上表的“用户 ID”列中，单击要修改的用户名称。

    将打开“编辑 File 领域用户”页面。

  7. 编辑“组列表”字段，以修改用户所属的组。

    使用逗号分隔组名。无需在其他位置定义这些组。

  8. 在“新密码”和“确认新密码”字段中，键入一个新密码，以更改用户的密码。
  9. 单击“保存”。
