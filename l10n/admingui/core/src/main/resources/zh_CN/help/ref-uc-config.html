更新工具配置

“配置”页面提供了有关更新工具的配置信息，并允许您更改更新工具的配置。

“更新工具配置”页面包含以下选项：

安装路径

安装映像的根目录或顶层目录的文件系统路径。

首选系统信息库

与此安装映像相关联的首选网络系统信息库的名称，例如 glassfish.org。

Internet 连接

Enterprise Server 连接到 Internet 的方式：

直接连接

此选项指定 Enterprise Server 直接连接到 Internet。

通过代理服务器

此选项指定 Enterprise Server 通过代理服务器连接到 Internet。选择此选项后，“代理主机”字段和“代理端口”字段都将添加到此页面中。

代理主机

运行 Enterprise Server 的计算机的代理服务器的主机名。

仅当运行 Enterprise Server 的计算机通过代理服务器连接到 Internet 时，此字段才适用。

代理端口

运行 Enterprise Server 的计算机的代理服务器的端口号。

仅当运行 Enterprise Server 的计算机通过代理服务器连接到 Internet 时，此字段才适用。

系统信息库

此表提供了与此安装映像相关联的每个网络系统信息库的以下信息：

系统信息库

系统信息库的名称，例如 glassfish.org。

相关任务

为访问网络系统信息库配置网络代理设置

相关的 asadmin 命令

无。
