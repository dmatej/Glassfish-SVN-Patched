JVM 类路径设置

类路径就是 JAR 文件的列表，Java 运行时环境在此列表中搜索类和其他资源。

“JVM 类路径设置”页面包含以下选项：

环境类路径

如果选中“忽略”复选框，将忽略 CLASSPATH 环境变量。对于编程基本教程而言，CLASSPATH 环境变量使用起来很方便，但建议不要将此变量用于企业环境。

服务器类路径

Enterprise Server 类路径。这是只读字段。

系统类路径

系统中的类的路径。

类路径前缀

要前置到服务器类路径的 JAR 文件完整路径，每行一个。

类路径后缀

要附加到服务器类路径的 JAR 文件完整路径，每行一个。

本地库路径前缀

要前置到本地库路径的 JAR 文件完整路径，每行一个。

本地库路径后缀

要附加到本地库路径的 JAR 文件完整路径，每行一个。

将服务器的本地共享库的相对路径、标准运行时环境本地库路径、shell 环境设置（UNIX® 系统中的 LD_LIBRARY_PATH）以及在 JVM 事件探查器设置页面中指定的任意路径连接起来就构成了本地库路径。
