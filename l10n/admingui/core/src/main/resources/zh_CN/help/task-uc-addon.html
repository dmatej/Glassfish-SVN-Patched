安装附加组件

安装附加组件涉及以下操作：

  • 传输来自网络系统信息库的组件内容和组件的依赖关系

  • 在系统上的 Enterprise Server 安装映像下安装组件

如果组件需要尚未安装的任何其他组件，则还须安装所需要的组件。

  1. 在导航树中，选择“更新工具”节点。

    此时将打开“可用的附加软件”页面。

  2. 在“可用的附加软件”页面上的可用附加组件表中，选择要安装的组件。

    如果没有可用的附加组件，则此表为空。

    • 要选择单个组件，请选择该组件名称旁边的复选框。
    • 要选择所有组件，单击表头中的“全选”图标。
    • 要取消选择所有组件，请单击表头中的“取消全选”图标。
  3. 单击“安装”。

    会出现一个对话框，要求您确认是否应该安装所选的组件以及其所需的组件。

  4. 在此对话框中，单击“确定”。

    系统将要求您接受要安装的组件的许可证协议。

  5. 接受要安装的组件的许可证协议。

    Enterprise Server 确认安装已完成。各组件将从附加组件表中删除，并添加到“安装的组件”页面上的已安装组件表中。

  6. 重新启动服务器。

    有关详细说明，请参见以下文档：

    • 《Sun GlassFish Enterprise Server v3 Administration Guide》中的"To Stop a Domain"

    • 《Sun GlassFish Enterprise Server v3 Administration Guide》中的"To Start a Domain"
