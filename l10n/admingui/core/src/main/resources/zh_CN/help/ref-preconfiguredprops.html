预定义系统属性

以下为 Enterprise Server 的预定义系统属性：

com.sun.aas.installRoot

安装 Enterprise Server 的目录。

com.sun.aas.instanceRoot

Enterprise Server 域的顶级目录。

com.sun.aas.hostName

主机（计算机）名称。

com.sun.aas.javaRoot

Java 运行时的安装目录。

com.sun.aas.imqLib

Sun GlassFish Message Queue 软件的库目录。

com.sun.aas.domainName

域名。在默认配置中，不使用此属性，但可以使用此属性来自定义配置。默认值为 domain1。
