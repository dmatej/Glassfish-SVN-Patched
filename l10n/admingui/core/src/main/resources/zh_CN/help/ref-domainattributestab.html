域属性

使用“域属性”页面可以设置 Enterprise Server 的高级域属性。

“域属性”页面包含以下选项：

应用程序根目录

已部署应用程序所在的目录的完整路径。默认值为 domain-dir/applications。

日志根目录

确定保存服务器日志文件的位置。默认值为 domain-dir/logs。

语言环境

如果将此字段留为空白，将使用主机的默认语言环境。

语言环境是一种标识符，用于指定特定的语言和区域组合。例如，美国英语的语言环境是 en_US，日语的语言环境是 ja_JP。要使用非英语语言环境，Enterprise Server 必须经过本地化，即必须将英语翻译成其他语言。
