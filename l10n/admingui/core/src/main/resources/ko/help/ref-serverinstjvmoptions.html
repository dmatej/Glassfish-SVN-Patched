JVM 옵션

JVM 옵션 페이지에서는 Enterprise Server를 실행하는 Java 응용 프로그램 실행 프로그램(java)을 지정할 수 있습니다. -D 옵션은 Enterprise Server 고유의 등록 정보를 지정합니다.

  • 옵션을 수정하려면 해당 옵션의 값 필드를 편집합니다.

  • 옵션을 추가하려면 JVM 옵션 추가 버튼을 누릅니다. 빈 행이 나타나면 값 필드에 옵션 정보를 입력합니다.

  • 옵션을 삭제하려면 삭제할 옵션의 값 필드 왼쪽에 있는 확인란을 선택한 다음 삭제를 누릅니다.
