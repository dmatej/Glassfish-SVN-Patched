활성 감사 모듈을 설정하는 방법

  1. 탐색 트리에서 구성 노드를 선택합니다.
  2. 구성 노드에서 보안 노드를 선택합니다.

    보안 페이지가 열립니다.

  3. 감사 모듈 드롭다운 목록에서 서버가 사용할 감사 모듈 이름을 선택합니다.

    기본적으로 default라는 이름의 감사 모듈 하나만 사용할 수 있습니다.

  4. 저장을 누릅니다.
참조

  • 감사 모듈을 만드는 방법

  • 감사 모듈을 편집하는 방법

  • 감사 모듈을 삭제하는 방법

  • 감사 로깅을 활성화하거나 비활성화하는 방법

  • 기본 감사 모듈을 사용하는 방법
