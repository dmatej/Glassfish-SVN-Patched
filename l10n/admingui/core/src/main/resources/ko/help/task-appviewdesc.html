응용 프로그램의 설명자를 보는 방법

  1. 탐색 트리에서 응용 프로그램 노드를 선택합니다.

    응용 프로그램 페이지가 열립니다.

  2. 응용 프로그램 페이지에서 설명자를 볼 응용 프로그램의 이름을 누릅니다.

    응용 프로그램 편집 페이지가 열립니다.

  3. 응용 프로그램 편집 페이지에서 설명자 탭을 누릅니다.

    모듈 설명자 페이지가 열립니다.

  4. 배포 설명자 테이블에서 보려는 XML 파일의 이름을 누릅니다.

    XML 파일이 새 창에서 열립니다.

  • 배포된 응용 프로그램의 목록을 보는 방법

  • 엔터프라이즈 응용 프로그램을 배포하는 방법

  • 응용 프로그램 클라이언트를 배포하는 방법

  • 커넥터 모듈을 배포하는 방법

  • EJB Jar를 배포하는 방법

  • Ruby 응용 프로그램을 배포하는 방법

  • 배포된 응용 프로그램을 편집하는 방법

  • 응용 프로그램을 시작하는 방법

  • 배포된 응용 프로그램을 재배포하는 방법

  • 배포된 응용 프로그램을 다시 시작하는 방법
