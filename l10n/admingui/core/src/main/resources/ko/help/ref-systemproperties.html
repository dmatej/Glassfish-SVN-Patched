시스템 등록 정보

시스템 등록 정보 페이지에서는 Enterprise Server 등록 정보를 보고 관리할 수 있습니다.

시스템 등록 정보는 서버 수준에서의 설정에 대한 공통 값입니다. 달러 기호와 굽은 괄호로 묶어 관리 콘솔 텍스트 필드의 시스템 등록 정보를 참조할 수 있습니다. 예를 들어 prop-name이라는 등록 정보를 참조하려면 구문 ${prop-name}을 사용합니다.

각 등록 정보에 대해 시스템 등록 정보 페이지에는 다음 정보가 포함됩니다.

이름

등록 정보의 이름입니다.

값

등록 정보의 값입니다.

설명

등록 정보에 대한 선택적 설명입니다.

여러 개의 시스템 등록 정보가 사전 정의되어 있지만 이 페이지에는 나와 있지 않습니다. 자세한 내용은 사전 정의된 시스템 등록 정보를 참조하십시오.
