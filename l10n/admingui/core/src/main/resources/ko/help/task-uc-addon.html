애드온 구성 요소 설치 방법

애드온 구성 요소 설치에는 다음 작업이 포함됩니다.

  • 네트워크 저장소에서 구성 요소 내용 및 구성 요소 종속성 전송

  • 시스템의 Enterprise Server 설치 이미지에 구성 요소 설치

구성 요소가 아직 설치되지 않은 다른 구성 요소가 필요한 경우 필요한 구성 요소도 설치됩니다.

  1. 탐색 트리에서 업데이트 도구 노드를 선택합니다.

    사용 가능한 애드온 페이지가 열립니다.

  2. 사용 가능한 애드온 페이지의 사용 가능한 애드온 구성 요소 테이블에서 설치할 구성 요소를 선택합니다.

    애드온 구성 요소를 사용할 수 없는 경우 테이블이 비어 있습니다.

    • 개별 구성 요소를 선택하려면 구성 요소 이름 옆에 있는 확인란을 선택합니다.
    • 모든 구성 요소를 선택하려면 테이블 헤더에서 모두 선택 아이콘을 누릅니다.
    • 모든 구성 요소를 선택 해제하려면 테이블 헤더에서 모두 선택 해제 아이콘을 누릅니다.
  3. 설치를 누릅니다.

    선택한 구성 요소 및 필요한 모든 구성 요소를 설치해야 함을 확인하라는 대화 상자가 나타납니다.

  4. 대화 상자에서 확인을 누릅니다.

    설치할 구성 요소의 사용권 계약에 동의할 것을 묻는 메시지가 표시됩니다.

  5. 설치할 구성 요소의 사용권 계약에 동의합니다.

    Enterprise Server는 설치가 완료되었음을 확인합니다. 구성 요소가 애드온 구성 요소 테이블에서 제거되고 설치된 구성 요소 페이지의 설치된 구성 요소 테이블에 추가됩니다.

  6. 서버를 다시 시작합니다.

    자세한 지침은 다음 설명서를 참조하십시오.

    • Sun GlassFish Enterprise Server v3 Administration Guide의 To Stop a Domain

    • Sun GlassFish Enterprise Server v3 Administration Guide의 To Start a Domain
