新建 File 範圍使用者

使用 [新建 File 範圍使用者] 頁面可授予新使用者對目前範圍的存取權。

[新建 File 範圍使用者] 頁面包含下列選項。

範圍名稱

目前所選範圍的名稱。此為唯讀欄位。

使用者 ID

新使用者帳號的使用者 ID。

群組清單

新使用者所屬的群組。以逗號分隔多個群組名稱。

針對 admin-realm 使用者，此為唯讀欄位，群組為 asadmin。

新密碼

新使用者的密碼。

確認新密碼

新使用者的密碼。

相關作業

  • 將使用者增加至 File 範圍

  • 編輯 File 範圍使用者資訊

  • 刪除 File 範圍中的使用者
