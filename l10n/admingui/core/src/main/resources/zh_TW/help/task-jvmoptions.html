配置 JVM 選項

使用 [JVM 選項] 頁面可指定執行 Enterprise Server 的 Java 應用程式啟動程式 (java) 的選項。-D 選項會指定專屬於 Enterprise Server 的特性。

  1. 在瀏覽樹狀結構中，選取 [配置] 節點。
  2. 在 [配置] 節點下，選取 [JVM 設定] 節點。

    [JVM 一般設定] 頁面會隨即開啟。

  3. 在 [JVM 一般設定] 頁面上，按一下 [JVM 選項] 標籤。

    [JVM 選項] 頁面會隨即開啟。

  4. 若要修改選項，請編輯選項的 [值] 欄位。
  5. 若要增加選項，請執行以下步驟：
    1. 按一下 [增加 JVM 選項] 按鈕。
    2. 在顯示的空白列中，在 [值] 欄位中鍵入選項資訊。
  6. 若要刪除選項：
    1. 選取要刪除的選項之 [值] 欄位左側的核取方塊。
    2. 按一下 [刪除]。
  7. 按一下 [儲存]。
  8. 重新啟動伺服器。
