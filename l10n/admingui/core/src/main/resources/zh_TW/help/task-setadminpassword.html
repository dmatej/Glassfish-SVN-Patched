變更管理員密碼

  1. 在瀏覽樹狀結構中，選取 [Enterprise Server] 節點。

    此時將開啟 [一般資訊] 頁面。

  2. 選取 [管理員密碼] 標籤。

    該頁面將顯示目前管理員帳號的使用者 ID 和群組名稱。根據預設，使用者 ID 為 admin，而群組名稱為 asadmin 。

  3. 在 [新密碼] 欄位中，鍵入目前管理員帳號的新密碼。
  4. 在 [確認新密碼] 欄位中，再次鍵入新密碼。
  5. 按一下 [儲存]。
