訊息安全性提供者配置專屬的特性

下列特性適用於訊息安全性提供者配置。

server.config

包含伺服器配置資訊的 XML 檔案名稱和所在目錄。預設值為 domain-dir/config/wss-server-config-1.0.xml 。

debug

若為 true，則會將伺服器提供者除錯訊息傾印至伺服器記錄。預設值為 false。

dynamic.username.password

若為 true，則會指示提供者執行階段針對每個請求，從 CallbackHandler 收集使用者名稱和密碼。若為 false，則會在模組初始化期間收集 wsse:UsernameToken(s) 的使用者名稱和密碼一次。此特性僅適用於 ClientAuthModule。預設值為 false。

encryption.key.alias

指定提供者使用的加密金鑰。金鑰會以其 keystore 別名指定。預設值為 s1as。

signature.key.alias

指定提供者使用的簽名金鑰。金鑰會以其 keystore 別名指定。預設值為 s1as。
