配置網路代理伺服器設定以存取網路儲存庫

網路代理伺服器設定會決定 Enterprise Server 連線至網際網路的方式。若要啟用更新工具以透過網際網路存取網路儲存庫，請配置網路代理伺服器設定。

  1. 在瀏覽樹狀結構中，選取 [更新工具] 節點。

    [可用的附加元件] 頁面會隨即開啟。

  2. 在 [可用的附加元件] 頁面上，按一下 [配置] 標籤。

    [更新工具配置] 頁面會隨即顯示。

  3. 在 [更新工具 配置] 頁面上，指定執行 Enterprise Server 的機器連線至網際網路的方式。
    • 如果機器直接連線至網際網路，請選取 [直接連線] 選項。
    • 如果機器透過代理伺服器連線至網際網路，請選取 [透過代理] 選項。
  4. 如果執行 Enterprise Server 的機器透過代理伺服器連線至網際網路，請提供代理伺服器的代理主機名稱和代理連接埠號碼。

    如果執行 Enterprise Server 的機器直接連線至網際網路，則請略過此步驟。

    Web 瀏覽器可能正在執行 Enterprise Server 的機器上執行。在此情況下，您可以從 Web 瀏覽器取得代理伺服器的主機名稱和連接埠號碼。否則，請詢問您的系統管理員，以取得此資訊。

  5. 按一下 [儲存]。
