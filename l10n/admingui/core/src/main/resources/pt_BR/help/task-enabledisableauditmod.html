Para ativar ou desativar logs de auditoria

  1. Na árvore de navegação, expanda o nó Configuração.
  2. Sob o nó Configuração, selecione o nó Segurança.

    A página Segurança é exibida.

  3. Na página Segurança, marque a caixa de seleção Logs de Auditoria Ativados para ativar os logs de auditoria.

    Para ativar o logs de auditoria, você deve definir também a propriedade auditOn do módulo de auditoria ativo para verdadeiro.

  4. Clique em Salvar.
