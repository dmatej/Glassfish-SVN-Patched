Editar Conector JMX

Use a página Editar conector JMX para editar o conector JMX compatível com JSR 360.

A página Editar conector JMX contém as seguintes opções.

Nome

O nome do conector JMX. Este campo é somente leitura.

Protocol JMX

O nome do protocolo que esse conector JMX suporta. Este campo é somente leitura.

JMXServiceURL

O URL de serviços JMX. Este campo é somente leitura.

Segurança

Se a caixa de seleção Disponível estiver marcada, a comunicação JMX será criptografada. Por padrão, a opção está desativada.

Endereço

O endereço IP do serviço de nomes no qual o stub do servidor do conector JMX está registrado. Esta não é a porta do soquete do servidor que faz a comunicação JMX real. Este é o endereço da interface de rede onde o registro do RMI foi iniciado. Se o seu sistema tem múltiplas interfaces de rede, modifique esse valor para que apenas uma determinada interface seja selecionada. O valor padrão é 0.0.0.0.

Porta

O número da porta na qual o serviço de nomes (registro RMI) escutam conexões do cliente RMI. O único uso deste serviço de nomes é para baixar os stubs RMI. Se a porta padrão estiver ocupada, uma porta livre será usada. Os valores legais são 1- 65535. Em sistemas UNIX®, a criação de soquetes que escutam nas portas 1 - 1024 requer privilégios de superusuário. O valor padrão é 8686.

Nome de Domínio

O nome que representa o domínio administrativo especial. Todas as autenticações são manuseadas por este domínio. O valor padrão é admin-realm .

Propriedades Adicionais

Propriedade adicional para o conector JMX. Diversas propriedades são especificadas por padrão.
