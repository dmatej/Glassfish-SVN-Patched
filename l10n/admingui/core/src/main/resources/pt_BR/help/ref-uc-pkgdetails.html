Detalhes do Componente

A página Detalhes do Componente fornece informações detalhadas sobre um componente.

A página Detalhes do Componente contém estas opções:

Voltar

Este botão permite voltar à página em que você selecionou o componente cujos detalhes está visualizando.

Nome do Pacote

O nome do pacote em que o componente foi fornecido.

Identificador Exclusivo

O identificador totalmente qualificado do componente, tal como apresentado no repositório.

Categoria

A categoria de software que é associada ao componente; por exemplo, Servidores de Aplicativos, Ferramentas do Sistema e Bancos de Dados. A categoria é atribuída pelo autor do componente quando o componente é criado.

Versão

O número de versão do pacote em que o componente foi fornecido.

Tamanho Instalado

A quantidade de espaço em disco que a instalação do componente consome.

Data da Publicação

A data e a hora em que a versão do componente foi publicada no repositório. Essa data não é a data e hora em que o componente foi instalado na imagem de instalação.

Origem

Para um componente instalado, o nome do Repositório de rede de onde o componente foi obtido.

Para um componente de suplemento ou para uma atualização, o nome do repositório de rede de onde o componente deve ser obtido.

Descrição

Uma visão geral do componente.
