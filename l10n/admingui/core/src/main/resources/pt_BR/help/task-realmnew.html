Para criar um domínio

O Enterprise Server vem pré-configurado com estes domínios: arquivo, certificado e domínio admin. Também é possível criar ldap, JDBC, solaris e domínios personalizados. Geralmente, você tem um domínio de cada tipo em um servidor, mas, no Enterprise Server, há dois domínios de arquivo: arquivo e domínio admin. Há dois domínios do mesmo tipo, mas usados para finalidades diferentes: É possível também ter um banco de dados de certificado para cada servidor virtual do sistema.

  1. Na árvore de navegação, expanda o nó Configuração.
  2. Expanda o nó Segurança.
  3. Selecione o nó Domínios.

    É aberta a página Domínios.

  4. Na página Domínios, clique em Novo.

    É aberta a página Novo Domínio.

  5. No campo Nome, digite um nome para o domínio.
  6. Para especificar o nome da classe do domínio, faça uma das seguintes coisas:
    • Na lista suspensa Nome da Classe, selecione o nome da classe do domínio que está sendo criado.

      As opções válidas são mostradas na seguinte tabela.

      Nome de Domínio

      Nome da Classe

      arquivo

      com.sun.enterprise.security.auth.realm.file.FileRealm

      certificado

      com.sun.enterprise.security.auth.realm.certificate.CertificateRealm

      ldap

      com.sun.enterprise.security.auth.realm.ldap.LDAPRealm

      JDBC

      com.sun.enterprise.security.auth.realm.jdbc.JDBCRealm

      solaris

      com.sun.enterprise.security.auth.realm.solaris.SolarisRealm
    • Selecione o botão de opções inferior e digite o nome da classe do domínio personalizado que está sendo criado.
  7. (Opcional) Na seção Propriedades Adicionais, especifique essas propriedades.

    Para adicionar uma propriedade, clique no botão Adicionar Propriedade. Na linha em brando que é exibida, digite o nome da propriedade no campo Nome e digite o valor da propriedade no campo Valor.

    As propriedades válidas dependem do tipo de domínio.

    • Para obter uma descrição das propriedades Domínio de Arquivo, consulte Propriedades específicas da classe FileRealm.

    • Para obter uma descrição das propriedades Domínio de Certificado, consulte Propriedades específicas da classe CertificateRealm .

    • Para obter uma descrição das propriedades Domínio JDBC, consulte Propriedades específicas para a classe JDBCRealm .

    • Para obter uma descrição das propriedades Domínio LDAP, consulte Propriedades específicas da classe LDAPRealm .

    • Para obter uma descrição das propriedades Domínio do Solaris, consulte Propriedades específicas da classe SolarisRealm .

  8. Clique em OK.
