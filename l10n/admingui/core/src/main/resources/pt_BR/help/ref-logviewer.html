Visualizador de Logs

Use a página Visualizador de Logs para exibir o log do servidor e para personalizar a exibição de log.

A página Visualizador de Logs contém três áreas:

  • Critérios de pesquisa

  • Pesquisa avançada

  • Resultados da pesquisa

A área Critérios de Pesquisa contém estas opções.

Pesquisa Advançada

Clicar neste link abre uma área que permite fazer ajustes adicionais ao visualizador de logs.

Carimbo de Hora

O botão de opção Mais Recente exibe as mensagens mais recentes. O botão de opção Intervalo Específico abre os campos De e Para que permite exibir mensagens apenas de um determinado período de tempo.

Para o valor Tempo, a sintaxe deve considerar que esta forma (SSS significa milissegundos):

hh:mm:ss.SSS

Por exemplo:

17:10:00.000

Se o valor De for posterior ao valor Para, aparecerá uma mensagem de erro.

Nível de Log

O nível de log selecionado para filtragem. Por padrão, a exibição inclui todas as mensagens que aparecem no log do servidor, no nível de log escolhido e nos níveis mais graves. Marque a caixa de seleção rotulada “Não incluir mais mensagens graves” para exibir mensagens apenas no nível escolhido

Para assegurar que as mensagens que você deseja exibir apareçam no log do servidor, primeiro defina os níveis de log apropriados na página Níveis de Log do Módulo. Ver Para configurar níveis de log

Se você escolher filtrar as mensagens de log com base no nível de log, apenas as mensagens que correspondem aos critérios de filtro específicos serão mostradas. Contudo, essa filtragem não afeta quais mensagens são registradas no log do servidor.

A área Pesquisa Avançada contém as seguintes opções.

Agente de Log

Espaços para nomes do Agente de Log a serem filtrados por módulo. Use shift-clique ou control-clique para escolher múltiplos espaços para nomes.

Selecionar um espaço para nomes em um nível mais elevado seleciona todos os espaços para nomes abaixo dele. Por exemplo, selecionar javax.enterprise.system também selecione os agentes de log de todos os módulos naquele espaço para nomes: javax.enterprise.system.core , javax.enterprise.system.tools.admin e assim por diante.

Agente de Log Personalizado

Os Agentes de Log específicos de um determinado aplicativo, um por linha.

Se o aplicativo tiver diversos módulos, você poderá exibir qualquer um deles. Por exemplo, suponha que o aplicativo tenha agentes de log com os seguintes nomes:

com.mycompany.myapp.module1
com.mycompany.myapp.module2
com.mycompany.myapp.module3

Para exibir as mensagens de todos os módulos no aplicativo, digite com.mycompany.myapp . Para exibir apenas mensagens do module2, digite com.mycompany.myapp.module2.

Quando você especifica um ou mais agentes de logs, as mensagens dos módulos do Enterprise Server aparecem apenas se você as especificar explicitamente na área do Agente de Log.

Pares nome-valor

O nome da chave de um segmento, para exibir a saída de um segmento específico. O nome da chave é _ThreadID. Por exemplo:

_ThreadID=13

Suponha que com.mycompany.myapp.module2 é executado em diversos segmentos. Para refinar o visualizador de logs para mostrar apenas a saída de um único segmento, especifique agente de log daquele módulo no campo Agente de Log Personalizado e especifique o ID do segmento neste campo.

Exibir

O número de mensagens a exibir por vez. O número padrão de mensagens exibidas é 40. Outras opções são 100, 250 e 1.000.

Para exibir os rastreamentos de pilha, desmarque a caixa de seleção “Limitar mensagens excessivamente longas”. Por padrão, os rastreamentos de pilha não aparecem no visualizador; para exibi-las, clique no link (detalhes) para obter uma mensagem.

A tabela Resultados da Pesquisa exibe as mensagens do log do servidor.

Para cada mensagem, a tabela Resultados da Pesquisa contém as seguintes informações.

Número do Registro

O número da mensagem no log do servidor.

Nível do Log

O nível do log da mensagem.

Mensagem

O texto breve da mensagem de log. Clique no link (detalhes) para exibir uma versão formatada de uma mensagem.

Agente de Log

O módulo do agente de log que gerou a mensagem de log.

Carimbo de Hora

A data e a hora em que a mensagem de log foi emitida.

Pares nome/valor

O nome e a ID do segmento da mensagem de log.
