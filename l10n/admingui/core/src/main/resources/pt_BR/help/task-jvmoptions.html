Para configurar as opções de JVM

Use a página Opções de JVM para especificar as opções do lançador de aplicativos Java (java) que executa o Enterprise Server. As opções -D designam as propriedades específicas do Enterprise Server.

  1. Na árvore de navegação, selecione o nó Configuração.
  2. Sob o nó Configuração, selecione o nó Configurações de JVM.

    A página Configurações Gerais de JVM é exibida.

  3. Na página Configurações Gerais de JVM, clique na guia Opções de JVM.

    A página Opções de JVM é exibida.

  4. Para modificar uma opção, edite o campo Valor dessa opção.
  5. Para adicionar uma opção:
    1. Clique no botão Adicionar Opção de JVM.
    2. Na linha em branco que é exibida, digite as informações da opção do campo Valor.
  6. Para excluir uma opção:
    1. Marque a caixa de seleção à esquerda do campo Valor da opção a ser excluída.
    2. Clique em Excluir.
  7. Clique em Salvar.
  8. Reinicie o servidor.
