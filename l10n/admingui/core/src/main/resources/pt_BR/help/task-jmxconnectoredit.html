Para editar o conector JMX

Use o Serviço de Administração para configurar um conector JMX remoto compatível com JSR-160.

O conector JMX remoto compatível com JSR-160 lida com a comunicação entre clientes JMX, como o JConsole, e o serviço JMX, para fins de gerenciamento de domínio.

  1. Na árvore de navegação, expanda o nó Configuração.
  2. Sob o nó Configuração, selecione o nó Serviço de Administração.

    A página Editar Conector JMX é exibida.

  3. Marque a caixa de seleção Segurança Ativada para que a comunicação JMX seja criptografada.

    Por padrão, a opção está desativada.

  4. No campo Endereço, digite o endereço IP do serviço de nomes onde o stub do servidor do conector JMX está registrado.

    Esta não é a porta do soquete do servidor que faz a comunicação JMX real. Este é o endereço da interface de rede onde o registro do RMI foi iniciado. Se o seu sistema tem múltiplas interfaces de rede, modifique esse valor para que apenas uma determinada interface seja selecionada. O valor padrão é 0.0.0.0.

  5. No campo Porta, digite o número da porta no qual o serviço de nomes (registro RMI) escuta as conexões cliente RMI.

    O único uso deste serviço de nomes é baixado para os stubs de RMI. Se a porta padrão estiver ocupada, uma porta livre será usada. Os valores legais são 1- 65535. Em sistemas UNIX®, a criação de soquetes que escutam nas portas 1 - 1024 requer privilégios de superusuário. O valor padrão é 8686.

  6. No campo Nome do Domínio, digite o nome que representa o domínio administrativo especial.

    Toda a autenticação é tratada por esse domínio. O valor padrão é admin-domínio.

  7. (Opcional) Na seção Propriedades Adicionais, especifique essas propriedades.

    O Enterprise Server especifica diversas propriedades por padrão.

    • Para adicionar uma propriedade, clique no botão Adicionar Propriedade. Na linha em branco que é exibida, digite o nome da propriedade no campo Nome e digite o valor da propriedade no campo Valor.
    • Para modificar uma propriedade, edite o campo Valor da propriedade.
    • Para excluir uma propriedade, marque a caixa de seleção à esquerda do campo Nome da propriedade a ser excluída. Clique no botão Excluir Propriedades.
  8. Clique em Salvar.
  9. Se você tiver marcado a caixa de seleção Segurança Ativada, siga as etapas em Para editar as configurações SSL do conector JMX.
