Propriedades específicas da classe FileRealm

As seguintes propriedades são necessárias a um domínio arquivo.

Contexto JAAS

O contexto JAAS (Java Authentication and Authorization Service) (o identificador do módulo de login para usar este domínio). O único valor válido é fileRealm.

Arquivo Principal

Caminho completo e nome do arquivo em que o servidor armazenará todas as informações de usuário, grupo e senha para esse domínio. O Valor padrão é domain-dir/config/keyfile para o domínio arquivo e domain-dir/config/admin-keyfile para domínio admin.

O arquivo principal do domínio arquivo inicialmente está vazio, portanto, os usuários devem ser adicionados na página Usuários do Arquivo para que o domínio arquivo possa ser usado.

O arquivo principal do domínio admin inicialmente contém o nome de usuário do administrador, a senha do administrador em um formato criptografado e o grupo ao qual este usuário pertence (asadmin, por padrão). Para obter mais informações sobre como adicionar usuários ao domínio admin, consulte Para dar acesso a ferramentas de administração.

Observação - Os usuários do grupo asadmin no domínio admin são autorizados a usar o Console de Administração e as ferramentas de asadmin. Adicione, a esse grupo, somente usuários com privilégios administrativos de servidor.

A propriedade opcional a seguir está disponível no domínio arquivo.

Atribuir Grupos

Uma lista de nomes de grupos, separados por vírgula. Todos os clientes que apresentarem certificados válidos serão atribuídos a esses grupos; por exemplo, funcionário,gerente, onde esses forem os nomes de grupos de usuários.
