Contrato de Licença

A página Contrato de Licença exibe os contratos de licença de todos os componentes selecionados para instalação ou atualização. O contrato de licença é fornecido com o componente, pelo autor do componente. Diferentes componentes no repositório de rede podem ser publicados sob diferentes licenças; por exemplo:

  • Common Development and Distribution License (CDDL)

  • GNU General Public License (GPL)

  • Apache License

  • wxWindows Library Licence

A página Contrato de Licença contém estas opções:

Aceitar

Este botão aceita os termos da licença de todos os componentes selecionados e dá continuidade à instalação ou atualização.

Recusar

Este botão recusa os termos da licença de todos os componentes selecionados e volta para a página em que os componentes foram selecionados.
