Sun Connection に Sun GlassFish Enterprise Server を登録する

Sun GlassFishTM Enterprise Server のユーザー登録をすると、次の情報を得ることができます。

  • パッチ情報およびバグのアップデート

  • スクリーンキャストおよびチュートリアル

  • ニュースおよびイベント

  • サポートおよびトレーニングの提供

Sun GlassFish Enterprise Server のインストールを Sun Connection に登録するには、Sun Online Account が必要です。Enterprise Server では、Sun Online Account を所有していない場合、登録時にアカウントを作成できます。

注 - Enterprise Server を登録する際に Sun Online Account を作成する場合は、登録に必要となる情報だけを入力するよう求められます。Enterprise Server の登録後に、Sun Online Account を更新すれば、その他オプションの詳細情報を設定できます。

始める前に

Enterprise Server を実行しているマシンがプロキシを介してインターネットに接続している場合、プロキシのホスト名とポート番号を取得します。Enterprise Server を実行しているマシンで Web ブラウザを実行している場合、それらの情報は Web ブラウザから取得できます。それ以外の場合は、システム管理者に問い合わせてください。

  1. ナビゲーションツリーで、「登録」ノードを選択します。

    「製品の登録」ページが開きます。

  2. 「製品の登録」ページで、必須情報を入力します。
    • Sun Online Account をすでに持っている場合は、次の方法で必須情報を入力します。
      1. 「Sun Online Account または Sun Developer Network (SDN) Account をすでに持っています」オプションを選択します。
      2. 「ユーザー名」フィールドに、Sun Online Account の登録時に指定したユーザー名を入力します。
      3. 「パスワード」フィールドに、Sun Online Account の登録時に指定したパスワードを入力します。
    • Sun Online Account を持っていない場合は、次の方法で必須情報を指定します。
      1. 「Sun Online Account を持っていません」オプションを選択します。
      2. 「電子メールアドレス」フィールドに電子メールアドレスを入力します。

        入力した電子メールアドレスが、Sun Online Account のユーザー名になります。

      3. 「パスワード」フィールドに、Sun Online Account で使用するパスワードを入力します。

        パスワードでは、大文字と小文字が区別されます。また、6 ～ 10 文字にする必要があります。

      4. 「パスワード再入力」フィールドに、前の手順で入力したパスワードをもう一度入力します。
      5. 「名」フィールドに、名前を入力します。
      6. 「姓」フィールドに、姓を入力します。
      7. 「会社」フィールドに、会社の名前を入力します (該当する場合)。
      8. ドロップダウンリストから国または地域を選択します。
  3. Enterprise Server を実行しているマシンがプロキシを介してインターネットに接続している場合、プロキシのホスト名とポート番号を入力します。

    Enterprise Server を実行しているマシンがインターネットに直接接続している場合は、この手順を省略してください。

    Enterprise Server を実行しているマシンで Web ブラウザを実行している場合があります。その場合は、プロキシのホスト名とポート番号を Web ブラウザから取得できます。それ以外の場合は、システム管理者に問い合わせてください。

  4. 「登録」をクリックします。

    登録の終了を確認するために、「Support」ページが開きます。
