ネットワークリポジトリにアクセスするためにネットワークプロキシを設定する

ネットワークプロキシ設定により、Enterprise Server のインターネットへの接続方法が指定されます。更新ツール がインターネット経由でネットワークリポジトリにアクセスできるようにするには、ネットワークプロキシを設定します。

  1. ナビゲーションツリーで、更新ツール ノードを選択します。

    「利用可能なアドオン」ページが開きます。

  2. 「利用可能なアドオン」ページで、「設定」タブをクリックします。

    「更新ツール の設定」ページが表示されます。

  3. 「更新ツールの設定」ページで、 Enterprise Server が実行されているマシンがインターネットに接続する方法を指定します。
    • マシンがインターネットに直接接続する場合は、「直接接続」オプションを選択します。
    • マシンがプロキシ経由でインターネットに接続する場合は、「プロキシ経由」オプションを選択します。
  4. Enterprise Server を実行しているマシンがプロキシを介してインターネットに接続している場合、プロキシのプロキシホスト名とプロキシポート番号を入力します。

    Enterprise Server を実行しているマシンがインターネットに直接接続している場合は、この手順を省略してください。

    Enterprise Server を実行しているマシンで Web ブラウザを実行している場合があります。その場合は、プロキシのホスト名とポート番号を Web ブラウザから取得できます。それ以外の場合は、システム管理者に問い合わせてください。

  5. 「保存」をクリックします。
