製品登録

Sun GlassFishTMEnterprise Server のインストールをSun Connection に登録するには、Sun Online Account が必要です。Enterprise Server では、Sun Online Account を所有していない場合、登録時にアカウントを作成できます。

「製品の登録」ページには、次のオプションがあります。

Sun Online Account または Sun Developer Network (SDN) Account をすでに持っています

すでにアカウントを持っている場合は選択します。このオプションを選択すると、次のフィールドが表示されます。

ユーザー名

Sun Online Account の登録時に指定したユーザー名。

パスワード

Sun Online Account の登録時に指定したパスワード。

プロキシホスト

Enterprise Server を実行しているマシンのプロキシサーバーのホスト名。

このフィールドは、Enterprise Server を実行しているマシンがプロキシサーバーを介してインターネットに接続する場合のみ適用可能です。

プロキシポート

Enterprise Server を実行しているマシンのプロキシサーバーのポート番号。

このフィールドは、Enterprise Server を実行しているマシンがプロキシサーバーを介してインターネットに接続する場合のみ適用可能です。

Sun Online Account を持っていません。登録します。

アカウントを持っておらず、Enterprise Server のインストールを Sun Connection に登録する場合、選択します。このオプションを選択すると、次のフィールドが表示されます。

電子メールアドレス

電子メールアドレスは、Sun Online Account ユーザー名になります。

パスワード

Sun Online Account で使用するパスワード。パスワードでは、大文字と小文字が区別されます。また、6 ～ 10 文字にする必要があります。

パスワード再入力

「パスワード」フィールドで指定したパスワードを再入力する必要があります。

名

名前を入力します。

姓

姓を入力します。

会社

該当する場合は勤務している会社。

国/地域

居住している国または地域。

プロキシホスト

Enterprise Server を実行しているマシンのプロキシサーバーのホスト名。

このフィールドは、Enterprise Server を実行しているマシンがプロキシサーバーを介してインターネットに接続する場合のみ適用可能です。

プロキシポート

Enterprise Server を実行しているマシンのプロキシサーバーのポート番号。

このフィールドは、Enterprise Server を実行しているマシンがプロキシサーバーを介してインターネットに接続する場合のみ適用可能です。
