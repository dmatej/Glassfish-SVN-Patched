システム設定

「システム設定」ページを使用して、Enterprise Server のプロパティーを表示および管理します。

システムプロパティーは、サーバーレベルの設定に対して共通の値を定義します。管理コンソールのテキストフィールドでは、システムプロパティーはドル記号と中括弧で囲んで表します。たとえば、prop-name という名前のプロパティーを表すには、${prop-name} の構文を使用します。

「システム設定」ページには、プロパティーごとに次の情報が表示されます。

名前

プロパティーの名前。

値

プロパティーの値

説明

プロパティーのその他の説明。

多数のシステムプロパティーがあらかじめ定義されていますが、これらはこのページに表示されません。詳細は、「定義済みのシステムプロパティー」を参照してください。
