IIOP 리스너를 편집하는 방법

  1. 트리 구성 요소에서 [구성] 노드를 확장합니다.

  2. 구성할 인스턴스를 선택합니다.

    • 특정 인스턴스를 구성하려면 해당 인스턴스의 구성 노드를 선택합니다. 예를 들어, 기본 인스턴스 server의 경우 server-config 노드를 선택합니다.

    • default-config 복사본을 사용하는 향후 인스턴스에 대해 기본 설정을 구성하려면 default-config 노드를 선택합니다.

  3. [구성] 노드에서 ORB 노드를 확장합니다.

  4. [IIOP 리스너] 노드를 선택합니다.

    [IIOP 리스너] 페이지가 열립니다.

  5. 표에서 편집할 리스너의 이름을 누릅니다.

    [IIOP 리스너 편집] 페이지가 열립니다.

  6. [네트워크 주소] 필드에 리스너의 네트워크 주소를 입력합니다.

    IP 주소 또는 DNS 확인 가능한 호스트 이름을 입력할 수 있습니다.

  7. [리스너 포트] 필드에 IIOP 리스너의 수신 위치에 해당하는 포트 번호를 입력합니다.

  8. GlassFish Server에 대한 ORB IIOP 인바운드 접속을 사용으로 설정하려면 [리스너 사용] 체크 박스를 선택합니다.

    이 옵션은 기본적으로 사용으로 설정되어 있습니다.

  9. IIOP 리스너에 대해 SSL을 사용하려면 [보안 사용] 체크 박스를 선택합니다.

    이 옵션은 기본적으로 사용 안함으로 설정되어 있습니다.

  10. [추가 속성] 섹션에서 추가 속성을 지정합니다.

    GlassFish Server에서는 IIOP 리스너에 대한 추가 속성을 정의하지 않습니다.

    • 속성을 추가하려면 [속성 추가] 단추를 누릅니다. 빈 행이 표시되면 [이름] 필드에 속성 이름을 입력하고 [값] 필드에 속성값을 입력합니다.

    • 속성을 수정하려면 속성의 [값] 필드를 편집합니다.

    • 속성을 삭제하려면 삭제할 속성의 [이름] 필드 왼쪽에 있는 체크 박스를 선택한 다음 [속성 삭제] 단추를 누릅니다.

  11. [저장]을 누릅니다.

    팁:

    기본 설정으로 돌아가려면 [기본값 로드] 단추를 누른 다음 [저장]을 누릅니다.

  12. [보안 사용] 체크 박스를 선택한 경우, 이 IIOP 리스너의 SSL 설정을 수정하려면 SSL 탭 영역을 누릅니다.

    자세한 내용은 보안 IIOP 리스너의 SSL 설정을 편집하는 방법을 참조하십시오.

  13. 리스너의 포트 번호를 변경한 경우에는 GlassFish Server를 재시작합니다.
