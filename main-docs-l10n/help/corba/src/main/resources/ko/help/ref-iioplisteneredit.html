IIOP 리스너 편집

IIOP 리스너 편집 페이지에서는 IIOP 리스너에 대한 설정을 수정할 수 있습니다.

[IIOP 리스너 편집] 페이지에는 다음 옵션이 포함되어 있습니다.

기본값 로드

기본값을 사용하는 설정을 기본값으로 복원하는 단추입니다. 기본값을 사용하지 않는 설정은 변경되지 않습니다.

구성 이름

이 페이지의 설정이 적용되는 구성의 이름입니다. 이 필드는 읽기 전용입니다.

이름

IIOP 리스너를 식별하는 고유한 이름입니다. [이름] 필드는 읽기 전용 필드입니다. IIOP 리스너를 생성할 때만 이름을 지정할 수 있습니다.

네트워크 주소

IIOP 리스너의 네트워크 주소입니다. 네트워크 주소는 IP 주소 또는 DNS에서 확인 가능한 호스트 이름입니다.

리스너 포트

IIOP 리스너의 수신 위치에 해당하는 포트 번호입니다.

리스너

리스너의 상태입니다. [사용] 체크 박스를 선택하면 GlassFish Server에 대한 ORB IIOP 인바운드 접속이 사용으로 설정됩니다. 이 옵션은 기본적으로 사용으로 설정되어 있습니다.

보안

[사용] 체크 박스를 선택하면 IIOP 리스너에 대해 SSL이 사용으로 설정됩니다. 이 옵션은 기본적으로 사용 안함으로 설정되어 있습니다.

추가 속성

IIOP 리스너에 대한 추가 속성입니다. GlassFish Server에서는 IIOP 리스너에 대한 추가 속성을 정의하지 않습니다.

관련 작업

  • IIOP 리스너를 생성하는 방법

  • IIOP 리스너를 편집하는 방법

  • 보안 IIOP 리스너의 SSL 설정을 편집하는 방법

  • IIOP 리스너를 삭제하는 방법
