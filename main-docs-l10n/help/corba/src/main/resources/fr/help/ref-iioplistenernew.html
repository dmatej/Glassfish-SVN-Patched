Nouveau processus d'écoute IIOP

La page Nouveau processus d'écoute IIOP permet de créer un processus d'écoute IIOP.

Elle contient les options ci-après.

Nom de configuration

Nom de la configuration à laquelle s'appliquent les paramètres de cette page. Ce champ est en lecture seule.

Nom

Nom unique identifiant le processus d'écoute IIOP.

Adresse réseau

Adresse réseau du processus d'écoute IIOP. L'adresse réseau peut être une adresse IP ou un nom d'hôte DNS pouvant être résolu.

Port du processus d'écoute

Numéro de port d'écoute que le processus d'écoute IIOP doit utiliser.

Processus d'écoute

Statut du processus d'écoute. Si la case Activé est cochée, les connexions entrantes IIOP de l'ORB à GlassFish Server sont activées. Cette option est activée par défaut.

Sécurité

Si la case Activé est cochée, le protocole SSL est activé pour le processus d'écoute IIOP. Cette option est désactivée par défaut.

Autres propriétés

Propriétés supplémentaires du processus d'écoute IIOP. GlassFish Server ne définit aucune autre propriété pour les processus d'écoute IIOP.
