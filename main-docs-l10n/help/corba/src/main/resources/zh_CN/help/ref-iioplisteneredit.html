编辑 IIOP 监听程序

使用“编辑 IIOP 监听程序”页可以修改 IIOP 监听程序的设置。

“编辑 IIOP 监听程序”页包含以下选项。

加载默认值

用于将具有默认值的设置还原为其默认值的按钮。没有默认值的设置不发生更改。

配置名称

此页上的设置应用到的配置的名称。此字段为只读。

名称

标识 IIOP 监听程序的唯一名称。“名称”字段是只读字段。只能在创建 IIOP 监听程序时指定名称。

网络地址

IIOP 监听程序的网络地址。网络地址可以是 IP 地址或 DNS 可解析的主机名。

监听程序端口

IIOP 监听程序要监听的端口号。

监听程序

监听程序的状态。如果选中“已启用”复选框，则将启用与 GlassFish Server 的 ORB IIOP 入站连接。默认情况下启用此选项。

安全性

如果选中“已启用”复选框，则将为 IIOP 监听程序启用 SSL。默认情况下禁用此选项。

其他属性

IIOP 监听程序的其他属性。GlassFish Server 不定义 IIOP 监听程序的任何其他属性。
