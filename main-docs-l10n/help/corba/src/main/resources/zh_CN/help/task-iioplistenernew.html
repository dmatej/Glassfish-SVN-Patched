创建 IIOP 监听程序

  1. 在树组件中，展开“配置”节点。

  2. 选择要配置的实例：

    • 要配置特定的实例，请选择该实例的配置节点。例如，对于默认实例 server，请选择 server-config 节点。

    • 要为以后使用 default-config 副本的实例配置默认设置，请选择 default-config 节点。

  3. 在“配置”节点下，展开“ORB”节点。

  4. 选择“IIOP 监听程序”。

    此时将打开“IIOP 监听程序”页。

  5. 请单击“新建”。

  6. 在“名称”字段中，键入用于标识监听程序的名称。

  7. 在“网络地址”字段中，键入监听程序的网络地址。

    可以键入 IP 地址，也可以键入 DNS 可解析的主机名。

  8. 在“监听程序端口”字段中，键入 IIOP 监听程序要监听的端口号。

  9. 选中“启用监听程序”复选框以启用至 GlassFish Server 的 ORB IIOP 入站连接。

    默认情况下启用此选项。

  10. 选中“已启用安全性”复选框，为 IIOP 监听程序启用 SSL。

    默认情况下禁用此选项。

  11. 在“其他属性”部分中，指定其他属性。

    GlassFish Server 不定义 IIOP 监听程序的任何其他属性。

    要添加属性，请单击“添加属性”按钮。在显示的空白行中，在“名称”字段中键入属性名称，在“值”字段中键入属性值。

  12. 单击“确定”。

  13. 如果已选中“已启用安全性”复选框，请按照编辑安全 IIOP 监听程序的 SSL 设置中的步骤操作。
