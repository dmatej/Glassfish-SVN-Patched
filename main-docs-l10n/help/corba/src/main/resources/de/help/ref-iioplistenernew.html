Neuer IIOP-Listener

Verwenden Sie die Seite "Neuer IIOP-Listener", um einen neuen IIOP-Listener zu erstellen.

Die Seite "Neuer IIOP-Listener" umfasst die folgenden Optionen.

Name der Konfiguration

Der Name der Konfiguration, für die die Einstellungen auf dieser Seite angewendet werden. Dieses Feld ist schreibgeschützt.

Name

Ein eindeutiger Name für den IIOP-Listener.

Netzwerkadresse

Die Netzwerkadresse für den IIOP-Listener. Die Netzwerkadresse kann eine IP-Adresse oder ein von DNS auflösbarer Hostname sein.

Listener-Port

Die Portnummer, auf der der IIOP-Listener horcht.

Listener

Der Status des Listeners. Wenn das Kontrollkästchen "Aktiviert" gewählt ist, sind die eingehenden IIOP-Verbindungen vom ORB zum GlassFish Server aktiviert. Diese Option ist standardmäßig aktiviert.

Sicherheit

Wenn das Kontrollkästchen "Aktiviert" gewählt ist, wird SSL für den IIOP-Listener aktiviert. Diese Option ist standardmäßig deaktiviert.

Weitere Eigenschaften

Weitere Eigenschaften für den IIOP-Listener. Der GlassFish Server definiert keine zusätzlichen Eigenschaften für die IIOP-Listener.

  • So wird ein IIOP-Listener bearbeitet

  • So werden SSL-Einstellungen für einen sicheren IIOP-Listener bearbeitet

  • So wird ein IIOP-Listener gelöscht
