Para Editar um Listener IIOP

  1. No componente da árvore, expanda o nó Configuração.

  2. Selecione a instância que será configurada:

    • Para configurar uma instância específica, selecione o nó de configuração da instância. Por exemplo, para a instância default, server, selecione o nó server-config.

    • Para configura as definições default das futuras instâncias que utilizam uma cópia de default-config, selecione o nó default-config.

  3. No nó Configuração, expanda o nó ORB.

  4. Selecione o nó Listeners IIOP.

    A página Listeners IIOP é aberta.

  5. Na tabela, clique no nome do listener a ser editado.

    A página Editar Listener IIOP é aberta.

  6. No campo Endereço de Rede, digite o endereço de rede do listener.

    Você pode digitar um endereço IP ou um nome de host resolúvel de DNS.

  7. No campo Porta do Listener, digite o número da porta que o listener IIOP utilizará.

  8. Marque a caixa de seleção Listener Ativado para ativar conexões de entrada IIOP do ORB com o GlassFish Server.

    Por default, a opção está ativada.

  9. Marque a caixa de seleção Segurança Ativada para ativar SSL para o listener IIOP.

    Por default, essa opção está desativada.

  10. Na seção Propriedades Adicionais, especifique as propriedades adicionais.

    O GlassFish Server não define propriedades adicionais para listeners IIOP.

    • Para adicionar uma propriedade, clique no botão Adicionar Propriedade. Na linha em branco que for exibida, digite o nome da propriedade no campo Nome e digite o valor da propriedade no campo Valor.

    • Para modificar uma propriedade, edite o campo Valor da propriedade.

    • Para deletar uma propriedade, marque a caixa de seleção à esquerda do campo Nome da Propriedade a ser deletada. Clique no botão Deletar Propriedades.

  11. Clique em Salvar.

    Dica:

    Para voltar às definições default, clique no botão Carregar Defaults e depois em Salvar.

  12. Se a caixa de seleção Segurança Ativada estiver marcada, clique no painel com abas SSL para modificar as configurações de Camada de Soquete Seguro desse listener de IIOP.

    Consulte Para Editar Definições SSL de um Listener IIOP Seguro para obter detalhes.

  13. Se você alterou o número da porta do listener, reinicie o GlassFish Server.
