Novo Listerner IIOP

Utilize a página Novo Listener IIOP para criar um novo listener IIP.

A página Novo Listener IIOP contém as opções a seguir.

Nome da Configuração

O nome da configuração à qual as definições nesta página se aplicam. Este campo é somente para leitura.

Nome

Um nome exclusivo que identifica o listener IIOP.

Endereço de Rede

O endereço de rede do listener IIOP. O endereço de rede pode ser um endereço IP ou um nome de host resolúvel de DNS.

Porta de Listener

O número da porta onde o listener IIOP fará listening.

Listener

O status do listener. Se a caixa de seleção Ativo estiver marcada, as conexões de entrada IIOP do ORB com o GlassFish Server serão ativadas. Por default, a opção está ativada.

Segurança

Se a caixa de seleção Ativado estiver marcada, a SSL será ativada para o listener IIOP. Por default, essa opção está desativada.

Propriedades Adicionais

Propriedades adicionais para o listener IIOP. O GlassFish Server não define propriedades adicionais para listeners IIOP.
