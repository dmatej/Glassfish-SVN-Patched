IIOPリスナーを作成するには

  1. ツリー・コンポーネントで、「構成」ノードを展開します。

  2. 構成するインスタンスを選択します:

    • 特定のインスタンスを構成するには、そのインスタンスの構成ノードを選択します。たとえば、デフォルト・インスタンスserverの場合は、server-configノードを選択します。

    • default-configのコピーを使用する将来のインスタンスのためにデフォルトの設定値を構成するには、default-configノードを選択します。

  3. 「構成」ノードで、「ORB」ノードを展開します。

  4. IIOPリスナーを選択します。

    「IIOPリスナー」ページが開きます。

  5. 「新規」をクリックします。

  6. 「名前」フィールドに、リスナーを識別する名前を入力します。

  7. 「ネットワーク・アドレス」フィールドに、リスナーのネットワーク・アドレスを入力します。

    IPアドレスまたはDNSで解決可能なホスト名を入力できます。

  8. 「リスナー・ポート」フィールドに、IIOPリスナーがリスニングするポート番号を入力します。

  9. 「リスナー」の「有効」チェック・ボックスを選択して、GlassFish ServerへのORB IIOPのインバウンド接続を有効にします。

    このオプションはデフォルトで有効になっています。

  10. 「セキュリティ有効」チェック・ボックスを選択して、IIOPリスナーでSSLを有効にします。

    このオプションは、デフォルトでは無効になっています。

  11. 「追加プロパティ」セクションで、追加プロパティを指定します。

    GlassFish ServerはIIOPリスナーの追加プロパティを定義しません。

    プロパティを追加するには、「プロパティを追加」ボタンをクリックします。表示される空白行で、「名前」フィールドにプロパティ名を入力し、「値」フィールドにプロパティ値を入力します。

  12. 「OK」をクリックします。

  13. 「セキュリティ有効」チェック・ボックスを選択した場合は、ソースIIOPリスナー用にSSL設定を編集するにはのステップに従ってください。
