建立 IIOP 監聽程式

  1. 在樹形元件中，展開「組態」節點。

  2. 選取要組態的執行處理：

    • 若要組態特定的執行處理，請選取該執行處理的組態節點。例如，對於預設執行處理 server，請選取 server-config 節點。

    • 若要組態使用 default-config 副本之未來執行處理的預設設定，請選取 default-config 節點。

  3. 在「組態」節點下，展開「ORB」節點。

  4. 選取「IIOP 監聽器」。

    「IIOP 監聽器」頁面會隨即開啟。

  5. 按一下「新建」。

  6. 在「名稱」欄位中，鍵入識別監聽器的名稱。

  7. 在「網路位址」欄位中，鍵入監聽器的網路位址。

    您可以鍵入 IP 位址或 DNS 可解析主機名稱。

  8. 在「監聽器連接埠」欄位中，鍵入 IIOP 監聽器要監聽的連接埠號碼。

  9. 選取「啟用監聽器」核取方塊可啟用通往 GlassFish Server 的 ORB IIOP 輸入連線。

    此選項預設為啟用。

  10. 選取「啟用安全性」核取方塊可為 IIOP 監聽器啟用 SSL。

    此選項預設為停用。

  11. 在「其他特性」區段中，指定其他特性。

    GlassFish Server 未定義 IIOP 監聽器的其他特性。

    若要增加特性，請按一下「增加特性」按鈕。在出現的空白列中，於「名稱」欄位中鍵入特性名稱，然後在「值」欄位中鍵入特性值。

  12. 按一下「確定」。

  13. 如果您選取「啟用安全性」核取方塊，請依照編輯安全 IIOP 監聽程式的 SSL 設定中的步驟進行。
