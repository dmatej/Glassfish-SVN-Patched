編輯 IIOP 監聽程式

使用「編輯 IIOP 監聽器」頁面可修改 IIOP 監聽器的設定。

「編輯 IIOP 監聽器」頁面包含下列選項。

載入預設值

用於將具有預設值的設定復原為設定值的按鈕。沒有預設值的設定則保持不變。

組態名稱

套用此頁面上之設定的組態名稱。本欄位是唯讀的。

名稱

可識別 IIOP 監聽器的唯一名稱。「名稱」欄位是唯讀欄位。您僅能在建立 IIOP 監聽器時指定名稱。

網路位址

IIOP 監聽器的網路位址。網路位址可以是 IP 位址或 DNS 可解析主機名稱。

監聽器連接埠

IIOP 監聽器進行監聽的連接埠號碼。

監聽器

監聽器的狀態。若選取了「啟用」核取方塊，會啟用通往 GlassFish Server 的 ORB IIOP 輸入連線。此選項預設為啟用。

安全性

若選取了「啟用」核取方塊，會為 IIOP 監聽器啟用 SSL。此選項預設為停用。

其他特性

IIOP 監聽器的其他特性。GlassFish Server 未定義 IIOP 監聽器的其他特性。
