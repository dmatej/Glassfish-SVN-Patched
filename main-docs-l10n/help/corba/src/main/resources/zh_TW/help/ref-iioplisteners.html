IIOP 監聽程式

使用「IIOP 監聽器」頁面可組態 IIOP 監聽器。

可以針對 ORB 組態一些 IIOP 監聽器，其中的每個監聽器均接受不同網路位址和連接埠組合上的連線。您可以為安全通訊和從屬端認證進行不同的監聽器組態。

依預設，會組態三個 IIOP 監聽程式：orb-listener-1、SSL 和 SSL_MUTUALAUTH。

對於每個 IIOP 監聽器，「IIOP 監聽器」頁面均包含以下資訊。

名稱

IIOP 監聽器的名稱。

已啟用

IIOP 監聽器啟用時為 True，監聽器未啟用時為 False。

網路位址

IIOP 監聽器的網際網路協定 (IP) 位址或 DNS 可解析主機名稱。

監聽器連接埠

IIOP 監聽器的連接埠號碼。
