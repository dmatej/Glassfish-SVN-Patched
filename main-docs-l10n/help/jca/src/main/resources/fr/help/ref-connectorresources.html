Ressources de connecteur

La page Ressources du connecteur permet de configurer les ressources de connecteur.

Une ressource de connecteur est un objet de programme fournissant une connexion vers un système EIS à une application.

Pour chaque ressource, les informations suivantes sont fournies.

Nom JNDI

Nom unique identifiant la ressource de connecteur.

Activé

True si la ressource de connecteur est activée ou False si la ressource de connecteur est désactivée.

Pool de connexions

Pool de connexions de connecteur associé à la ressource de connecteur.

Description

Description de la ressource de connecteur.

La table Ressources contient également les options suivantes :

Nouveau

Bouton permettant de créer une ressource de connecteur.

Supprimer

Bouton permettant de supprimer les ressources de connecteur sélectionnées.

Activer

Bouton permettant d'activer les ressources de connecteur sélectionnées.

Désactiver

Bouton permettant de désactiver les ressources de connecteur sélectionnées.
