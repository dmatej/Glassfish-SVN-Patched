Modifier une ressource de connecteur

La page Modifier une ressource de connecteur permet de modifier les paramètres d'une ressource de connecteur.

Elle contient les options ci-après.

Charger les valeurs par défaut

Bouton permettant de restaurer les valeurs par défaut des paramètres qui en ont. Les paramètres qui n'ont pas de valeurs par défaut ne sont pas modifiés.

Nom JNDI

Nom unique identifiant la ressource de connecteur. Le champ Nom JNDI est en lecture seule. Vous ne pouvez spécifier le nom JNDI qu'à la création d'une ressource de connecteur.

Nom du pool

Pool de connexions de connecteur associé à cette ressource.

Description

Description de la ressource de connecteur.

Statut

Si cette option est sélectionnée, la ressource de connecteur est activée. Cette option est activée par défaut.

Autres propriétés

Propriétés supplémentaires de la ressource de connecteur. Les propriétés disponibles dépendent de l'adaptateur de ressources du pool de connexions de connecteur. Les paires nom-valeur spécifiées dans cette table peuvent être utilisées pour remplacer les valeurs par défaut des propriétés définies par le fournisseur de l'adaptateur de ressources.
