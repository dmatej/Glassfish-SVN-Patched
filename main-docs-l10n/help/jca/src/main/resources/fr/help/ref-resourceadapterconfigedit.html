Modifier la configuration de l'adaptateur de ressources

La page Modifier la configuration de l'adaptateur de ressources permet de modifier les paramètres d'une configuration d'adaptateur de ressources.

Elle contient les options ci-après.

Nom de l'adaptateur de ressources

Nom de l'adaptateur de ressources que vous configurez. Le champ Nom est en lecture seule. Vous ne pouvez spécifier ce nom qu'à la création d'une configuration d'adaptateur de ressources.

ID du pool de threads

Pool de threads à utiliser pour la configuration d'adaptateur de ressources.

Autres propriétés

Propriétés supplémentaires de l'adaptateur de ressources. Les propriétés disponibles sont les noms des méthodes set de la classe référencée par l'élément resourceadapter-class dans le fichier ra.xml, qui définit le nom de la classe du composant JavaBeans de l'adaptateur de ressources. Toutes les modifications apportées aux propriétés définies ici remplacent les valeurs par défaut présentes dans le fichier ra.xml.
