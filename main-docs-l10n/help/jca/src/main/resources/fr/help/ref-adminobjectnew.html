Nouvelle ressource d'objet d'administration

La page Nouvelle ressource d'objet d'administration permet de créer une ressource d'objet administré.

Elle contient les options ci-après.

Nom JNDI

Nom unique identifiant la ressource d'objet administré.

Adaptateur de ressources

Adaptateur de ressources pour la ressource d'objet administré.

Type de ressource

Type Java de la ressource d'objet administré (par exemple, javax.jms.Topic).

Nom de classe

Nom de la classe d'implémentation associée au type de ressource.

Description

Description de la ressource d'objet administré.

Statut

Si cette option est sélectionnée, la ressource d'objet administré est activée. Cette option est activée par défaut.

Autres propriétés

Propriétés supplémentaires de la ressource d'objet administré. Les propriétés disponibles dépendent de l'adaptateur de ressources du pool de connexions de connecteur. Les paires nom-valeur spécifiées dans cette table peuvent être utilisées pour remplacer les valeurs par défaut des propriétés définies par le fournisseur de l'adaptateur de ressources.

Cibles

Clusters et instances autonomes de la ressource d'objet administré. Déplacez les cibles souhaitées dans la colonne Cibles sélectionnées à l'aide des boutons Ajouter et Tout ajouter. Déplacez toutes les cibles inutiles dans la colonne Cibles disponibles à l'aide des boutons Enlever et Tout enlever.

Cette option apparaît uniquement si des clusters ou des instances autonomes ont été créés dans le domaine.
