Modification d'une ressource de connecteur

  1. Dans l'arborescence de navigation, développez le noeud Ressources.

  2. Développez le noeud Connecteurs.

  3. Sous le noeud Connecteurs, sélectionnez le noeud Ressources du connecteur.

    La page Ressources du connecteur apparaît.

  4. Sur la page Ressources du connecteur, cliquez sur le nom de la ressource à modifier.

    La page Modifier une ressource de connecteur apparaît.

  5. Dans la liste déroulante Nom du pool, sélectionnez le pool de connexions de connecteur auquel la ressource du connecteur appartient.

  6. Dans le champ Ordre de déploiement, saisissez un nombre pour indiquer l'ordre de déploiement de la ressource de connecteur.

    Les ressources avec les plus petits numéros sont chargés en premier au démarrage du serveur. La valeur par défaut est 100.

  7. Dans le champ Description, saisissez la description de la ressource de connecteur.

  8. Dans l'onglet Cible, sélectionnez les cibles souhaitées, puis cliquez sur Activer ou Désactiver.

    Pour ajouter des cibles à la liste ou en enlever, cliquez sur Gérer les cibles.

    Revenez à la page Modifier une ressource de connecteur en cliquant sur l'onglet Général.

  9. Sélectionnez la case Activé du champ Statut pour activer la ressource de connecteur lors de l'exécution.

    Cette option est activée par défaut.

  10. Dans la section Autres propriétés, spécifiez les propriétés supplémentaires de la ressource de connecteur.

    Les propriétés disponibles dépendent de l'adaptateur de ressources du pool de connexions de connecteur.

    • Pour ajouter une propriété, cliquez sur le bouton Ajouter une propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur.

    • Pour modifier une propriété, modifiez le champ Valeur de cette propriété.

    • Pour supprimer une propriété, sélectionnez la case située à gauche du champ Nom de la propriété à supprimer, puis cliquez sur le bouton Supprimer des propriétés.

  11. Cliquez sur Enregistrer.

    Conseil:

    Pour revenir aux paramètres par défaut, cliquez sur le bouton Charger les valeurs par défaut, puis sur Enregistrer.
