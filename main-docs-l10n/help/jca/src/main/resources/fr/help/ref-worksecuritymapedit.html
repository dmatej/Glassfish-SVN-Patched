Modifier une mappe de sécurité de travail

La page Modifier une mappe de sécurité de travail permet de modifier les paramètres d'une mappe de sécurité de travail.

Elle contient les options ci-après.

Nom de la mappe de sécurité

Nom de la mappe de sécurité de travail. Le champ Nom est en lecture seule. Vous ne pouvez spécifier ce nom qu'à la création d'une mappe de sécurité de travail.

Groupe

Si le bouton radio Groupe est sélectionné, le champ de texte doit spécifier une mappe pour chaque groupe d'utilisateurs EIS back-end avec le groupe d'utilisateurs correspondant défini dans le domaine de sécurité GlassFish Server. Utilisez une liste séparée par des virgules pour spécifier plusieurs mappings. Exemple :

eis-group-1=server-group-1,eis-group-2=server-group-2
Principal

Si le bouton radio Principal est sélectionné, le champ de texte doit spécifier une mappe pour chaque principal EIS back-end avec le principal correspondant défini dans le domaine de sécurité GlassFish Server. Utilisez une liste séparée par des virgules pour spécifier plusieurs mappings. Exemple :

eis-principal-1=server-principal-1,eis-principal-2=server-principal-2
Adaptateur de ressources

Adaptateur de ressources pour le système EIS.

Description

Description de la mappe de sécurité de travail.

Statut

Si cette option est sélectionnée, la mappe de sécurité de travail est activée. Cette option est activée par défaut.
