Editar Mapa de Segurança do Pool de Conexões do Conector

Use a página Editar Mapa de Segurança do Pool de Conexões do Conector para editar o mapa de segurança de um pool de conexões do conector.

A página Editar Mapa de Segurança do Pool de Conexões do Conector contém as opções a seguir.

Nome do Pool de Conexões

O nome do pool de conexões do conector ao qual pertence este mapa de segurança. O nome do pool é um campo somente para leitura.

Nome do Mapa de Segurança

O nome do mapa de segurança. O nome é um campo somente para leitura. Você só pode especificar um nome de mapa de segurança quando cria um novo mapa de segurança.

Nome do Mapa de Segurança

O nome do mapa de segurança.

Grupos de Usuários

Se este botão de opção for selecionado, o campo de texto precisará conter os nomes dos grupos de usuários, separados por vírgulas.

Principais

Se este botão de opção for selecionado, o campo de texto precisará conter os nomes dos principais, separados por vírgulas.

Nome de usuário

O nome de usuário do principal do backend exigido pelo EIS.

Senha

A senha do principal back-end exigido pelo EIS.
