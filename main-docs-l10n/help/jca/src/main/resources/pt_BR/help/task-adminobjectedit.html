Para Editar um Recurso do Objeto Administrado

  1. Na árvore de navegação, expanda o nó Recursos.

  2. Expando o nó Conectores

  3. Sob o nó Conectores, selecione o nó Recursos de Objeto Admin.

    A página Recursos de Objeto Admin será aberta.

  4. Na página Recursos de Objeto Admin, clique no novo recurso de objeto a ser editado.

    A página Editar o Recurso de Objeto Administrado será aberta.

  5. Na lista drop-down Adaptador de Recursos, selecione o adaptador de recursos para o recurso de objeto administrado.

  6. Na lista drop-down Tipo de Recurso, selecione o tipo Java para o recurso de objeto administrado.

  7. Na lista drop-down Nome da Classe, selecione o nome da classe de implementação associado ao tipo de recurso.

  8. No campo Descrição, digite uma descrição do recurso de objeto administrado.

  9. Marque a caixa de seleção Status Ativado, para ativar o recurso de objeto administrado.

    Por default, a opção está ativada.

  10. Na tabela Propriedades Adicionais, especifique propriedades adicionais para o recurso de objeto administrado.

    As propriedades disponíveis dependem do adaptador de recursos para o pool de conexões do conector.

    • Para adicionar uma propriedade, clique no botão Adicionar Propriedade. Na linha em branco que for exibida, digite o nome da propriedade no campo Nome e digite o valor da propriedade no campo Valor.

    • Para modificar uma propriedade, edite o campo Valor da propriedade.

    • Para deletar uma propriedade, marque a caixa de seleção à esquerda do campo Nome da Propriedade a ser deletada. Clique no botão Deletar Propriedades.

  11. Na guia Alvo, clique em Gerenciar Alvos para editar os alvos nos quais o objeto administrado está implantado.

    Volte à página Editar o Recurso de Objeto Administrado clicando na guia Geral.

  12. Clique em Salvar.

    Dica:

    Para voltar às definições default, clique no botão Carregar Defaults e depois em Salvar.
