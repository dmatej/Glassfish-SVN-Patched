Para Criar um Recurso do Objeto Administrado

  1. Na árvore de navegação, expanda o nó Recursos.

  2. Expando o nó Conectores

  3. Sob o nó Conectores, selecione o nó Recursos de Objeto Admin.

    A página Recursos de Objeto Admin será aberta.

  4. Na página Recursos de Objeto Admin, clique no botão Novo.

    A página Novo Recurso de Objeto Administrado será aberta.

  5. No campo Nome JNDI, digite um nome JNDI único que identifica o recurso de objeto administrado.

  6. Na lista drop-down Adaptador de Recursos, selecione o adaptador de recursos para o recurso de objeto administrado.

  7. Na lista drop-down Tipo de Recurso, selecione o tipo Java para o recurso de objeto administrado.

  8. Na lista drop-down Nome da Classe, selecione o nome da classe de implementação associado ao tipo de recurso.

  9. No campo Descrição, digite uma descrição do recurso de objeto administrado.

  10. Marque a caixa de seleção Status Ativado, para ativar o recurso de objeto administrado.

    Por default, a opção está ativada.

  11. Na tabela Propriedades Adicionais, especifique propriedades adicionais para o recurso de objeto administrado.

    As propriedades disponíveis dependem do adaptador de recursos para o pool de conexões do conector.

    Para adicionar uma propriedade, clique no botão Adicionar Propriedade. Na linha em branco que for exibida, digite o nome da propriedade no campo Nome e digite o valor da propriedade no campo Valor.

  12. Na seção Alvo Disponíveis, selecione o domínio, cluster ou instâncias de servidor em que o objeto administrado residirá e clique em Adicionar.

    Para remover um dos domínios, clusters ou instâncias de servidor listados no campo Selecionado, selecione-o no campo e clique em Remover.

  13. Clique em OK.
