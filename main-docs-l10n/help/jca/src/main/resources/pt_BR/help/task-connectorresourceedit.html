Para Editar um Recurso do Conector

  1. Na árvore de navegação, expanda o nó Recursos.

  2. Expanda o nó Conectores.

  3. Sob o nó Conectores, selecione o nó Recursos do Conector.

    A página Recursos do Conector é aberta.

  4. Na página Recursos do Conector, clique no nome do recurso a ser editado.

    A página Editar Recurso Conector é aberta.

  5. Na lista drop-down Nome do Pool, selecione um pool de conexões ao qual pertence o recurso do conector.

  6. No campo Ordem de Implantação, digite um número para especificar a ordem de implantação do recurso conector.

    Recursos com um número mais baixo são carregados primeiro na inicialização do servidor. O valor default é 100.

  7. No campo Descrição, digite uma descrição do recurso do conector.

  8. Na guia Alvo, selecione os alvos necessários e clique em Ativar ou Desativar.

    Para adicionar ou remover alvo da lista, selecione Gerenciar Alvos.

    Volte à página Editar Recurso Conector clicando na guia Geral.

  9. Marque a caixa de seleção Status Ativado para ativar o recurso do conector durante o runtime.

    Por default, a opção está ativada.

  10. Na seção Propriedades |Adicionais, especifique propriedades adicionais para o recurso do conector.

    As propriedades disponíveis dependem do adaptador de recursos para o pool de conexões do conector.

    • Para adicionar uma propriedade, clique no botão Adicionar Propriedade. Na linha em branco que for exibida, digite o nome da propriedade no campo Nome e digite o valor da propriedade no campo Valor.

    • Para modificar uma propriedade, edite o campo Valor da propriedade.

    • Para deletar uma propriedade, marque a caixa de seleção à esquerda do campo Nome da Propriedade a ser deletada. Clique no botão Deletar Propriedades.

  11. Clique em Salvar.

    Dica:

    Para voltar às definições default, clique no botão Carregar Defaults e depois em Salvar.
