Serviço do Conector

Use a página Serviço de Conector para modificar as definições gerais para o serviço de conector que governa os adaptadores de recursos.

A página Serviço de Conector contém as seguintes opções:

Carregar Defaults

Botão para restaurar definições para seus valores default. Definições que não tiveram seus valores default alterados.

Nome da Configuração

O nome da configuração à qual as definições nesta página se aplicam. Este campo é somente para leitura.

Time-out para Shutdown

O número máximo de segundos permitidos durante o shutdown do GlassFish Server, para que o método ResourceAdapter.stop de uma instância do módulo do conector seja completada. Os adaptadores de recursos que levam mais tempo para serem submetidos a shutdown são ignorados e o shutdown do GlassFish Server continua. O valor default é 30 segundos.

Política de Carga de Classes para Conector

A política a ser usada para a carga de classes. As opções disponíveis são:

derived

Indica que os adaptadores de recursos são fornecidos de acordo com as referências da aplicação para qualquer um dos recursos do adaptador de recursos. Essa política é a default.

global

Indica que todos os adaptadores de recursos estarão visíveis para todas as aplicações.

Esta política não se aplica ao adaptador de recursos JMS pré-instalado, jmsra, que sempre está visível para todas as aplicações.
