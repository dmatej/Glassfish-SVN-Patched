Editar Configuração do Adaptador de Recursos

Use a página Editar Configuração do Adaptador de Recursos, para modificar as definições do adaptador de recurso.

A página Editar Configuração do Adaptador de Recursos contém as seguintes opções.

Nome do Adaptador de Recursos

O nome do adaptador de recursos que você está configurando. O nome é um campo somente para leitura. Você só pode especificar um nome de adaptador de recursos quando cria uma nova configuração de adaptador de recursos.

ID do Pool de Threads

O nome de um pool de threads a ser usado para a configuração do adaptador de recursos.

Propriedades Adicionais

Propriedades adicionais para o adaptador de recursos. As propriedades disponíveis são os nomes dos métodos setter da classe referenciada pelo elemento resourceadapter-class no arquivo ra.xml, que define o nome da classe do componente JavaBeans do adaptador de recursos. Quaisquer modificações das propriedades aqui definidas substituem os valores presentes no ra.xml
