Recursos de Objeto Admin

Use a página Recursos de Objeto Admin para configurar os recursos de objetos administrados.

Um recurso de objeto administrado fornece funcionalidade especializada definida pelo adaptador de recursos do módulo de conector implantado.

Para cada recurso de objeto administrado, as seguintes informações são fornecidas.

Nome JNDI

Um nome exclusivo que identifica o recurso do objeto administrado.

Ativado

Verdadeiro se o recurso do objeto administrado estiver ativado ou falso se o recurso estiver desativado.

Tipo de Recurso

O tipo Java do recurso do objeto administrado.

Descrição

A descrição do recurso do objeto administrado.

A tabela Recursos também contém as opções a seguir.

Novo

Botão para criar um novo recurso de objeto administrado.

Deletar

Botão para deletar um ou mais recursos de objeto administrado selecionados.

Ativar

Botão para ativar um ou mais recursos de objeto administrado selecionados.

Desativar

Botão para desativar um ou mais recursos de objeto administrado selecionados.
