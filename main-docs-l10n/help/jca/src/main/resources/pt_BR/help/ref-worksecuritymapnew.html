Novo Mapa de Segurança de Trabalho

Use a página Novo Mapa de Segurança de Trabalho para criar um novo mapa de segurança de trabalho.

A página Novo Mapa de Segurança de Trabalho contém as opções a seguir.

Nome do Mapa de Segurança

O nome do mapa de segurança de trabalho.

Grupo

Se o botão de rádio Mapeamento do Grupo for selecionado, o campo de texto precisará especificar um mapa para de cada grupo de usuários EIS back-end para o correspondente grupo de usuários definido no domínio de segurança do GlassFish Server. Use uma lista separada por vírgulas para especificar mais de um mapeamento. Por exemplo:

eis-group-1=server-group-1,eis-group-2=server-group-2
Principal

Se o botão de rádio Mapeamento do Principal for selecionado, o campo de texto precisará especificar um mapa para de cada grupo de usuários EIS back-end para o correspondente principal definido no domínio de segurança do GlassFish Server. Use uma lista separada por vírgulas para especificar mais de um mapeamento. Por exemplo:

eis-principal-1=server-principal-1,eis-principal-2=server-principal-2
Adaptador do Recurso

O adaptador de recurso para o EIS.

Descrição

Uma descrição para o mapa de segurança do trabalho.

Status

Se esta opção está selecionada, o mapa de segurança do trabalho será ativado. Por default, a opção está ativada.
