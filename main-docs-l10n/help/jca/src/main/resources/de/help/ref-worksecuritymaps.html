Arbeitssicherheitszuordnungen

Auf der Seite "Arbeitssicherheitszuordnungen" können Sie Arbeitssicherheitszuordnungen für Ressourcenadapter konfigurieren.

Eine Arbeitssicherheitszuordnung des Connectors ordnet die Identität des aufrufenden Programms der von dem Ressourcenadapter-EIS-Principal bzw. der -EIS-Benutzergruppe übergebenen Arbeit einem geeigneten Principal bzw. einer geeigneten Benutzergruppe in der GlassFish Server-Sicherheitsdomain zu. Mit einem Ressourcenadapter können eine oder mehrere Arbeitssicherheitszuordnungen verknüpft werden. In der Konfiguration einer Arbeitssicherheitszuordnungen eines Connectors kann das Platzhalterzeichen * (Sternchen) verwendet werden, um alle Benutzer oder Benutzergruppen anzugeben.

Eine Arbeitssicherheitszuordnung ist nicht dasselbe wie eine Sicherheitszuordnung für einen Connector Connection-Pool; diese ordnet den während einer Servlet- oder EJB-Authentifizierung empfangenen Principal den vom EIS akzeptierten Zugangsdaten zu. In Connector Connection-Pool - Sicherheitszuordnungen finden Sie weitere Informationen über Sicherheitszuordnungen für Connector Connection-Pools.

Als EIS (Enterprise Information System) wird jedes System bezeichnet, das die Daten einer Organisation enthält. Dabei kann es sich um ein Mainframe-, Messaging- oder Datenbanksystem bzw. um eine Anwendung handeln.

Für jede Arbeitssicherheitszuordnung werden die folgenden Informationen zur Verfügung gestellt.

Name

Der Name der Arbeitssicherheitszuordnung.

Die Tabelle "Arbeitssicherheitszuordnungen" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen einer neuen Arbeitssicherheitszuordnung.

Löschen

Schaltfläche zum Löschen von mindestens einer ausgewählten Arbeitssicherheitszuordnung.

Verwandte Aufgaben

  • So wird eine Arbeitssicherheitszuordnung erstellt

  • So wird eine Arbeitssicherheitszuordnung bearbeitet

  • So wird eine Arbeitssicherheitszuordnung gelöscht
