Neue Sicherheitszuordnung für Connector Connection-Pool

Auf der Seite "Neue Sicherheitszuordnung für Connector Connection-Pool" können Sie eine neue Sicherheitszuordnung für einen Connector Connection-Pool erstellen.

Die Seite "Neue Sicherheitszuordnung für Connector Connection-Pool" enthält die folgenden Optionen.

Connection-Poolname

Der Name des Connector Connection-Pools, zu dem diese Sicherheitszuordnung gehört. Der Poolname ist ein schreibgeschütztes Feld.

Name der Sicherheitszuordnung

Der Name der Sicherheitszuordnung.

Benutzergruppen

Wenn dieses Optionsfeld aktiviert ist, muss das Textfeld die Namen der Benutzergruppen, mit Kommas getrennt, enthalten.

Principals

Wenn dieses Optionsfeld aktiviert ist, muss das Textfeld die Namen der Principals, mit Kommas getrennt, enthalten.

Benutzername

Der vom EIS benötigte Benutzername des Backend Principals.

Kennwort

Das vom EIS benötigte Kennwort des Backend Principals.
