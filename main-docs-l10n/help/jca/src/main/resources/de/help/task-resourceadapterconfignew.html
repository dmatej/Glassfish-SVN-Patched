1. Blenden Sie in der Navigationsbaumstruktur den Knoten "Ressourcen" ein.

  2. Wählen Sie unter dem Ressourcenknoten den Knoten "Ressourcenadapterkonfigurationen".

    Die Seite "Ressourcenadapterkonfigurationen" wird geöffnet.

  3. Klicken Sie auf der Seite "Ressourcenadapterkonfigurationen" auf "Neu".

    Die Seite "Neue Ressourcenadapterkonfiguration" wird geöffnet.

  4. Wählen Sie in der Dropdown-Liste "Ressourcenadaptername" den Ressourcenadapter.

  5. Wählen Sie in der Dropdown-Liste "Threadpool-ID" den Threadpool, der für die Ressourcenadapterkonfiguration verwendet werden soll.

  6. Geben Sie im Abschnitt "Weitere Eigenschaften" zusätzliche Eigenschaften für den Ressourcenadapter an.

    Verfügbare Eigenschaften sind die Namen von Setter-Methoden der Klassen, die vom Element resourceadapter-class in der Datei ra.xml referenziert werden; dieses Element definiert den Klassennamen der JavaBeans-Komponente des Ressourcenadapters. Hier definierte Eigenschaftenänderungen überschreiben die in der Datei ra.xml vorhandenen Standardwerte.

    • Um eine Eigenschaft hinzuzufügen, klicken Sie auf die Schaltfläche "Eigenschaft hinzufügen". Geben Sie in die daraufhin angezeigte leere Zeile den Eigenschaftsnamen in das Feld "Name" und den Eigenschaftswert in das Feld "Wert" ein.

    • Um eine Eigenschaft zu ändern, bearbeiten Sie das Feld "Wert" dieser Eigenschaft.

    • Um eine Eigenschaft zu löschen, aktivieren Sie das Kontrollkästchen links neben dem Feld "Name" der zu löschenden Eigenschaft und klicken anschließend auf die Schaltfläche "Eigenschaften löschen".

  7. Klicken Sie auf "OK".
