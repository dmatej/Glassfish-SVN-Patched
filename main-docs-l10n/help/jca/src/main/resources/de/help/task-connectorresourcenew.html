So wird eine Connector-Ressource erstellt

Bevor Sie beginnen

Bevor Sie eine Connector-Ressource erstellen, müssen Sie zunächst einen Connector Connection-Pool erstellen.

  1. Blenden Sie in der Navigationsbaumstruktur den Knoten "Ressourcen" ein.

  2. Blenden Sie unter dem Ressourcenknoten den Knoten "Connector" ein.

  3. Wählen Sie unter dem Knoten "Connector" den Knoten "Connector-Ressourcen".

    Die Seite "Connector-Ressourcen" wird geöffnet.

  4. Klicken Sie auf der Seite "Connector-Ressourcen" auf die Schaltfläche "Neu".

    Die Seite "Neue Connector-Ressource" wird geöffnet.

  5. Geben Sie im Feld "JNDI-Name" einen eindeutigen JNDI-Namen für die Connector-Ressource ein.

    Beispiel: eis/myERP.

  6. Wählen Sie in der Dropdown-Liste "Poolname" den Connection-Pool, dem die neue Connector-Ressource angehört.

  7. Geben Sie im Feld "Beschreibung" eine Beschreibung der Connector-Ressource ein.

  8. Aktivieren Sie das Kontrollkästchen "Status", um die Connector-Ressource zur Laufzeit zu aktivieren.

    Diese Option ist standardmäßig aktiviert.

  9. Wählen Sie unter "Verfügbare Ziele" die Domain, das Cluster oder die Serverinstanzen, in denen die Connector-Ressource gespeichert werden soll, und klicken Sie auf "Hinzufügen".

    Wenn Sie die Connector-Ressource nicht für eine Domain, ein Cluster oder eine Serverinstanz, die im Feld "Ausgewählte Ziele" aufgelistet wird, bereitstellen möchten, wählen Sie sie aus dem Feld und klicken auf "Entfernen".

  10. Geben Sie im Abschnitt "Weitere Eigenschaften" zusätzliche Eigenschaften für die Connector-Ressource an.

    Die verfügbaren Eigenschaften richten sich nach dem Ressourcenadapter, der von dem Connector Connection-Pool verwendet wird.

    Um eine Eigenschaft hinzuzufügen, klicken Sie auf die Schaltfläche "Eigenschaft hinzufügen". Geben Sie in die daraufhin angezeigte leere Zeile den Eigenschaftsnamen in das Feld "Name" und den Eigenschaftswert in das Feld "Wert" ein.

  11. Klicken Sie auf "OK".
