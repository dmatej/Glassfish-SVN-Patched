Neue Arbeitssicherheitszuordnung

Auf der Seite "Neue Arbeitssicherheitszuordnung" können Sie eine neue Arbeitssicherheitszuordnung erstellen.

Die Seite "Neue Arbeitssicherheitszuordnung" enthält die folgenden Optionen.

Name der Sicherheitszuordnung

Der Name der Arbeitssicherheitszuordnung.

Gruppe

Wenn das Optionsfeld "Gruppenzuordnung" aktiviert ist, muss im Textfeld eine Zuordnung von jeder Backend-EIS-Benutzergruppe zur entsprechenden, in der GlassFish Server-Sicherheitsdomain definierten Benutzergruppe angegeben werden. Verwenden Sie eine durch Komma getrennte Liste, wenn Sie mehr als eine Zuordnung festlegen möchten. Beispiel:

eis-group-1=server-group-1,eis-group-2=server-group-2
Principal

Wenn das Optionsfeld "Principal-Zuordnung" aktiviert ist, muss im Textfeld eine Zuordnung von jedem Backend-EIS-Principal zum entsprechenden, in der GlassFish Server-Sicherheitsdomain definierten Principal angegeben werden. Verwenden Sie eine durch Komma getrennte Liste, wenn Sie mehr als eine Zuordnung festlegen möchten. Beispiel:

eis-principal-1=server-principal-1,eis-principal-2=server-principal-2
Ressourcenadapter

Der Ressourcenadapter für das EIS.

Beschreibung

Eine Beschreibung der Arbeitssicherheitszuordnung.

Status

Bei aktivierter Option ist die Arbeitssicherheitszuordnung aktiviert. Diese Option ist standardmäßig aktiviert.

Verwandte Aufgaben

  • So wird eine Arbeitssicherheitszuordnung erstellt

  • So wird eine Arbeitssicherheitszuordnung bearbeitet

  • So wird eine Arbeitssicherheitszuordnung gelöscht
