Connector Connection-Pools

Auf der Seite "Connector Connection-Pools" können Sie Connector Connection-Pools konfigurieren.

Ein Connector Connection-Pool besteht aus einer Gruppe wiederverwendbarer Verbindungen für ein spezielles EIS.

Für jede Ressource werden die folgenden Informationen zur Verfügung gestellt.

Poolname

Ein eindeutiger Name, der den Connector Connection-Pool identifiziert.

Ressourcenadapter

Der mit dem Connector Connection-Pool verknüpfte Ressourcenadapter.

Verbindungsdefinition

Der Klassenname der Verbindungsdefinition für den Connector Connection-Pool.

Beschreibung

Eine Beschreibung des Connector Connection-Pools.

Die Tabelle "Pools" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen eines neuen Connector Connection-Pools.

Löschen

Schaltfläche zum Löschen von mindestens einem ausgewählten Connector Connection-Pool.
