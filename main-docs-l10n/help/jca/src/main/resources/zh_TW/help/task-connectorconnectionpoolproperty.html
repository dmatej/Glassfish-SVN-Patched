編輯連線器連線集區特性

  1. 在瀏覽樹狀結構中，展開「資源」節點。

  2. 展開「連線器」節點。

  3. 在「連線器」節點下，選取「連線器連線集區」節點。

    「連線器連線集區」頁面會隨即開啟。

  4. 在「連線器連線集區」頁面上，按一下要編輯的連線器連線集區名稱。

    「編輯連線器連線集區」頁面會隨即開啟。

  5. 在「編輯連線器連線集區」頁面上，按一下「其他特性」標籤。

    「編輯連線器連線集區特性」頁面會隨即開啟。

  6. 在「其他特性」表中，指定連線器連線集區的其他特性。

    指定的特性取決於該集區使用的資源介面。由使用此表的使用者所指定的名/值配對，可用於置換資源轉接器供應商定義的特性預設值。

    若您未針對特性指定值，連線集區編輯完成時，會從表中移除該特性。若您不想立即為特性設定值，但希望將該特性保留在表中，待稍後為其設定值，請在「值」欄位中鍵入一組空的圓括號：

    ()

    • 若要增加特性，請按一下「增加特性」按鈕。在出現的空白列中，於「名稱」欄位中鍵入特性名稱，然後在「值」欄位中鍵入特性值。

    • 若要修改特性，請編輯特性的「值」欄位。

    • 若要刪除特性，請選取您要刪除之特性「名稱」欄位左側的核取方塊，然後按一下「刪除特性」按鈕。

  7. 按一下「儲存」。
