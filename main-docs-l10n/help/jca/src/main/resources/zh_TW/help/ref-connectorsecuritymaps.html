連線器連線集區安全性對應

使用「連線器連線集區安全性對應」頁面可管理連線器連線集區的安全性對應。

連線器連線集區的安全性對應可以在 Java EE 使用者和群組以及 EIS 使用者和群組之間建立對應。安全性對應會將在 Servlet 或 EJB 認證期間接收的主體對應到 EIS 接受的憑證。您可以將多個 GlassFish Server 主體對應到相同的後端主體。

連線集區的安全性對應與連線器工作安全性對應不同，後者會將與內送工作執行處理關聯的主體對應到 GlassFish Server 網域之安全性範圍中的主體。請參閱工作安全性對應以取得有關連線器工作安全性對應的資訊。

對於每個安全性對應，都將提供以下資訊。

名稱

安全性對應的名稱。

安全性對應表也包含下列選項。

新建

建立新的安全性對應的按鈕。

刪除

刪除一或多個所選取安全性對應的按鈕。
