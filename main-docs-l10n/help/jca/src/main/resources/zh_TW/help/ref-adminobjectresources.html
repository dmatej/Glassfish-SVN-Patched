管理物件資源

使用「管理物件資源」頁面可組態管理物件資源。

管理物件資源提供的特殊功能，可由已建置連線器模組的資源轉接器定義。

對於每個管理物件資源，均提供以下資訊。

JNDI 名稱

可識別管理物件資源的唯一名稱。

已啟用

如果管理物件資源已啟用，則為 True；如果資源已停用，則為 False。

資源類型

管理物件資源的 Java 類型。

描述

管理物件資源的描述。

資源表也包含下列選項。

新建

建立新管理物件資源的按鈕。

刪除

刪除一或多個所選取管理物件資源的按鈕。

啟用

啟用一或多個所選取管理物件資源的按鈕。

停用

停用一或多個所選取管理物件資源的按鈕。
