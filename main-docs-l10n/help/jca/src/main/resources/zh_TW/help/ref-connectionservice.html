連線器服務

使用「連線器服務」頁面可修改控制資源轉接器之連線器服務的一般設定。

「連線器服務」頁面包含下列選項。

載入預設值

用於將具有預設值的設定復原為設定值的按鈕。沒有預設值的設定則保持不變。

組態名稱

套用此頁面上之設定的組態名稱。本欄位是唯讀的。

關機逾時

GlassFish Server 關機期間可等候連線器模組執行處理 ResourceAdapter.stop 方法完成的最大秒數。會忽略需要更長時間關機的資源轉接器並繼續 GlassFish Server 關機動作。預設值為 30 秒。

連線器類別載入策略

用於載入類別的策略。可用的選項包括：

derived

代表會將資源轉接器根據應用程式的參照，提供給任何資源轉接器的資源。此策略為預設值。

global

代表會對所有應用程式顯示所有資源轉接器。

此策略不會套用到預先安裝的 JMS 系統資源轉接器 jmsra，該資源轉接器會永遠對所有應用程式顯示。
