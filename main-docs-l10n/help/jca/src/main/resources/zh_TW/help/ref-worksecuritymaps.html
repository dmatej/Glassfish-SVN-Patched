工作安全性對應

使用「工作安全性對應」頁面可為資源轉接器組態工作安全性對應。

連線器工作安全性對應會將資源轉接器 EIS 主體或 EIS 使用者群組所提交之工作的呼叫者身份，對應到 GlassFish Server 安全性網域中適當的主體或使用者群組。可以將一或多個工作安全性對應與一個資源轉接器建立關聯。連線器工作安全性對應組態支援使用萬用字元星號 (*) 來表示所有使用者或所有使用者群組。

工作安全性對應與連線器連線集區的安全性對應不同，後者會將 Servlet 或 EJB 認證期間接收的主體對應到 EIS 接受的憑證。請參閱連線器連線集區安全性對應以取得有關連線器連線集區安全性對應的資訊。

企業資訊系統 (EIS) 是指控管組織資料的任何系統。它可以是主機、訊息傳送系統、資料庫系統或應用程式。

對於每個工作安全性對應，都將提供以下資訊。

名稱

工作安全性對應的名稱。

工作安全性對應表也包含下列選項。

新建

建立新的工作安全性對應的按鈕。

刪除

刪除一或多個所選取工作安全性對應的按鈕。
