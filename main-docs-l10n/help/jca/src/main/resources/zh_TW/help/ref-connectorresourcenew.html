新建連線器資源

使用「新建連線器資源」頁面可建立新的連線器資源。

「新建連線器資源」頁面包含下列選項。

JNDI 名稱

可識別連線器資源的唯一名稱。

集區名稱

新的連線器資源所屬的連線集區。

描述

連線器資源的描述。

狀態

若選取了此選項，則會啟用連線器資源。此選項預設為啟用。

其他特性

連線器資源的其他特性。可用的特性會視連線器連線集區的資源轉接器而定。此表中指定的名/值配對，可用於置換資源轉接器供應商定義的特性預設值。

目標

(已在網域中建立叢集或獨立執行處理時才會顯示此選項。)

資源的叢集和獨立執行處理。可使用「加入」與「全部加入」按鈕，將所需的目標移至「選取的目標」欄。可使用「移除」與「全部移除」按鈕，將任何不需要的目標移至「可用的目標」欄。
