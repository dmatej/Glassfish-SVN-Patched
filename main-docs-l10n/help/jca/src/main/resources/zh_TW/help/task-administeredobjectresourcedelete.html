刪除管理物件資源

  1. 在瀏覽樹狀結構中，展開「資源」節點。

  2. 展開「連線器」節點。

  3. 在「連線器」節點下，選取「管理物件資源」節點。

    「管理物件資源」頁面會隨即開啟。

  4. 在「管理物件資源」頁面上，選取要刪除的管理物件資源名稱旁邊的核取方塊。

  5. 按一下「刪除」。
