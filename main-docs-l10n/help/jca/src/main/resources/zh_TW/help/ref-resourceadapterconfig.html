資源轉接器組態

使用「資源轉接器組態」頁面可管理連線器 (資源轉接器) 組態。

「資源轉接器組態」提供資源轉接器的組態資訊。您可以在建置資源轉接器後，使用管理主控台建立資源轉接器組態。在這個情況下，會使用新的組態重新啟動資源轉接器。若要在建置資源轉接器前建立資源轉接器組態，請使用 asadmin create-resource-adapter-config 命令。

您必須建立供資源轉接器使用的繫線集區，才能建立資源轉接器組態。

對於每個資源轉接器組態，都將提供以下資訊。

名稱

資源轉接器的名稱。

組態表也包含下列選項。

新建

建立新的資源轉接器組態的按鈕。

刪除

刪除一或多個所選取的資源轉接器組態的按鈕。
