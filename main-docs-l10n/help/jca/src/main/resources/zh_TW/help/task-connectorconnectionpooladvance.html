編輯連線器連線集區進階屬性

  1. 在瀏覽樹狀結構中，展開「資源」節點。

  2. 展開「連線器」節點。

  3. 在「連線器」節點下，選取「連線器連線集區」節點。

    「連線器連線集區」頁面會隨即開啟。

  4. 在「連線器連線集區」頁面上，按一下要編輯的連線器連線集區名稱。

    「編輯連線器連線集區」頁面會隨即開啟。

  5. 在「編輯連線器連線集區」頁面上，按一下「進階」標籤。

    「編輯連線器連線集區進階屬性」頁面會隨即開啟。

  6. 在「驗證間隔最久時間」欄位中，鍵入到達驗證連線間隔最長時間之前，等待的時間長度 (以秒為單位)。

    將此屬性設定為適當值，可將驗證連線請求的數目減至最小。預設值為 0，表示未啟用此屬性。

  7. 在「錯誤逾時」欄位中，鍵入追蹤連線集區中錯誤連線的時間長度 (以秒為單位)。

    0 值表示已停用連線洩漏追蹤。預設值為 0。

  8. 選取「錯誤收回」核取方塊，可在連線錯誤追蹤完成之後，將錯誤連線復原至連線集區。

    此選項預設為停用。

  9. 在「建立重新嘗試次數」欄位中，鍵入建立新連線失敗之後，將進行的嘗試次數。

    0 值表示不會再次嘗試建立連線。預設值為 0。

  10. 在「重試間隔」欄位中，鍵入兩次嘗試建立連線之間的時間間隔 (以秒為單位)。

    僅當「建立重新嘗試次數」的值大於 0 時，才使用該屬性。預設值為 10 秒。

  11. 選取「集區管理」核取方塊可針對集區啟用連線集區管理。

    此選項預設為啟用。

  12. 選取「惰性關聯」核取方塊可在作業於連線上執行時，為連線建立惰性關聯。

    當作業事件完成且元件方法結束時，這些連線便會取消關聯，從而可以重複使用實體連線。此選項預設為停用。

  13. 選取「惰性連線登記」核取方塊，可僅在資源實際用於方法中時，才將資源登記至作業事件。

    此選項預設為停用。

  14. 選取「與繫線建立關聯」核取方塊，可將連線與繫線建立關聯。

    當相同的繫線需要連線時，便可以重複使用已與該繫線建立關聯的連線，而省下從集區中取得連線的經常性耗用時間。此選項預設為停用。

  15. 選取「比對連線」核取方塊，可啟用集區的連線比對功能。

    如果您知道集區中的連線一定是同質連線，因此不需要資源轉接器對從集區中挑選的連線進行比對，可以停用此選項。此選項預設為啟用。

  16. 在「比對連線使用率」欄位中，鍵入集區重複使用連線的次數。

    重複使用指定次數的連線後，就會關閉連線。0 值表示沒有啟用此屬性。預設值為 0。

  17. 按一下「儲存」。

    提示:

    若要回到預設設定，請按一下「載入預設值」按鈕，然後按一下「儲存」。
