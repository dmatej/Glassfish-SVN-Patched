建立工作安全性對應

  1. 在瀏覽樹狀結構中，展開「資源」節點。

  2. 展開「連線器」節點。

  3. 在「連線器」節點下，選取「工作安全性對應」節點。

    「工作安全性對應」頁面會隨即開啟。

  4. 在「工作安全性對應」頁面上，按一下「新建」。

    「新建工作安全性對應」頁面會隨即開啟。

  5. 在「安全性對應名稱」欄位中，鍵入工作安全性對應的名稱。

  6. 以如下方式使用「群組」欄位或「主體」欄位。

    • 如果您選取「群組」單選按鈕，請鍵入每個後端 EIS 使用者群組與 GlassFish Server 安全性網域中定義的對應使用者群組的對應。

      使用以逗號分隔的清單指定一個以上的對應。例如︰

      eis-group-1=server-group-1,eis-group-2=server-group-2

    • 如果您選取「主體」單選按鈕，請鍵入每個後端 EIS 主體與 GlassFish Server 安全性網域中定義的對應主體的對應。

      使用以逗號分隔的清單指定一個以上的對應。例如︰

      eis-principal-1=server-principal-1,eis-principal-2=server-principal-2

  7. 從「資源轉接器」下拉式清單中，選取 EIS 的資源轉接器。

  8. 在「描述」欄位中，鍵入工作安全性對應的描述。

  9. 選取「啟用狀態」核取方塊啟用工作安全性對應。

    此選項預設為啟用。

  10. 按一下「確定」。
