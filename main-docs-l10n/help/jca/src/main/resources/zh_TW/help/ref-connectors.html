連線器

使用「連線器」頁面可組態下列類型的連線器資源：

  • 連線器資源

  • 連線器連線集區

  • 管理物件資源

  • 工作安全性對應

連線器模組 (也稱為資源轉接器) 是一個 Java 元件，它可讓應用程式與企業資訊 (EIS) 軟體互動。EIS 軟體包含各種類型的系統：企業資源規劃 (ERP)、主機作業事件處理、非關聯式資料庫和訊息傳送系統 (例如 Java 訊息服務 (JMS)) 以及其他系統。

連線器連線集區是針對特定 EIS 的一組可重複使用的連線。

連線器資源是為應用程式提供 EIS 連線的程式物件。

管理物件資源會為應用程式提供專用化功能。

工作安全性對應會將與內送工作執行處理關聯的主體對應到 GlassFish Server 安全性網域中的主體。
