編輯資源轉接器組態

  1. 在瀏覽樹狀結構中，展開「資源」節點。

  2. 在「資源」節點下，選取「資源轉接器組態」節點。

    「資源轉接器組態」頁面會隨即開啟。

  3. 在「資源轉接器組態」頁面上，按一下要編輯的資源轉接器組態名稱。

    「編輯資源轉接器組態」頁面會隨即開啟。

  4. 從「繫線集區 ID」下拉式清單中，選取要用於資源轉接器組態的繫線集區。

  5. 在「其他特性」區段中，指定資源轉接器的其他特性。

    可用的特性包括 ra.xml 檔案中 resourceadapter-class 元素所參照的類別之 Setter 方法名稱，該檔案定義了資源轉接器 JavaBeans 元件的類別名稱。對此處定義的特性所做的任何修改都會置換 ra.xml 中顯示的預設值。

    • 若要增加特性，請按一下「增加特性」按鈕。在出現的空白列中，於「名稱」欄位中鍵入特性名稱，然後在「值」欄位中鍵入特性值。

    • 若要修改特性，請編輯特性的「值」欄位。

    • 若要刪除特性，請選取您要刪除之特性「名稱」欄位左側的核取方塊，然後按一下「刪除特性」按鈕。

  6. 按一下「儲存」。
