編輯連線器連線集區的安全性對應

  1. 在瀏覽樹狀結構中，展開「資源」節點。

  2. 展開「連線器」節點。

  3. 在「連線器」節點下，選取「連線器連線集區」節點。

    「連線器連線集區」頁面會隨即開啟。

  4. 在「連線器連線集區」頁面上，按一下要編輯的連線集區名稱。

    「編輯連線器連線集區」頁面會隨即開啟。

  5. 在「編輯連線器連線集區」頁面上，按一下「安全性對應」標籤。

    「連線器連線集區安全性對應」頁面會隨即開啟。

  6. 在「連線器連線集區安全性對應」頁面上，按一下要編輯的安全性對應名稱。

    「編輯連線器連線集區安全性對應」頁面會隨即開啟。

  7. 進行以下其中一項作業 (不要兩者都進行)：

    • 選取「使用者群組」單選按鈕並鍵入使用者群組名稱 (以逗號分隔)。

    • 選取「主體」單選按鈕並鍵入主體名稱 (以逗號分隔)。

  8. 在「使用者名稱」欄位中，指定 EIS 所需的後端主體使用者名稱。

  9. 在「密碼」欄位中，指定 EIS 所需的後端主體密碼。

  10. 按一下「儲存」。
