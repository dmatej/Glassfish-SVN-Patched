新建資源轉接器組態

使用「新建資源轉接器組態」頁面可建立新的資源轉接器組態。

「新建資源轉接器組態」頁面包含下列選項。

資源轉接器名稱

您要組態的資源轉接器名稱。

繫線集區 ID

要用於資源轉接器組態的繫線集區。

其他特性

資源轉接器的其他特性。可用的特性包括 ra.xml 檔案中 resourceadapter-class 元素所參照的類別之 Setter 方法名稱，該檔案定義了資源轉接器 JavaBeans 元件的類別名稱。對此處定義的特性所做的任何修改都會置換 ra.xml 中顯示的預設值。
