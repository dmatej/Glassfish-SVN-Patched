連線器連線集區

使用「連線器連線集區」頁面可組態連線器連線集區。

連線器連線集區是針對特定 EIS 的一組可重複使用的連線。

對於每個資源，都將提供以下資訊。

集區名稱

可識別連線器連線集區的唯一名稱。

資源轉接器

與連線器連線集區關聯的資源轉接器。

連線定義

連線器連線集區的連線定義類別名稱。

描述

連線器連線集區的描述。

連線集區表也包含下列選項。

新建

建立新連線器連線集區的按鈕。

刪除

刪除一或多個所選取連線器連線集區的按鈕。
