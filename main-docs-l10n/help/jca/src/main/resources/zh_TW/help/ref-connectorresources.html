連線器資源

使用「連線器資源」頁面可組態連線器資源。

連線器資源是為應用程式提供 EIS 連線的程式物件。

對於每個資源，都將提供以下資訊。

JNDI 名稱

可識別連線器資源的唯一名稱。

已啟用

如果連線器資源已啟用，則為 True；如果連線器資源已停用，則為 False。

連線集區

與連線器資源相關聯的連線器連線集區。

描述

連線器資源的描述。

資源表也包含下列選項。

新建

建立新連線器資源的按鈕。

刪除

刪除一或多個所選取連線器資源的按鈕。

啟用

啟用一或多個所選取連線器資源的按鈕。

停用

停用一或多個所選取連線器資源的按鈕。
