新建連線器連線集區安全性對應

使用「新建連線器連線集區安全性對應」頁面可針對連線器連線集區建立新的安全性對應。

「新建連線器連線集區安全性對應」頁面包含下列選項。

連線集區名稱

此安全性對應所屬的連線器連線集區名稱。「集區名稱」是唯讀欄位。

安全性對應名稱

安全性對應的名稱。

使用者群組

若選取了此單選按鈕，則文字欄位中必須包含使用者群組的名稱 (以逗號分隔)。

主體

若選取了此單選按鈕，則文字欄位中必須包含主體的名稱 (以逗號分隔)。

使用者名稱

EIS 所需的後端主體使用者名稱。

密碼

EIS 所需的後端主體密碼。
