新建連線器連線集區 (步驟 1，共 2 個步驟)

使用「新建連線器連線集區 (步驟 1，共 2 步)」頁面可建立新的連線器連線集區。

「新建連線器連線集區 (步驟 1，共 2 步)」頁面包含下列選項。

名稱

可識別連線器連線集區的唯一名稱。

資源轉接器

與連線器連線集區關聯的資源轉接器。

連線定義

連線器連線集區的連線定義。清單中提供的選項會視您所選取的資源轉接器而定。連線定義會識別資源轉接器在 ra.xml 檔案中的 connection-definition 元素。
