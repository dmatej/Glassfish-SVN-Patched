組態連線器服務

  1. 在瀏覽樹狀結構中，展開「組態」節點。

  2. 在「組態」節點下，選取「連線器服務」節點。

    將開啟「連線器服務」頁面。

  3. 在「關機逾時」欄位中鍵入 GlassFish Server 關機期間可等候連線器模組執行處理的 ResourceAdapter.stop 方法完成的最大秒數。

    會忽略需要更長時間關機的資源轉接器並繼續 GlassFish Server 關機動作。預設值為 30 秒。

  4. 從「連線器類別載入策略」下拉式清單中，選取要使用的策略。

    • 選取 derived 可指定根據應用程式的參照將資源轉接器提供給任何資源轉接器的資源。

      此策略為預設值。

    • 選取 global 可指定對所有應用程式顯示所有資源轉接器。

    此策略不會套用到預先安裝的 JMS 系統資源轉接器 jmsra，該資源轉接器會永遠對所有應用程式顯示。

  5. 按一下「儲存」。

    提示:

    若要回到預設設定，請按一下「載入預設值」按鈕，然後按一下「儲存」。
