커넥터 리소스

[커넥터 리소스] 페이지에서는 커넥터 리소스를 구성할 수 있습니다.

커넥터 리소스는 응용 프로그램과의 EIS 접속을 제공하는 프로그램 객체입니다.

각 리소스에 대해 제공되는 정보는 다음과 같습니다.

JNDI 이름

커넥터 리소스를 식별하는 고유한 이름입니다.

사용

(이 열은 기본 서버 인스턴스인 server가 도메인에 존재하는 경우에만 표시됩니다.)

리소스가 사용으로 설정된 경우 선택 표시가 나타나고 사용 안함으로 설정된 경우 X 표시가 나타납니다.

상태

(이 열은 클러스터나 독립형 인스턴스가 도메인에 생성된 경우에만 표시됩니다.)

리소스의 대상으로 지정된 클러스터 및 독립형 인스턴스의 수, 그리고 리소스가 사용으로 설정된 이러한 대상의 수입니다. 예를 들어, "2/4개의 대상에서 사용"은 리소스의 대상으로 4개의 클러스터 및 독립형 인스턴스가 지정되었고 이 4개의 대상 중 2개에서 리소스가 사용으로 설정되었음을 의미합니다.

접속 풀

커넥터 리소스와 연관된 커넥터 접속 풀입니다.

설명

커넥터 리소스에 대한 설명입니다.

리소스 테이블에는 다음 옵션도 포함되어 있습니다.

새로 만들기

새 커넥터 리소스를 생성하는 단추입니다.

삭제

선택한 커넥터 리소스를 하나 이상 삭제하는 단추입니다.

사용

선택한 하나 이상의 커넥터 리소스를 사용으로 설정하는 단추입니다.

사용 안함

선택한 하나 이상의 커넥터 리소스를 사용 안함으로 설정하는 단추입니다.

관련 작업

  • 커넥터 리소스를 편집하는 방법

  • 커넥터 리소스를 삭제하는 방법
