커넥터 접속 풀 속성 편집

[커넥터 접속 풀 속성 편집] 페이지에서는 커넥터 접속 풀에 대한 속성을 수정할 수 있습니다.

[커넥터 접속 풀 속성 편집] 페이지에는 다음 옵션이 포함되어 있습니다.

추가 속성

커넥터 접속 풀에 대한 추가 속성입니다. 사용 가능한 속성은 커넥터 접속 풀의 리소스 어댑터에 따라 다릅니다. 이 테이블에 지정된 이름-값 쌍을 사용하여 리소스 어댑터 공급업체에서 정의한 속성의 기본값을 대체할 수 있습니다.

속성값을 지정하지 않으면 접속 풀 편집을 완료할 경우 이 값이 테이블에서 제거됩니다. 지금은 속성값을 설정하지 않지만 나중에 설정할 수 있도록 속성을 테이블에 유지하려면 [값] 필드에 일련의 빈 괄호를 입력합니다.
