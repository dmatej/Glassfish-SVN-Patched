커넥터 접속 풀 고급 속성을 편집하는 방법

  1. 탐색 트리에서 [리소스] 노드를 확장합니다.

  2. [커넥터] 노드를 확장합니다.

  3. [커넥터] 노드에서 [커넥터 접속 풀] 노드를 선택합니다.

    [커넥터 접속 풀] 페이지가 열립니다.

  4. [커넥터 접속 풀] 페이지에서 편집할 커넥터 접속 풀의 이름을 누릅니다.

    [커넥터 접속 풀 편집] 페이지가 열립니다.

  5. [커넥터 접속 풀 편집] 페이지에서 [고급] 탭을 누릅니다.

    [커넥터 접속 풀 고급 속성 편집] 페이지가 열립니다.

  6. [최대 1회 검증 기간] 필드에서 접속이 최대 1회 검증되기까지의 시간(초)을 입력합니다.

    이 속성을 적절한 값으로 설정하면 접속의 검증 요청 횟수가 최소화됩니다. 기본값은 0입니다. 값이 0이면 속성이 사용으로 설정되지 않습니다.

  7. [누수 시간 초과] 필드에서 접속 풀의 접속 누수를 추적할 시간(초)을 식별합니다.

    값이 0이면 접속 누수 추적을 사용 안함으로 설정되어 있음을 나타냅니다. 기본값은 0입니다.

  8. 누수 접속 추적이 완료된 후 누수된 접속을 풀로 복원하려면 [누수 재생] 체크 박스를 선택합니다.

    이 옵션은 기본적으로 사용 안함으로 설정되어 있습니다.

  9. [생성 재시도 횟수] 필드에서 새 접속 생성에 실패할 경우 재시도할 횟수를 입력합니다.

    값이 0이면 접속 생성을 다시 시도하지 않음을 의미합니다. 기본값은 0입니다.

  10. [재시도 간격] 필드에서 접속을 생성하기 위한 두 시도 간의 간격(초)을 입력합니다.

    이 속성은 생성 재시도 횟수의 값이 0보다 큰 경우에만 사용됩니다. 기본값은 10초입니다.

  11. 풀에 대한 접속 풀링을 사용으로 설정하려면 [풀링] 체크 박스를 선택합니다.

    이 옵션은 기본적으로 사용으로 설정되어 있습니다.

  12. 접속에서 작업을 수행할 때 접속이 지연되도록 하려면 [느린 연결] 체크 박스를 선택합니다.

    트랜잭션이 완료되고 구성 요소 메소드가 종료되면 접속이 해제되어 물리적 접속을 재사용할 수 있습니다. 이 옵션은 기본적으로 사용 안함으로 설정되어 있습니다.

  13. 실제 메소드에서 사용할 경우에만 트랜잭션에 리소스를 등록하려면 [지연 접속 참여] 체크 박스를 선택합니다.

    이 옵션은 기본적으로 사용 안함으로 설정되어 있습니다.

  14. 접속을 스레드와 연결하려면 [스레드와 연결] 체크 박스를 선택합니다.

    동일한 스레드에서 접속을 필요로 하는 경우 이미 연관된 접속을 다시 사용할 수 있으므로 풀에서 접속을 가져오기 위한 오버헤드가 발생하지 않습니다. 이 옵션은 기본적으로 사용 안함으로 설정되어 있습니다.

  15. 풀에 대해 일치하는 접속을 사용으로 설정하려면 [접속 일치] 체크 박스를 선택합니다.

    풀의 접속이 항상 같은 종류이며 따라서 풀에서 선택된 접속을 리소스 어댑터와 일치시킬 필요가 없음을 알고 있는 경우 이 옵션을 사용 안함으로 설정합니다. 이 옵션은 기본적으로 사용으로 설정되어 있습니다.

  16. [접속 사용 일치] 필드에 풀에서 접속을 재사용할 횟수를 입력합니다.

    접속을 지정된 횟수만큼 재사용한 다음에는 해당 접속이 닫힙니다. 값이 0이면 이 속성을 사용하지 않음을 나타냅니다. 기본값은 0입니다.

  17. [저장]을 누릅니다.

    팁:

    기본 설정으로 돌아가려면 [기본값 로드] 단추를 누른 다음 [저장]을 누릅니다.
