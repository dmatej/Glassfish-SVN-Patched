리소스 어댑터 구성을 편집하는 방법

  1. 탐색 트리에서 [리소스] 노드를 확장합니다.

  2. [리소스] 노드에서 [리소스 어댑터 구성] 노드를 선택합니다.

    [리소스 어댑터 구성] 페이지가 열립니다.

  3. [리소스 어댑터 구성] 페이지에서 편집할 리소스 어댑터 구성의 이름을 누릅니다.

    [리소스 어댑터 구성 편집] 페이지가 열립니다.

  4. [스레드 풀 ID] 드롭다운 목록에서 리소스 어댑터 구성에 대해 사용할 스레드 풀을 선택합니다.

  5. [배치 순서] 필드에 리소스 어댑터의 배치 순서를 지정하는 숫자를 입력합니다.

    서버 시작 시 낮은 번호의 리소스부터 로드됩니다. 기본값은 100입니다.

  6. [추가 속성] 섹션에서 리소스 어댑터에 대한 추가 속성을 지정합니다.

    사용 가능한 속성은 리소스 어댑터 JavaBeans 구성 요소의 클래스 이름을 정의하는 ra.xml 파일의 resourceadapter-class 요소로 참조되는 클래스의 setter 메소드의 이름입니다. 여기에 정의된 속성에 대한 모든 수정 사항은 ra.xml에 있는 기본값을 대체합니다.

    • 속성을 추가하려면 [속성 추가] 단추를 누릅니다. 빈 행이 표시되면 [이름] 필드에 속성 이름을 입력하고 [값] 필드에 속성값을 입력합니다.

    • 속성을 수정하려면 속성의 [값] 필드를 편집합니다.

    • 속성을 삭제하려면 삭제할 속성의 [이름] 필드 왼쪽에 있는 체크 박스를 선택한 다음 [속성 삭제] 단추를 누릅니다.

  7. [저장]을 누릅니다.
