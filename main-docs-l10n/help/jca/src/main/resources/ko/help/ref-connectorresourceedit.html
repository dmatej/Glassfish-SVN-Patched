커넥터 리소스 편집

[커넥터 리소스 편집] 페이지에서는 커넥터 리소스에 대한 설정을 수정할 수 있습니다.

[커넥터 리소스 편집] 페이지에는 다음 옵션이 포함되어 있습니다.

기본값 로드

기본값을 사용하는 설정을 기본값으로 복원하는 단추입니다. 기본값을 사용하지 않는 설정은 변경되지 않습니다.

JNDI 이름

커넥터 리소스를 식별하는 고유한 이름입니다. [JNDI 이름]은 읽기 전용 필드입니다. 새 커넥터 리소스를 생성할 때만 JNDI 이름을 지정할 수 있습니다.

풀 이름

이 리소스와 연관된 커넥터 접속 풀입니다.

설명

커넥터 리소스에 대한 설명입니다.

상태

이 옵션을 선택하면 커넥터 리소스가 사용으로 설정됩니다. 이 옵션은 기본적으로 사용으로 설정되어 있습니다.

추가 속성

커넥터 리소스에 대한 추가 속성입니다. 사용 가능한 속성은 커넥터 접속 풀의 리소스 어댑터에 따라 다릅니다. 이 테이블에 지정된 이름-값 쌍을 사용하여 리소스 어댑터 공급업체에서 정의한 속성의 기본값을 대체할 수 있습니다.

관련 작업

  • 커넥터 리소스를 생성하는 방법

  • 커넥터 리소스를 편집하는 방법

  • 커넥터 리소스를 삭제하는 방법

  • create-connector-resource(1)

  • list-connector-resources(1)
