커넥터 서비스

[커넥터 서비스] 페이지에서는 리소스 어댑터를 제어하는 커넥터 서비스에 대한 일반 설정을 수정할 수 있습니다.

[커넥터 서비스] 페이지에는 다음 옵션이 포함되어 있습니다.

기본값 로드

기본값을 사용하는 설정을 기본값으로 복원하는 단추입니다. 기본값을 사용하지 않는 설정은 변경되지 않습니다.

구성 이름

이 페이지의 설정이 적용되는 구성의 이름입니다. 이 필드는 읽기 전용입니다.

종료 시간 초과

GlassFish Server 종료 중 커넥터 모듈 인스턴스의 ResourceAdapter.stop 메소드가 완료될 때까지 허용되는 최대 시간(초)입니다. 종료하는 데 시간이 더 오래 걸리는 리소스 어댑터는 무시되며 GlassFish Server 종료가 계속됩니다. 기본값은 30초입니다.

커넥터 클래스 로드 정책

클래스를 로드하는 데 사용할 정책입니다. 사용 가능한 선택 항목은 다음과 같습니다.

derived

리소스 어댑터의 리소스에 대한 응용 프로그램의 참조에 따라 리소스 어댑터가 제공됨을 나타냅니다. 이 정책은 기본값입니다.

global

모든 응용 프로그램에 모든 리소스 어댑터가 표시됨을 나타냅니다.

이 정책은 항상 모든 응용 프로그램에 표시되는 사전 설치된 JMS 시스템 리소스 어댑터 jmsra에는 적용되지 않습니다.
