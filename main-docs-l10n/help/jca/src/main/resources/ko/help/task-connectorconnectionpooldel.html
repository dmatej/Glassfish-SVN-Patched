커넥터 접속 풀을 삭제하는 방법

커넥터 접속 풀을 삭제할 때 해당 커넥터 접속 풀에 종속된 모든 커넥터 리소스도 삭제됩니다.

  1. 탐색 트리에서 [리소스] 노드를 확장합니다.

  2. [커넥터] 노드를 확장합니다.

  3. [커넥터] 노드에서 [커넥터 접속 풀] 노드를 선택합니다.

    [커넥터 접속 풀] 페이지가 열립니다.

  4. [커넥터 접속 풀] 페이지에서 삭제할 접속 풀 이름 옆의 체크 박스를 선택합니다.

  5. [삭제]를 누릅니다.
