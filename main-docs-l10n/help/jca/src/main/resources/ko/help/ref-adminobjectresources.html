관리 객체 리소스

[관리 객체 리소스] 페이지에서는 관리 객체 리소스를 구성할 수 있습니다.

관리 객체 리소스는 배치된 커넥터 모듈용 리소스 어댑터로 정의된 특수 기능을 제공합니다.

각 관리 객체 리소스에 대해 제공되는 정보는 다음과 같습니다.

JNDI 이름

관리 객체 리소스를 식별하는 고유한 이름입니다.

사용

(이 열은 기본 서버 인스턴스인 server가 도메인에 존재하는 경우에만 표시됩니다.)

리소스가 사용으로 설정된 경우 선택 표시가 나타나고 사용 안함으로 설정된 경우 X 표시가 나타납니다.

상태

(이 열은 클러스터나 독립형 인스턴스가 도메인에 생성된 경우에만 표시됩니다.)

리소스의 대상으로 지정된 클러스터 및 독립형 인스턴스의 수, 그리고 리소스가 사용으로 설정된 이러한 대상의 수입니다. 예를 들어, "2/4개의 대상에서 사용"은 리소스의 대상으로 4개의 클러스터 및 독립형 인스턴스가 지정되었고 이 4개의 대상 중 2개에서 리소스가 사용으로 설정되었음을 의미합니다.

리소스 유형

관리 객체 리소스의 Java 유형입니다.

설명

관리 객체 리소스에 대한 설명입니다.

리소스 테이블에는 다음 옵션도 포함되어 있습니다.

새로 만들기

새 관리 객체 리소스를 생성하는 단추입니다.

삭제

선택한 관리 객체 리소스를 하나 이상 삭제하는 단추입니다.

사용

선택한 하나 이상의 관리 객체 리소스를 사용으로 설정하는 단추입니다.

사용 안함

선택한 하나 이상의 관리 객체 리소스를 사용 안함으로 설정하는 단추입니다.

관련 작업

  • 관리 객체 리소스를 편집하는 방법

  • 관리 객체 리소스를 삭제하는 방법
