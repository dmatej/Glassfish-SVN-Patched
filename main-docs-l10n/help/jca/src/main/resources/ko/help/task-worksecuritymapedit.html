작업 보안 맵을 편집하는 방법

  1. 탐색 트리에서 [리소스] 노드를 확장합니다.

  2. [커넥터] 노드를 확장합니다.

  3. [커넥터] 노드에서 [작업 보안 맵] 노드를 선택합니다.

    [작업 보안 맵] 페이지가 열립니다.

  4. [작업 보안 맵] 페이지에서 편집할 작업 보안 맵의 이름을 누릅니다.

    [작업 보안 맵 편집] 페이지가 열립니다.

  5. 다음과 같이 [그룹] 필드 또는 [주체] 필드를 사용합니다.

    • [그룹] 라디오 단추를 선택한 경우, 각 백엔드 EIS 사용자 그룹과 GlassFish Server 보안 도메인에서 정의된 해당 사용자 그룹 간의 매핑을 입력합니다.

      콤마로 구분된 목록을 사용하여 매핑을 두 개 이상 지정합니다. 예를 들면 다음과 같습니다.

      eis-group-1=server-group-1,eis-group-2=server-group-2

    • [주체] 라디오 단추를 선택한 경우, 각 백엔드 EIS 주체와 GlassFish Server 보안 도메인에서 정의한 해당 주체 간의 매핑을 입력합니다.

      콤마로 구분된 목록을 사용하여 매핑을 두 개 이상 지정합니다. 예를 들면 다음과 같습니다.

      eis-principal-1=server-principal-1,eis-principal-2=server-principal-2

  6. [리소스 어댑터] 드롭다운 목록에서 EIS에 대한 리소스 어댑터를 선택합니다.

  7. [설명] 필드에 작업 보안 맵에 대한 설명을 입력합니다.

  8. 작업 보안 맵을 사용으로 설정하려면 [상태 사용] 체크 박스를 선택합니다.

    이 옵션은 기본적으로 사용으로 설정되어 있습니다.

  9. [확인]을 누릅니다.
