资源适配器配置

使用“资源适配器配置”页可以管理连接器 (资源适配器) 配置。

资源适配器配置提供资源适配器的配置信息。使用管理控制台，您可在部署资源适配器之后创建资源适配器配置。在此情况下，资源适配器将使用新配置重新启动。要在部署资源适配器之前创建资源适配器配置，请使用 asadmin create-resource-adapter-config 命令。

在创建资源适配器配置之前，必须为要使用的资源适配器创建一个线程池。

对于每个资源适配器配置，将提供以下信息。

名称

资源适配器的名称。

“配置”表还包含以下选项。

新建

用于创建新资源适配器配置的按钮。

删除

用于删除一个或多个所选资源适配器配置的按钮。

  • 删除资源适配器配置
