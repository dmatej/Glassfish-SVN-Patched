新建工作安全映射

使用“新建工作安全映射”页可以创建新的工作安全映射。

“新建工作安全映射”包含以下选项。

安全映射名称

工作安全映射的名称。

组

如果选中“组映射”单选按钮，则文本字段必须指定每个后端 EIS 用户组与在 GlassFish Server 安全域中定义的相应用户组的映射。可使用逗号分隔列表指定多个映射。例如：

eis-group-1=server-group-1,eis-group-2=server-group-2
主用户

如果选中“主用户映射”单选按钮，则文本字段必须指定每个后端 EIS 主用户与在 GlassFish Server 安全域中定义的相应主用户的映射。可使用逗号分隔列表指定多个映射。例如：

eis-principal-1=server-principal-1,eis-principal-2=server-principal-2
资源适配器

EIS 的资源适配器。

说明

工作安全映射的说明。

状态

如果选中此选项，则将启用工作安全映射。默认情况下启用此选项。
