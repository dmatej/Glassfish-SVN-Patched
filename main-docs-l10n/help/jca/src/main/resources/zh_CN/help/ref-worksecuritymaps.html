工作安全映射

使用“工作安全映射”页可以配置资源适配器的工作安全映射。

连接器工作安全映射将由资源适配器 EIS 主用户或 EIS 用户组提交的工作的调用方标识映射到 GlassFish Server 安全域中的合适主用户或用户组。可将一个或多个工作安全映射与资源适配器关联。连接器工作安全映射配置支持使用通配符星号 (*) 来表示所有用户或所有用户组。

工作安全映射与连接器连接池的安全映射有所不同，后者将在 Servlet 或 EJB 验证过程中接收的主用户映射到 EIS 接受的身份证明。有关连接器连接池的安全映射的信息，请参阅连接器连接池安全映射。

企业信息系统 (EIS) 是保存组织数据的任何系统。它可以是大型机、消息传送系统、数据库系统或应用程序。

对于每个工作安全映射，将提供以下信息。

名称

工作安全映射的名称。

“工作安全映射”表还包含以下选项。

新建

用于创建新工作安全映射的按钮。

删除

用于删除一个或多个所选工作安全映射的按钮。
