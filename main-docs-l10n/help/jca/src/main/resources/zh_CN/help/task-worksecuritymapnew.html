创建工作安全映射

  1. 在导航树中，展开“资源”节点。

  2. 展开“连接器”节点。

  3. 在“连接器”节点下，选择“工作安全映射”节点。

    此时将打开“工作安全映射”页。

  4. 在“工作安全映射”页上，单击“新建”。

    此时将打开“新建工作安全映射”页。

  5. 在“安全映射名称”字段中，键入工作安全映射的名称。

  6. 按以下所述使用“组”字段或“主用户”字段。

    • 如果选中“组”单选按钮，请键入每个后端 EIS 用户组与在 GlassFish Server 安全域中定义的相应用户组的映射。

      可使用逗号分隔列表指定多个映射。例如：

      eis-group-1=server-group-1,eis-group-2=server-group-2

    • 如果选中“主用户”单选按钮，请键入每个后端 EIS 主用户与在 GlassFish Server 安全域中定义的相应主用户的映射。

      可使用逗号分隔列表指定多个映射。例如：

      eis-principal-1=server-principal-1,eis-principal-2=server-principal-2

  7. 从“资源适配器”下拉列表中，选择 EIS 的资源适配器。

  8. 在“说明”字段中，键入工作安全映射的说明。

  9. 选择“启用状态”复选框以启用工作安全映射。

    默认情况下启用此选项。

  10. 单击“确定”。
