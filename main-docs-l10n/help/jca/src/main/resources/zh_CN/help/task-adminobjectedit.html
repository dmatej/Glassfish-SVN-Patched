编辑受管理对象资源

  1. 在导航树中，展开“资源”节点。

  2. 展开“连接器”节点

  3. 在“连接器”节点下，选择“管理对象资源”节点。

    此时将打开“管理对象资源”页。

  4. 在“管理对象资源”页上，单击要编辑的受管理对象资源的名称。

    此时将打开“编辑管理对象资源”页。

  5. 从“资源适配器”下拉列表中，选择受管理对象资源的资源适配器。

  6. 从“资源类型”下拉列表中，选择受管理对象资源的 Java 类型。

  7. 从“类名”下拉列表中，选择与资源类型关联的实现类名。

  8. 在“说明”字段中，键入受管理对象资源的说明。

  9. 选择“启用状态”复选框以启用受管理对象资源。

    默认情况下启用此选项。

  10. 在“其他属性”表中，指定受管理对象资源的其他属性。

    可用属性取决于连接器连接池的资源适配器。

    • 要添加属性，请单击“添加属性”按钮。在显示的空白行中，在“名称”字段中键入属性名称，在“值”字段中键入属性值。

    • 要修改属性，请编辑该属性的“值”字段。

    • 要删除属性，请选中位于您要删除属性的“名称”字段左边的复选框，然后单击“删除属性”按钮。

  11. 在“目标”选项卡上，单击“管理目标”可编辑部署了受管理对象的目标。

    通过单击“常规”选项卡返回到“编辑管理对象资源”页。

  12. 单击“保存”。

    提示:

    要返回到默认设置，请单击“加载默认值”按钮，然后单击“保存”。
