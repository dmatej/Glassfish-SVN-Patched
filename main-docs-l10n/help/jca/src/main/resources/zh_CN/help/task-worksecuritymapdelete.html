删除工作安全映射

  1. 在导航树中，展开“资源”节点。

  2. 展开“连接器”节点

  3. 在“连接器”节点下，选择“工作安全映射”节点。

    此时将打开“工作安全映射”页。

  4. 在“工作安全映射”页上，选中要删除的工作安全映射名称旁边的复选框。

  5. 单击“删除”。
