创建受管理对象资源

  1. 在导航树中，展开“资源”节点。

  2. 展开“连接器”节点

  3. 在“连接器”节点下，选择“管理对象资源”节点。

    此时将打开“管理对象资源”页。

  4. 在“管理对象资源”页上，单击“新建”按钮。

    此时将打开“新建管理对象资源”页。

  5. 在“JNDI 名称”字段中，键入标识受管理对象资源的唯一名称。

  6. 从“资源适配器”下拉列表中，选择受管理对象资源的资源适配器。

  7. 从“资源类型”下拉列表中，选择受管理对象资源的 Java 类型。

  8. 从“类名”下拉列表中，选择与资源类型关联的实现类名。

  9. 在“说明”字段中，键入受管理对象资源的说明。

  10. 选择“启用状态”复选框以启用受管理对象资源。

    默认情况下启用此选项。

  11. 在“其他属性”表中，指定受管理对象资源的其他属性。

    可用属性取决于连接器连接池的资源适配器。

    要添加属性，请单击“添加属性”按钮。在显示的空白行中，在“名称”字段中键入属性名称，在“值”字段中键入属性值。

  12. 在“可用目标”部分中，选择受管理对象将驻留的域、集群或服务器实例，然后单击“添加”。

    要删除“已选择”字段中列出的某个域、集群或服务器实例，请从该字段中选择该目标并单击“删除”。

  13. 单击“确定”。
