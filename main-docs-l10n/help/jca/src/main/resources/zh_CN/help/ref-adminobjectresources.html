管理对象资源

使用“管理对象资源”页可以配置受管理对象资源。

受管理对象资源提供了专门的功能，该功能由已部署连接器模块的资源适配器定义。

对于每个受管理对象资源，将提供以下信息。

JNDI 名称

标识受管理对象资源的唯一名称。

已启用

如果受管理对象资源已启用，则为“真”；如果受管理对象资源已禁用，则为“假”。

资源类型

受管理对象资源的 Java 类型。

说明

受管理对象资源的说明。

“资源”表还包含以下选项。

新建

用于创建新受管理对象资源的按钮。

删除

用于删除一个或多个所选受管理对象资源的按钮。

启用

用于启用一个或多个所选受管理对象资源的按钮。

禁用

用于禁用一个或多个所选受管理对象资源的按钮。
