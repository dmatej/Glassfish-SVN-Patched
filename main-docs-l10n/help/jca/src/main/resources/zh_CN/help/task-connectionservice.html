配置连接器服务

  1. 在导航树中，展开“配置”节点。

  2. 在“配置”节点下，选择“连接器服务”节点。

    此时将打开“连接器服务”页。

  3. 在“关闭超时”字段中，键入在 GlassFish Server 关闭过程中，允许用于完成连接器模块实例的 ResourceAdapter.stop 方法的最大秒数。

    需要更长时间关闭的资源适配器将被忽略，而 GlassFish Server 关闭操作将继续。默认值为 30 秒。

  4. 从“连接器类加载策略”下拉列表中，选择要使用的策略。

    • 选择 derived 可以指定根据应用程序对任何资源适配器资源的引用来提供资源适配器。

      此策略为默认策略。

    • 选择 global 可以指定所有资源适配器对所有应用程序都将是可见的。

    此策略不适用于预安装的 JMS 系统资源适配器 jmsra，该适配器对所有应用程序始终都是可见的。

  5. 单击“保存”。

    提示:

    要返回到默认设置，请单击“加载默认值”按钮，然后单击“保存”。
