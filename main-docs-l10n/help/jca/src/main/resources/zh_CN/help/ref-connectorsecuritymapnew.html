新建连接器连接池安全映射

使用“新建连接器连接池安全映射”页可以为连接器连接池创建新的安全映射。

“新建连接器连接池安全映射”页包含以下选项。

连接池名称

此安全映射所属的连接器连接池的名称。池名称是只读字段。

安全映射名称

安全映射的名称。

用户组

如果选中此单选按钮，则文本字段必须包含用户组的名称，以逗号分隔。

主用户

如果选中此单选按钮，则文本字段必须包含主用户的名称，以逗号分隔。

用户名

EIS 所需的后端主用户的用户名。

口令

EIS 所需的后端主用户口令。
