编辑连接器连接池属性

  1. 在导航树中，展开“资源”节点。

  2. 展开“连接器”节点。

  3. 在“连接器”节点下，选择“连接器连接池”节点。

    此时将打开“连接器连接池”页。

  4. 在“连接器连接池”页上，单击要编辑的连接器连接池的名称。

    此时将打开“编辑连接器连接池”页。

  5. 在“编辑连接器连接池”页上，单击“其他属性”选项卡。

    此时将打开“编辑连接器连接池属性”页。

  6. 在“其他属性”表中，指定连接器连接池的其他属性。

    指定的属性取决于此池使用的资源适配器。可以使用用户通过此表指定的名称/值对，覆盖由资源适配器提供方定义的属性默认值。

    如果您不指定某个属性的值，则在编辑完连接池时，该属性将从表中删除。如果您现在不希望设置某个属性的值，但希望将该属性保留在表中以便在以后设置值，请在“值”字段中键入一对空括号：

    ()

    • 要添加属性，请单击“添加属性”按钮。在显示的空白行中，在“名称”字段中键入属性名称，在“值”字段中键入属性值。

    • 要修改属性，请编辑该属性的“值”字段。

    • 要删除属性，请选中位于您要删除属性的“名称”字段左边的复选框，然后单击“删除属性”按钮。

  7. 单击“保存”。
