Editar configuración de adaptador de recursos

Utilice la página Editar configuración de adaptador de recursos para modificar la configuración del adaptador de recursos.

La página Editar configuración de adaptador de recursos contiene las siguientes opciones.

Nombre del adaptador de recursos

Nombre del adaptador de recursos que está configurando. Este campo es de sólo lectura. Sólo se puede especificar un nombre de adaptador de recursos cuando cree una nueva configuración del adaptador de recursos.

ID de Pool de Threads

Pool de threads que se va a utilizar para la configuración del adaptador de recursos.

Orden de despliegue

Orden de despliegue del adaptador de recursos.

Propiedades Adicionales

Propiedades adicionales para el adaptador de recursos. Las propiedades disponibles son los nombres de métodos setter de la clase a la que hace referencia el elemento resourceadapter-class del archivo ra.xml, que define el nombre de clase del componente JavaBeans del adaptador de recursos. Las modificaciones en las propiedades definidas aquí sustituyen los valores por defecto presentes en ra.xml.
