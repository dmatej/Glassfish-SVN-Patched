Configuraciones del Adaptador de Recursos

Utilice la página Configuraciones del adaptador de recursos para gestionar las configuraciones del conector (adaptador de recursos).

La configuración del adaptador de recursos proporciona información de configuración para un adaptador de recursos. Con la Consola de administración puedes crear una configuración del adaptador de recursos después de desplegar dicho adaptador. En este caso, el adaptador de recursos se reinicia con la nueva configuración. Para crear una configuración del adaptador de recursos antes de desplegar dicho adaptador, utilice el comando asadmin create-resource-adapter-config.

Antes de que pueda crear una configuración del adaptador de recursos, debe crear un pool de threads para que lo utilice dicho adaptador.

Para la configuración de cada adaptador de recursos, se proporciona la siguiente información.

Nombre

Nombre del adaptador de recursos.

La tabla Configuraciones también contiene las siguientes opciones.

Nuevo

Botón para crear una nueva configuración del adaptador de recursos.

Suprimir

Botón para suprimir una o más configuraciones seleccionadas del adaptador de recursos.
