コネクタ・リソースを削除するには

リソースに関連付けられたコネクタ接続プールを削除すると、コネクタ・リソースは自動的に削除されます。

  1. ナビゲーション・ツリーで、「リソース」ノードを展開します。

  2. 「コネクタ」ノードを展開します。

  3. 「コネクタ」ノードで、「コネクタ・リソース」ノードを選択します。

    「コネクタ・リソース」ページが開きます。

  4. 「コネクタ・リソース」ページで、削除するリソースの名前の隣にあるチェック・ボックスを選択します。

  5. 「削除」をクリックします。
