コネクタ

次のタイプのコネクタ・リソースを構成するには、「コネクタ」ページを使用します。

  • コネクタ・リソース

  • コネクタ接続プール

  • 管理オブジェクト・リソース

  • 作業セキュリティ・マップ

コネクタ・モジュール(リソース・アダプタとも呼ばれる)は、アプリケーションとエンタープライズ情報システム(EIS)ソフトウェアとの対話を可能にするJavaコンポーネントです。EISソフトウェアには、統合業務ソフトウェア(ERP)、メインフレーム・トランザクション処理、非リレーショナル・データベース、Java Message Service (JMS)などのメッセージング・システムなど、様々なタイプのシステムが含まれます。

コネクタ接続プールとは、特定のEISのための再利用可能な接続のグループです。

コネクタ・リソースとは、アプリケーションにEISへの接続を提供するプログラム・オブジェクトです。

管理対象オブジェクト・リソースは、アプリケーションに特別な機能を提供します。

作業セキュリティ・マップは、受信した作業インスタンスに関連付けられているプリンシパルを、GlassFish Serverのセキュリティ・ドメインにあるプリンシパルにマッピングします。
