コネクタ接続プールのセキュリティ・マップの編集

コネクタ接続プールのセキュリティ・マップを編集するには、「コネクタ接続プールのセキュリティ・マップの編集」ページを使用します。

「コネクタ接続プールのセキュリティ・マップの編集」ページには、次のオプションが含まれます。

接続プール名

このセキュリティ・マップを作成するコネクタ接続プールの名前。プール名は読取り専用フィールドです。

セキュリティ・マップ名

セキュリティ・マップの名前名前は読取り専用フィールドです。セキュリティ・マップ名を指定できるのは、新規セキュリティ・マップの作成時のみです。

セキュリティ・マップ名

セキュリティ・マップの名前

ユーザー・グループ

このラジオ・ボタンを選択する場合は、テキスト・フィールドにユーザー・グループの名前をカンマで区切って入力する必要があります。

プリンシパル

このラジオ・ボタンを選択する場合は、テキスト・フィールドにプリンシパルの名前をカンマで区切って入力する必要があります。

ユーザー名

EISで必要なバックエンド・プリンシパルのユーザー名。

パスワード

EISで必要なバックエンド・プリンシパルのパスワード。
