新規コネクタ接続プール(ステップ1/2)

新規コネクタ接続プールを作成するには、「新規コネクタ接続プール(ステップ1/2)」ページを使用します。

「新規コネクタ接続プール(ステップ1/2)」ページには、次のオプションが含まれます。

名前

コネクタ接続プールを識別する一意の名前。

リソース・アダプタ

コネクタ接続プールに関連付けられたリソース・アダプタ。

接続定義

コネクタ接続プールの接続定義。リストに表示される項目は、選択したリソース・アダプタによって異なります。接続定義によって、リソース・アダプタのconnection-definition要素がra.xmlファイル内で識別されます。
