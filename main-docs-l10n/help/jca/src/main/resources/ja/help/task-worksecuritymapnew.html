作業セキュリティ・マップを作成するには

  1. ナビゲーション・ツリーで、「リソース」ノードを展開します。

  2. 「コネクタ」ノードを展開します。

  3. 「コネクタ」ノードで、「作業セキュリティ・マップ」ノードを選択します。

    「作業セキュリティ・マップ」ページが開きます。

  4. 「作業セキュリティ・マップ」ページで、「新規」をクリックします。

    「新規作業セキュリティ・マップ」ページが開きます。

  5. 「セキュリティ・マップ名」フィールドに、作業セキュリティ・マップの名前を入力します。

  6. 「グループ」フィールドまたは「プリンシパル」フィールドのいずれかを、次のように使用します。

    • 「グループ」ラジオ・ボタンを選択する場合は、各バックエンドEISのユーザー・グループと、GlassFish Serverセキュリティ・ドメインで定義された対応するユーザー・グループのマッピングを入力します。

      カンマ区切りリストを使用して、複数のマッピングを指定します。例:

      eis-group-1=server-group-1,eis-group-2=server-group-2

    • 「プリンシパル」ラジオ・ボタンを選択する場合は、各バックエンドEISのプリンシパルと、GlassFish Serverセキュリティ・ドメインで定義された対応するプリンシパルのマッピングを入力します。

      カンマ区切りリストを使用して、複数のマッピングを指定します。例:

      eis-principal-1=server-principal-1,eis-principal-2=server-principal-2

  7. 「リソース・アダプタ」ドロップダウン・リストから、EISのリソース・アダプタを選択します。

  8. 「説明」フィールドに、作業セキュリティ・マップの説明を入力します。

  9. 「ステータス」の「有効」チェック・ボックスを選択して、作業セキュリティ・マップを有効にします。

    このオプションはデフォルトで有効になっています。

  10. 「OK」をクリックします。
