Recursos Externos

Um recurso JNDI externo acessa um repositório JNDI externo, como um servidor LDAP.

Use a página Recursos Externos para configurar recursos JNDI externos.

Para cada recurso externo, as seguintes informações são fornecidas.

Nome JNDI

Um nome exclusivo que identifica o recurso externo.

Ativado

Verdadeiro se o recurso externo estiver ativado ou falso se o recurso externo estiver desativado.

Tipo de Recurso

O tipo do recurso externo.

Descrição

Uma descrição do recurso externo.

A tabela Recursos também contém as opções a seguir.

Novo

Botão para criar um novo recurso externo.

Deletar

Botão para deletar um ou mais recursos externos selecionados.

Ativar

Botão para ativar um ou mais recursos externos selecionados.

Desativar

Botão para desativar um ou mais recursos externos selecionados.
