刪除自訂 JNDI 資源

  1. 在瀏覽樹狀結構中，展開「資源」節點。

  2. 在「資源」節點下，展開「JNDI」節點。

  3. 在「JNDI」節點下，選取「自訂資源」。

    「自訂資源」頁面會隨即開啟。

  4. 在「自訂資源」頁面上，選取要刪除的資源名稱旁邊的核取方塊。

  5. 按一下「刪除」。
