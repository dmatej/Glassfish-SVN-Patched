刪除 JavaMail 階段作業

  1. 在瀏覽樹狀結構中，展開「資源」節點。

  2. 在「資源」節點下，選取「JavaMail 階段作業」。

    「JavaMail 階段作業」頁面會隨即開啟。

  3. 在「JavaMail 階段作業」頁面上，選取要刪除的階段作業名稱旁的核取方塊。

  4. 按一下「刪除」。
