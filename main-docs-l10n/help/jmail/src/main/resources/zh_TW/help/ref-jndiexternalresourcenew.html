新建外部資源

使用「新建外部資源」頁面可建立外部 JNDI 資源。

「新建外部資源」頁面包含下列選項。

JNDI 名稱

可識別資源的唯一名稱。

資源類型

完全合格的資源類型定義。

工廠類別

使用者指定的工廠類別名稱。此類別會實行 javax.naming.spi.InitialContextFactory 介面。

JNDI 查找

要在外部儲存區域中查找的 JNDI 值。

例如，建立連線至外部儲存區域的外部資源之後，為了測試 Bean 類別，可進行如下 JNDI 查詢︰cn=testmybean。

描述

此外部資源的描述。

狀態

外部資源的狀態。您可啟用或停用外部資源。預設值為「啟用」。

其他特性

外部資源的其他特性。GlassFish Server 未定義外部資源的任何其他特性。

目標

(已在網域中建立叢集或獨立執行處理時才會顯示此選項。)

資源的叢集和獨立執行處理。可使用「加入」與「全部加入」按鈕，將所需的目標移至「選取的目標」欄。可使用「移除」與「全部移除」按鈕，將任何不需要的目標移至「可用的目標」欄。
