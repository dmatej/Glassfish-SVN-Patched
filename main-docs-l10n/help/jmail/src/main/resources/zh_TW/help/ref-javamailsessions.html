JavaMail 階段作業

GlassFish Server 包括 JavaMail API。JavaMail 可讓 Java 應用程式存取您的網路或網際網路上，能使用網際網路郵件存取協定 (IMAP) 與簡易郵件傳輸協定 (SMTP) 之郵件伺服器。

使用「JavaMail 階段作業」頁面可組態 JavaMail 階段作業。

對於每個 JavaMail 階段作業，都將提供以下資訊：

JNDI 名稱

可識別 JavaMail 階段作業的唯一名稱。

已啟用

如果階段作業已啟用，則為 True；如果階段作業已停用，則為 False。

描述

JavaMail 階段作業的描述。

階段作業表也包含下列選項。

新建

建立新的 JavaMail 階段作業的按鈕。

刪除

刪除一或多個所選取 JavaMail 階段作業的按鈕。

啟用

啟用一或多個所選取 JavaMail 階段作業的按鈕。

停用

停用一或多個所選取 JavaMail 階段作業的按鈕。
