Nouvelle ressource externe

La page Nouvelle ressource externe permet de créer une ressource JNDI externe.

Elle contient les options ci-après.

Nom JNDI

Nom unique identifiant la ressource.

Type de ressource

Définition qualifiée complète du type de la ressource.

Classe de fabrique

Nom de classe de fabrique spécifié par l'utilisateur. Cette classe implémente l'interface javax.naming.spi.InitialContextFactory.

Recherche JNDI

Valeur JNDI à rechercher dans le référentiel externe.

Par exemple, si vous créez une ressource externe pour vous connecter à un référentiel externe, pour tester une classe de bean, la recherche JNDI peut prendre la forme suivante : cn=testbean.

Description

Description de cette ressource externe.

Statut

Statut de la ressource externe. La ressource externe peut être activée ou désactivée. La valeur par défaut est Activé.

Autres propriétés

Propriétés supplémentaires de la ressource externe. GlassFish Server ne définit aucune autre propriété pour les ressources externes.

Cibles

Clusters et instances autonomes de la ressource externe. Déplacez les cibles souhaitées dans la colonne Cibles sélectionnées à l'aide des boutons Ajouter et Tout ajouter. Déplacez toutes les cibles inutiles dans la colonne Cibles disponibles à l'aide des boutons Enlever et Tout enlever.

Cette option apparaît uniquement si des clusters ou des instances autonomes ont été créés dans le domaine.
