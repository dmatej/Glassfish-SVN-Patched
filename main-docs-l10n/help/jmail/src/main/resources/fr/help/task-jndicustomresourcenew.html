Création d'une ressource JNDI personnalisée

Une ressource JNDI personnalisée spécifie une fabrique personnalisée d'objets de ressource au niveau du serveur.

  1. Dans l'arborescence de navigation, développez le noeud Ressources.

  2. Sous le noeud Ressources, développez le noeud JNDI.

  3. Sous le noeud JNDI, sélectionnez Ressources personnalisées.

    La page Ressources personnalisées apparaît.

  4. Sur la page Ressources personnalisées, cliquez sur Nouveau.

    La page Nouvelle ressource personnalisée apparaît.

  5. Dans le champ Nom JNDI, entrez un nom unique.

    Il sera inscrit auprès du service de dénomination JNDI. Le nom JNDI organise et localise les composants dans un environnement informatique distribué de la même manière que des catalogues sur fiches organisent et représentent des emplacements de livres dans une bibliothèque. Par conséquent, le nom JNDI est une méthode importante d'accès à la ressource personnalisée.

  6. Dans le champ Type de ressource, effectuez l'une des opérations ci-après.

    • Sélectionnez une définition de type dans la liste déroulante.

    • Sélectionnez le deuxième bouton radio et entrez une définition de type qualifiée complète au format suivant : xxx.xxx.

  7. Dans le champ Classe de fabrique, entrez un nom de classe de fabrique pour la ressource personnalisée à créer.

    La classe de fabrique correspond au nom spécifié par l'utilisateur pour la classe de fabrique. Cette classe implémente l'interface javax.naming.spi.ObjectFactory.

  8. Dans le champ Description, entrez une brève description de la ressource.

    Cette description doit se présenter sous la forme d'une chaîne pouvant inclure un maximum de 250 caractères.

  9. Cochez la case Activé du champ Statut pour activer la ressource.

    Par défaut, la ressource est activée, ce qui signifie qu'elle est disponible dès sa création.

  10. Dans la section Autres propriétés, spécifiez les propriétés supplémentaires de la ressource personnalisée.

    GlassFish Server ne définit aucune autre propriété pour les ressources personnalisées.

    Pour ajouter une propriété, cliquez sur le bouton Ajouter une propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur.

  11. Cliquez sur OK.
