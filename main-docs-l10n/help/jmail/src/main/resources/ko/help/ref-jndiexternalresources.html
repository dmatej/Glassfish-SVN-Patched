외부 리소스

외부 JNDI 리소스는 LDAP 서버와 같은 외부 JNDI 저장소에 액세스합니다.

[외부 리소스] 페이지에서는 외부 JNDI 리소스를 구성할 수 있습니다.

각 외부 리소스에 대해 제공되는 정보는 다음과 같습니다.

JNDI 이름

외부 리소스를 식별하는 고유한 이름입니다.

사용

외부 리소스가 사용으로 설정된 경우 true, 사용 안함으로 설정된 경우 false입니다.

리소스 유형

외부 리소스의 유형입니다.

설명

외부 리소스에 대한 설명입니다.

리소스 테이블에는 다음 옵션도 포함되어 있습니다.

새로 만들기

새 외부 리소스를 생성하는 단추입니다.

삭제

선택한 외부 리소스를 하나 이상 삭제하는 단추입니다.

사용

선택한 하나 이상의 외부 리소스를 사용으로 설정하는 단추입니다.

사용 안함

선택한 하나 이상의 외부 리소스를 사용 안함으로 설정하는 단추입니다.

관련 작업

  • 외부 JNDI 리소스를 생성하는 방법

  • 외부 JNDI 리소스를 편집하는 방법

  • 외부 JNDI 리소스를 삭제하는 방법

관련 asadmin 명령

  • create-jndi-resource(1)

  • list-jndi-resources(1)

  • delete-jndi-resource(1)
