새 사용자 정의 리소스

[새 사용자 정의 리소스] 페이지에서는 사용자 정의 JNDI 리소스를 생성할 수 있습니다.

[새 사용자 정의 리소스] 페이지에는 다음 옵션이 포함되어 있습니다.

JNDI 이름

리소스를 식별하는 고유한 이름입니다.

리소스 유형

리소스에 대한 전체 유형 정의입니다.

팩토리 클래스

팩토리 클래스에 대한 사용자 지정 이름입니다. 이 클래스는 javax.naming.spi.ObjectFactory 인터페이스를 구현합니다.

설명

이 사용자 정의 리소스에 대한 설명입니다.

상태

사용자 정의 리소스의 상태입니다. 사용자 정의 리소스는 사용 또는 사용 안함으로 설정할 수 있습니다. 기본값은 사용입니다.

추가 속성

사용자 정의 리소스에 대한 추가 속성입니다. GlassFish Server에서는 사용자 정의 리소스에 대한 추가 속성을 정의하지 않습니다.

대상

사용자 정의 리소스에 대한 클러스터 및 독립형 인스턴스입니다. 원하는 대상을 [선택한 대상] 열로 이동하려면 [추가] 및 [모두 추가] 단추를 사용합니다. 필요하지 않은 대상을 [사용 가능한 대상] 열로 이동하려면 [제거] 및 [모두 제거] 단추를 사용합니다.

이 옵션은 클러스터나 독립형 인스턴스가 도메인에 생성된 경우에만 표시됩니다.

관련 작업

  • 사용자 정의 JNDI 리소스를 생성하는 방법

  • 사용자 정의 JNDI 리소스를 편집하는 방법

  • 사용자 정의 JNDI 리소스를 삭제하는 방법

관련 asadmin 명령

  • create-custom-resource(1)

  • list-custom-resources(1)

  • delete-custom-resource(1)
