カスタム・リソースを編集

カスタムJNDIリソースを編集するには、「カスタム・リソースを編集」ページを使用します。

「カスタム・リソースを編集」ページには、次のオプションが含まれます。

JNDI名

リソースを識別する一意の名前。JNDI名は読取り専用フィールドです。JNDI名を指定できるのは、新規カスタム・リソースの作成時のみです。

リソース・タイプ

リソースの完全修飾タイプ定義。

ファクトリ・クラス

ファクトリ・クラスのユーザー指定の名前。このクラスはjavax.naming.spi.ObjectFactoryインタフェースを実装する必要があります。

説明

カスタム・リソースの説明。

ステータス

カスタム・リソースのステータス。カスタム・リソースは、有効または無効の状態になります。デフォルト値は「有効」です。

追加プロパティ

カスタム・リソースの追加プロパティ。GlassFish Serverはカスタム・リソースの追加プロパティを定義しません。
