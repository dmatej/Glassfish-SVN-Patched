JavaMailセッション・ターゲットを管理するには

  1. ナビゲーション・ツリーで、「リソース」ノードを展開します。

  2. 「リソース」ノードで、「JavaMailセッション」を選択します。

    「JavaMailセッション」ページが開きます。

  3. 「セッション」の表で、ターゲットを管理するセッションの名前をクリックします。

    「JavaMailセッションを編集」ページが開きます。

  4. 「ターゲット」タブをクリックします。

    「JavaMailセッション・ターゲット」ページが開きます。

  5. 「ターゲット」の表で、必要なアクションを選択します。

    • ターゲットを追加または除去するには、「ターゲットを管理」をクリックします。

    • ターゲットを有効にするには、目的のサーバー・インスタンスまたはクラスタの隣のチェック・ボックスをクリックし、「有効」をクリックします。

    • ターゲットを無効にするには、目的のサーバー・インスタンスまたはクラスタの隣のチェック・ボックスをクリックし、「無効」をクリックします。
