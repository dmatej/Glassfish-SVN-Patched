So wird eine JavaMail-Session bearbeitet

  1. Blenden Sie in der Navigationsbaumstruktur den Knoten "Ressourcen" ein.

  2. Wählen Sie unter dem Ressourcenknoten den Knoten "JavaMail-Sessions" aus.

    Die Seite "JavaMail-Sessions" wird geöffnet.

  3. Klicken Sie auf der Seite "JavaMail-Sessions" auf den Namen der zu bearbeitenden Session.

    Die Seite "JavaMail-Session bearbeiten" wird geöffnet.

  4. Geben Sie im Feld "Mail-Host" den Hostnamen des standardmäßigen Mailservers an.

    Die Verbindungsmethoden der Speicher- und Transportobjekte verwenden diesen Wert, wenn keine protokollspezifische Hosteigenschaft angegeben ist. Der Name muss in einen tatsächlichen Hostnamen auflösbar sein.

  5. Geben Sie im Feld "Standardbenutzer" einen anderen Benutzernamen ein, der bei der Verbindungsherstellung mit einem Mailserver angegeben werden soll.

    Die Verbindungsmethode der Speicher- und Transportobjekte verwendet diesen Wert, wenn keine protokollspezifische Benutzernamenseigenschaft angegeben ist. Der Benutzername darf nur alphanumerische Zeichen, Unterstriche, Gedankenstriche oder Punkte enthalten.

  6. Geben Sie im Feld "Standardmäßige Absenderadresse" die E-Mail-Adresse des Standardbenutzers ein.

    Das Format ist username@host.domain.

  7. Geben Sie im Feld "Deployment-Reihenfolge" eine Nummer ein, um die Deployment-Reihenfolge der JavaMail-Sessionressource anzugeben.

    Ressourcen mit einer niedrigeren Nummer werden beim Hochfahren des Servers zuerst geladen. Der Standardwert ist 100.

  8. Ändern Sie im Feld "Beschreibung" die Beschreibung für die Mailsession.

  9. Aktivieren Sie das Kontrollkästchen "Status", um die Mailsession zu aktivieren.

    Der Standardwert ist "Aktiviert".

  10. Legen Sie im Feld "Speicherprotokoll" die zu verwendende Kommunikationsmethode für das Speicherobjekt fest.

    Standardmäßig wird das Speicherprotokoll imap verwendet.

  11. Geben Sie im Feld "Speicherprotokollklasse" die Kommunikationsmethodenklasse für die Speicherung an, die das gewünschte Speicherprotokoll implementiert.

    Standardmäßig lautet die Speicherprotokollklasse com.sun.mail.imap.IMAPStore.

  12. Geben Sie im Feld "Transportprotokoll" die Kommunikationsmethode für den Transport an.

    Standardmäßig wird das Transportprotokoll smtp verwendet.

  13. Geben Sie im Feld "Transportprotokollklasse" die Kommunikationsmethodenklasse für den Transport an

    Standardmäßig lautet die Transportprotokollklasse com.sun.mail.smtp.SMTPTransport.

  14. Aktivieren Sie das Kontrollkästchen "Debug", um eine zusätzliche Debugging-Ausgabe einschließlich einem Protokoll-Tracing für diese Mailsession zu aktivieren.

    Wenn die JavaMail-Logebene auf FINE (Genau) oder höher festgelegt ist, wird eine Debugging-Ausgabe generiert und in die Systemlogdatei geschrieben. Weitere Informationen zur Festlegung der Logebene finden Sie unter So konfigurieren Sie Logebenen.

  15. Geben Sie in der Tabelle "Weitere Eigenschaften" zusätzliche Eigenschaften an, die von Anwendungen benötigt werden, wie z.B. eine Eigenschaft für einen protokollspezifischen Host oder Benutzernamen.

    Eine Liste der verfügbaren Eigenschaften finden Sie in der JavaMail-API-Dokumentation (http://java.sun.com/products/javamail/javadocs/index.html). Zusätzliche Eigenschaften müssen mit mail- beginnen und Bindestriche anstelle von Punkten als Trennzeichen verwenden. Beispiel: Richtig ist mail-debug, während mail.debug falsch ist. Das GlassFish Server-Backend wandelt die Bindestriche in die von der JavaMail-API erwarteten Punkte um.

    • Um eine Eigenschaft hinzuzufügen, klicken Sie auf die Schaltfläche "Eigenschaft hinzufügen". Geben Sie in die daraufhin angezeigte leere Zeile den Eigenschaftsnamen in das Feld "Name" und den Eigenschaftswert in das Feld "Wert" ein.

    • Um eine Eigenschaft zu ändern, bearbeiten Sie das Feld "Wert" dieser Eigenschaft.

    • Um eine Eigenschaft zu löschen, aktivieren Sie das Kontrollkästchen links neben dem Feld "Name" der zu löschenden Eigenschaft und klicken anschließend auf die Schaltfläche "Eigenschaften löschen".

  16. Klicken Sie auf "Speichern".

    Tipp:

    Um zu den Standardeinstellungen zurückzukehren, klicken Sie auf die Schaltfläche "Standardwerte laden" und anschließend auf "Speichern".
