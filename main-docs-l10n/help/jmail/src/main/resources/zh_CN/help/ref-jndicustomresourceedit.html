编辑定制资源

使用“编辑定制资源”页可以编辑定制 JNDI 资源。

“编辑定制资源”页包含以下选项。

JNDI 名称

标识资源的唯一名称。JNDI 名称是只读字段。只能在创建新的定制资源时指定 JNDI 名称。

资源类型

资源的全限定类型定义。

工厂类

用户指定的工厂类的名称。此类实现 javax.naming.spi.ObjectFactory 接口。

说明

此定制资源的说明。

状态

定制资源的状态。可以启用或禁用定制资源。默认值为“已启用”。

其他属性

定制资源的其他属性。GlassFish Server 不定义定制资源的任何其他属性。
