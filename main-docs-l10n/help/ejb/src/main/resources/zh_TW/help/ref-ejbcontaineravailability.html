EJB 容器可用性

使用「EJB 容器可用性」頁面可組態 EJB 容器的可用性設定。

「EJB 容器可用性」頁面包含下列選項。

載入預設值

用於將具有預設值的設定復原為設定值的按鈕。沒有預設值的設定則保持不變。

組態名稱

套用此頁面上之設定的組態名稱。本欄位是唯讀的。

可用性服務

如果選取「已啟用」核取方塊，則會為 EJB 容器啟用可用性。

HA 持續性類型

這是勾選「可用性服務」的「已啟用」方塊時使用的持續性機制。預設值為 replicated，以叢集中的容錯移轉為基礎來提供持續性。請勿變更此值。

SFSB 持續性類型

這是未勾選「啟用可用性服務」時使用的持續性機制。預設值為 file，以檔案系統為基礎來提供持續性。請勿變更此值。

「階段作業儲存位置」欄位可決定儲存 SFSB 檢查點狀態的位置。若要設定「階段作業儲存位置」，請參閱 To Configure EJB Settings。

其他特性

EJB 容器可用性的其他特性。

GlassFish Server 未定義任何其他 EJB 容器可用性特性。
