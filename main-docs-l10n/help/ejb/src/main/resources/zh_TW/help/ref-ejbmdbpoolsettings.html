MDB 預設集區設定

使用「MDB 預設集區設定」頁面可組態 MDB 集區。

「MDB 預設集區設定」頁面包含下列選項。

載入預設值

用於將具有預設值的設定復原為設定值的按鈕。沒有預設值的設定則保持不變。

組態名稱

套用此頁面上之設定的組態名稱。本欄位是唯讀的。

集區的初始大小和最小大小

保留在集區中的最小訊息 Bean 數目。預設值為 0。

集區的最大大小

容器一次可保留在集區中的最大 Bean 數目。預設值為 32。

集區設定大小數量

閒置時間超過「集區閒置逾時」欄位中指定的時間時，要從集區中移除的 Bean 數目。此值必須至少為 0 且小於「集區的最大大小」值。預設值為 8。

集區閒置逾時

Bean 可在集區中維持閒置而不加以刪除的時間 (以秒為單位)。超過這段時間後，將會銷毀 Bean。0 值表示 Bean 可以無限期保持閒置。預設值為 600。

其他特性

MDB 集區的其他特性。GlassFish Server 未定義 MDB 集區的任何其他特性。
