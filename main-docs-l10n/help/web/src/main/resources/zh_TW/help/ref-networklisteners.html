網路監聽器

使用「網路監聽器」頁面可組態網路監聽器。

當您第一次安裝 GlassFish Server 時，預設已存在三個網路監聽器，名稱為 admin-listener、http-listener1 和 http-listener2。使用 SSL 組態的 http-listener2 預設會停用。

對於每個網路監聽器，均提供以下資訊。

名稱

網路監聽器的名稱。

連接埠號碼

網路監聽器監聽的連接埠號碼。

協定

網路監聽器所使用的協定。

繫線集區

網路監聽器所使用的繫線集區。

已啟用

網路監聽器啟用時為 True，網路監聽器未啟用時為 False。

網路監聽器表也包含下列選項。

新建

建立新網路監聽器的按鈕。

刪除

刪除一或多個選取的網路監聽器的按鈕。
