HTTP 監聽器

使用「HTTP 監聽器」頁面可組態 HTTP 監聽器。

HTTP 監聽器是一種網路監聽器。如果要存取進階功能，您可以使用「網路監聽器」頁面來編輯 HTTP 監聽器。

當您第一次安裝 GlassFish Server 時，預設已存在三個 HTTP 監聽器，名稱為 admin-listener、http-listener1 和 http-listener2。使用 SSL 組態的 http-listener2 預設會停用。

每個 HTTP 監聽器均提供以下資訊。

名稱

HTTP 監聽器的名稱。

連接埠號碼

HTTP 監聽器監聽的連接埠號碼。

位址

HTTP 監聽器使用的 IP 位址。

已啟用

HTTP 監聽器啟用時為 True，HTTP 監聽器未啟用時為 False。

HTTP 監聽器表也包含下列選項。

新建

建立新 HTTP 監聽器的按鈕。

刪除

刪除一或多個所選之 HTTP 監聽器的按鈕。
