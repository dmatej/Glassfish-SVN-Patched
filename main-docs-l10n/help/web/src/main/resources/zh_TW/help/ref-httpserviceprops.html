HTTP 服務特定的特性

以下其他特性適用於 HTTP 服務。如果您指定這些特性，它們會套用到所有網路監聽器。

ssl-cache-entriese

要快取的 SSL 階段作業數目。預設值為 10,000。

ssl3-session-timeout

SSL3 階段作業快取的間隔。預設值為 86,400。

ssl-session-timeout

SSL2 階段作業快取的間隔。預設值為 100。

proxyHandler

完全合格的 com.sun.appserv.ProxyHandler 抽象類別自訂實行之類別名稱，能讓後端 GlassFish Server 執行處理擷取 SSL 終端代理伺服器 (例如負載平衡程式) 所截取的原始從屬端請求相關資訊。此抽象類別的實行會檢視自訂請求標頭的指定請求，而代理伺服器就是透過該自訂請求標頭，將原始從屬端請求的相關資訊傳送至企業伺服器執行處理，並將該資訊傳回至其呼叫者。

預設實行會從名為 Proxy-ip 的 HTTP 請求標頭讀取從屬端 IP 位址、從名為 Proxy-keysize 的 HTTP 請求標頭讀取 SSL keysize，並從名為 Proxy-auth-cert 的 HTTP 請求標頭讀取 SSL 從屬端憑證鏈。Proxy-auth-cert 值必須包含 BASE-64 編碼的從屬端憑證鏈且無 BEGIN CERTIFICATE 和 END CERTIFICATE 邊界，並以 % d% a 取代 \n。

此設定僅會在網路監聽器協定之 HTTP 頁面的「認證傳送」核取方塊設定為 true 時使用。

connectionTimeout

指定在接受連線之後，於提交請求 URI 行之前，網路監聽器要等待的秒數。預設值為 30。
