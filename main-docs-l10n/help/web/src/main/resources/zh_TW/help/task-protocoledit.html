編輯協定

  1. 在瀏覽樹狀結構中，展開「組態」節點。

  2. 在「組態」節點下，展開「網路組態」節點。

  3. 在「網路組態」節點下，選取「協定」節點。

    「協定」頁面會隨即開啟。

  4. 在「協定」頁面上，按一下您要編輯的協定名稱。

    「編輯協定」頁面會隨即開啟。

  5. 選取「啟用安全性」核取方塊，以啟用協定的安全性。

    如果已啟用安全性，您可以使用「SSL」標籤修改安全性設定。

  6. 按一下「儲存」。

    提示:

    若要回到預設設定，請按一下「載入預設值」按鈕，然後按一下「儲存」。

  7. 如果選取「啟用安全性」核取方塊，請依照編輯協定的 SSL 設定中的步驟進行。
