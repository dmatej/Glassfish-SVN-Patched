刪除協定

  1. 在瀏覽樹狀結構中，展開「組態」節點。

  2. 在「組態」節點下，展開「網路組態」節點。

  3. 在「網路組態」節點下，選取「協定」節點。

    「協定」頁面會隨即開啟。

  4. 在「協定」頁面上，選取您要刪除的協定左側的核取方塊。

  5. 按一下「刪除」。
