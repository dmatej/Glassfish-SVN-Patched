階段作業特性

使用「階段作業特性」頁面可增加或修改 Web 容器階段作業的特性。

「階段作業特性」頁面包含下列選項。

載入預設值

用於將具有預設值的設定復原為設定值的按鈕。沒有預設值的設定則保持不變。

組態名稱

套用此頁面上之設定的組態名稱。本欄位是唯讀的。

階段作業逾時

指定非使用中階段作業保持有效的最大秒數。若此值設為 0 或負值，則階段作業永遠不會過期。預設值為 1800。

其他特性

Web 容器階段作業的其他特性。如需可用特性的描述，請參閱Web 容器階段作業特定的特性。
