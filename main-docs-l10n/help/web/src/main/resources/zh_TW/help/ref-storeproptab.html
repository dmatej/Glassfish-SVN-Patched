儲存特性

使用「儲存特性」頁面可增加或修改 Web 容器的階段作業持續性 (儲存) 特性。

「儲存特性」頁面包含下列選項。

組態名稱

套用此頁面上之設定的組態名稱。本欄位是唯讀的。

目錄

會寫入個別階段作業檔案的目錄的絕對或相對路徑名稱。預設值為 domain-dir/generated/jsp/j2ee-apps/appname/appname_war。相對路徑是相對於此 Web 應用程式暫存工作目錄的路徑。

其他特性

儲存的其他特性。GlassFish Server 未定義儲存的任何其他特性。
