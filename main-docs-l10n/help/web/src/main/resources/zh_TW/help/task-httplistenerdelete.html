刪除 HTTP 監聽程式

  1. 在瀏覽樹狀結構中，展開「組態」節點。

  2. 在「組態」節點下，展開「HTTP 服務」節點。

  3. 在「網路組態」節點下，選取「HTTP 監聽器」節點。

    「HTTP 監聽器」頁面會隨即開啟。

  4. 在「HTTP 監聽器」頁面上，選取您要刪除的 HTTP 監聽器左側的核取方塊。

  5. 按一下「刪除」。
