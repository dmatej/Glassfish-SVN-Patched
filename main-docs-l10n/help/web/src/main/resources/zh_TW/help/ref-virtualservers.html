虛擬伺服器

虛擬伺服器 (有時稱為虛擬主機) 是一種物件，允許同一實體伺服器託管多個網際網路網域名稱。同一個實體伺服器上託管的所有虛擬伺服器共用該實體伺服器的網際網路協定 (IP) 位址。虛擬伺服器會建立伺服器的網域名稱 (例如 www.sun.com) 與執行 GlassFish Server 之特定伺服器的關聯。

使用「虛擬伺服器」頁面可組態虛擬伺服器。

對於每個虛擬伺服器，均提供以下資訊。

名稱

虛擬伺服器的名稱。

狀態

下列其中之一：on、off 或 disabled。

預設 Web 模組

已建置的 Web 模組 (如果有)，此模組將回應所有無法對應至其他 Web 模組 (已建置到虛擬伺服器) 的請求。

虛擬伺服器表也包含下列選項。

新建

建立新虛擬伺服器的按鈕。

刪除

刪除一或多個選取之虛擬伺服器的按鈕。
