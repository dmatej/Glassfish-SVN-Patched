編輯協定

使用「編輯協定」頁面可修改現有網路協定的設定。

「編輯協定」頁面包含下列選項。

載入預設值

用於將具有預設值的設定復原為設定值的按鈕。沒有預設值的設定則保持不變。

組態名稱

套用此頁面上之設定的組態名稱。本欄位是唯讀的。

名稱

協定的名稱。「名稱」欄位是唯讀欄位。您僅能在建立新的協定時指定名稱。

安全性

如果選取此選項，則啟用協定的安全性。如果已啟用安全性，您可以使用「SSL」標籤修改安全性設定。

如果您正在組態使用 server-config 組態的網域管理伺服器 (DAS) admin-listener，此選項會是唯讀的。如果要為此協定啟用安全性，您必須啟用安全管理。請參閱 Secure Administration。
