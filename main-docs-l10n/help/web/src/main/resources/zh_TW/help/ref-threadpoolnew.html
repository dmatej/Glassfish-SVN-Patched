新建繫線集區

使用「新建繫線集區」頁面可建立繫線集區。

「新建繫線集區」頁面包含下列選項。

組態名稱

套用此頁面上之設定的組態名稱。本欄位是唯讀的。

名稱

繫線集區的名稱。

類別名稱

實行此繫線集區的類別之完全合格名稱。預設值為 com.sun.grizzly.http.StatsThreadPool。

佇列最大大小

在有繫線可處理訊息之前，可排入佇列的最大訊息數目。-1 值表示佇列大小沒有限制。預設值為 4096。

最大繫線集區大小

繫線集區中繫線的最大數目。預設值為 5。此為網路監聽器繫線集區的建議值。

最小繫線集區大小

繫線集區中繫線的最小數目。預設值為 2。

閒置繫線逾時

繫線可以在集區中閒置的最長時間 (以秒為單位)。一旦超過此時間，即從集區中移除該繫線。預設值為 900。
