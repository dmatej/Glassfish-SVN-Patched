Webコンテナに固有のプロパティ

Webコンテナで使用可能な追加の一般プロパティは次のとおりです。

dispatcher-max-depth

入れ子の最大ディスパッチ・レベルを設定し、再帰的インクルード文、またはフォワード文で無限ループが作成されないようにします。このレベルを超過すると、次のメッセージがサーバー・ログに書き込まれます。

Exceeded maximum depth for nested request dispatches

デフォルト値は20です。
