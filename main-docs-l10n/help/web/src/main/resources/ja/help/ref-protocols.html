プロトコル

ネットワーク・プロトコルを構成するには、「プロトコル」ページを使用します。

プロトコルごとに、次の情報が表示されます。

名前

プロトコルの名前。

セキュリティ有効

プロトコルに対してセキュリティが有効な場合はTrue、無効な場合はFalseです。

「プロトコル」の表には、次のオプションも表示されます。

新規

新規プロトコルを作成するボタン。

削除

選択した1つまたは複数のプロトコルを削除するボタン。
