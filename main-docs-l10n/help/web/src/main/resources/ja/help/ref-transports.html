トランスポート

トランスポートを構成するには、「トランスポート」ページを使用します。各ネットワーク・リスナーは、トランスポートに関連付けられます。

GlassFish Serverを初めてインストールする場合は、tcpという名前のTCPトランスポートがデフォルトで存在します。TCPまたはUDPトランスポートを変更または作成するか、カスタム・トランスポート実装を指定できます。

トランスポートごとに、次の情報が表示されます。

トランスポート名

トランスポートの名前。

クラス名

トランスポート実装のクラス名。

バイト・バッファのタイプ

トランスポートで使用するByteBufferのタイプ。

「トランスポート」の表には、次のオプションも表示されます。

新規

新規トランスポートを作成するボタン。

削除

選択した1つまたは複数のトランスポートを削除するボタン。
