プロトコルを編集するには

  1. ナビゲーション・ツリーで、「構成」ノードを展開します。

  2. 「構成」ノードの下で、「ネットワーク構成」ノードを展開します。

  3. 「ネットワーク構成」ノードの下で、「プロトコル」ノードを選択します。

    「プロトコル」ページが開きます。

  4. 「プロトコル」ページで、編集するプロトコルの名前をクリックします。

    「プロトコルの編集」ページが開きます。

  5. 「セキュリティ有効」チェック・ボックスを選択し、プロトコルに対してセキュリティを有効にします。

    セキュリティが有効な場合は、「SSL」タブを使用してセキュリティ設定を変更できます。

  6. 「保存」をクリックします。

    ヒント:

    デフォルト設定に返すには、「デフォルトのロード」ボタンをクリックし、「保存」をクリックします。

  7. 「セキュリティ有効」チェック・ボックスを選択した場合は、プロトコルのSSL設定を編集するにはのステップを実行します。
