プロトコルの編集

既存のネットワーク・プロトコルの設定を変更するには、「プロトコルの編集」ページを使用します。

「プロトコルの編集」ページには、次のオプションが含まれます。

デフォルトのロード

デフォルト値のある設定をデフォルト値に復元するボタン。デフォルト値のない設定は変更されません。

構成名

このページの設定が適用される構成の名前。このフィールドは読取り専用です。

名前

プロトコルの名前。「名前」フィールドは読取り専用です。「名前」を指定できるのは、新規プロトコルの作成時のみです。

セキュリティ

このオプションが選択されている場合、プロトコルに対してセキュリティは有効です。セキュリティが有効な場合は、「SSL」タブを使用してセキュリティ設定を変更できます。

server-config構成を使用するドメイン管理サーバー(DAS)に対してadmin-listenerを構成している場合、このオプションは読取り専用です。このプロトコルに対してセキュリティを有効にするには、セキュリティ保護された管理を有効にする必要があります。Secure Administrationを参照してください。
