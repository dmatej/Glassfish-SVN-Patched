Webコンテナ・セッションに固有のプロパティ

Webコンテナ・セッションで使用できるプロパティは次のとおりです。

enableCookies

trueに設定されている場合、セッション・トラッキングにcookieを使用します。デフォルト値はtrueです。

enableURLRewriting

URLのリライトを有効にします。これにより、ブラウザがcookieを受け入れない場合にURLをリライトする方法でセッション・トラッキングを行います。また、encodeURLまたはencodeRedirectURLコールをサーブレットやJavaServer Pages(JSP)ページで使用する必要があります。デフォルト値はtrueです。

idLengthBytes

このWebモジュールのセッションIDのバイト数を指定します。デフォルト値は128です。
