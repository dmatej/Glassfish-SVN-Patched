ストア・プロパティ

「ストア・プロパティ」ページを使用して、Webコンテナのセッション永続性(記憶域)プロパティを追加または変更します。

「ストア・プロパティ」ページには、次のオプションが含まれます。

構成名

このページの設定が適用される構成の名前。このフィールドは読取り専用です。

ディレクトリ

各セッション・ファイルが書き込まれるディレクトリの絶対パス名または相対パス名。デフォルト値はdomain-dir/generated/jsp/j2ee-apps/appname/appname_warです。相対パスは、このWebアプリケーションの一時作業ディレクトリからの相対で指定します。

追加プロパティ

ストアの追加プロパティ。GlassFish Serverはストアの追加プロパティを定義しません。
