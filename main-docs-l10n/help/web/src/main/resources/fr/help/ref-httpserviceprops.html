Propriétés propres au service HTTP

Les propriétés supplémentaires suivantes sont disponibles pour le service HTTP. Si vous spécifiez ces propriétés, elles s'appliquent à tous les éléments Processus d'écoute réseau.

ssl-cache-entriese

Nombre de sessions SSL à mettre en mémoire cache. La valeur par défaut est 10 000.

ssl3-session-timeout

Intervalle de mise en mémoire cache des sessions SSL3. La valeur par défaut est 86 400.

ssl-session-timeout

Intervalle de mise en mémoire cache des sessions SSL2. La valeur par défaut est 100.

proxyHandler

Nom de classe qualifié complet d'une implémentation personnalisée de la classe abstraite com.sun.appserv.ProxyHandler, permettant à une instance GlassFish Server back-end d'extraire des informations sur la demande client d'origine qui a été interceptée par un serveur proxy à terminaison SSL (un équilibreur de charge, par exemple). Une implémentation de cette classe abstraite recherche dans une demande donnée les en-têtes de demande personnalisés via lesquels le serveur proxy transmet les informations sur la demande client d'origine à l'instance Enterprise Server, puis renvoie ces informations à l'appelant.

L'implémentation par défaut lit l'adresse IP du client à partir d'un en-tête de demande HTTP appelé Proxy-ip, la valeur keysize SSL à partir d'un en-tête de demande HTTP appelé Proxy-keysize et la chaîne de certificats client SSL à partir d'un en-tête de demande HTTP appelé Proxy-auth-cert. La valeur Proxy-auth-cert doit contenir la chaîne de certificats client encodée en BASE-64 sans les limites BEGIN CERTIFICATE et END CERTIFICATE et avec \n remplacé par % d% a.

Ce paramètre est utilisé uniquement si la valeur Pass-through d'auth. de la page HTTP du protocole de processus d'écoute réseau est True.

connectionTimeout

Spécifie le nombre de secondes pendant lesquelles les processus d'écoute réseau attendent que la ligne d'URI de demande soit présentée, après avoir accepté une connexion. La valeur par défaut est 30.
