Propriétés générales

La page Propriétés générales permet d'ajouter ou de modifier les propriétés du conteneur Web.

Le conteneur Web héberge des applications Web. Il fournit l'environnement dans lequel sont exécutés les servlets et les fichiers JavaServer Pages (JSP). Pour obtenir une description des propriétés disponibles, reportez-vous à Propriétés propres au conteneur Web.

Pour chaque propriété, la page Propriétés générales contient les informations suivantes :

Nom

Nom de la propriété.

Valeur

Valeur de la propriété.

Description

Description de la propriété.
