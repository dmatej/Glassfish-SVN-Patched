Serveurs virtuels

Un serveur virtuel, parfois appelé hôte virtuel, est un objet qui permet à un même serveur physique d'héberger plusieurs noms de domaine Internet. Tous les serveurs virtuels hébergés sur le même serveur physique partagent la même adresse IP (Internet Protocol) que ce serveur. Un serveur virtuel associe le nom de domaine d'un serveur (tel que www.sun.com) au serveur sur lequel GlassFish Server est exécuté.

La page Serveurs virtuels permet de configurer des serveurs virtuels.

Pour chaque serveur virtuel, les informations suivantes sont fournies.

Nom

Nom du serveur virtuel.

Etat

Valeurs possibles : on, off et disabled.

Module Web par défaut

Module Web déployé (le cas échéant) qui doit répondre à toutes les demandes ne pouvant pas être mappées avec d'autres modules Web déployés sur le serveur virtuel.

La table Serveurs virtuels contient également les options suivantes :

Nouveau

Bouton permettant de créer un serveur virtuel.

Supprimer

Bouton permettant de supprimer les serveurs virtuels sélectionnés.
