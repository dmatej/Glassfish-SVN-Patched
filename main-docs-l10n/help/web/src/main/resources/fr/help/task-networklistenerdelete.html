Suppression d'un processus d'écoute réseau

  1. Dans l'arborescence de navigation, développez le noeud Configuration.

  2. Sous le noeud Configuration, développez le noeud Configuration réseau.

  3. Sous le noeud Configuration réseau, sélectionnez le noeud Processus d'écoute réseau.

    La page Processus d'écoute réseau apparaît.

  4. Sur la page Processus d'écoute réseau, cochez la case située à gauche du processus d'écoute que vous supprimez.

  5. Cliquez sur Supprimer.
