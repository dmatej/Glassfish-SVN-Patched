Transports

Utilisez la page Transports pour configurer les transports. Chaque processus d'écoute réseau est associé à un transport.

Lorsque vous installez GlassFish Server pour la première fois, il existe un transport TCP par défaut. Il porte le nom tcp. Vous pouvez modifier ou créer un transport TCP ou UDP, ou fournir une implémentation de transport personnalisée.

Pour chaque transport, les informations suivantes sont fournies.

Nom du transport

Nom du transport

Nom de la classe

Nom de classe de l'implémentation de transport.

Type de tampon d'octet

Type ByteBuffer à utiliser avec le transport.

La table Transports contient également les options ci-après.

Nouveau

Bouton permettant de créer un transport.

Supprimer

Bouton permettant de supprimer les transports sélectionnés.
