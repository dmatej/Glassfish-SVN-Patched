Dateicache

Auf der Seite "Dateicache" können Sie Dateicacheeinstellungen für ein Netzwerkprotokoll ändern.

Die Seite "Dateicache" enthält die folgenden Optionen.

Standardeinstellungen laden

Schaltfläche zur Wiederherstellung der Standardwerte von Einstellungen, die Standardwerte haben. Einstellungen, die keine Standardwerte haben, werden nicht geändert.

Name der Konfiguration

Der Name der Konfiguration, für die die Einstellungen auf dieser Seite angewendet werden. Dieses Feld ist schreibgeschützt.

Protokollname

Der Name des Protokolls. Das Feld "Protokollname" ist ein schreibgeschütztes Feld. Sie können den Namen nur festlegen, wenn Sie ein neues Protokoll erstellen.

Status

Wenn diese Option ausgewählt wird, wird der Status des Dateicache aktiviert. Diese Option ist standardmäßig aktiviert.

Maximales Alter

Das maximale Alter in Sekunden für einen gültigen Cacheeintrag.

Dieser Parameter kontrolliert, wie lange gecachte Informationen verwendet werden, nachdem eine Datei gecacht wurde. Ein Eintrag, der älter ist als das angegebene Maximalalter, wird durch einen neuen Eintrag für dieselbe Datei ersetzt. Wenn sich der verwendete Content nur selten ändert, erhöhen Sie diesen Wert, um die Performance zu verbessern.

Das optimale maximale Alter hängt davon ab, ob die vorhandenen Dateien regelmäßig geändert werden. Wenn die Dateien beispielsweise vier Mal täglich in regelmäßigen Abständen geändert werden, können Sie das maximale Alter auf 21600 Sekunden (6 Stunden) festlegen. Andernfalls sollten Sie das maximale Alter auf den längsten Zeitraum einstellen, für den die frühere Version einer Inhaltsdatei auf dem Server verbleiben soll, nachdem die Datei geändert wurde.

Der Standardwert ist 30.

Maximale Zwischenspeichergröße

Die maximale Gesamtgröße (in Byte) des Dateicache auf dem Datenträger. Der Standardwert ist 10485760.

Maximale Dateizahl

Die maximale Anzahl an Dateien, die im Dateicache gespeichert werden können.

Wenn dieser Wert zu hoch gewählt ist, cacht der Server selten verwendete Dateien, wodurch Arbeitsspeicher vergeudet wird. Ist dieser Wert zu niedrig eingestellt, gehen die Vorteile der Cachings verloren. Probieren Sie verschiedene Werte für dieses Attribut aus, um die optimale Einstellung für spezifische Anwendungen zu finden.

Der Standardwert ist 1024.

Verwandte Aufgaben

  • So wird ein Protokoll erstellt

  • So wird ein Protokoll bearbeitet

  • So werden SSL-Einstellungen für ein Protokoll bearbeitet

  • So werden HTTP-Einstellungen für ein Protokoll bearbeitet

  • So werden Dateicacheeinstellungen für ein Protokoll bearbeitet

  • So wird ein Protokoll gelöscht
