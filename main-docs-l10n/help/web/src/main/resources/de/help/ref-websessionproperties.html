Für Webcontainersessions spezifische Eigenschaften

Für eine Webcontainersession stehen die folgenden Eigenschaften zur Verfügung.

enableCookies

Wenn der Wert auf True gesetzt ist, werden Cookies für das Session-Tracking verwendet. Der Standardwert ist True.

enableURLRewriting

Aktiviert das Umschreiben der URL. Ermöglicht das Session-Tracking durch das Umschreiben der URL, wenn der Browser keine Cookies akzeptiert. Sie müssen auch einen Aufruf encodeURL oder encodeRedirectURL im Servlet oder der JavaServer Pages-(JSP-)Seite verwenden. Der Standardwert ist True.

idLengthBytes

Gibt die Anzahl Byte in der Session-ID in diesem Webmodul an. Der Standardwert ist 128.
