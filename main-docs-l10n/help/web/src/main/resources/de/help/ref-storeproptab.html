Speichereigenschaften

Auf der Seite "Speichereigenschaften" können Sie Eigenschaften der Sessionpersistenz für den Webcontainer hinzufügen oder ändern.

Die Seite "Speichereigenschaften" umfasst die folgenden Optionen.

Name der Konfiguration

Der Name der Konfiguration, für die die Einstellungen auf dieser Seite angewendet werden. Dieses Feld ist schreibgeschützt.

Verzeichnis

Der absolute oder relative Pfadname des Verzeichnisses, in das einzelne Sessiondateien geschrieben werden. Der Standardwert ist domain-dir/generated/jsp/j2ee-apps/appname/appname_war. Ein relativer Pfad wird relativ zum temporären Arbeitsverzeichnis für diese Webanwendung angegeben.

Weitere Eigenschaften

Weitere Eigenschaften für den Speicher. GlassFish Server definiert keine zusätzlichen Eigenschaften für den Speicher.

Verwandte Aufgaben

  • So werden allgemeine Eigenschaften für den Webcontainer konfiguriert

  • So werden Sessioneigenschaften für den Webcontainer konfiguriert

  • So werden Managereigenschaften für den Webcontainer konfiguriert

  • So werden Speichereigenschaften für den Webcontainer konfiguriert

  • So werden Verfügbarkeitseinstellungen für den Webcontainer konfiguriert
