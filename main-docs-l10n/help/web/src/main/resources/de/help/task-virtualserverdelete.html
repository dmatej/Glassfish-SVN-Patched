So wird ein virtueller Server gelöscht

  1. Blenden Sie in der Navigationsbaumstruktur den Knoten "Konfiguration" ein.

  2. Wählen Sie die zu ändernde Konfiguration.

  3. Wählen Sie unter dem Konfigurationsknoten den Knoten "Virtuelle Server".

    Die Seite "Virtuelle Server" wird geöffnet.

  4. Aktivieren Sie auf der Seite "Virtuelle Server" das Kontrollkästchen links neben dem Namen des zu löschenden virtuellen Servers.

  5. Klicken Sie auf "Löschen".

    Es wird davon abgeraten, die Standard-Virtual Server, __asadmin und server zu löschen.
