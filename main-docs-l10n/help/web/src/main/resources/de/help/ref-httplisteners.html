HTTP-Listener

Verwenden Sie die Seite "HTTP-Listener", um HTTP-Listener zu konfigurieren.

Ein HTTP-Listener ist ein Netzwerk-Listener-Typ. Für den Zugriff auf erweiterte Features können Sie einen HTTP-Listener mit den Seiten "Netzwerk-Listener" bearbeiten.

Wenn Sie GlassFish Server das erste Mal installieren, sind standardmäßig drei HTTP-Listener mit folgenden Namen vorhanden: admin-listener, http-listener1 und http-listener2. Der Netzwerk-Listener http-listener2 ist mit SSL konfiguriert und standardmäßig deaktiviert.

Für jeden HTTP-Listener werden die folgenden Informationen zur Verfügung gestellt.

Name

Der Name des HTTP-Listeners.

Port

Die Portnummer, auf der der HTTP-Listener horcht.

Adresse

Die IP-Adresse, die der HTTP-Listener verwendet.

Aktiviert

"True", wenn der HTTP-Listener aktiviert ist, bzw. "False", wenn er nicht aktiviert ist.

Die Tabelle "HTTP-Listener" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen eines neuen HTTP-Listeners.

Löschen

Schaltfläche zum Löschen von mindestens einem gewählten HTTP-Listener.
