Netzwerk-Listener

Auf der Seite "Netzwerk-Listener" können Sie Netzwerk-Listener konfigurieren.

Wenn Sie GlassFish Server erstmalig installieren, sind standardmäßig drei Netzwerk-Listener vorhanden: admin-listener, http-listener1 und http-listener2. Der Netzwerk-Listener http-listener2 ist mit SSL konfiguriert und standardmäßig deaktiviert.

Für jeden Netzwerk-Listener werden die folgenden Informationen zur Verfügung gestellt.

Name

Der Name des Netzwerk-Listeners.

Port

Die Nummer des Ports, auf dem der Netzwerk-Listener horcht.

Protokoll

Das vom Netzwerk-Listener verwendete Protokoll.

Threadpool

Der vom Netzwerk-Listener verwendete Threadpool.

Aktiviert

"True", wenn der Netzwerk-Listener aktiviert ist, bzw. "False", wenn er nicht aktiviert ist.

Die Tabelle "Netzwerk-Listener" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen eines neuen Netzwerk-Listeners.

Löschen

Schaltfläche zum Löschen von mindestens einem ausgewählten Netzwerk-Listener.

Verwandte Aufgaben

  • So wird ein Netzwerk-Listener erstellt

  • So wird ein Netzwerk-Listener bearbeitet

  • So wird ein Netzwerk-Listener gelöscht

  • So wird ein virtueller Server erstellt

Zugehörige asadmin-Befehle

  • create-network-listener(1)

  • list-network-listeners(1)

  • delete-network-listener(1)
