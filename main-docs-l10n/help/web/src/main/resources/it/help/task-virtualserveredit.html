Per modificare un server virtuale

  1. Espandere il nodo Configurazione nell'albero di navigazione.

  2. Selezionare la configurazione da modificare.

  3. Nel nodo Configurazione, selezionare il nodo Server virtuali.

    Viene visualizzata la pagina Server virtuali.

  4. Nella pagina Server virtuali fare clic sul nome del server virtuale da modificare.

    Viene visualizzata la pagina Modifica server virtuale.

  5. Nel campo Host digitare i nomi host o gli indirizzi IP per il computer in cui è in esecuzione il server.

    Utilizzare i nomi host effettivi o virtuali registrati con il server DNS per la rete (e, in un sistema UNIX, nel file /etc/hosts). Il valore predefinito è il valore della proprietà di sistema ${com.sun.aas.hostName}.

  6. Selezionare un'opzione Stato per il server virtuale.

    Il valore può essere uno di quelli riportati di seguito.

    On

    Il server virtuale è attivo. Questo è il valore predefinito.

    Off

    Il server virtuale è inattivo. Ogni tentativo di accedere al server comporterà la restituzione del codice di errore 404 (risorsa non disponibile).

    Disabilitato

    Il server virtuale è disabilitato. Ogni tentativo di accedere al server comporterà la restituzione del codice di errore 403 (elaborazione richiesta rifiutata).

  7. Selezionare un'opzione SSO per il nuovo server virtuale.

    Per impostazione predefinita, un server virtuale eredita le impostazioni SSO (Single Sign-On) del servizio HTTP. Selezionare Abilitato o Disabilitato per sovrascrivere l'impostazione del servizio HTTP abilitando o disabilitando SSO per questo server virtuale.

    L'autorizzazione SSO nelle applicazioni di GlassFish Server è supportata da servlet e file JSP. Questa funzione consente la condivisione di informazioni SSO utente tra più applicazioni che le richiedono, anziché l'accesso separato dei singoli utenti per ogni applicazione.

  8. Selezionare uno o più listener di rete nell'elenco a discesa Listener di rete oppure lasciare il campo vuoto.

  9. Nell'elenco a discesa Modulo Web predefinito selezionare il modulo da utilizzare per rispondere a tutte le richieste di cui non è possibile eseguire il mapping agli altri moduli Web distribuiti nel server virtuale.

    Se non viene specificato un modulo Web predefinito, viene utilizzato il modulo Web con una radice contesto vuota. Se non sono disponibili moduli Web con una radice contesto vuota, viene creato e utilizzato un modulo Web predefinito per il sistema.

  10. Nel campo File di log digitare il nome del percorso del file in cui visualizzare i messaggi di log provenienti da questo server virtuale.

    Per impostazione predefinita i messaggi di log vengono inviati al log del server predefinito domain-dir/logs/server.log.

  11. Nel campo Docroot digitare il percorso assoluto della directory radice dei documenti per il server.

    Il valore predefinito è domain-dir/docroot.

  12. Selezionare un'opzione Log degli accessi per abilitare o disabilitare il log degli accessi.

    Per impostazione predefinita, un server virtuale eredita le impostazioni di log degli accessi del servizio HTTP. Selezionare Abilitato o Disabilitato per sovrascrivere l'impostazione del servizio HTTP abilitando o disabilitando il log degli accessi per questo server virtuale.

  13. Nel campo Directory digitare il percorso assoluto della directory dei log degli accessi del server.

    Il valore predefinito è domain-dir/logs/access.

  14. Nella sezione Proprietà aggiuntive specificare le proprietà aggiuntive per il server virtuale.

    Per informazioni sulle proprietà disponibili, vedere Proprietà specifiche dei server virtuali.

    • Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

    • Per modificare una proprietà, è necessario modificare il valore del campo Valore della proprietà.

    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Nome corrispondente alla proprietà da eliminare, quindi fare clic sul pulsante Elimina proprietà.

  15. Fare clic su Salva.

    Suggerimento:

    Per ripristinare le impostazioni predefinite, fare clic sul pulsante Carica impostazioni predefinite, quindi su Salva.
