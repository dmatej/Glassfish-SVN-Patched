Server virtuali

Un server virtuale, talvolta denominato host virtuale, è un oggetto che consente allo stesso server fisico di ospitare più nomi di domini Internet. Tutti i server virtuali ospitati sullo stesso server fisico condividono l'indirizzo IP (Internet Protocol) di tale server fisico. Un server virtuale associa un nome di dominio per un server, ad esempio www.sun.com, al server specifico su cui è in esecuzione GlassFish Server.

Utilizzare la pagina Server virtuali per configurare i server virtuali.

Per ogni server virtuale vengono fornite le informazioni riportate di seguito.

Nome

Il nome del server virtuale.

Stato

Sono disponibili i valori On, Off e Disabled.

Modulo Web predefinito

L'eventuale modulo Web distribuito che deve rispondere a tutte le richieste che non possono essere mappate ad altri moduli Web distribuiti sul server virtuale.

La tabella Server virtuali contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare un nuovo server virtuale.

Elimina

Pulsante che consente di eliminare uno o più server virtuali selezionati.
