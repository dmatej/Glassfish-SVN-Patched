Listener HTTP

Utilizzare la pagina Listener HTTP per configurare i listener HTTP.

Un listener HTTP è un tipo di listener di rete. Per accedere alle funzioni avanzate, è possibile modificare un listener HTTP utilizzando le pagine Listener di rete.

Quando si esegue la prima installazione di GlassFish Server, per impostazione predefinita sono disponibili tre listener HTTP denominati admin-listener, http-listener1 e http-listener2. Il listener http-listener2, configurato tramite SSL, è disabilitato per impostazione predefinita.

Per ogni listener HTTP vengono fornite le informazioni riportate di seguito.

Nome

Il nome del listener HTTP.

Porta n.

Il numero della porta di ascolto del listener HTTP.

Indirizzo

L'indirizzo IP usato dal listener HTTP.

Abilitato

L'impostazione è True se il listener HTTP è abilitato oppure è False se il listener HTTP è disabilitato.

La tabella Listener HTTP contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare un nuovo listener HTTP.

Elimina

Pulsante che consente di eliminare uno o più listener HTTP selezionati.
