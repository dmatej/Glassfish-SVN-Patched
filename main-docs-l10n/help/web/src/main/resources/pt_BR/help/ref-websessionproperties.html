Propriedades Específicas para as Sessões de Contêiner Web

As propriedades a seguir estão disponíveis para uma sessão de contêiner Web.

enableCookies

Se definida como true, usa cookies para rastreamento de sessão. O valor default é true.

enableURLRewriting

Ativa regravação de URL. Isso fornece um rastreamento de sessão por meio de regravação de URL quando o browser não aceita cookies. Você deve usar também uma chamada de encodeURL ou de encodeRedirectURL na página do servlet or JavaServer Pages (JSP). O valor default é true.

idLengthBytes

Especifica o número de bytes do ID de sessão do módulo Web. O valor default é 128.
