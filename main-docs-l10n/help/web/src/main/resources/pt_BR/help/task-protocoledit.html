Para Editar um Protocolo

  1. Na árvore de navegação, expanda o nó Configuração.

  2. No nó Configuração, expanda o nó Configuração da Rede.

  3. No nó Configuração de Rede, selecione o nó Protocolos.

    É aberta a página Protocolos.

  4. Na página Protocolos, clique no nome do protocolo a ser editado.

    A página Editar Protocolo será aberta.

  5. Marque a caixa de seleção Segurança Ativada para ativar a segurança do protocolo.

    Se a segurança está ativada, você pode usar a guia SSL para modificar as definições de segurança.

  6. Clique em Salvar.

    Dica:

    Para voltar às definições default, clique no botão Carregar Defaults e depois em Salvar.

  7. Se a caixa de seleção Segurança Ativada estiver marcada, siga as etapas contidas em Para Editar Definições de SSL de um Protocolo.
