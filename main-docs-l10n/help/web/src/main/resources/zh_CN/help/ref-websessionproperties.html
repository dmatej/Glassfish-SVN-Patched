特定于 Web 容器会话的属性

以下属性可用于 Web 容器会话。

enableCookies

如果设置为真，将使用 cookie 进行会话跟踪。默认值为真。

enableURLRewriting

启用 URL 重写。这样可以在浏览器不接受 cookie 时通过 URL 重写的方式提供会话跟踪。您还必须在 servlet 或 JavaServer Pages (JSP) 页中使用 encodeURL 或 encodeRedirectURL 调用。默认值为真。

idLengthBytes

指定此 Web 模块会话 ID 中的字节数。默认值为 128。
