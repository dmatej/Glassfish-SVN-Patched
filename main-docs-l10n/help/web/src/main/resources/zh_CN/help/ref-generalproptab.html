常规属性

使用“常规属性”页可为 Web 容器添加或修改属性。

Web 容器用于托管 Web 应用程序。它为 Servlet 和 JavaServer Pages (JSP) 文件提供了运行环境。有关可用属性的说明，请参阅特定于 Web 容器的属性。

对于每个属性，“常规属性”页包含以下信息。

名称

属性名称。

值

属性值。

说明

属性说明。
