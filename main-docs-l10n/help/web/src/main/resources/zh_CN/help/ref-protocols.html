协议

使用“协议”页可以配置网络协议。

对于每个协议，将提供以下信息。

名称

协议名称。

已启用安全性

如果已为协议启用了安全性，则为“真”；如果未为协议启用安全性，则为“假”。

“协议”表还包含以下选项。

新建

用于创建新协议的按钮。

删除

用于删除一个或多个所选协议的按钮。

相关任务

  • 创建协议

  • 编辑协议

  • 编辑协议的 SSL 设置

  • 编辑协议的 HTTP 设置

  • 编辑协议的文件高速缓存设置

  • 删除协议
