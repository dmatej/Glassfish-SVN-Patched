编辑协议

使用“编辑协议”页可以修改现有网络协议的设置。

“编辑协议”页包含以下选项。

加载默认值

用于将具有默认值的设置还原为其默认值的按钮。没有默认值的设置不发生更改。

配置名称

此页上的设置应用到的配置的名称。此字段为只读。

名称

协议名称。“名称”字段是只读字段。只能在创建新协议时指定“名称”。

安全性

如果选中此选项，则将为协议启用安全性。如果启用安全性，则可以使用 SSL 选项卡来修改安全性设置。

如果正在为域管理服务器 (DAS) 配置 admin-listener，而该服务器使用 server-config 配置，则此选项是只读的。要为此协议启用安全性，必须启用安全管理。请参阅安全管理。
