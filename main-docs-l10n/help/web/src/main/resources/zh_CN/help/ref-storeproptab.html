存储属性

使用“存储属性”页可以为 Web 容器添加或修改会话持久性 (存储) 属性。

“存储属性”页包含以下选项。

配置名称

此页上的设置应用到的配置的名称。此字段为只读。

目录

各个会话文件写入到的目录的绝对路径名或相对路径名。默认值为 domain-dir/generated/jsp/j2ee-apps/appname/appname_war。相对路径是相对于此 Web 应用程序的临时工作目录的路径。

其他属性

存储的其他属性。GlassFish Server 不定义存储的任何其他属性。
