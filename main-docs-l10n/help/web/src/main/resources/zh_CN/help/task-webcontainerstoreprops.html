配置 Web 容器的存储属性

  1. 在导航树中，展开“配置”节点。

  2. 选择您要更改的配置。

  3. 在配置的节点下，选择“Web 容器”节点。

    此时将打开“常规属性”页。

  4. 在“常规属性”页上，单击“存储属性”选项卡。

    此时将打开“存储属性”页。

  5. 在“目录”字段中，键入各个会话文件写入到的目录的绝对路径名或相对路径名。

    默认值为 domain-dir/generated/jsp/j2ee-apps/appname/appname_war。

    相对路径是相对于此 Web 应用程序的临时工作目录的路径。

    仅当“持久类型”为文件时，该设置才适用。要设置持久类型，请参阅配置 Web 容器的可用性设置。

  6. 在“其他属性”部分中，指定其他属性。

    GlassFish Server 不定义存储的任何其他属性。

    • 要添加属性，请单击“添加属性”按钮。在显示的空白行中，在“名称”字段中键入属性名称，在“值”字段中键入属性值。

    • 要修改属性，请编辑该属性的“值”字段。

    • 要删除属性，请选中位于您要删除属性的“名称”字段左边的复选框，然后单击“删除属性”按钮。

  7. 单击“保存”。
