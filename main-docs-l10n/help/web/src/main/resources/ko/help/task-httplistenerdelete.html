HTTP 리스너를 삭제하는 방법

  1. 탐색 트리에서 [구성] 노드를 확장합니다.

  2. [구성] 노드에서 [HTTP 서비스] 노드를 확장합니다.

  3. [네트워크 구성] 노드에서 [HTTP 리스너] 노드를 선택합니다.

    [HTTP 리스너] 페이지가 열립니다.

  4. [HTTP 리스너] 페이지에서 삭제할 HTTP 리스너의 왼쪽에 있는 체크 박스를 선택합니다.

  5. [삭제]를 누릅니다.
