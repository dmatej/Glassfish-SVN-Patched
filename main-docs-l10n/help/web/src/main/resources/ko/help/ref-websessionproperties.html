웹 컨테이너 세션 관련 속성

웹 컨테이너 세션에 사용할 수 있는 속성은 다음과 같습니다.

enableCookies

true로 설정하면 쿠키를 사용하여 세션을 추적할 수 있습니다. 기본값은 true입니다.

enableURLRewriting

URL 다시 쓰기를 사용합니다. 그러면 브라우저에서 쿠키를 승인하지 않는 경우에 URL 다시 쓰기를 사용하여 세션 추적 기능을 제공합니다. 서블릿이나 JSP(JavaServer Pages) 페이지에서 encodeURL 또는 encodeRedirectURL 호출도 사용해야 합니다. 기본값은 true입니다.

idLengthBytes

웹 모듈의 세션 ID에 있는 바이트 수를 지정합니다. 기본값은 128입니다.
