가상 서버를 삭제하는 방법

  1. 탐색 트리에서 [구성] 노드를 확장합니다.

  2. 변경 중인 구성을 선택합니다.

  3. [구성] 노드에서 [가상 서버] 노드를 선택합니다.

    [가상 서버] 페이지가 열립니다.

  4. [가상 서버] 페이지에서 삭제할 가상 서버 이름 왼쪽의 체크 박스를 선택합니다.

  5. [삭제]를 누릅니다.

    기본 가상 서버 __asadmin 및 server는 삭제하지 않는 것이 좋습니다.
