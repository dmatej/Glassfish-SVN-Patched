HTTP 리스너

[HTTP 리스너] 페이지에서는 HTTP 리스너를 구성할 수 있습니다.

HTTP 리스너는 네트워크 리스너의 한 유형입니다. [네트워크 리스너] 페이지에서 HTTP 리스너를 편집하면 고급 기능에 액세스할 수 있습니다.

GlassFish Server를 처음 설치하면 기본적으로 admin-listener, http-listener1 및 http-listener2라는 세 개의 HTTP 리스너가 있습니다. SSL로 구성된 http-listener2는 기본적으로 사용 안함으로 설정되어 있습니다.

각 HTTP 리스너에 대해 제공되는 정보는 다음과 같습니다.

이름

HTTP 리스너의 이름입니다.

포트 #

HTTP 리스너에서 수신하는 포트 번호입니다.

주소

HTTP 리스너에서 사용되는 IP 주소입니다.

사용

HTTP 리스너를 사용하는 경우 true이고 HTTP 리스너를 사용하지 않는 경우 false입니다.

HTTP 리스너 테이블에는 다음 옵션도 포함되어 있습니다.

새로 만들기

새 HTTP 리스너를 생성하는 단추입니다.

삭제

선택한 HTTP 리스너를 하나 이상 삭제하는 단추입니다.

관련 작업

  • HTTP 리스너를 생성하는 방법

  • HTTP 리스너를 편집하는 방법

  • HTTP 리스너를 삭제하는 방법

  • HTTP 리스너의 SSL 설정을 편집하는 방법

  • 가상 서버를 생성하는 방법

  • 네트워크 리스너를 생성하는 방법

  • 프로토콜의 HTTP 설정을 편집하는 방법
