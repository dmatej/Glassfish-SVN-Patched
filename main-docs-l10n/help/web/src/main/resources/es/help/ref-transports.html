Transportes

Utilice la página Transportes para configurar los transportes. Cada listener de red se encuentra asociado a un transporte.

Al instalar por primera vez GlassFish Server, un transporte TCP existe por defecto con el nombre tcp. Puede modificar o crear un transporte TCP o UDP, o bien proporcionar una implantación de transporte personalizada.

Para cada transporte, se muestra la siguiente información.

Nombre de Transporte

Nombre del transporte.

Nombre de clase

Nombre de clase de la implantación del transporte.

Tipo de Buffer de Bytes

El tipo de ByteBuffer que se debe usar con el transporte.

La tabla Transportes contiene además las siguientes opciones.

Nuevo

Botón para crear un nuevo transporte.

Suprimir

Botón para suprimir uno o varios transportes seleccionados.
