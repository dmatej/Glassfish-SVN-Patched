Para Definir a Localização de Logs da Transação

O log da transação registra as informações sobre cada transação, a fim de manter a integridade dos dados dos recursos envolvidos e se recuperar de falhas. Os logs de transação são mantidos no sub diretório tx do diretório especificado pelo campo Localização do Log de Transações. Esses logs não são legíveis para as pessoas.

  1. Na árvore de navegação, expanda o nó Configuração.

  2. Selecione a instância que será configurada:

    • Para configurar uma instância específica, selecione o nó de configuração da instância. Por exemplo, na instância default, server, selecione o nó server-config.

    • Para definir as definições default de todas as instâncias, selecione o nó default-config.

  3. Selecione o nó Serviço de Transação.

    A página Serviço de Transação é aberta.

  4. No campo Localização do Log de Transações, digite a localização dos logs de transação.

    É criado um subdiretório tx e os logs de transação são mantidos nesse diretório.

  5. Clique em Salvar.
