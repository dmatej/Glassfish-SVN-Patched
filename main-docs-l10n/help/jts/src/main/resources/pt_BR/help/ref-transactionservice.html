Serviço de Transação

Use a página Serviço de Transação para configurar recuperação, time-out e log de transação.

A página Serviço de Transação contém as seguintes opções.

Carregar Defaults

Botão para restaurar definições para seus valores default. Definições que não tiveram seus valores default alterados.

Nome da Configuração

O nome da configuração à qual as definições nesta página se aplicam. Este campo é somente para leitura.

Ao Reiniciar

Se a caixa de seleção Ativado estiver marcada, o GlassFish Server tentará recuperar as transações incompletas quando o servidor for reiniciado. Por default, essa opção está desativada.

Time-out da Transação

Número de segundos que o servidor aguarda antes de reverter uma transação não concluída. O valor default é 0, significando que o servidor aguarda indefinidamente que uma transação seja concluída.

Time-out p/Novas Tentativas

Número de segundos em que o GlassFish Server tenta se conectar a um servidor inacessível. O valor default é 600 (10 minutos).

Localização do Log de Transações

O diretório no qual os logs do servidor são mantidos. Os logs de transação são mantidos no subdiretório tx do diretório especificado neste campo. O valor default é o diretório especificado pelo campo Raiz de Logs da página Atributos de Domínio, que está localizada na guia Avançado do nó do GlassFish Server.

Decisão Heurística

Define se as transações envolvendo servidores inacessíveis devem ser confirmadas ou revertidas. O valor default é Reverter. A confirmação de transações indeterminadas pode comprometer a integridade dos dados da aplicação.

Intervalo de Keypoint

O número de transações entre as operações de ponto-chave, que compacta o arquivo de log de transação. O valor default é 65536.

Propriedades Adicionais

As propriedades adicionais do Serviço de Transação.

Para obter uma descrição das propriedades disponíveis, consulte Propriedades Específicas para o Serviço de Transação.
