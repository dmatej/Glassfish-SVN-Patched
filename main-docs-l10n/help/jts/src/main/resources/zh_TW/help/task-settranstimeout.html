設定作業事件逾時值

依預設，伺服器不會使作業事件逾時。也就是說，伺服器無限期地等待作業事件完成。為作業事件設定逾時值後，如果作業事件未在組態的時間內完成，則 GlassFish Server 會倒回該作業事件。

  1. 在瀏覽樹狀結構中，展開「組態」節點。

  2. 選取要組態的執行處理：

    • 若要組態特定的執行處理，請選取該執行處理的組態節點。例如，對於預設執行處理 server，請選取「server-config」節點。

    • 若要組態所有執行處理的預設設定，請選取「default-config」節點。

  3. 選取「作業事件服務」節點。

    「作業事件服務」頁面會隨即開啟。

  4. 在「作業事件逾時」欄位中，鍵入作業事件逾時之前需等待的秒數。

    預設值為 0。此值將停用作業事件逾時。

  5. 按一下「儲存」。

  6. 重新啟動 GlassFish Server。
