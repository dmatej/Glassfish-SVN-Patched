Transaktionsservice

Auf der Seite "Transaktionsservice" können Sie Transaktions-Recovery, Timeouts und Logging konfigurieren.

Die Seite "Transaktionsservice" umfasst die folgenden Optionen.

Standardeinstellungen laden

Schaltfläche zur Wiederherstellung der Standardwerte von Einstellungen, die Standardwerte haben. Einstellungen, die keine Standardwerte haben, werden nicht geändert.

Name der Konfiguration

Der Name der Konfiguration, für die die Einstellungen auf dieser Seite angewendet werden. Dieses Feld ist schreibgeschützt.

Beim Neustart

Wenn das Kontrollkästchen "Aktiviert" gewählt ist, versucht GlassFish Server unvollständige Transaktionen wiederherzustellen, sobald der Server neu gestartet wird. Diese Option ist standardmäßig deaktiviert.

Transaktionstimeout

Der Zeitraum (in Sekunden), die der Server wartet, bevor er eine unvollständige Transaktion zurücksetzt. Der Standardwert ist 0; das bedeutet, dass es von Seiten des Servers keine Zeitbeschränkung für den Abschluss einer Transaktion gibt.

Wiederholungstimeout

Der Zeitraum (in Sekunden), in dem GlassFish Server versucht, eine Verbindung zu einem nicht erreichbaren Server herzustellen. Der Standardwert ist 600 (10 Minuten).

Verzeichnis des Transaktionslogs

Das Verzeichnis, in dem die Serverlogs gespeichert sind. Transaktionslogs werden im Unterverzeichnis tx des Verzeichnisses gespeichert, das in diesem Feld angegeben ist. Der Standardwert ist das Verzeichnis, das auf der Seite "Domainattribute" im Feld "Log-Root" angegeben ist, das sich im Knoten GlassFish Server in der Registerkarte "Erweitert" befindet.

Heuristische Entscheidung

Die Entscheidung, ob Transaktionen, die nicht erreichbare Server betreffen, festgeschrieben oder zurückgesetzt werden. Der Standardwert ist "Zurücksetzen". Das Festschreiben unbestimmter Transaktionen kann die Datenintegrität Ihrer Anwendung gefährden.

Schlüsselpunktintervall

Die Anzahl der Transaktionen zwischen Schlüsselpunktvorgängen, die die Transaktionslogdatei komprimieren. Der Standardwert ist 65.536.

Weitere Eigenschaften

Weitere Eigenschaften für den Transaktionsservice.

Eine Beschreibung der verfügbaren Eigenschaften finden Sie unter Für den Transaktionsservice spezifische Eigenschaften.

Verwandte Aufgaben

  • So wird der Transaktionsservice konfiguriert

  • So wird konfiguriert, wie GlassFish Server Transaktionen wiederherstellt

  • So wird ein Wert für das Transaktionstimeout festgelegt

  • So wird das Verzeichnis der Transaktionslogs festgelegt

  • So wird das Schlüsselpunktintervall festgelegt
