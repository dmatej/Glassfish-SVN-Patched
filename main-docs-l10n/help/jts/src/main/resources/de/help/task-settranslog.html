So wird das Verzeichnis der Transaktionslogs festgelegt

Im Transaktionslog werden Informationen zu den einzelnen Transaktionen aufgezeichnet, um die Datenintegrität der beteiligten Ressourcen sicherzustellen und ein Recovery nach Ausfällen zu ermöglichen. Transaktionslogs werden im Unterverzeichnis tx des im Feld "Verzeichnis des Transaktionslogs" angegebenen Verzeichnisses gespeichert. Diese Logs sind nicht visuell lesbar.

  1. Blenden Sie in der Navigationsbaumstruktur den Knoten "Konfiguration" ein.

  2. Wählen Sie die zu konfigurierende Instanz:

    • Um eine bestimmte Instanz zu konfigurieren, wählen Sie den Konfigurationsknoten der Instanz. Beispiel: Für die Standardinstanz server wählen Sie den Knoten server-config.

    • Um die Standardeinstellungen für alle Instanzen zu konfigurieren, wählen Sie den Knoten default-config.

  3. Wählen Sie den Knoten "Transaktionsservice".

    Die Seite "Transaktionsservice" wird geöffnet.

  4. Geben Sie im Feld "Verzeichnis des Transaktionslogs" den Speicherort der Transaktionslogs ein.

    Es wird ein Unterverzeichnis tx zum Speichern der Transaktionslogs erstellt.

  5. Klicken Sie auf "Speichern".
