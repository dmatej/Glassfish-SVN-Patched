So wird eine JDBC-Ressource erstellt

Eine Java DataBase Connectivity-(JDBC-)Ressource (Datenquelle) stellt Anwendungen über die Verbindung zu einer Datenbank bereit.

Bevor Sie beginnen

Um eine JDBC-Ressource zu erstellen, müssen Sie zuerst einen JDBC-Connection-Pool erstellen. Siehe So wird einen JDBC-Connection-Pool erstellt

  1. Blenden Sie in der Navigationsbaumstruktur den Knoten "Ressourcen" ein.

  2. Blenden Sie unter dem Ressourcenknoten den JDBC-Knoten ein.

  3. Wählen Sie unter dem JDBC-Knoten den JDBC-Ressourcenknoten.

    Die Seite "JDBC-Ressourcen" wird geöffnet.

  4. Klicken Sie auf der Seite "JDBC-Ressourcen" auf "Neu".

    Die Seite "Neue JDBC-Ressource" wird geöffnet.

  5. Geben Sie im Feld "JNDI-Name" einen eindeutigen Ressourcennamen ein.

    Der JNDI-Name organisiert und ordnet Komponenten innerhalb einer verteilten Computing-Umgebung in ähnlicher Weise an wie ein Karteikartensystem die Bücher in einer Bibliothek katalogisiert und deren Standort festhält. Folglich stellt der JNDI-Name ein wichtiges Mittel für den Zugriff auf eine JDBC-Ressource dar. Gemäß Konvention beginnt der Name mit der Zeichenfolge jdbc/, Beispiel: jdbc/payrolldb. Vergessen Sie nicht den Schrägstrich (/).

    Der Name darf ausschließlich alphanumerische Zeichen, Unterstriche, Gedankenstriche oder Punkte enthalten.

  6. Wählen Sie in der Dropdown-Liste "Poolname" einen Connection-Pool, der der neuen JDBC-Ressource zuzuordnen ist.

  7. Geben Sie im Feld "Beschreibung" eine kurze Beschreibung der neuen JDBC-Ressource ein.

  8. Aktivieren Sie das Kontrollkästchen "Status", um die Ressource zur Laufzeit zur Verfügung zu stellen.

    Diese Option ist standardmäßig aktiviert.

  9. Geben Sie im Abschnitt "Weitere Eigenschaften" weitere Eigenschaften an.

    In GlassFish Server werden keine weiteren Eigenschaften für JDBC-Ressourcen definiert.

    Um eine Eigenschaft hinzuzufügen, klicken Sie auf die Schaltfläche "Eigenschaft hinzufügen". Geben Sie in die daraufhin angezeigte leere Zeile den Eigenschaftsnamen in das Feld "Name" und den Eigenschaftswert in das Feld "Wert" ein.

  10. Wählen Sie ein verfügbares Ziel, um es zur Liste "Ausgewählte Ziele" hinzuzufügen oder aus dieser Liste zu entfernen.

  11. Klicken Sie auf "OK".
