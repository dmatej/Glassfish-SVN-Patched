JDBC-Ressource bearbeiten

Sie verwenden die Seite "JDBC-Ressource bearbeiten", um eine vorhandene JDBC-Ressource zu ändern.

Die Seite "JDBC-Ressource bearbeiten" bietet die folgenden Optionen.

JNDI-Name

Ein eindeutiger Name, der die JDBC-Ressource identifiziert. Der JNDI-Name ist ein schreibgeschütztes Feld. Sie können den JNDI-Namen nur festlegen, wenn Sie eine neue JDBC-Ressource erstellen.

Poolname

Der mit der JDBC-Ressource verbundene Connection-Pool.

Beschreibung

Eine kurze Beschreibung der JDBC-Ressource.

Status

Wenn diese Option aktiviert ist, steht die Ressource zur Laufzeit zur Verfügung. Diese Option ist standardmäßig aktiviert.

Weitere Eigenschaften

Weitere Eigenschaften für die JDBC-Ressource. In GlassFish Server werden keine weiteren Eigenschaften für JDBC-Ressourcen definiert.

Verwandte Aufgaben

  • So wird der Datenbankzugriff eingerichtet

  • So wird ein JDBC-Treiber integriert

  • So wird eine JDBC-Ressource erstellt

  • So wird eine JDBC-Ressource bearbeitet

  • So wird eine JDBC-Ressource gelöscht
