So werden Eigenschaften des JDBC-Connection-Pool bearbeitet

  1. Blenden Sie in der Navigationsbaumstruktur den Knoten "Ressourcen" ein.

  2. Blenden Sie unter dem Ressourcenknoten den JDBC-Knoten ein.

  3. Wählen Sie unter dem JDBC-Knoten den Knoten "Connection-Pools".

    Die Seite "Connection-Pools" wird geöffnet.

  4. Klicken Sie auf den Namen des zu bearbeitenden Connection-Pools.

    Die Seite "Connection-Pool bearbeiten" wird geöffnet.

  5. Klicken Sie auf der Seite "Connection-Pool bearbeiten" auf die Registerkarte "Weitere Eigenschaften".

    Die Seite "Connection-Pooleigenschaften bearbeiten" wird geöffnet.

  6. Geben Sie im Abschnitt "Weitere Eigenschaften" weitere Eigenschaften an.

    Zu erforderlichen Eigenschaften gehören zum Beispiel Datenbankname (URL), Benutzername und Kennwort. Information zu verfügbaren Eigenschaften finden Sie unter Für JDBC-Connection-Pools spezifische Eigenschaften.

    Wenn Sie für eine Eigenschaft keinen Wert angeben, wird die Eigenschaft nach Abschluss der Bearbeitung des Connection-Pools aus der Tabelle entfernt. Falls Sie zum jetzigen Zeitpunkt keinen Wert für eine Eigenschaft festlegen möchten, die Eigenschaft jedoch in der Tabelle verbleiben soll, damit Sie den Wert später angeben können, geben Sie ein leeres Klammernpaar in das Wertefeld ein:

    ()

    • Um eine Eigenschaft hinzuzufügen, klicken Sie auf die Schaltfläche "Eigenschaft hinzufügen". Geben Sie in die daraufhin angezeigte leere Zeile den Eigenschaftsnamen in das Feld "Name" und den Eigenschaftswert in das Feld "Wert" ein.

    • Um eine Eigenschaft zu ändern, bearbeiten Sie das Feld "Wert" dieser Eigenschaft.

    • Um eine Eigenschaft zu löschen, aktivieren Sie das Kontrollkästchen links neben dem Feld "Name" der zu löschenden Eigenschaft und klicken anschließend auf die Schaltfläche "Eigenschaften löschen".

  7. Klicken Sie auf "Speichern".
