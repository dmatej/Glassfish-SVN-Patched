So wird eine JDBC-Ressource bearbeitet

  1. Blenden Sie in der Navigationsbaumstruktur den Knoten "Ressourcen" ein.

  2. Blenden Sie unter dem Ressourcenknoten den JDBC-Knoten ein.

  3. Wählen Sie unter dem JDBC-Knoten den JDBC-Ressourcenknoten.

    Die Seite "JDBC-Ressourcen" wird geöffnet.

  4. Klicken auf der Seite "JDBC-Ressourcen" auf den Namen der zu bearbeitenden Ressource.

    Die Seite "JDBC-Ressource bearbeiten" wird geöffnet.

  5. Wählen Sie in der Dropdown-Liste "Poolname" einen Connection-Pool, der dieser JDBC-Ressource zuzuordnen ist.

  6. Geben Sie im Feld "Beschreibung" eine kurze Beschreibung dieser neuen JDBC-Ressource ein.

  7. Geben Sie im Abschnitt "Weitere Eigenschaften" weitere Eigenschaften an.

    In GlassFish Server werden keine weiteren Eigenschaften für JDBC-Ressourcen definiert.

    • Um eine Eigenschaft hinzuzufügen, klicken Sie auf die Schaltfläche "Eigenschaft hinzufügen". Geben Sie in die daraufhin angezeigte leere Zeile den Eigenschaftsnamen in das Feld "Name" und den Eigenschaftswert in das Feld "Wert" ein.

    • Um eine Eigenschaft zu ändern, bearbeiten Sie das Feld "Wert" dieser Eigenschaft.

    • Um eine Eigenschaft zu löschen, aktivieren Sie das Kontrollkästchen links neben dem Feld "Name" der zu löschenden Eigenschaft und klicken anschließend auf die Schaltfläche "Eigenschaften löschen".

  8. Klicken Sie auf die Registerkarte "Ziel", um Ressourcenziele zu verwalten.

    1. Aktivieren oder deaktivieren Sie definierte Ziele für die Ressource, wenn erforderlich.

    2. Klicken Sie auf die Schaltfläche "Ziele verwalten", um Ziele für die Ressource hinzuzufügen oder zu entfernen.

    Weitere Informationen finden Sie unter So werden Ziele für JDBC-Ressourcen bearbeitet.

  9. Klicken Sie auf die Registerkarte "Allgemein", um zur Seite "JDBC-Ressource bearbeiten" zurückzukehren.

  10. Klicken Sie auf "Speichern".
