Eigenschaften des Connection- Pools bearbeiten

Verwenden Sie die Seite "Eigenschaften des Connection-Pools bearbeiten", um zusätzliche Eigenschaften für einen vorhandenen JDBC-Connection-Pool zu ändern.

Weitere Eigenschaften

Weitere Eigenschaften für den JDBC-Connection-Pool.

Eine Beschreibung von einigen verfügbaren Eigenschaften finden Sie unter Für JDBC-Connection-Pools spezifische Eigenschaften.

Wenn Sie für eine Eigenschaft keinen Wert angeben, wird die Eigenschaft nach Abschluss der Bearbeitung des Connection-Pools aus der Tabelle entfernt. Falls Sie zum jetzigen Zeitpunkt keinen Wert für eine Eigenschaft festlegen möchten, die Eigenschaft jedoch in der Tabelle verbleiben soll, damit Sie den Wert später angeben können, geben Sie ein leeres Klammernpaar in das Wertefeld ein:
