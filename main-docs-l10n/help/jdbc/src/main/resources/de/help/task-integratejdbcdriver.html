So wird ein JDBC-Treiber integriert

Ein Treiber, der die JDBC-API (JDBC-Treiber) unterstützt, übersetzt die JDBC-Aufrufe einer Anwendung in das Protokoll des Datenbankservers.

Zum Integrieren des JDBC-Treibers in eine GlassFish Server-Domain führen Sie einen der folgenden Schritte aus:

  • Machen Sie den Treiber dem Ladeprogramm für gemeinsame Klassen zugänglich.

    1. Kopieren Sie die JAR- und ZIP-Dateien des Treibers in das Verzeichnis domain-dir/lib.

      Hinweis:

      Es gibt eine Ausnahme zu dieser Regel. Wenn Sie einen Oracle JDBC-Treiber verwenden und die EclipseLink-Erweiterungen nutzen möchten, die für Oracle-Datenbanken zur Verfügung stehen, müssen Sie die JAR-Dateien für den Oracle JDBC-Treiber im Verzeichnis domain-dir/lib/ext ablegen.

    2. Starten Sie die Domain neu.

  • Machen Sie den Treiber dem Ladeprogramm für Systemklassen zugänglich.

    1. Blenden Sie in der Navigationsbaumstruktur den Knoten "Konfiguration" ein.

    2. Wählen Sie unter dem Konfigurationsknoten den JVM-Einstellungsknoten.

      Die Seite "Allgemeine JVM-Einstellungen" wird geöffnet.

    3. Klicken Sie auf der Seite "Allgemeine JVM-Einstellungen" auf die Registerkarte "Pfadeinstellungen".

    4. Geben Sie im Feld "Classpath-Suffix" den vollständig angegebenen Pfadnamen für die JAR-Datei des Treibers ein.

    5. Klicken Sie auf "Speichern".

    6. Starten Sie den Server neu.
