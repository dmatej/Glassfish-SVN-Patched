编辑 JDBC 资源

  1. 在导航树中，展开“资源”节点。

  2. 在“资源”节点下，展开“JDBC”节点。

  3. 在“JDBC”节点下，选择“JDBC 资源”节点。

    此时将打开“JDBC 资源”页。

  4. 在“JDBC 资源”页上，单击要编辑的资源的名称。

    此时将打开“编辑 JDBC 资源”页。

  5. 从“池名称”下拉列表中，选择要与此 JDBC 资源关联的连接池。

  6. 在“说明”字段中，键入此 JDBC 资源的简短说明。

  7. 在“其他属性”部分中，指定其他属性。

    GlassFish Server 不定义 JDBC 资源的任何其他属性。

    • 要添加属性，请单击“添加属性”按钮。在显示的空白行中，在“名称”字段中键入属性名称，在“值”字段中键入属性值。

    • 要修改属性，请编辑该属性的“值”字段。

    • 要删除属性，请选中位于您要删除属性的“名称”字段左边的复选框，然后单击“删除属性”按钮。

  8. 单击“目标”选项卡以管理资源目标。

    1. 根据需要启用或禁用资源的已定义目标。

    2. 单击“管理目标”按钮为资源添加或删除目标。

    有关详细信息，请参阅编辑 JDBC 资源目标。

  9. 单击“常规”选项卡返回到“编辑 JDBC 资源”页。

  10. 单击“保存”。
