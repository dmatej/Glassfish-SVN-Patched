编辑 JDBC 连接池属性

  1. 在导航树中，展开“资源”节点。

  2. 在“资源”节点下，展开“JDBC”节点。

  3. 在“JDBC”节点下，选择“连接池”节点。

    此时将打开“连接池”页。

  4. 单击您要编辑的连接池的名称。

    此时将打开“编辑连接池”页。

  5. 在“编辑连接池”页上，单击“其他属性”按钮。

    此时将打开“编辑连接池属性”页。

  6. 在“其他属性”部分中，指定其他属性。

    必需属性的示例包括数据库名称 (URL)、用户名和口令。有关可用属性的信息，请参阅特定于 JDBC 连接池的属性。

    如果您不指定某个属性的值，则在编辑完连接池时，该属性将从表中删除。如果您现在不希望设置某个属性的值，但希望将该属性保留在表中以便在以后设置值，请在“值”字段中键入一对空括号：

    ()

    • 要添加属性，请单击“添加属性”按钮。在显示的空白行中，在“名称”字段中键入属性名称，在“值”字段中键入属性值。

    • 要修改属性，请编辑该属性的“值”字段。

    • 要删除属性，请选中位于您要删除属性的“名称”字段左边的复选框，然后单击“删除属性”按钮。

  7. 单击“保存”。
