JDBC 접속 풀을 삭제하는 방법

JDBC 접속 풀을 삭제하는 경우 접속 풀에 의존하는 JDBC 리소스도 삭제됩니다.

  1. 탐색 트리에서 [리소스] 노드를 확장합니다.

  2. [리소스] 노드에서 JDBC 노드를 확장합니다.

  3. JDBC 노드에서 [접속 풀] 노드를 선택합니다.

    [접속 풀] 페이지가 열립니다.

  4. [접속 풀] 페이지에서 삭제할 풀의 왼쪽에 있는 체크 박스를 선택합니다.

  5. [삭제]를 누릅니다.
