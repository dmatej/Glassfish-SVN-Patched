새 JDBC 접속 풀 만들기(단계 1/2)

[새 JDBC 접속 풀(단계 1/2)] 페이지에서는 새 JDBC 접속 풀을 생성할 수 있습니다.

[새 JDBC 접속 풀(단계 1/2)] 페이지에는 다음 옵션이 포함되어 있습니다.

풀 이름

접속 풀의 이름입니다.

리소스 유형

사용 가능한 리소스 유형은 javax.sql.XADataSource(전역 트랜잭션), java.sql.ConnectionPoolDataSource(로컬 트랜잭션, 성능 향상 가능), javax.sql.DataSource(로컬 트랜잭션만) 및 java.sql.Driver입니다.

데이터베이스 드라이버 공급업체

데이터베이스 제품의 공급업체 이름입니다.

검사

이 옵션을 사용으로 설정하면 리소스 유형 및 데이터 드라이버 공급업체에 따라 데이터베이스 드라이버가 검사되며, [다음] 단추를 누르면 검색된 데이터 소스 또는 드라이버 클래스 이름이 새 JDBC 접속 풀 만들기(단계 2/2) 페이지에 표시됩니다.

검사가 작동하려면 GlassFish Server에서 JDBC 드라이버를 사용할 수 있어야 합니다. JDBC 드라이버를 통합하는 방법을 참조하십시오.
