JDBC

JDBC 페이지에서는 다음 유형의 리소스를 구성할 수 있습니다.

  • JDBC 리소스

  • 접속 풀

JDBC(Java DataBase Connectivity) 기술은 관계형 데이터베이스에 대한 접속을 제공합니다. GlassFish Server에서 도메인에 JDBC 리소스(데이터 소스)를 추가하여 데이터베이스 접속을 구성합니다. 데이터 소스는 데이터베이스에 대한 접속을 구성하는 Java 플랫폼, Enterprise Edition(Java EE 플랫폼)의 표준 메소드입니다. 각 데이터 소스에는 데이터베이스 접속의 풀이 포함되어 있습니다. 응용 프로그램은 JNDI(Java Naming and Directory Interface) API 트리 또는 로컬 응용 프로그램 컨텍스트에서 데이터 소스를 찾은 다음 getConnection 메소드로 데이터베이스 접속을 검색합니다. 데이터 소스 및 접속 풀은 시스템을 실행 상태로 유지하는 데 도움을 주는 접속 관리 프로세스를 제공합니다.

GlassFish Server에는 Java DB 데이터베이스의 복사본이 포함되고 이 데이터베이스에 액세스할 수 있는 기능이 사전 구성되어 있습니다.
