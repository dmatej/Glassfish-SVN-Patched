JDBC 리소스 대상을 편집하는 방법

  1. 탐색 트리에서 [리소스] 노드를 확장합니다.

  2. JDBC 노드를 확장합니다.

  3. [JDBC 리소스] 노드를 확장합니다.

  4. 편집할 JDBC 리소스를 선택합니다.

  5. [JDBC 리소스 편집] 페이지에서 [대상] 탭을 선택합니다.

  6. 새 대상을 리소스에 지정하려면 [대상 관리] 단추를 선택합니다.

  7. [대상 관리] 페이지에서 사용 가능한 대상을 선택하여 추가하거나 [선택한 대상] 목록에서 제거합니다.

  8. 편집 내용을 적용하려면 [확인]을 누릅니다.
