JDBCドライバを統合するには

JDBC APIをサポートするドライバ(JDBCドライバ)は、アプリケーションのJDBCコールを、データベース・サーバーのプロトコルに変換します。

JDBCドライバをGlassFish Serverドメインに統合するには、次のいずれかの操作を実行します。

  • 共通クラス・ローダーからアクセス可能なドライバを作成します。

    1. ドライバのJARファイルおよびZIPファイルをdomain-dir/libディレクトリにコピーします。

      注意:

      この手順では、例外が1つあります。Oracle JDBCドライバを使用しているときに、Oracleデータベースで使用可能なEclipseLink拡張機能を利用する場合は、Oracle JDBCドライバのJARファイルをdomain-dir/lib/extディレクトリに配置する必要があります。

    2. ドメインを再起動します。

  • システム・クラス・ローダーからアクセス可能なドライバを作成します。

    1. ナビゲーション・ツリーで、「構成」ノードを展開します。

    2. 「構成」ノードの下で、「JVM設定」ノードを選択します。

      「JVMの全般的な設定」ページが開きます。

    3. 「JVMの全般的な設定」ページで、「パス設定」タブをクリックします。

    4. 「クラスパスの接尾辞」フィールドで、ドライバのJARファイルの完全修飾パス名を入力します。

    5. 「保存」をクリックします。

    6. サーバーを再起動します。
