JDBCリソースを削除するには

リソースに関連付けられているJDBC接続プールを削除すると、JDBCリソースが自動的に削除されます。

  1. ナビゲーション・ツリーで、「リソース」ノードを展開します。

  2. 「リソース」ノードで、「JDBC」ノードを展開します。

  3. 「JDBC」ノードで、「JDBCリソース」ノードを選択します。

    「JDBCリソース」ページが開きます。

  4. 「JDBCリソース」ページで、削除するリソースの左側にあるチェック・ボックスを選択します。

  5. 「削除」をクリックします。
