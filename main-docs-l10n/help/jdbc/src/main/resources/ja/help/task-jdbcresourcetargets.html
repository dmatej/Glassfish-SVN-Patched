JDBCリソースのターゲットを編集するには

  1. ナビゲーション・ツリーで、「リソース」ノードを展開します。

  2. 「JDBC」ノードを展開します。

  3. 「JDBCリソース」ノードを展開します。

  4. 編集するJDBCリソースを選択します。

  5. 「JDBCリソースを編集」ページで、「ターゲット」タブを選択します。

  6. リソースに新規ターゲットを割り当てるには、「ターゲットの管理」ボタンを選択します。

  7. 「ターゲットの管理」ページで、使用可能なターゲットを選択してターゲットを追加するか、「選択したターゲット」リストからターゲットを除去します。

  8. 「OK」をクリックして編集を適用します。
