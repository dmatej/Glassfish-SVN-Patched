JDBC

「JDBC」ページを使用して、次のタイプのリソースを構成します。

  • JDBCリソース

  • 接続プール

Java Database Connectivity (JDBC)テクノロジは、リレーショナル・データベースへの接続を提供します。GlassFish Serverでは、JDBCリソース(データソース)をドメインに追加することで、データベースの接続を構成します。データソースは、データベースへの接続を構成するJava Platform,Enterprise Edition (Java EEプラットフォーム)の標準メソッドです。各データソースには、データベース接続のプールが格納されます。アプリケーションは、Java Naming and Directory Interface (JNDI)APIツリー上またはローカルのアプリケーション・コンテキスト内でデータソースを検索し、getConnectionメソッドを使用してデータベース接続を取得します。データソースと接続プールによって、システムの稼働を保持する接続管理プロセスが提供されます。

GlassFish ServerにはJava DBデータベースのコピーが含まれ、このデータベースにアクセスする機能が事前に構成されています。
