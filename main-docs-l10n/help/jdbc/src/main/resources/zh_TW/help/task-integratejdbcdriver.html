整合 JDBC 驅動程式

支援 JDBC API 的驅動程式 (JDBC 驅動程式) 會將應用程式的 JDBC 呼叫轉譯為資料庫伺服器的協定。

若要將 JDBC 驅動程式整合到 GlassFish Server 網域中，請執行以下操作之一：

  • 使通用類別載入器可以存取該驅動程式。

    1. 將驅動程式的 JAR 和 ZIP 檔案複製到 domain-dir/lib 目錄中。

      備註:

      此規則有一個例外。如果您使用 Oracle JDBC 驅動程式，且希望運用可與 Oracle 資料庫共同使用的 EclipseLink 延伸，您必須將 Oracle JDBC 驅動程式的 JAR 檔案置於 domain-dir/lib/ext 目錄中。

    2. 重新啟動網域。

  • 使系統類別載入器可以存取該驅動程式。

    1. 在瀏覽樹狀結構中，展開「組態」節點。

    2. 在「組態」節點下，選取「JVM 設定」節點。

      「JVM 一般設定」頁面會隨即開啟。

    3. 在「JVM 一般設定」頁面上，按一下「路徑設定」標籤。

    4. 在「類別路徑字尾」欄位中，鍵入驅動程式的 JAR 檔案之完全合格路徑名稱。

    5. 按一下「儲存」。

    6. 重新啟動伺服器。
