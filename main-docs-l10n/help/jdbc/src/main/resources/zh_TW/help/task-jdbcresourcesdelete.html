刪除 JDBC 資源

如果您刪除與資源關聯的 JDBC 連線集區，則會自動刪除 JDBC 資源。

  1. 在瀏覽樹狀結構中，展開「資源」節點。

  2. 在「資源」節點下，展開「JDBC」節點。

  3. 在「JDBC」節點下，選取「JDBC 資源」節點。

    「JDBC 資源」頁面會隨即開啟。

  4. 在「JDBC 資源」頁面上，選取要刪除之資源左側的核取方塊。

  5. 按一下「刪除」。
