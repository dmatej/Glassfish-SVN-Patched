從獨立 GlassFish Server 執行處理移除應用程式

移除應用程式只會從目前目標中移除應用程式。從目標中移除應用程式時，在顯示 GlassFish Server 中已建置之所有應用程式的 「應用程式」 頁面上，仍會繼續列出該應用程式。在已建置該應用程式的任何其他目標上，該應用程式仍然保持已建置狀態。

  1. 在瀏覽樹狀結構中，選取「獨立執行處理」節點。

    此時將開啟「獨立伺服器執行處理」頁面。

  2. 在「獨立伺服器執行處理」頁面中，選取執行處理。

    此時將開啟執行處理的「一般資訊」頁面。

  3. 在執行處理的「一般資訊」頁面上，按一下「應用程式」標籤。

    此時將開啟執行處理的「應用程式」頁面。

  4. 對於您要移除的每個應用程式，請在「已建置的應用程式」表格中，選取該應用程式旁邊的核取方塊。

  5. 按一下「移除」。

    系統會要求您確認是否要移除應用程式。

  6. 請確認您要移除應用程式。

    此時便會從此頁面上的已建置應用程式表格中移除應用程式。
