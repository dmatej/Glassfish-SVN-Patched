啟用群組管理服務

群組管理服務 (GMS) 是一種進程服務，可提供叢集監視、叢集成員身份和群組通訊服務。GMS 會執行下列功能：

  • 從叢集增加或移除一或多個成員執行處理，或者一或多個成員執行處理疑似故障或確認故障時，會通知已在 GlassFish Server 執行處理中註冊的模組。

  • 在叢集成員之間傳送和接收訊息。

  1. 在瀏覽樹狀結構中，展開「叢集」節點。

  2. 選取要為其啟用群組管理服務 (GMS) 的叢集。

    此時將開啟叢集的「一般資訊」頁面。

  3. 選取「GMS 已啟用」核取方塊，為叢集啟用 GMS。

    此選項預設為選取。若要停用叢集的 GMS，請取消勾選「GMS 已啟用」核取方塊。

    如果啟用，就會在叢集與網域管理伺服器(DAS) 的每個伺服器執行處理中啟動群組管理服務。DAS 會參與每個已啟用 GMS 的叢集。

  4. 按一下「儲存」。
