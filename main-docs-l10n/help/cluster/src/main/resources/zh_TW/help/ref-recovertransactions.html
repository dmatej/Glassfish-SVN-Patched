復原作業事件

使用「復原作業事件」頁面可復原 GlassFish Server 執行處理的作業事件。

如果執行處理在執行中，「復原作業事件」頁面只包含文字「復原是由正在執行的伺服器自行執行」與「確定」按鈕。

如果執行處理不在執行中，「復原作業事件」頁面會包含下列選項。

執行處理名稱

建立執行處理時指定給執行處理的名稱。本欄位是唯讀的。

目的地伺服器

可在目前執行處理上復原作業事件的另一個執行處理。

作業事件日誌目錄

儲存作業事件日誌的目錄。預設是位於網域目錄底下的 /logs/tx。

作業事件日誌保存在 tx 中，它是叢集組態之作業事件服務「作業事件日誌位置」設定所指定目錄的子目錄。「作業事件日誌位置」預設值為「網域屬性」頁面上的「日誌檔根目錄」欄位所指定的目錄，該頁面位於「GlassFish Server」節點的「進階」標籤之下。
