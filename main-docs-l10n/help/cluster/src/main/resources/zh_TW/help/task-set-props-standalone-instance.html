只設定獨立 GlassFish Server 執行處理的執行處理特性

GlassFish Server 執行處理的特性可增加有關執行處理的選擇性組態資訊。

  1. 在瀏覽樹狀結構中，選取「獨立執行處理」節點。

    此時將開啟「獨立伺服器執行處理」頁面。

  2. 選取只要修改其執行處理特性的執行處理。

    此時將開啟執行處理的「一般資訊」頁面。

  3. 在「一般資訊」頁面上，按一下「特性」標籤。

  4. 如果需要，請按一下「執行處理特性」標籤。

    此時將開啟「執行處理特性」頁面。

    此頁面會顯示為所選取 GlassFish Server 執行處理設定的特性清單。

  5. 對於您要修改的每個現有特性，請在該特性的「值」欄位中鍵入新值。

  6. 為所需的任何其他特性設定值。

    1. 在「其他特性」表格中，按一下「增加特性」。

      隨即會在其他特性表中增加一列。

    2. 在「名稱」欄位中，鍵入特性名稱。

    3. 在「值」欄位中，鍵入值。

    4. 在「描述」欄位中，鍵入特性的簡短描述。

  7. 按一下「儲存」。
