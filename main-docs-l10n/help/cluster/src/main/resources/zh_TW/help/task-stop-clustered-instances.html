停止叢集 GlassFish Server 執行處理

  1. 在瀏覽樹狀結構中，展開「叢集」節點。

  2. 在展開的「叢集」節點下，選取該叢集的節點。

    此時將開啟叢集的「一般資訊」頁面。

  3. 在「一般資訊」頁面上，按一下「執行處理」標籤。

    此時將開啟該叢集的「叢集伺服器執行處理」頁面。

  4. 針對您要停止的每個執行處理，請在伺服器執行處理表中選取該執行處理旁邊的核取方塊。

  5. 按一下「停止」。

    系統會要求您確認是否要停止這些執行處理。

  6. 請確認您要停止這些執行處理。

    執行處理停止後，便會更新伺服器執行處理表中各個執行處理的狀態。
