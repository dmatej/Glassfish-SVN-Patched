修改叢集的組態

  1. 在瀏覽樹狀結構中，展開「叢集」節點。

  2. 選擇叢集的節點。

  3. 按一下組態的名稱。

    此時將開啟叢集的 「組態」 頁面。

  4. 在「組態」頁面上，選取要修改的組態元件。

    此時將開啟所選取組態元件的編輯頁面。

  5. 在所選取元件的編輯頁面上，進行必要的變更，然後儲存變更。

    如需詳細資訊，請參閱所選取元件之編輯頁面的線上說明。
