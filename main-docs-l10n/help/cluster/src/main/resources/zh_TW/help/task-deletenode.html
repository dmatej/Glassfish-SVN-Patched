刪除節點

刪除節點會將該節點從網域管理伺服器(DAS) 的組態中移除。刪除節點上存在的最後一個 GlassFish Server 執行處理時，會刪除節點的目錄和檔案。

您也可以選擇在刪除節點時，解除安裝節點主機中的 GlassFish Server 軟體。

開始前

請確認要刪除的節點上沒有任何 GlassFish Server 執行處理。

  1. 在瀏覽樹狀結構中，選取「節點」節點。

    此時將開啟「節點」頁面。

  2. 對於您要刪除的每個節點，請在節點表中選取該節點旁邊的核取方塊。

    備註:

    無法刪除預先定義的節點 localhost-domain。

  3. 按一下此按鈕可刪除節點，或是刪除節點並解除安裝節點主機中的 GlassFish Server 軟體。

    • 若要刪除節點，而不解除安裝 GlassFish Server 軟體，請按一下「刪除」。

      系統會要求您確認是否要刪除這些節點。

    • 若要刪除節點並解除安裝節點主機中的 GlassFish Server 軟體，請按一下「刪除並解除安裝」。

      系統會要求您確認是否要刪除這些節點並解除安裝節點主機中的 GlassFish Server 軟體。

      備註:

      GlassFish Server 軟體不會從 CONFIG 類型之節點所代表的任何主機上解除安裝。只會刪除節點。

  4. 請確認您是要刪除這些節點，還是要刪除節點並解除安裝節點主機中的 GlassFish Server 軟體。

    系統會刪除這些節點，並從「節點」頁面上的節點表格中移除它們。如果您按一下「刪除並解除安裝」，則也會解除安裝節點主機中的 GlassFish Server 軟體。
