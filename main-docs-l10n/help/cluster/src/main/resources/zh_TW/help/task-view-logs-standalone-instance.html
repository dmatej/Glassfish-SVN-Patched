檢視獨立 GlassFish Server 執行處理的日誌檔

開始前

您要檢視其日誌檔的執行處理必須正在執行中。

  1. 在瀏覽樹狀結構中，選取「獨立執行處理」節點。

    此時將開啟「獨立伺服器執行處理」頁面。

  2. 選取要檢視其日誌檔的執行處理。

    此時將開啟執行處理的「一般資訊」頁面。

  3. 在「一般資訊」頁面上，按一下「檢視日誌檔」。

    此時將開啟「日誌檢視器」視窗。

    備註:

    執行處理必須已在執行中。如果執行處理未在執行中，便會停用「檢視日誌檔」按鈕。
