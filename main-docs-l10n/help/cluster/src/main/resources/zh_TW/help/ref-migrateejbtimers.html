遷移 EJB 計時器

「遷移 EJB 計時器」頁面包含下列選項。

來源

叢集中已停止之 GlassFish Server 執行處理的下拉式清單。EJB 計時器只能從未執行的執行處理進行遷移。

目的地

叢集內執行中 GlassFish Server 執行處理的下拉式清單。EJB 計時器只能遷移到執行中的執行處理。

備註:

如果叢集中的所有執行處理均未執行，此頁面只會顯示一個警告，說明沒有執行中的執行處理可供遷移 EJB 計時器。

「遷移 EJB 計時器」頁面包含 EJB 計時器表格與下列資訊。

伺服器名稱

叢集中 GlassFish Server 執行處理的清單。

計時器計數

每個執行處理上的永久性計時器數目。

相關作業
