執行處理特性

「執行處理特性」頁面會顯示為目前 GlassFish Server 執行處理設定的特性清單。這些特性可增加有關執行處理的選擇性組態資訊。

「執行處理名稱」欄位是唯讀欄位，會顯示目前執行處理的名稱。

對於每個特性，都會顯示以下資訊：

名稱

特性的名稱。

值

為所選取執行處理設定的特性值。

描述

提供有關特性之更多資訊的文字描述。

其他特性表也包含下列選項。

增加特性

增加特性的按鈕。按一下此按鈕即可在其他特性表中增加一列。

刪除特性

刪除一或多個所選取特性的按鈕。刪除的任何特性都會復原成其預設值或未定義 (如果未設定任何預設值)。

GlassFish Server 會定義下列執行處理特性：

rendezvousOccurred

指定執行處理是否已與網域管理伺服器(DAS) 取得連絡。可能值如下：

true

執行處理已與 DAS 取得連絡。

false

執行處理尚未與 DAS 取得連絡。
