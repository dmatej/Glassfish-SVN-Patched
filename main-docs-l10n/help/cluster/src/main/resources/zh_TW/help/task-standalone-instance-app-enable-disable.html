啟用或停用建置至 GlassFish Server 執行處理的應用程式

  1. 在瀏覽樹狀結構中，選取「獨立執行處理」節點。

    此時將開啟「獨立伺服器執行處理」頁面。

  2. 在「獨立伺服器執行處理」頁面中，選取執行處理。

    此時將開啟執行處理的「一般資訊」頁面。

  3. 在執行處理的「一般資訊」頁面上，按一下「應用程式」標籤。

    此時將開啟執行處理的「應用程式」頁面。

  4. 選取應用程式旁邊的核取方塊。

  5. 從「更多動作」下拉式功能表中，選擇適當的命令。

    • 若要啟用應用程式，請選擇「啟用」。

    • 若要停用應用程式，請選擇「停用」。
