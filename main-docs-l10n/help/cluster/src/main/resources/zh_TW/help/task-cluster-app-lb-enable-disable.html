啟用或停用建置至叢集之應用程式的負載平衡

  1. 在瀏覽樹狀結構中，展開「叢集」節點。

  2. 在展開的「叢集」節點下，選取該叢集的節點。

    此時將開啟叢集的「一般資訊」頁面。

  3. 在「一般資訊」頁面上，按一下「應用程式」標籤。

    此時將開啟叢集的「應用程式」頁面。

  4. 選取應用程式旁邊的核取方塊。

  5. 從「更多動作」下拉式功能表中，選擇適當的命令。

    • 若要啟用應用程式的負載平衡，請選擇「啟用負載平衡器」。

    • 若要停用應用程式的負載平衡，請選擇「停用負載平衡器」。
