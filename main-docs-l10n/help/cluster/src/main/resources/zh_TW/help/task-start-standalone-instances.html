啟動獨立 GlassFish Server 執行處理

開始前

請確保符合以下先決條件︰

  • 執行處理所在的所有節點都已啟用，以進行遠端通訊。

  • 網域管理伺服器 (DAS) 使用者可以使用分散式元件物件模型 (DCOM) 遠端協定或安全 Shell (SSH) 登入叢集中執行處理所在之任何節點的主機。

如果不符合上述任一個先決條件，請使用 start-local-instance(1) 子命令，從命令行個別啟動各個執行處理。

  1. 在瀏覽樹狀結構中，選取「獨立執行處理」節點。

    此時將開啟「獨立伺服器執行處理」頁面。

  2. 針對您要啟動的每個執行處理，請在伺服器執行處理表中選取該執行處理旁邊的核取方塊。

  3. 按一下「啟動」。

    系統會要求您確認是否要啟動這些執行處理。

  4. 請確認您要啟動這些執行處理。

    執行處理啟動後，便會更新伺服器執行處理表中各個執行處理的狀態。
