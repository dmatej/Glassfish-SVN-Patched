資源

使用「資源」頁面，可對選取的獨立 GlassFish Server 執行處理執行下列作業：

  • 檢視和管理執行處理的現有資源

  • 為執行處理建立其他資源

「執行處理名稱」欄位是唯讀欄位，會顯示目前執行處理的名稱。

「資源」頁面會顯示執行處理的現有資源清單。對於每個資源，都將提供以下資訊。

資源名稱

建立資源時指定給資源的名稱。按一下名稱，隨即會根據資源類型，開啟下列其中一個頁面來編輯資源：

  • 編輯 JDBC 資源

  • 編輯 JavaMail 階段作業

  • 編輯自訂資源

  • 新建自訂資源

  • 編輯外部資源

  • 編輯連線器資源

  • 編輯管理物件資源

已啟用

指示資源是否已啟用：

  • 如果已啟用資源，便會顯示勾選記號 (✓)。

  • 如果已停用資源，便會顯示叉號 (✗)。

類型

資源的類型。

資源表也包含下列選項。

啟用

啟用一或多個所選取資源的按鈕。

停用

停用一或多個所選取資源的按鈕。

新建

建立所選取類型之資源的下拉式清單。從清單中選取資源類型，隨即會開啟下列其中一個頁面，以建立所選取類型的資源：

  • 新建 JDBC 資源

  • 新建 JMS 連線工廠

  • 新建 JMS 目的地資源

  • 新建 JavaMail 階段作業

  • 新建自訂資源

  • 新建外部資源

  • 新建連線器資源

  • 新建管理物件資源

篩選

依類型篩選所顯示資源清單的下拉式清單。

相關作業

  • 建立獨立 GlassFish Server 執行處理的資源

  • 列出獨立 GlassFish Server 執行處理的資源

  • 啟用或停用獨立 GlassFish Server 執行處理的資源

  • 編輯獨立 GlassFish Server 執行處理的資源
