檢視叢集 GlassFish Server 執行處理的日誌檔

開始前

您要檢視其日誌檔的執行處理必須正在執行中。

  1. 在瀏覽樹狀結構中，展開「叢集」節點。

  2. 在展開的「叢集」節點下，選取該叢集的節點。

    此時將開啟叢集的「一般資訊」頁面。

  3. 在「一般資訊」頁面上，按一下「執行處理」標籤。

    此時將開啟該叢集的「叢集伺服器執行處理」頁面。

  4. 選取要檢視其日誌檔的執行處理。

    此時將開啟執行處理的「一般資訊」頁面。

  5. 在「一般資訊」頁面上，按一下「檢視日誌檔」。

    此時將開啟「日誌檢視器」視窗。

    備註:

    執行處理必須已在執行中。如果執行處理未在執行中，便會停用「檢視日誌檔」按鈕。
