將應用程式建置至獨立 GlassFish Server 執行處理

  1. 在瀏覽樹狀結構中，選取「獨立執行處理」節點。

    此時將開啟「獨立伺服器執行處理」頁面。

  2. 在「獨立伺服器執行處理」頁面中，選取執行處理。

    此時將開啟執行處理的「一般資訊」頁面。

  3. 在執行處理的「一般資訊」頁面上，按一下「應用程式」標籤。

    此時將開啟執行處理的「應用程式」頁面。

  4. 在「應用程式」頁面上，按一下「建置」。

    「建置應用程式或模組」頁面會隨即開啟。

  5. 在「建置應用程式或模組」頁面上，提供有關應用程式的必要資訊，然後按一下「確定」。

    如需詳細資訊，請參閱 「建置應用程式或模組」 線上說明。
