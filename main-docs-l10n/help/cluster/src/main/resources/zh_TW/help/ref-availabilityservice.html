可用性服務

使用「可用性服務」頁面可變更叢集的可用性設定。

「可用性服務」頁面包含下列選項。

組態名稱

套用此頁面上之設定的組態名稱。本欄位是唯讀的。

載入預設值

用於將具有預設值的設定復原為設定值的按鈕。沒有預設值的設定則保持不變。

可用性服務

如果選取「已啟用」核取方塊，則會為叢集啟用可用性。此選項預設為選取。

其他特性

可用服務的其他特性表格。

GlassFish Server 未定義任何其他可用服務特性。

對於每個特性，都會顯示以下資訊：

名稱

特性的名稱。

值

為所選取執行處理設定的特性值。

描述

提供有關特性之更多資訊的文字描述。

其他特性表也包含下列選項。

增加特性

增加特性的按鈕。按一下此按鈕即可在其他特性表中增加一列。

刪除特性

刪除一或多個所選取特性的按鈕。刪除的任何特性都會復原成其預設值或未定義 (如果未設定任何預設值)。
