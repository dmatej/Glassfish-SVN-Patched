Nodos

La página Nodos muestra una lista de los nodos del dominio. Para cada nodo, se muestra la información siguiente:

Nombre

Nombre asignado al nodo durante su creación. Al hacer clic en el nombre se abrirá la página Editar nodo del nodo.

Host de nodo

Nombre del host que el nodo representa.

Tipo

Tipo de nodo, que es uno de los siguientes:

CONFIG

El nodo no soporta la comunicación remota.

DCOM

El nodo soporta comunicación a través del protocolo remoto Distributed Component Object Model (DCOM). El protocolo DCOM está disponible sólo en los sistemas Windows.

SSH

El nodo soporta comunicación sobre shell seguro (SSH).

Instancias

Lista de las instancias que residen en el nodo con una indicación de si se están ejecutando:

  • Si la instancia se está ejecutando, se muestra una marca de control (✓).

  • Si la instancia se para, se muestra un signo de exclamación (!).

Al hacer clic en el nombre de una instancia se abrirá la página Información general de la instancia.

Acción

Sólo los nodos del tipo DCOM y SSH: un enlace a la acción Ping para el nodo. Al hacer clic en el enlace Ping se prueba si el nodo es accesible mediante el protocolo de comunicación que el nodo soporta.

No existen acciones posibles para los nodos CONFIG.

La tabla Nodos también contiene las opciones siguientes.

Nuevo

Botón para crear un nodo.

Suprimir

Botón para suprimir uno o varios nodos seleccionados.

Suprimir y desinstalar

Botón para suprimir uno o más nodos seleccionados y desinstalar el software GlassFish Server del host que cada nodo suprimido representa.

Nota:

El software GlassFish Server no se desinstala de ningún host representado por un nodo de tipo CONFIG. En el caso de nodos de tipo CONFIG, la acción de este botón es idéntica a la del botón Suprimir.
