Suprimir nodos

Al suprimir un nodo se elimina el nodo de la configuración del servidor de administración de dominio (DAS). Los archivos y directorios del nodo se suprimirán cuando se suprima la última instancia de GlassFish Server que resida en el nodo.

También puede desinstalar el software GlassFish Server del host del nodo cuando se suprime el nodo.

Antes de Empezar

Asegúrese de que no resida ninguna instancia de GlassFish Server en el nodo que se va a suprimir.

  1. En el árbol de navegación, seleccione el nodo Nodos.

    Se abrirá la página Nodos.

  2. Para cada nodo que desee suprimir, active la casilla de control que aparece junto al nodo en la tabla Nodos.

    Nota:

    El nodo predefinido localhost-dominio no se puede suprimir.

  3. Haga clic en el botón para suprimir los nodos o suprimir los nodos y desinstalar el software GlassFish Server de los hosts del nodo.

    • Para suprimir los nodos sin desinstalar el software GlassFish Server, haga clic en Suprimir.

      Se le pedirá que confirme que desea suprimir los nodos.

    • Para suprimir los nodos y desinstalar el software GlassFish Server de los hosts del nodo, haga clic en Suprimir y Desinstalar.

      Se le pedirá que confirme que desea suprimir los nodos y desinstalar el software GlassFish Server de los hosts del nodo.

      Nota:

      El software GlassFish Server no se desinstala de ningún host representado por un nodo de tipo CONFIG. Sólo se suprimirá el nodo.

  4. Confirme que desea suprimir los nodos o suprimir los nodos y desinstalar el software GlassFish Server de los hosts del nodo.

    Los nodos se suprimen y se eliminan de la tabla de nodos en la página Nodos. Si hace clic en Suprimir y Desinstalar, el software GlassFish Server se desinstala también de los hosts del nodo.
