Crear un nodo

Un nodo representa un host en el que está instalado el software GlassFish Server. Debe existir un nodo en cada host en el que residan instancias de GlassFish Server.

  1. En el árbol de navegación, seleccione el nodo Nodos.

    Se abrirá la página Nodos.

  2. En la página Nodos, haga clic en Nuevo.

    Se abre la página Nuevo Nodo.

  3. En el campo Nombre, escriba el nombre que desee para el nodo.

  4. En la lista desplegable Tipo, seleccione el tipo de nodo que vaya a crear.

    • Para activar el nodo para que admita comunicaciones mediante el protocolo remoto Distributed Component Object Model (DCOM), seleccione DCOM.

    • Para crear un nodo que soporte la comunicación sobre Secure Shell (SSH), seleccione SSH.

    • Para crear un nodo que no soporte la comunicación remota, seleccione CONFIG.

  5. En el campo Host de Nodo, escriba el nombre del host que representa el nodo.

    Este campo es obligatorio o no en función del tipo de nodo:

    • Si el tipo de nodo es CONFIG., el nombre del host puede omitirse. El nombre del host se puede determinar al crear las instancias que residen en el nodo.

    • Si el tipo de nodo es DCOM o SSH, se debe especificar el nombre del host. De lo contrario, se produce un error.

  6. En el campo Directorio de Nodos, indique la ruta de acceso al directorio que contendrá las instancias de GlassFish Server que se crean en el nodo.

    Si el tipo de nodo es CONFIG., deje este campo en blanco. El directorio del nodo se puede determinar al crear las instancias que residen en el nodo.

    Si especifica una ruta de acceso relativa, la ruta de acceso es relativa con respecto al directorio as-install, donde as-install es el directorio base de instalación del software GlassFish Server en el host.

  7. En el campo Directorio de Instalación, especifique la ruta de acceso completa al directorio principal del directorio base de instalación del software GlassFish Server en el host, por ejemplo /export/glassfish3.

    Si el tipo de nodo es CONFIG., deje este campo en blanco. El directorio de instalación se puede determinar al crear las instancias que residen en el nodo.

  8. Finalice la creación del nodo tal como lo exija el tipo de nodo.

    1. Si el tipo de nodo es CONFIG, haga clic en Guardar.

    2. Si el tipo de nodo es DCOM, siga las instrucciones de Terminar la creación de un nodo DCOM.

    3. Si el tipo de nodo es SSH, siga las instrucciones de Finalizar la Creación de un Nodo SSH.
