Instancias de servidor independientes

Utilice la página Instancias de servidor independientes para crear y gestionar instancias de GlassFish Server independientes.

La página Instancias de servidor independientes muestra una lista de todas las instancias independientes del dominio. Para cada instancia se proporciona la información siguiente.

Nombre

Nombre que se asignó a la instancia en el momento de su creación. Al hacer clic en el nombre se abrirá la página Información general de la instancia.

Peso de equilibrador de carga

Entero que representa el peso de equilibrio de carga de la instancia.

El peso de equilibrio de carga determina la proporción de todas las solicitudes con respecto al cluster que debe procesar la instancia.

Por ejemplo, en un cluster de dos instancias, puede necesitar que una instancia procese una solicitud de cuatro, y que otra procese las otras tres. En este caso, configure el peso de la instancia que debe procesar una de las cuatro solicitudes como 1 y el peso de la otra instancia como 3. Si prefiere usar porcentajes, defina los pesos de las instancias como 25 y 75, respectivamente.

El peso por defecto es 100.

Configuración

Configuración que se seleccionó para la instancia en el momento de su creación. Al hacer clic en la configuración se abrirá la página Configuraciones para la instancia.

Nodo

Nombre del nodo en el que reside la instancia. Al hacer clic en el nombre se abrirá la página Editar nodo del nodo.

Estado

Indicación de si la instancia está en ejecución o no.

La tabla Instancias de servidor también contiene las opciones siguientes.

Nuevo

Botón para crear una instancia independiente.

Suprimir

Botón para suprimir una o varias instancias independientes seleccionadas.

Iniciar

Botón para iniciar una o varias instancias independientes seleccionadas.

Parar

Botón para parar una o varias instancias independientes seleccionadas.
