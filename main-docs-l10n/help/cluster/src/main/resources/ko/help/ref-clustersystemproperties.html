클러스터 시스템 속성

[클러스터 시스템 속성] 페이지에는 선택된 클러스터에 대해 설정된 Java 시스템 속성 목록이 표시됩니다. Java 시스템 속성은 GlassFish Server 시작 시 Java 응용 프로그램 실행기의 -D 옵션을 통해 Java 응용 프로그램 실행기로 전달됩니다.

[클러스터 이름] 필드는 선택한 클러스터의 이름을 표시하는 읽기 전용 필드입니다.

각 속성에 대해 표시되는 정보는 다음과 같습니다.

인스턴스 변수 이름

시스템 속성의 이름입니다.

현재 값

현재 속성에 대해 설정된 값입니다. 이 필드는 읽기 전용입니다.

대체 값

변경 사항이 저장되면 속성에 설정되는 값입니다.

추가 속성 테이블에는 다음 옵션도 포함되어 있습니다.

속성 추가

속성을 추가하는 단추입니다. 이 단추를 누르면 추가 속성 테이블에 행이 추가됩니다.

속성 삭제

선택한 속성을 하나 이상 삭제하는 단추입니다. 삭제된 속성은 해당 기본값으로 되돌려지거나 기본값이 설정되어 있지 않은 경우 정의되지 않습니다.

관련 작업

구성 시스템 속성의 기본값을 설정하는 방법

동일한 asadmin 하위 명령

  • create-system-properties(1)

  • delete-system-property(1)

  • list-system-properties(1)
