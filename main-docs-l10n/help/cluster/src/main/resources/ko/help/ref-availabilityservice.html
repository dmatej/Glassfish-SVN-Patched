가용성 서비스

[가용성 서비스] 페이지에서는 클러스터에 대한 가용성 설정을 변경할 수 있습니다.

[가용성 서비스] 페이지에는 다음 옵션이 포함되어 있습니다.

구성 이름

이 페이지의 설정이 적용되는 구성의 이름입니다. 이 필드는 읽기 전용입니다.

기본값 로드

기본값을 사용하는 설정을 기본값으로 복원하는 단추입니다. 기본값을 사용하지 않는 설정은 변경되지 않습니다.

가용성 서비스

[사용] 체크 박스를 선택하면 클러스터에 대해 가용성이 사용으로 설정됩니다. 이 옵션은 기본적으로 선택되어 있습니다.

추가 속성

가용성 서비스에 대한 추가 속성 테이블입니다.

GlassFish Server에서는 추가 가용성 서비스 속성을 정의하지 않습니다.

각 속성에 대해 표시되는 정보는 다음과 같습니다.

이름

속성의 이름입니다.

값

선택된 인스턴스에 대해 설정된 속성의 값입니다.

설명

속성에 대한 자세한 정보를 제공하는 텍스트 설명입니다.

추가 속성 테이블에는 다음 옵션도 포함되어 있습니다.

속성 추가

속성을 추가하는 단추입니다. 이 단추를 누르면 추가 속성 테이블에 행이 추가됩니다.

속성 삭제

선택한 속성을 하나 이상 삭제하는 단추입니다. 삭제된 속성은 해당 기본값으로 되돌려지거나 기본값이 설정되어 있지 않은 경우 정의되지 않습니다.
