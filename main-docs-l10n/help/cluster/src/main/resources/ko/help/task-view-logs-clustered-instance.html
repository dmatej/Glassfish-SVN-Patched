클러스터된 GlassFish Server 인스턴스에 대한 로그 파일을 보는 방법

시작하기 전에

로그 파일을 볼 인스턴스가 실행되고 있어야 합니다.

  1. 탐색 트리에서 [클러스터] 노드를 확장합니다.

  2. 확장된 [클러스터] 노드에서 클러스터에 대한 노드를 선택합니다.

    클러스터에 대한 [일반 정보] 페이지가 열립니다.

  3. [일반 정보] 페이지에서 [인스턴스] 탭을 누릅니다.

    클러스터에 대한 [클러스터된 서버 인스턴스] 페이지가 열립니다.

  4. 로그 파일을 표시할 인스턴스를 선택합니다.

    인스턴스에 대한 [일반 정보] 페이지가 열립니다.

  5. [일반 정보] 페이지에서 [로그 파일 보기]를 누릅니다.

    [로그 뷰어] 창이 열립니다.

    주:

    인스턴스가 이미 실행 중이어야 합니다. 인스턴스가 실행되고 있지 않으면 [로그 파일 보기] 단추가 비활성화됩니다.
관련 항목

  • 표시된 로그 레코드를 필터링하는 방법

  • 클러스터에 GlassFish Server 인스턴스를 추가하는 방법

  • 클러스터된 GlassFish Server 인스턴스를 삭제하는 방법

  • 클러스터된 GlassFish Server 인스턴스를 시작하는 방법

  • 클러스터된 GlassFish Server 인스턴스를 정지하는 방법

  • 클러스터된 GlassFish Server 인스턴스에 대한 트랜잭션을 복구하는 방법

  • 클러스터된 GlassFish Server 인스턴스의 포트 번호를 수정하는 방법

  • 클러스터된 GlassFish Server 인스턴스에 대한 인스턴스 전용 속성을 설정하는 방법

  • 클러스터된 GlassFish Server 인스턴스에 대한 응용 프로그램 모니터링 데이터를 보는 방법

  • 클러스터된 GlassFish Server 인스턴스에 대한 가상 서버 모니터링 데이터를 보는 방법

  • 클러스터된 GlassFish Server 인스턴스에 대한 리소스 모니터링 데이터를 보는 방법
