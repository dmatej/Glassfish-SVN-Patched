인스턴스 속성

[인스턴스 속성] 페이지에는 현재 GlassFish Server 인스턴스에 대해 설정된 속성 목록이 표시됩니다. 이러한 속성은 인스턴스에 대한 선택적 구성 정보를 추가합니다.

[인스턴스 이름] 필드는 현재 인스턴스의 이름을 표시하는 읽기 전용 필드입니다.

각 속성에 대해 표시되는 정보는 다음과 같습니다.

이름

속성의 이름입니다.

값

선택된 인스턴스에 대해 설정된 속성의 값입니다.

설명

속성에 대한 자세한 정보를 제공하는 텍스트 설명입니다.

추가 속성 테이블에는 다음 옵션도 포함되어 있습니다.

속성 추가

속성을 추가하는 단추입니다. 이 단추를 누르면 추가 속성 테이블에 행이 추가됩니다.

속성 삭제

선택한 속성을 하나 이상 삭제하는 단추입니다. 삭제된 속성은 해당 기본값으로 되돌려지거나 기본값이 설정되어 있지 않은 경우 정의되지 않습니다.

GlassFish Server에 정의된 인스턴스 속성은 다음과 같습니다.

rendezvousOccurred

인스턴스가 DAS(도메인 관리 서버)에 액세스했는지 여부를 지정합니다. 가능한 값은 다음과 같습니다.

true

인스턴스가 DAS에 액세스했습니다.

false

인스턴스가 DAS에 액세스하지 않았습니다.

관련 작업

클러스터된 GlassFish Server 인스턴스에 대한 인스턴스 전용 속성을 설정하는 방법
