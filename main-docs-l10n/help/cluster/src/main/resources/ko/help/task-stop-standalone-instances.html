독립형 GlassFish Server 인스턴스를 정지하는 방법

  1. 탐색 트리에서 [독립형 인스턴스] 노드를 선택합니다.

    [독립형 서버 인스턴스] 페이지가 열립니다.

  2. 정지할 각 인스턴스에 대해 [서버 인스턴스] 테이블에서 인스턴스 옆에 있는 체크 박스를 선택합니다.

  3. [정지]를 누릅니다.

    인스턴스를 정지할 것인지 확인하는 메시지가 표시됩니다.

  4. 인스턴스를 정지할 것임을 확인합니다.

    인스턴스가 정지되면 [서버 인스턴스] 테이블에서 각 인스턴스의 상태가 업데이트됩니다.
