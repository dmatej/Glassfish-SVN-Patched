클러스터를 정지하는 방법

클러스터를 정지하면 클러스터의 모든 실행되고 있는 GlassFish Server 인스턴스가 정지됩니다.

  1. 탐색 트리에서 [클러스터] 노드를 선택합니다.

    [클러스터] 페이지가 열립니다.

  2. [클러스터] 페이지에서 클러스터 이름 옆에 있는 체크 박스를 선택합니다.

  3. [클러스터 정지]를 누릅니다.

    클러스터를 정지할 것인지 확인하는 메시지가 표시됩니다.

  4. 클러스터를 정지할 것임을 확인합니다.

    클러스터가 정지됩니다. 인스턴스가 정지되면 [클러스터] 페이지에서 클러스터에 있는 각 인스턴스의 상태가 업데이트됩니다.

  • 클러스터를 삭제하는 방법

  • 클러스터를 시작하는 방법

  • 클러스터 구성을 수정하는 방법

  • 클러스터에 응용 프로그램을 배치하는 방법

  • 클러스터에 배치된 응용 프로그램을 나열하는 방법

  • 클러스터에서 응용 프로그램 배치를 해제하는 방법

  • 클러스터에서 응용 프로그램을 제거하는 방법

  • 클러스터에 배치된 응용 프로그램을 편집하는 방법

  • 클러스터에 배치된 응용 프로그램을 사용 또는 사용 안함으로 설정하는 방법

  • 클러스터에 배치된 응용 프로그램에 대한 로드 균형 조정을 사용 또는 사용 안함으로 설정하는 방법

  • 클러스터에 대한 리소스를 생성하는 방법

  • 클러스터에 대한 리소스를 나열하는 방법

  • 리소스를 전역적으로 사용 또는 사용 안함으로 설정하는 방법

  • 클러스터에 대한 리소스를 편집하는 방법
