노드

[노드] 페이지에는 도메인의 노드 목록이 표시됩니다. 각 노드에 대해 표시되는 정보는 다음과 같습니다.

이름

노드가 생성될 때 노드에 지정된 이름입니다. 이름을 누르면 해당 노드에 대한 노드 편집 페이지가 열립니다.

노드 호스트

노드가 나타내는 호스트의 이름입니다.

유형

다음 유형 중 하나인 노드의 유형:

CONFIG

노드가 원격 통신을 지원하지 않습니다.

DCOM

노드가 DCOM(Distributed Component Object Model) 원격 프로토콜을 통한 통신을 지원합니다. DCOM 프로토콜은 Windows 시스템에서만 사용할 수 있습니다.

SSH

노드가 SSH(보안 셸)를 통한 통신을 지원합니다.

인스턴스

노드에 있는 인스턴스와 각 인스턴스가 실행 중인지 여부가 표시된 목록입니다.

  • 인스턴스가 실행 중인 경우 선택 표시 기호(✓)가 나타납니다.

  • 인스턴스가 정지된 경우 느낌표(!)가 표시됩니다.

인스턴스의 이름을 누르면 인스턴스에 대한 [일반 정보] 페이지가 열립니다.

작업

DCOM 및 SSH 유형의 노드에만 해당: 노드의 핑 작업에 대한 링크입니다. 핑 링크를 누르면 노드에서 지원하는 통신 프로토콜을 통해 노드에 연결할 수 있는 여부가 테스트됩니다.

CONFIG 노드의 경우에는 작업을 사용할 수 없습니다.

노드 테이블에는 다음 옵션도 포함되어 있습니다.

새로 만들기

노드를 생성하는 단추입니다.

삭제

선택한 노드를 하나 이상 삭제하는 단추입니다.

삭제 및 설치 해제

선택한 노드를 한 개 이상 삭제한 다음 삭제된 각 노드가 나타내는 호스트에서 GlassFish Server 소프트웨어의 설치를 제거하는 단추입니다.

주:

CONFIG 유형의 노드가 나타내는 호스트에서는 GlassFish Server 소프트웨어의 설치를 제거할 수 없습니다. CONFIG 유형 노드의 경우 이 단추는 [삭제] 단추와 동일하게 작동합니다.

  • 노드를 편집하는 방법
