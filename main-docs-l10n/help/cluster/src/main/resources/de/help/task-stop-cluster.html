So wird ein Cluster gestoppt

Durch Anhalten eines Clusters werden alle ausgeführten GlassFish Server-Instanzen im Cluster angehalten.

  1. Wählen Sie in der Navigationsbaumstruktur den Knoten "Cluster".

    Die Seite "Cluster" wird geöffnet.

  2. Aktivieren Sie auf der Seite "Cluster" das Kontrollkästchen neben dem Namen des Clusters.

  3. Klicken Sie auf "Cluster anhalten".

    Sie müssen das Anhalten des Clusters bestätigen.

  4. Bestätigen Sie das Anhalten des Clusters.

    Das Cluster wird angehalten. Der Status jeder Instanz im Cluster wird auf der Seite "Cluster" aktualisiert, wenn die Instanz angehalten wird.
