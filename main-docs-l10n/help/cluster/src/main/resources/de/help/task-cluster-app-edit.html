So wird eine Anwendung bearbeitet, die in einem Cluster bereitgestellt ist

  1. Blenden Sie in der Navigationsbaumstruktur den Knoten "Cluster" ein.

  2. Wählen Sie unter dem eingeblendeten Knoten "Cluster" den Knoten für das Cluster.

    Die Seite "Allgemeine Informationen" für das Cluster wird geöffnet.

  3. Klicken Sie auf der Seite "Allgemeine Informationen" auf die Registerkarte "Anwendungen".

    Die Seite "Anwendungen" für das Cluster wird geöffnet.

  4. Klicken Sie auf der Seite "Anwendungen" in der Tabelle "Bereitgestellte Anwendungen" auf den Anwendungsnamen.

    Die Seite "Anwendung bearbeiten" wird geöffnet.

  5. Geben Sie auf der Seite "Anwendung bearbeiten" die erforderlichen Informationen zur Anwendung an, und speichern Sie die Änderungen.

    Weitere Informationen finden Sie in der Onlinehilfe unter Edit Application.
