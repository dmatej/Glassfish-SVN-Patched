So werden Ressourcen für eine Standalone-GlassFish Server-Instanz aufgelistet

  1. Wählen Sie in der Navigationsbaumstruktur den Knoten "Standalone-Instanzen".

    Die Seite "Standalone-Serverinstanz" wird geöffnet.

  2. Wählen Sie auf der Seite "Standalone-Serverinstanzen" die Instanz.

    Die Seite "Allgemeine Informationen" für die Instanz wird geöffnet.

  3. Klicken Sie auf der Seite "Allgemeine Informationen" auf die Registerkarte "Ressourcen".

    Die Seite "Ressourcen" für die Instanz wird geöffnet.

  4. Um nur Ressourcen eines bestimmten Typs anzuzeigen, wählen Sie in der Pulldown-Liste "Filter" den anzuzeigenden Ressourcentyp.
