So wird eine Ressource für ein Cluster erstellt

  1. Blenden Sie in der Navigationsbaumstruktur den Knoten "Cluster" ein.

  2. Wählen Sie unter dem eingeblendeten Knoten "Cluster" den Knoten für das Cluster.

    Die Seite "Allgemeine Informationen" für das Cluster wird geöffnet.

  3. Klicken Sie auf der Seite "Allgemeine Informationen" auf die Registerkarte "Ressourcen".

    Die Seite "Ressourcen" für das Cluster wird geöffnet.

  4. Wählen Sie im Pulldown-Menü "Neu" den zu erstellenden Ressourcentyp.

    Die Seite zum Erstellen des ausgewählten Ressourcentyps wird geöffnet.

  5. Nehmen Sie auf der Seite zum Erstellen des ausgewählten Ressourcentyps die erforderlichen Änderungen vor, und speichern Sie die Änderungen.

    Weitere Informationen zum Erstellen des ausgewählten Ressourcentyps finden Sie in der Onlinehilfe.

    Hinweis:

    Stellen Sie beim Erstellen der Ressource sicher, dass Sie das Cluster als Ziel angeben.
