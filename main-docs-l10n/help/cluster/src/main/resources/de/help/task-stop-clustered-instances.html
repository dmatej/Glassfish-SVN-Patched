So werden geclusterte GlassFish Server-Instanzen angehalten

  1. Blenden Sie in der Navigationsbaumstruktur den Knoten "Cluster" ein.

  2. Wählen Sie unter dem eingeblendeten Knoten "Cluster" den Knoten für das Cluster.

    Die Seite "Allgemeine Informationen" für das Cluster wird geöffnet.

  3. Klicken Sie auf der Seite "Allgemeine Informationen" auf die Registerkarte "Instanzen".

    Die Seite "Clusterserverinstanzen" wird für das Cluster geöffnet.

  4. Aktivieren Sie für jede anzuhaltende Instanz das Kontrollkästchen neben der Instanz in der Tabelle "Serverinstanzen".

  5. Klicken Sie auf "Anhalten".

    Sie müssen das Anhalten der Instanzen bestätigen.

  6. Bestätigen Sie das Anhalten der Instanzen.

    Der Status der Instanzen wird in der Tabelle "Serverinstanzen" aktualisiert, wenn die Instanz angehalten wird.
