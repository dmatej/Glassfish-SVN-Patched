Clustersystemeigenschaften

Auf der Seite "Cluster-Systemeigenschaften" werden die Java-Systemeigenschaften aufgelistet, die für das gewählte Cluster festgelegt sind. Java-Systemeigenschaften werden an das Java-Anwendungsstarttool über die Option -D des Java-Anwendungsstarttools weitergeleitet, wenn GlassFish Server gestartet wird.

Das Feld "Clustername" ist ein schreibgeschütztes Feld, in dem der Name des ausgewählten Clusters angezeigt wird.

Für jede Instanz werden folgende Informationen angezeigt:

Name der Instanzenvariablen

Der Name der Systemeigenschaft.

Aktueller Wert

Der Wert, der aktuell für die Eigenschaft festgelegt ist. Dieses Feld ist schreibgeschützt.

Überschreibungswert

Der Wert, auf den die Eigenschaft gesetzt wird, nachdem die Änderungen gespeichert wurden.

Die Tabelle "Weitere Eigenschaften" enthält darüber hinaus die folgenden Optionen.

Eigenschaft hinzufügen

Schaltfläche zum Hinzufügen einer Eigenschaft. Durch Klicken auf diese Schaltfläche wird eine Zeile zur Tabelle "Weitere Eigenschaften" hinzugefügt.

Eigenschaften löschen

Schaltfläche zum Löschen von mindestens einer ausgewählten Eigenschaft. Eine gelöschte Eigenschaft wird auf den Standardwert zurückgesetzt oder, falls kein Standardwert vorhanden ist, nicht definiert.
