So werden geclusterte GlassFish Server-Instanzen gelöscht

Achtung:

Löschen Sie nicht die Instanz, die mit dem Master-Broker verknüpft ist, wenn Sie ein Java Message Service-(JMS-)Cluster mit einem Master-Broker verwenden. Wenn diese Instanz gelöscht werden muss, verwenden Sie den Unterbefehl change-master-broker 1, um den Master-Broker einer anderen Instanz zuzuweisen.

  1. Blenden Sie in der Navigationsbaumstruktur den Knoten "Cluster" ein.

  2. Wählen Sie unter dem eingeblendeten Knoten "Cluster" den Knoten für das Cluster.

    Die Seite "Allgemeine Informationen" für das Cluster wird geöffnet.

  3. Klicken Sie auf der Seite "Allgemeine Informationen" auf die Registerkarte "Instanzen".

    Die Seite "Clusterserverinstanzen" wird für das Cluster geöffnet.

  4. Aktivieren Sie für jede zu löschende Instanz das Kontrollkästchen neben der Instanz in der Tabelle "Serverinstanzen".

  5. Klicken Sie auf "Löschen".

    Sie müssen das Löschen der Instanzen bestätigen.

  6. Bestätigen Sie das Löschen der Instanzen.

    Die Instanzen werden gelöscht und aus der Tabelle der Instanzen auf der Seite entfernt.
