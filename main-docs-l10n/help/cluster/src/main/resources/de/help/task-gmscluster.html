So werden Group Management Service-Einstellungen für ein Cluster konfiguriert

Der Group Management Service (GMS) ist ein prozessinterner Service, der Services für Clusterüberwachung, Clustermitgliedschaft und Gruppenkommunikation bereitstellt. Der GMS führt folgende Funktionen aus:

  • Benachrichtigt registrierte Module in einer GlassFish Server-Instanz, wenn mindestens eine Mitgliedsinstanz zum Cluster hinzugefügt bzw. aus dem Cluster entfernt wurde oder bei einer Mitgliedsinstanz Fehler vermutet bzw. bestätigt wurden.

  • Bietet die Möglichkeit, Nachrichten zwischen Cluster-Membern zu senden und zu empfangen.

  1. Blenden Sie in der Navigationsbaumstruktur den Knoten "Cluster" ein.

  2. Wählen Sie das Cluster, für das Sie Einstellungen für den Group Management Service (GMS) konfigurieren möchten.

    Die Seite "Allgemeine Informationen" für das Cluster wird geöffnet.

  3. Aktivieren Sie das Kontrollkästchen "Aktiviert" für den GMS, um den GMS für das Cluster zu aktivieren.

    Diese Option ist standardmäßig aktiviert. Um den GMS für das Cluster zu deaktivieren, deaktivieren Sie das Kontrollkästchen "Aktiviert" für den GMS.

    Wenn diese Option aktiviert ist, wird der Group Management Service (GMS) in jeder Serverinstanz im Cluster und im Domainadministrationsserver (DAS, Domain Administration Server) gestartet. Der DAS ist an jedem Cluster beteiligt, für den GMS aktiviert ist.

  4. Geben Sie im Feld "Multicast-Port" den Port ein, den der Group Management Service zum Überwachen der Gruppenereignisse verwendet.

    Geben Sie eine standardmäßige UDP-Portnummer im Bereich 2048-49151 an. Der Standardwert wird automatisch in diesem Bereich generiert.

  5. Geben Sie im Feld "Multicast-Adresse" die Adresse an, die der Group Management Service zum Überwachen der Gruppenereignisse verwendet.

    Geben Sie eine IP-Multicast-Adresse der Klasse D an. IP-Adressen der Klasse D liegen im Bereich von 224.0.0.0 bis einschließlich 239.255.255.255. Die Adresse 224.0.0.0 ist reserviert und sollte nicht verwendet werden. Der Standardwert ist 228.9.XX. YY. Dabei stehen XX und YY für automatisch generierte unabhängige Werte zwischen 0 und 255.

  6. Geben Sie im Feld "Bind-Schnittstellenadresse" die Netzwerkschnittstellenadresse für den Group Management Service (GMS) an, wenn mehrere Netzwerkschnittstellen vorhanden sind.

    Durch diese Einstellung können Sie festlegen, welche Netzwerkschnittstelle für den GMS verwendet wird, wenn mehrere Netzwerkschnittstellen auf den Rechnern vorhanden sind, auf denen die Serverinstanzen im Cluster gehostet werden. Bei diesem Wert muss es sich um die IP-Adresse einer lokalen Netzwerkschnittstelle handeln. Standardmäßig werden alle öffentlichen Netzwerkschnittstellenadressen verwendet.

    Um eine Adresse anzugeben, die für alle GlassFish Server-Instanzen im Cluster gültig ist, verwenden Sie eine Systemeigenschaft, mit der die Adresse einzeln für jede Instanz festgelegt werden kann.

    Beispiel: Erstellen Sie die Systemeigenschaft GMS-BIND-INTERFACE-ADDRESS-cluster-name auf dem DAS. Danach legen Sie diese Option "Bind-Schnittstellenadresse" auf ${GMS-BIND-INTERFACE-ADDRESS-cluster-name} fest, um die Systemeigenschaft anzugeben. Legen Sie abschließend für jede Instanz im Cluster die Systemeigenschaft GMS-BIND-INTERFACE-ADDRESS- cluster-name auf die erforderliche Netzwerkschnittstellenadresse im Rechner der Instanz fest.

  7. Um weitere Eigenschaften anzugeben, wählen Sie in der Registerkarte "Eigenschaften" die untergeordnete Registerkarte "Clustereigenschaften".

    • Um eine Eigenschaft hinzuzufügen, klicken Sie auf die Schaltfläche "Eigenschaft hinzufügen". Geben Sie in die daraufhin angezeigte leere Zeile den Eigenschaftsnamen in das Feld "Name" und den Eigenschaftswert in das Feld "Wert" ein.

    • Um eine Eigenschaft zu ändern, bearbeiten Sie das Feld "Wert" dieser Eigenschaft.

    • Um eine Eigenschaft zu löschen, aktivieren Sie das Kontrollkästchen links neben dem Feld "Name" der zu löschenden Eigenschaft und klicken anschließend auf die Schaltfläche "Eigenschaften löschen".

  8. Klicken Sie auf "Speichern".
