So zeigen Sie Logdateien für eine Standalone-GlassFish Server-Instanz an

Bevor Sie beginnen

Die Instanz, für die Sie Logdateien anzeigen möchten, muss ausgeführt werden.

  1. Wählen Sie in der Navigationsbaumstruktur den Knoten "Standalone-Instanzen".

    Die Seite "Standalone-Serverinstanz" wird geöffnet.

  2. Wählen Sie die Instanz, für die Logdateien angezeigt werden sollen.

    Die Seite "Allgemeine Informationen" für die Instanz wird geöffnet.

  3. Klicken Sie auf der Seite "Allgemeine Informationen" auf "Logdateien anzeigen".

    Ein Log-Viewer-Fenster wird geöffnet.

    Hinweis:

    Die Instanz muss bereits ausgeführt werden. Wenn die Instanz nicht ausgeführt wird, ist die Schaltfläche "Logdateien anzeigen" deaktiviert.
