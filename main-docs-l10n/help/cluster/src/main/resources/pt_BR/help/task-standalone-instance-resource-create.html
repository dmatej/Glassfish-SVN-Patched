Para Criar um Recurso para uma Instância do GlassFish Server Standalone

  1. Na árvore de navegação, selecione o nó Instâncias Standalone.

    A página Instância de Servidor Standalone será aberta.

  2. Na página Instância de Servidor Standalone, selecione a instância.

    A página Informações Gerais da instância se abre.

  3. Na página Informações Gerais, clique na guia Recursos.

    A página Recursos da instância se abre.

  4. No menu pull-down Novo, selecione o tipo do recurso a ser criado.

    É exibida a página para a criação de um recurso do tipo selecionado.

  5. Na página para edição de um recurso do tipo selecionado, faça as alterações necessárias e salve-as.

    Para obter detalhes, consulte a ajuda on-line da página de criação de recursos do tipo selecionado.

    Observação:

    Ao criar um recurso, certifique-se de ter especificado a instância como alvo.
