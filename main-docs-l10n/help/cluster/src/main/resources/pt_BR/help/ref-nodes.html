Nós

A página Nós exibe uma lista de nós no domínio. São exibidas as seguintes informações para cada nó:

Nome

Nome designado ao nó quando o nó foi criado. Clicar no nome abre a página Editar Nó para o nó.

Host do Nó

O nome do host que o nó representa.

Tipo

Tipo do nó, que é um dos seguintes tipos:

CONFIG

O nó não oferece suporte à comunicação remota.

DCOM

O nó suporta comunicação pelo protocolo remoto do DCOM (Distributed Component Object Model). O protocolo DCOM está disponível somente nos sistemas do Windows.

SSH

O nó oferece suporte a comunicações com shell seguro (SSH).

Instâncias

Lista das instâncias que residem no nó com uma indicação se a instância está em execução:

  • Se a instância estiver em execução, uma marca de verificação (✓) será exibida.

  • Se a instância estiver interrompida, um ponto de exclamação (!) será exibido.

Ao clicar no nome da instância, será aberta a página Informações Gerais.

Ação

Só nós do tipo DCOM e SSH: um link para a ação Ping do nó. Clicar no link Ping testa se o nó pode ser alcançado pelo protocolo de comunicação que o nó suporta.

Não é possível realizar nenhuma ação para os nós CONFIG.

A tabela Nós também contém as opções seguintes.

Novo

Botão para criar um nó.

Deletar

Botão para deletar um ou mais nós selecionados.

Deletar e Desinstalar

Botão para deletar um ou mais nós selecionados e desinstalar o software GlassFish Server do host que cada nó deletado representa.

Observação:

O software GlassFish Server não é desinstalado de qualquer host representado por um nó do tipo CONFIG. Para os nós do tipo CONFIG, a ação deste botão é idêntica à ação do botão Deletar.
