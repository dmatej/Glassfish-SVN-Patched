Clusters

A página Clusters exibe uma lista de clusters no domínio. São exibidas as seguintes informações para cada cluster:

Nome

Nome designado ao cluster quando o cluster é criado. Clicar no nome abre a página Informações Gerais para o cluster.

Configuração

A configuração selecionada para o cluster no momento em que ele foi criado. Clicar na configuração abre a página Configurações do cluster.

Instâncias

O nome e o status de cada instância do cluster. Clicar no nome da instância abre a página Informações Gerais da instância.

A tabela Clusters também contém as opções a seguir.

Novo

Botão para criar um cluster.

Deletar

Botão para deletar um cluster.

Iniciar Cluster

Botão para iniciar um cluster.

Interromper Cluster

Botão para interromper um cluster.
