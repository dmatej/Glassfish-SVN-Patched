Aplicações

Utilize a página Aplicações para realizar as seguintes tarefas no cluster atual:

  • Exibindo e gerenciando aplicações que já estão implantadas no cluster

  • Implantando mais aplicações no cluster

O campo Nome do Cluster é um campo somente para leitura que exibe o nome do cluster atual.

A página Aplicações exibe uma lista das aplicações que estão implantadas no cluster atual. Para cada aplicação, as informações a seguir são mostradas.

Nome

O nome da aplicação.

O nome pode incluir um identificador de versão opcional, que segue o nome e é separado do nome por dois-pontos (:). Para obter mais informações sobre versões de módulo e da aplicação, consulte Module and Application Versions em Oracle GlassFish Server Application Deployment Guide.

Ativado

Uma indicação que informa se a aplicação está ativada:

  • Se a aplicação estiver ativada, uma marca de verificação (✓) será exibida.

  • Se a aplicação for desativada, um ✗ () é exibido.

Balanceador de Carga Ativado

Indicação de que a aplicação está ativada para balanceamento de carga:

  • Se a aplicação estiver ativada para balanceamento de carga, uma marca de verificação (✓) será exibida.

  • Se a aplicação for desativada para balanceamento de carga, um (✗) é exibido.

Mecanismos

Os tipos de contêineres que a aplicação utiliza. Os tipos de contêineres podem ser os seguintes:

  • web

  • webservices

  • ejb

  • connector

  • appclient

  • weld — o contêiner para Contextos e Injeção de Dependência para aplicações da Plataforma Java EE

A tabela Aplicações Implantadas também contém as opções a seguir.

Implantar

Botão para implantar uma aplicação.

Cancelar Implantação

Botão para cancelar a implantação de uma ou mais aplicações selecionadas.

O cancelamento da implantação de uma aplicação remove a aplicação de todos os alvos e do domínio. Quando a implantação de uma aplicação é cancelada, a aplicação deixa de ser listada na página Aplicações que exibe todas as aplicações implantadas no GlassFish Server.

Para remover aplicações somente do alvo atual, utilize o botão Remover desta página.

Remover

Botão para remover um ou mais aplicações selecionadas.

A remoção de uma aplicação remove a aplicação somente do alvo atual. Quando uma aplicação é removida de um alvo, a aplicação continua a ser listada na página Aplicações que exibe todas as aplicações implantadas no GlassFish Server. A aplicação também permanece implantada nos outros alvos nos quais já havia sido implantada.

Para remover aplicações de todos os alvos e do domínio, utilize o botão Cancelar Implantação desta página.

Mais Ações

Lista drop-down que realiza ações adicionais em um ou mais aplicações selecionadas.

Ativar

Ativa uma ou mais aplicações selecionadas.

Desativar

Desativa um ou mais aplicações selecionadas.

Ativar Balanceador de Carga

Ativa um ou mais aplicações selecionadas para balanceamento de carga.

Desativar Balanceador de Carga

Desativa uma ou mais aplicações selecionadas para balanceamento de carga.
