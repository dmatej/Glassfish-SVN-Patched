Para Ativar ou Desativar o Balanceamento de Carga de uma Aplicação que foi Implantada em um Cluster

  1. Na árvore de navegação, expanda o nó Clusters.

  2. No nó Clusters expandidos, selecione o nó do cluster.

    A página Informações Gerais do cluster será aberta.

  3. Na página Informações Gerais, clique na guia Aplicações.

    A página Aplicações do cluster se abre.

  4. Marque a caixa de seleção ao lado da aplicação.

  5. No menu pull-down Mais Ações, escolha o comando apropriado.

    • Para ativar o balanceamento de carga da aplicação, escolha Ativar Balanceador de Carga.

    • Para desativar o balanceamento de carga da aplicação, escolha Desativar Balanceador de Carga.
