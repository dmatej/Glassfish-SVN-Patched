Nova Instância de Servidor Standalone

Utilize a página Nova Instância de Servidor Standalone para criar uma instância standalone do GlassFish Server.

A página Nova Instância de Servidor Standalone contém as seguintes opções.

Nome da Instância

O nome da instância que está sendo criada.

O nome deve atender aos seguintes requisitos:

  • O nome só pode conter caracteres ASCII.

  • O nome deve começar com uma letra, um número ou um sublinhado.

  • O nome pode conter só os seguintes caracteres:

    • Letras minúsculas

    • Letras maiúsculas

    • Números

    • Hífen

    • Ponto

    • Sublinhado

  • O nome deve ser único no domínio e não deve ser o nome de outra instância, um cluster, uma configuração nomeada ou um nó do GlassFish Server.

  • O nome não deve ser domain, server, nem nenhuma outra palavra-chave que esteja reservada pelo GlassFish Server.

Nó

Lista drop-down dos nós existentes que definem os hosts nos quais a instância pode estar. É necessário selecionar um nó na lista. Se a instância for criada no host em que o servidor de administração de domínios (DAS) estiver em execução, selecione o domínio do nó predefinido localhost-.

Configuração

Lista drop-down das configurações nomeadas existentes que a instância pode utilizar. É necessário selecionar uma configuração na lista. A instância utilizará a configuração selecionada quando a instância for criada.

Criar uma Cópia da Configuração Selecionada

Se esta opção for selecionada, a configuração selecionada será copiada quando a instância for criada. Foi designado à cópia da configuração o nome instance-name-config, em que instance-name é o nome da instância, conforme digitado no campo Nome da Instância.

Se default-config estiver selecionado na lista drop-down Configuração, é necessário selecionar esta opção. A configuração default-config pode ser apenas copiada, não referenciada.

Referenciar a Configuração Selecionada

Se esta opção for selecionada, a instância utilizará a configuração nomeada existente especificada.

Se default-config estiver selecionado na lista drop-down Configuração, esta opção não deve ser selecionada. A configuração default-config não pode ser referenciada, somente copiada.
