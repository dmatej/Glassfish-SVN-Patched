Informações Gerais

A página Informações Gerais contém as seguintes opções.

Iniciar

Botão para iniciar a instância. Se a instância estiver em execução, este botão será desativado.

Interromper

Botão para interromper a instância. Se a instância estiver interrompida, este botão está desativado.

Reiniciar

Botão para reiniciar a instância. Se a instância estiver interrompida, este botão está desativado.

Exibir Arquivos de Log

Botão para exibir os arquivos de log da instância.

Recuperar Transações

Botão para recuperar transações da instância.

Nome do Cluster

Nome designado ao cluster quando o cluster é criado. Clicar no nome abre a página Informações Gerais para o cluster.

Nome da Instância

Nome designado à instância quando a instância foi adicionada ao cluster. Este campo é somente para leitura.

Status

Indica se a instância está em execução, se requer reinicialização para que seja atualizada com as alterações na configuração ou se não está em execução. Este campo é somente para leitura.

JVM

Se você clicar em Relatório da JVM, será aberta uma janela separada que exibe relatórios sobre a Java Virtual Machine, incluindo um relatório de resumo, um relatório de gerenciamento de memória e de coleta de erros,um relatório de carregamento de classe e um dump de threads atuais.

Nó

O nome do nó no qual a instância reside. Clicar no nome abre a página Editar Nó para o nó.

Configuração

A configuração à qual a instância faz referência. Ao clicar na configuração, a página Configurações da instância se abre.

Portas HTTP

Uma lista separada por vírgulas dos números das portas na qual a instância faz listening dos seguintes tipos de solicitações:

  • Solicitações de administração

  • Solicitações HTTP

  • Solicitações HTTPS

Este campo é somente para leitura.

Portas IIOP

Uma lista separada por vírgulas dos números das portas que a instância utiliza nos seguintes tipos de conexões:

  • Conexão IIOP segura

  • Conexões IIOP seguras com autenticação do cliente

  • Conexões IIOP

Este campo é somente para leitura.
