Para Ativar ou Desativar um Recurso Globalmente

  1. Na árvore de navegação, expanda o nó Clusters.

  2. No nó Clusters expandidos, selecione o nó do cluster.

    A página Informações Gerais do cluster será aberta.

  3. Na página Informações Gerais, clique na guia Recursos.

    A página Recursos do cluster será aberta.

  4. Marque a caixa de seleção ao lado do recurso.

  5. Clique no botão apropriado.

    • Para ativar o recurso, escolha Ativar.

    • Para desativar o recurso, escolha Desativar.

      A desativação de um recurso não o remove.
