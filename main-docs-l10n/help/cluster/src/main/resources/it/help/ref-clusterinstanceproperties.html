Proprietà istanza

Nella pagina Proprietà istanza viene visualizzata una lista delle proprietà impostate per l'istanza di GlassFish Server corrente. Queste proprietà aggiungono informazioni di configurazione opzionali relative all'istanza.

Il campo Nome istanza è un campo di sola lettura in cui viene visualizzato il nome dell'istanza corrente.

Per ogni proprietà vengono visualizzate le informazioni riportate di seguito.

Nome

Il nome della proprietà.

Valore

Il valore della proprietà impostato per l'istanza selezionata.

Descrizione

Descrizione di testo che fornisce ulteriori informazioni sulla proprietà.

La tabella Proprietà aggiuntive contiene inoltre le opzioni riportate di seguito.

Aggiungi proprietà

Pulsante che consente di aggiungere una proprietà. Se si fa clic su questo pulsante, viene aggiunta una riga alla tabella Proprietà aggiuntive.

Elimina proprietà

Pulsante che consente di eliminare una o più proprietà selezionate. L'eliminazione comporta il ripristino del valore predefinito della proprietà. Se il valore predefinito non è stato impostato, la proprietà rimane non definita.

GlassFish Server definisce le proprietà di istanza riportate di seguito.

rendezvousOccurred

Specifica se l'istanza ha contattato il server di amministrazione del dominio (DAS). I possibili valori sono:

true

L'istanza ha contattato il server DAS.

false

L'istanza non ha contattato il server DAS.
