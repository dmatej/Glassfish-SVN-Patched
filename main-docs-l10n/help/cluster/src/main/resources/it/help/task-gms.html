Per configurare il servizio GMS (Group Management Service)

GMS (Group Management Service) è un servizio in-process che offre servizi di monitoraggio del cluster, appartenenza del cluster e comunicazione di gruppo. Il servizio GMS esegue le funzioni seguenti:

  • notifica ai moduli registrati in un'istanza di GlassFish Server l'aggiunta o la rimozione di una o più istanze membro da un cluster oppure un errore sospettato o confermato di tali istanze;

  • consente di inviare e ricevere messaggi tra i membri del cluster.

  1. Espandere il nodo Configurazione nell'albero di navigazione.

  2. Selezionare la configurazione per il cluster da modificare.

  3. Nel nodo della configurazione, selezionare il nodo Group Management Service.

    Viene visualizzata la pagina Group Management Service.

  4. Nel campo Numero massimo heartbeat mancati digitare il numero massimo di heartbeat mancati da un membro prima che il servizio GMS confermi un probabile errore.

    L'impostazione predefinita è 3.

  5. Nel campo Frequenza heartbeat digitare il periodo di tempo che deve trascorrere tra le trasmissioni successive dell'heartbeat di un membro del gruppo.

    L'impostazione predefinita è 2000 millisecondi.

  6. Nel campo Timeout individuazione gruppo digitare il periodo di tempo durante il quale il servizio GMS rimane in attesa dell'individuazione di altri membri del cluster.

    L'impostazione predefinita è 5000 millisecondi.

  7. Nel campo Tempo di attesa verifica errore digitare il periodo di tempo di attesa prima di verificare se segnalare come in stato di errore un membro nel quale si sospetta si siano verificati errori.

    L'impostazione predefinita è 1500 millisecondi.

  8. Per ripristinare i valori predefiniti, fare clic sul pulsante Carica impostazioni predefinite.

  9. Nella sezione Proprietà aggiuntive, specificare le proprietà aggiuntive.

    • Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

    • Per modificare una proprietà, è necessario modificare il valore del campo Valore della proprietà.

    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Nome corrispondente alla proprietà da eliminare, quindi fare clic sul pulsante Elimina proprietà.

  10. Fare clic su Salva.
