Per visualizzare i dati di monitoraggio dei server virtuali per un'istanza standalone di GlassFish Server

Prima di iniziare

Per visualizzare i dati di monitoraggio, è necessario configurare il servizio di monitoraggio.

  1. Selezionare il nodo Istanze standalone nell'albero di navigazione.

    Viene visualizzata la pagina Istanze server standalone.

  2. Selezionare l'istanza per la quale visualizzare i dati di monitoraggio per i server virtuali.

    Viene visualizzata la pagina Informazioni generali per l'istanza.

  3. Nella pagina Informazioni generali, fare clic sulla scheda Monitoraggio.

    Viene visualizzata la pagina Monitoraggio applicazioni.

  4. Fare clic sulla scheda Server.

    Viene visualizzata la pagina Monitoraggio server.

  5. Nell'elenco a discesa Visualizza, selezionare il componente per il quale si desidera visualizzare i dati di monitoraggio.

  6. Per aggiornare i dati di monitoraggio visualizzati, fare clic su Aggiorna.
