Per impostare le proprietà specifiche per un'istanza di GlassFish Server in cluster

Le proprietà di un'istanza di GlassFish Server aggiungono informazioni di configurazione opzionali relative all'istanza.

  1. Espandere il nodo Cluster nell'albero di navigazione.

  2. Selezionare il nodo per il cluster sotto il nodo Cluster espanso.

    Viene visualizzata la pagina Informazioni generali per il cluster.

  3. Nella pagina Informazioni generali, fare clic sulla scheda Istanze.

    Viene visualizzata la pagina Istanze server in cluster per il cluster.

  4. Selezionare l'istanza per la quale modificare le proprietà specifiche.

    Viene visualizzata la pagina Informazioni generali per l'istanza.

  5. Nella pagina Informazioni generali, fare clic sulla scheda Proprietà.

  6. Se necessario, fare clic sulla scheda Proprietà istanza.

    Viene visualizzata la pagina Proprietà istanza.

    Nella pagina viene visualizzata una lista delle proprietà impostate per l'istanza di GlassFish Server selezionata.

  7. Per ogni proprietà esistente da modificare, digitare il nuovo valore nel campo Valore per la proprietà.

  8. Impostare i valori per eventuali proprietà aggiuntive necessarie.

    1. Nella tabella Proprietà aggiuntive fare clic su Aggiungi proprietà.

      Viene aggiunta una riga alla tabella Proprietà aggiuntive.

    2. Nel campo Nome, digitare il nome della proprietà.

    3. Nel campo Valore digitare il valore della proprietà.

    4. Nel campo Descrizione digitare una breve descrizione della proprietà.

  9. Fare clic su Salva.
