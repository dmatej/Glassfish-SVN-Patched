启动集群 GlassFish Server 实例

开始之前

应确保满足以下先决条件：

  • 为远程通信启用实例驻留的所有节点。

  • 域管理服务器 (DAS) 用户可使用分布式组件对象模型 (DCOM) 远程协议或安全 Shell (SSH) 登录到实例驻留的任何节点的主机。

如果不满足其中任一先决条件，可通过使用 start-local-instance(1) 子命令从命令行单独启动每个实例。

  1. 在导航树中，展开“集群”节点。

  2. 在展开的“集群”节点下，选择某个集群的节点。

    此时将打开集群的“一般信息”页。

  3. 在“一般信息”页中，单击“实例”选项卡。

    此时将打开集群的“集群服务器实例”页。

  4. 对于要启动的每个实例，选择“服务器实例”表中该实例旁边的复选框。

  5. 单击“启动”。

    系统会要求您确认是否要启动实例。

  6. 确认您要启动实例。

    启动实例后，将会更新“服务器实例”表中每个实例的状态。
