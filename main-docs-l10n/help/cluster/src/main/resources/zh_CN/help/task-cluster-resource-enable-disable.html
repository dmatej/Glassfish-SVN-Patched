全局启用或禁用资源

  1. 在导航树中，展开“集群”节点。

  2. 在展开的“集群”节点下，选择某个集群的节点。

    此时将打开集群的“一般信息”页。

  3. 在“一般信息”页中，单击“资源”选项卡。

    此时将打开集群的“资源”页。

  4. 选择资源旁边的复选框。

  5. 单击相应的按钮。

    • 要启用资源，可选择“启用”。

    • 要禁用资源，可选择“禁用”。

      禁用某个资源不会删除该资源。
