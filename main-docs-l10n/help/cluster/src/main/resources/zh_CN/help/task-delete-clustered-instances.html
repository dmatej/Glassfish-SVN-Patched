删除集群 GlassFish Server 实例

注意:

如果使用的是带有主中介的 Java 消息服务 (JMS) 集群，请不要删除与该主中介关联的实例。如果必须删除该实例，请使用 change-master-broker(1) 子命令将该主中介指定给另一个实例。

  1. 在导航树中，展开“集群”节点。

  2. 在展开的“集群”节点下，选择某个集群的节点。

    此时将打开集群的“一般信息”页。

  3. 在“一般信息”页中，单击“实例”选项卡。

    此时将打开集群的“集群服务器实例”页。

  4. 对于要删除的每个实例，选择“服务器实例”表中该实例旁边的复选框。

  5. 单击“删除”。

    系统会要求您确认是否要删除实例。

  6. 确认您要删除实例。

    将删除该实例，并将其从该页的实例表中删除。
