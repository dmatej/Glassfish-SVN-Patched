查看集群 GlassFish Server 实例的日志文件

开始之前

您要查看其日志文件的实例必须正在运行。

  1. 在导航树中，展开“集群”节点。

  2. 在展开的“集群”节点下，选择某个集群的节点。

    此时将打开集群的“一般信息”页。

  3. 在“一般信息”页中，单击“实例”选项卡。

    此时将打开集群的“集群服务器实例”页。

  4. 选择要查看日志文件的实例。

    此时将打开实例的“一般信息”页。

  5. 在“一般信息”页中，单击“查看日志文件”。

    此时将打开“日志查看器”窗口。

    注:

    实例必须已在运行。如果实例没有运行，将取消激活“查看日志文件”按钮。
