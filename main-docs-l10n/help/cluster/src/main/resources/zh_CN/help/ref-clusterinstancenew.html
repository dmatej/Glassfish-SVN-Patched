新建集群服务器实例

使用“新建集群服务器实例”页可向现有集群添加 GlassFish Server 实例。

“新建集群服务器实例”包含以下选项。

集群名称

创建集群时为其指定的名称。此字段为只读。

实例名称

要创建的实例的名称。

名称必须满足以下要求：

  • 名称中只能包含 ASCII 字符。

  • 名称开头必须是字母、数字或下划线。

  • 名称只能包含以下字符：

    • 小写字母

    • 大写字母

    • 数字

    • 连字符

    • 句点

    • 下划线

  • 名称在域中必须是唯一的，不能是另一个 GlassFish Server 实例、集群、命名配置或节点的名称。

  • 名称不得是 domain、server 或 GlassFish Server 保留的任何其他关键字。

节点

现有节点的下拉列表，定义实例可驻留的主机。必须从列表中选择一个节点。如果要在运行域管理服务器 (DAS) 的主机上创建实例，请选择预定义节点 localhost-domain。
