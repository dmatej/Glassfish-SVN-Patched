查看独立 GlassFish Server 实例的虚拟服务器监视数据

开始之前

必须先配置监视服务，才可以查看监视数据。

  1. 在导航树中，选择“独立实例”节点。

    将打开“独立服务器实例”页。

  2. 选择要查看虚拟服务器监视数据的实例。

    此时将打开实例的“一般信息”页。

  3. 在“一般信息”页中，单击“监视”选项卡。

    此时将打开“应用程序监视”页。

  4. 单击“服务器”选项卡。

    此时将打开“服务器监视”页。

  5. 在“查看”下拉列表中，选择要查看监视数据的组件。

  6. 要更新显示的监视数据，可单击“刷新”。
