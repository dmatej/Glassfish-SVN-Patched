停止集群 GlassFish Server 实例

  1. 在导航树中，展开“集群”节点。

  2. 在展开的“集群”节点下，选择某个集群的节点。

    此时将打开集群的“一般信息”页。

  3. 在“一般信息”页中，单击“实例”选项卡。

    此时将打开集群的“集群服务器实例”页。

  4. 对于要停止的每个实例，选择“服务器实例”表中该实例旁边的复选框。

  5. 单击“停止”。

    系统会要求您确认是否要停止实例。

  6. 确认您要停止实例。

    停止实例后，将会更新“服务器实例”表中每个实例的状态。
