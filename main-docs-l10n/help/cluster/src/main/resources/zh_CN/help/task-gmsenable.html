启用组管理服务

组管理服务 (GMS) 是一种进程内服务，可提供集群监视、集群成员和组通信服务。GMS 执行以下功能：

  • 在集群中添加或删除一个或多个成员实例，或者怀疑或确认一个或多个成员实例已失败时，通知 GlassFish Server 实例中的已注册模块。

  • 提供在集群成员之间发送和接收消息的功能。

  1. 在导航树中，展开“集群”节点。

  2. 选择要为其启用组管理服务 (GMS) 的集群。

    此时将打开集群的“一般信息”页。

  3. 选中“已启用 GMS”复选框为集群启用 GMS。

    默认情况下，此选项已选中。要为集群禁用 GMS，则将复选标记从“已启用 GMS”复选框中删除。

    如果启用，则会在集群的每个服务器实例和域管理服务器 (DAS) 中启动组管理服务。DAS 会参与到启用 GMS 的每个集群中。

  4. 单击“保存”。
