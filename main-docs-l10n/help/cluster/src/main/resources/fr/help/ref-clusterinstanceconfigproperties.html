Propriétés système d'instance

La page Propriétés système d'instance affiche la liste des propriétés système Java définies pour l'instance GlassFish Server en cours. Les propriétés système Java sont transmises au lanceur d'application Java via son option -D au démarrage de GlassFish Server.

Ces propriétés remplacent les définitions de propriété pour les paramètres de port dans la configuration de l'instance. Les paramètres de port prédéfinis doivent être remplacés si, par exemple, deux instances incluses dans un cluster résident sur le même hôte. Dans ce cas, les paramètres de port d'une instance doivent être remplacés car les deux instances partagent la même configuration.

Pour plus d'informations sur les paramètres de port prédéfinis dans une configuration, reportez-vous à Propriétés système de configuration prédéfinies.

Le champ Nom d'instance est en lecture seule. Il indique le nom de l'instance en cours.

Pour chaque propriété, les informations suivantes sont affichées :

Nom de la variable d'instance

Nom de la propriété système.

Valeur en cours

Valeur actuellement définie pour la propriété. Ce champ est en lecture seule.

Valeur de remplacement

Valeur sur laquelle la propriété sera définie après l'enregistrement des modifications.

La table Autres propriétés contient également les options ci-après.

Ajouter une propriété

Bouton permettant d'ajouter une propriété. Un clic sur ce bouton permet d'ajouter une ligne à la table Autres propriétés.

Supprimer des propriétés

Bouton permettant de supprimer les propriétés sélectionnées. Toute propriété supprimée revient à sa valeur par défaut ou, si aucune valeur par défaut n'est configurée, n'est pas définie.
