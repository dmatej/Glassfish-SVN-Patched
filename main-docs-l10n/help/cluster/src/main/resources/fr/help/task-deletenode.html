Suppression de noeuds

Lorsque vous supprimez un noeud, celui-ci est enlevé de la configuration du serveur d'administration de domaine (DAS). Les répertoires et fichiers du noeud sont supprimés lorsque la dernière instance GlassFish Server résidant sur le noeud est supprimée.

Vous pouvez également choisir de désinstaller le logiciel GlassFish Server de l'hôte du noeud lorsque le noeud est supprimé.

Avant de commencer

Assurez-vous qu'aucune instance GlassFish Server ne réside sur le noeud à supprimer.

  1. Dans l'arborescence de navigation, sélectionnez le noeud Noeuds.

    La page Noeuds apparaît.

  2. Pour chaque noeud à supprimer, cochez la case en regard du noeud dans la table Noeuds.

    Remarque:

    Le noeud prédéfini localhost-domain ne peut pas être supprimé.

  3. Cliquez sur le bouton pour supprimer les noeuds, ou supprimez les noeuds et désinstallez le logiciel GlassFish Server des hôtes des noeuds.

    • Pour supprimer les noeuds sans désinstaller le logiciel GlassFish Server, cliquez sur Supprimer.

      Vous êtes invité à confirmer que vous souhaitez supprimer les noeuds.

    • Pour supprimer les noeuds et désinstaller le logiciel GlassFish Server des hôtes des noeuds, cliquez sur Supprimer et désinstaller.

      Vous êtes invité à confirmer que vous souhaitez supprimer les noeuds et désinstaller le logiciel GlassFish Server des hôtes des noeuds.

      Remarque:

      Le logiciel GlassFish Server n'est pas désinstallé des hôtes représentés par un noeud de type CONFIG. Seul le noeud est supprimé.

  4. Confirmez la suppression des noeuds, ou la suppression des noeuds et la désinstallation du logiciel GlassFish Server des hôtes des noeuds.

    Les noeuds sont supprimés et enlevés de la table des noeuds sur la page Noeuds. Si vous avez cliqué sur Supprimer et désinstaller, le logiciel GlassFish Server est également désinstallé des hôtes des noeuds.
