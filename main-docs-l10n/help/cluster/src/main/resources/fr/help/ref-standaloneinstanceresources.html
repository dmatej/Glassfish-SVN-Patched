Ressources

La page Ressources permet d'effectuer les tâches suivantes sur l'instance GlassFish Server autonome sélectionnée :

  • Affichage et gestion des ressources existantes de l'instance

  • Création d'autres ressources pour l'instance

Le champ Nom d'instance est en lecture seule. Il indique le nom de l'instance en cours.

La page Ressources affiche la liste des ressources existantes sur l'instance. Pour chaque ressource, les informations suivantes sont fournies.

Nom de ressource

Nom affecté à la ressource lors de sa création. Lorsque vous cliquez sur le nom, l'une des pages suivantes pour modifier la ressource s'ouvre, en fonction de son type :

  • Modifier une ressource JDBC

  • Modifier une session JavaMail

  • Modifier une ressource personnalisée

  • Nouvelle ressource personnalisée

  • Modifier une ressource externe

  • Modifier une ressource de connecteur

  • Modifier une ressource d'objet d'administration

Activé

Mention indiquant si la ressource est activée :

  • Si la ressource est activée, une coche (✓) s'affiche.

  • Si la ressource est désactivée, une croix (✗) s'affiche.

Type

Type de la ressource.

La table Ressources contient également les options suivantes :

Activer

Bouton permettant d'activer les ressources sélectionnées.

Désactiver

Bouton permettant de désactiver les ressources sélectionnées.

Nouveau

Liste déroulante permettant de créer une ressource du type sélectionné. La sélection d'un type de ressource dans la liste entraîne l'ouverture de l'une des pages suivantes, qui permet de créer une ressource du type sélectionné :

  • Nouvelle ressource JDBC

  • Nouvelle fabrique de connexions JMS

  • Nouvelle ressource de destination JMS

  • Nouvelle session JavaMail

  • Nouvelle ressource personnalisée

  • Nouvelle ressource externe

  • Nouvelle ressource de connecteur

  • Nouvelle ressource d'objet d'administration

Filtre

Liste déroulante qui filtre la liste des ressources affichées par type.

Tâches connexes

  • Création d'une ressource pour une instance GlassFish Server autonome

  • Enumération des ressources pour une instance GlassFish Server autonome

  • Activation ou désactivation d'une ressource pour une instance GlassFish Server autonome

  • Modification d'une ressource pour une instance GlassFish Server autonome
