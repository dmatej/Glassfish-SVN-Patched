Définition de propriétés propres à l'instance pour une instance GlassFish Server incluse dans un cluster

Les propriétés d'une instance GlassFish Server ajoutent des informations de configuration facultatives sur l'instance.

  1. Dans l'arborescence de navigation, développez le noeud Clusters.

  2. Sous le noeud Clusters développé, sélectionnez le noeud correspondant au cluster.

    La page Informations générales du cluster apparaît.

  3. Sur la page Informations générales, cliquez sur l'onglet Instances.

    La page Instances de serveur clusterisées du cluster apparaît.

  4. Sélectionnez l'instance dont vous souhaitez modifier les propriétés.

    La page Informations générales de l'instance apparaît.

  5. Sur la page Informations générales, cliquez sur l'onglet Propriétés.

  6. Si nécessaire, cliquez sur l'onglet Propriétés d'instance.

    La page Propriétés d'instance apparaît.

    Cette page affiche la liste des propriétés définies pour l'instance GlassFish Server sélectionnée.

  7. Indiquez la nouvelle valeur de chaque propriété existante à modifier dans le champ Valeur.

  8. Définissez les valeurs des propriétés supplémentaires requises.

    1. Dans la table Autres propriétés, cliquez sur Ajouter une propriété.

      Une ligne est ajoutée à la table Autres propriétés.

    2. Dans le champ Nom, entrez le nom de la propriété.

    3. Dans le champ Valeur, indiquez la valeur.

    4. Dans le champ Description, entrez une brève description de la propriété.

  9. Cliquez sur Enregistrer.
