Service de disponibilité

La page Service de disponibilité permet de modifier les paramètres de disponibilité d'un cluster.

Elle contient les options ci-après.

Nom de configuration

Nom de la configuration à laquelle s'appliquent les paramètres de cette page. Ce champ est en lecture seule.

Charger les valeurs par défaut

Bouton permettant de restaurer les valeurs par défaut des paramètres qui en ont. Les paramètres qui n'ont pas de valeurs par défaut ne sont pas modifiés.

Service de disponibilité

Si la case Activé est cochée, la disponibilité est activée pour le cluster. Cette option est sélectionnée par défaut.

Autres propriétés

Table des propriétés supplémentaires pour le service de disponibilité.

GlassFish Server ne définit aucune autre propriété pour le service de disponibilité.

Pour chaque propriété, les informations suivantes sont affichées :

Nom

Nom de la propriété.

Valeur

Valeur de la propriété définie pour l'instance sélectionnée.

Description

Texte de description qui fournit de plus amples informations sur la propriété.

La table Autres propriétés contient également les options ci-après.

Ajouter une propriété

Bouton permettant d'ajouter une propriété. Un clic sur ce bouton permet d'ajouter une ligne à la table Autres propriétés.

Supprimer des propriétés

Bouton permettant de supprimer les propriétés sélectionnées. Toute propriété supprimée revient à sa valeur par défaut ou, si aucune valeur par défaut n'est configurée, n'est pas définie.
