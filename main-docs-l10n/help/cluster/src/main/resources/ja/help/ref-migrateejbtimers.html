EJBタイマーの移行

「EJBタイマーの移行」ページには、次のオプションが含まれます。

ソース

クラスタ内の停止しているGlassFish Serverインスタンスのドロップダウン・リスト。EJBタイマーは、実行中ではないインスタンスからのみ移行できます。

宛先

クラスタ内の実行中のGlassFish Serverインスタンスのドロップダウン・リスト。EJBタイマーは、実行中のインスタンスにしか移行できません。

注意:

クラスタ内のインスタンスがすべて実行されていない場合は、EJBタイマーを移行できる実行中のインスタンスがないという警告のみが表示されます。

「EJBタイマーの移行」ページには、EJBタイマーの表が含まれ、次の情報が表示されます。

サーバー名

クラスタ内のGlassFish Serverインスタンスのリスト。

タイマー・カウント

各インスタンスの永続タイマーの数。
