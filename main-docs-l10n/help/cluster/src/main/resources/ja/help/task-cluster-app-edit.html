クラスタにデプロイされるアプリケーションを編集するには

  1. ナビゲーション・ツリーで、「クラスタ」ノードを展開します。

  2. 展開した「クラスタ」ノードで、クラスタのノードを選択します。

    クラスタの「一般情報」ページが開きます。

  3. 「一般情報」ページで、「アプリケーション」タブをクリックします。

    クラスタの「アプリケーション」ページが開きます。

  4. 「アプリケーション」ページの「デプロイされたアプリケーション」の表で、アプリケーション名をクリックします。

    「アプリケーションを編集」ページが開きます。

  5. 「アプリケーションを編集」ページで、アプリケーションに関する必要な情報を入力し、変更を保存します。

    詳細は、「アプリケーションの編集」オンライン・ヘルプを参照してください。
