スタンドアロンGlassFish Serverインスタンスのリソースを有効または無効にするには

  1. ナビゲーション・ツリーで、「スタンドアロン・インスタンス」ノードを選択します。

    「スタンドアロン・サーバー・インスタンス」ページが開きます。

  2. 「スタンドアロン・サーバー・インスタンス」ページで、インスタンスを選択します。

    インスタンスの「一般情報」ページが開きます。

  3. 「一般情報」ページで、「リソース」タブをクリックします。

    インスタンスの「リソース」ページが開きます。

  4. リソースの横にあるチェック・ボックスを選択します。

  5. 該当するボタンをクリックします。

    • リソースを有効にする場合は、「有効」を選択します。

    • リソースを無効にする場合は、「無効」を選択します。

      リソースを無効にしても、リソースは除去されません。
