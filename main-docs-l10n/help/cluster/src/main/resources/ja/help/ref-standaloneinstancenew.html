新規スタンドアロン・サーバー・インスタンス

「新規スタンドアロン・サーバー・インスタンス」ページを使用して、スタンドアロンGlassFish Serverインスタンスを作成します。

「新規スタンドアロン・サーバー・インスタンス」ページには、次のオプションが含まれます。

インスタンス名

作成されるインスタンスの名前。

名前は次の要件を満たす必要があります:

  • 名前にはASCII文字のみを使用できます。

  • 名前は英数字またはアンダースコアで始める必要があります。

  • 名前に含めることができるのは次の文字のみです:

    • 小文字

    • 大文字

    • 数値

    • ハイフン

    • ピリオド

    • アンダースコア

  • 名前はドメイン内で一意にします。また、別のGlassFish Serverインスタンス、クラスタ、名前付き構成、またはノードの名前にすることはできません。

  • ドメイン、サーバー、またはGlassFish Serverで予約されている他のキーワードを名前に使用しないでください。

ノード

インスタンスを配置できるホストを定義する既存のノードのドロップダウン・リスト。リストからいずれかのノードを選択する必要があります。ドメイン管理サーバー(DAS)が実行されているホストにインスタンスを作成する場合は、定義済ノードlocalhost-domainを選択します。

構成

インスタンスで使用できる既存の名前付き構成のドロップダウン・リスト。リストからいずれかの構成を選択する必要があります。インスタンスでは、インスタンスの作成時に選択した構成が使用されます。

選択している構成のコピーを作成する

このオプションを選択すると、インスタンスの作成時に選択した構成がコピーされます。構成のコピーには、instance-name-configという名前が割り当てられます。instance-nameは、「インスタンス名」フィールドに入力されているインスタンスの名前です。

「構成」ドロップダウン・リストから「default-config」を選択した場合は、このオプションを選択する必要があります。default-config構成は、コピーのみが可能で、参照することはできません。

選択している構成を参照する

このオプションを選択すると、指定した既存の名前付き構成が使用されます。

「構成」ドロップダウン・リストから「default-config」を選択した場合は、このオプションを選択しないでください。default-config構成は、コピーのみが可能で、参照することはできません。
