スタンドアロンGlassFish Serverインスタンスの仮想サーバーのモニター・データを表示するには

開始前

モニター・データを表示する前に、モニター・サービスを構成する必要があります。

  1. ナビゲーション・ツリーで、「スタンドアロン・インスタンス」ノードを選択します。

    「スタンドアロン・サーバー・インスタンス」ページが開きます。

  2. 仮想サーバー・モニター・データを表示するインスタンスを選択します。

    インスタンスの「一般情報」ページが開きます。

  3. 「一般情報」ページで、「モニター」タブをクリックします。

    「アプリケーションのモニター」ページが開きます。

  4. 「サーバー」タブをクリックします。

    「サーバーのモニター」ページが開きます。

  5. 「表示」ドロップダウン・リストから、モニター・データを表示するコンポーネントを選択します。

  6. 表示されるモニター・データを更新するには、「リフレッシュ」をクリックします。
