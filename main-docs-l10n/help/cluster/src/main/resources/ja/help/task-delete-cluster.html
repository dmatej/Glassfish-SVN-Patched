クラスタを削除するには

クラスタを削除すると、そのクラスタとクラスタ内のすべてのGlassFish Serverインスタンスが削除されます。クラスタを削除する前に、クラスタ内のインスタンスを削除する必要はありません。

クラスタが使用する構成を共有しない場合、構成も削除されます。

開始前

削除するクラスタが停止していることを確認します。詳細は、クラスタを停止するにはを参照してください。

  1. ナビゲーション・ツリーで、「クラスタ」ノードを選択します。

    「クラスタ」ページが開きます。

  2. 「クラスタ」ページで、クラスタ名の横にあるチェック・ボックスを選択します。

  3. 「削除」をクリックします。

    クラスタおよびクラスタ内のすべてのインスタンスを削除するかどうかを確認するメッセージが表示されます。

  4. クラスタおよびクラスタ内のすべてのインスタンスを削除することを確認します。

    クラスタは削除されて、「クラスタ」ページのクラスタの表からも除去されます。
