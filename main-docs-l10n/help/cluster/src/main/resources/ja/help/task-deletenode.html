ノードを削除するには

ノードを削除すると、ノードはドメイン管理サーバー(DAS)の構成から除去されます。ノードのディレクトリとファイルは、ノードに存在する最後のGlassFish Serverインスタンスが削除されるときに削除されます。

ノードの削除時にノード・ホストからGlassFish Serverソフトウェアをアンインストールするように選択することもできます。

開始前

GlassFish Serverインスタンスが削除されるノードに存在しないことを確認します。

  1. ナビゲーション・ツリーで、「ノード」ノードを選択します。

    「ノード」ページが開きます。

  2. 削除するノードごとに、「ノード」の表のノードの横にあるチェック・ボックスを選択します。

    注意:

    定義済ノードlocalhost-domainは削除できません。

  3. ボタンをクリックし、ノードを削除、またはノードを削除してノード・ホストからGlassFish Serverソフトウェアをアンインストールします。

    • GlassFish Serverソフトウェアをアンインストールせずにノードを削除するには、「削除」をクリックします。

      ノードを削除するかどうかを確認するメッセージが表示されます。

    • ノードを削除してノード・ホストからGlassFish Serverソフトウェアをアンインストールするには、「削除とアンインストール」をクリックします。

      ノードを削除してノード・ホストからGlassFish Serverソフトウェアをアンインストールするかどうかを確認するメッセージが表示されます。

      注意:

      「CONFIG」タイプのノードが表すホストからGlassFish Serverソフトウェアがアンインストールされることはありません。ノードのみが削除されます。

  4. ボタンをクリックし、ノードを削除、またはノードを削除してノード・ホストからGlassFish Serverソフトウェアをアンインストールすることを確認します。

    ノードが削除され、「ノード」ページのノードの表からも除去されます。「削除とアンインストール」をクリックした場合は、GlassFish Serverソフトウェアもノード・ホストからアンインストールされます。
