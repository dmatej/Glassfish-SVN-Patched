クラスタからアプリケーションを除去するには

アプリケーションの除去では、現在のターゲットからのみアプリケーションを除去します。ターゲットからアプリケーションを除去しても、そのアプリケーションはGlassFish Serverにデプロイされているすべてのアプリケーションが表示される「アプリケーション」ページに引き続きリストされます。デプロイが行われた他のターゲットのアプリケーションもデプロイされたままとなります。

  1. ナビゲーション・ツリーで、「クラスタ」ノードを展開します。

  2. 展開した「クラスタ」ノードで、クラスタのノードを選択します。

    クラスタの「一般情報」ページが開きます。

  3. 「一般情報」ページで、「アプリケーション」タブをクリックします。

    クラスタの「アプリケーション」ページが開きます。

  4. 削除するアプリケーションごとに、「デプロイされたアプリケーション」の表のアプリケーションの横にあるチェック・ボックスを選択します。

  5. 「除去」をクリックします。

    アプリケーションを除去するかどうかを確認するメッセージが表示されます。

  6. アプリケーションを除去することを確認します。

    ページのデプロイされたアプリケーションの表からアプリケーションが除去されます。
