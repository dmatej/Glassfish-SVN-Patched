スタンドアロンGlassFish Serverインスタンスを削除するには

  1. ナビゲーション・ツリーで、「スタンドアロン・インスタンス」ノードを選択します。

    「スタンドアロン・サーバー・インスタンス」ページが開きます。

  2. 削除するインスタンスごとに、「サーバー・インスタンス」の表のインスタンスの横にあるチェック・ボックスを選択します。

  3. 「削除」をクリックします。

    インスタンスを削除するかどうかを確認するメッセージが表示されます。

  4. インスタンスを削除することを確認します。

    インスタンスが削除され、ページのインスタンスの表からも除去されます。
