So werden detaillierte Informationen über eine Komponente angezeigt

Updatetool stellt detaillierte Informationen zu installierten Komponenten, verfügbaren Updates für installierte Komponenten und Add-on-Komponenten bereit.

  1. Wählen Sie in der Navigationsbaumstruktur den Knoten "Updatetool".

    Die Seite "Verfügbare Add-ons" wird angezeigt.

  2. Klicken Sie auf die Registerkarte für die Seite, auf der die Komponente aufgeführt ist.

    Die Komponente wird auf einer der folgenden Seiten aufgeführt:

    • Verfügbare Add-ons

    • Verfügbare Updates

    • Installierte Komponenten

  3. Wählen Sie in der Tabelle der Komponenten die Komponente.

    Die Seite Komponentendetails für die ausgewählte Komponente wird angezeigt.
