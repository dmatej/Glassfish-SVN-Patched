Überwachungsservice

Verwenden Sie die Seite "Überwachungsservice" um Überwachungsoptionen für einzelne Serverkomponenten oder Services zu konfigurieren.

Sie müssen eine Anwendung, eine Komponente oder einen Service bereitstellen, bevor Sie die Überwachung hierfür konfigurieren können.

Die Seite "Überwachungsservice" umfasst die folgenden Optionen.

Name der Konfiguration

Der Name der Konfiguration, für die die Einstellungen auf dieser Seite angewendet werden. Dieses Feld ist schreibgeschützt.

Überwachungsservice

Wenn das Kontrollkästchen "Überwachungsservice" ausgewählt ist, ist die Überwachung für GlassFish Server aktiviert. Diese Option ist standardmäßig aktiviert.

Überwachungs-MBeans

Wenn das Kontrollkästchen "Überwachungs-MBeans" aktiviert ist, werden alle für die Überwachung benötigten MBeans auf GlassFish Server bereitgestellt. Diese Option ist standardmäßig aktiviert.

Wenn das Kontrollkästchen nicht aktiviert ist, können Sie keine Überwachungsdaten in der Administrationskonsole anzeigen, auch dann nicht, wenn das Kontrollkästchen "Überwachungsservice" aktiviert ist. Sie müssen die Überwachungsdaten stattdessen über die Befehlszeile aufrufen. Wenn Sie die Auswahl des Kontrollkästchens aufheben, wird in GlassFish Server weniger Speicher belegt.

DTrace

Wenn das Kontrollkästchen "DTrace aktiviert" gewählt ist und Sie Oracle Enterprise Manager DTrace Monitoring verwenden, können Sie die DTrace-Funktionalität von Solaris verwenden. Diese Option ist standardmäßig deaktiviert.

Folgende Informationen werden für jede verfügbare Komponente zur Verfügung gestellt.

Komponente

Der Name der GlassFish Server-Komponente.

Überwachungsebene

Die Ebene der Überwachungseinstellung für die Komponente. Für jede Komponente sind die Ebenen LOW (Niedrig), HIGH (Hoch) und OFF (Deaktiviert) verfügbar. Der Standardwert ist OFF (Deaktiviert). In dieser Version besteht kein Unterschied zwischen den Überwachungsebenen der Optionen "NIEDRIG" und "HOCH".

Die Tabelle "Komponentenebeneneinstellungen" enthält darüber hinaus die folgenden Optionen.

Ebene ändern

Schaltfläche zum Ändern der Überwachungsebene für mindestens eine ausgewählte Komponente.

Ebene

Dropdown-Liste, aus der Sie eine Überwachungsebene auswählen können, die auf alle ausgewählten Komponenten angewendet werden soll.

Verwandte Aufgaben

  • So wird der Überwachungsservice konfiguriert

  • So werden Anwendungsüberwachungsdaten angezeigt

  • So werden Serverüberwachungsdaten angezeigt

  • So werden Ressourcenüberwachungsdaten angezeigt

Zugehörige asadmin-Befehle

  • enable-monitoring(1)

  • disable-monitoring(1)
