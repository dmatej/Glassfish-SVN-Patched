Lizenzvereinbarung

Die Lizenzvereinbarungsseite zeigt die Lizenzvereinbarungen aller Komponenten an, die zur Installation oder zum Update ausgewählt sind. Die Lizenzvereinbarung für eine Komponente wird vom Autor der Komponente mit der Komponente geliefert. Verschiedene Komponenten im Netzwerk-Repository könnten unter unterschiedlichen Lizenzen veröffentlicht werden; Beispiel:

  • Common Development and Distribution License (CDDL) (http://www.sun.com/cddl/)

  • GNU General Public License (GPL) (http://www.gnu.org/copyleft/gpl.html)

  • Apache License (http://www.apache.org/licenses/)

  • wxWindows Library License (http://www.opensource.org/licenses/wxwindows.php)

Die Seite "Lizenzvereinbarung" enthält die folgenden Optionen.

Accept (Akzeptieren)

Diese Schaltfläche akzeptiert die Bestimmungen der Lizenzen aller ausgewählten Komponenten und setzt die Installation oder das Update fort.

Decline (Ablehnen)

Diese Schaltfläche lehnt die Bestimmungen der Lizenzen aller ausgewählten Komponenten ab und wechselt zurück zu der Seite, auf der Sie die Komponenten ausgewählt haben.

Verwandte Aufgaben

  • So werden verfügbare Updates für Komponenten angezeigt

  • So werden Add-on-Komponenten installiert
