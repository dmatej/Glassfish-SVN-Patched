Admin-Benutzer

Auf der Seite "Admin-Benutzer" können Sie Benutzer von admin-realm verwalten. admin-realm ist eine File Realm. Benutzer und Gruppen in der Datei-Realm sind in der Schlüsseldatei aufgeführt, deren Verzeichnis durch die Eigenschaft file angegeben wird.

Nach der Installation von GlassFish Server ist der während der Installation eingegebene Administrator zunächst der einzige Benutzer. Dieser Benutzer ist standardmäßig Mitglied der Gruppe asadmin in der Realm admin-realm und damit berechtigt, Änderungen an GlassFish Server vorzunehmen. Benutzer, die dieser Gruppe zugewiesen sind, verfügen über Administratorberechtigungen und können damit auf das asadmin-Tool und die Administrationskonsole zugreifen.

Die Seite "Admin-Benutzer" enthält eine Liste mit Benutzern für die ausgewählte Realm. Für jeden Benutzer werden die folgenden Informationen zur Verfügung gestellt:

Benutzer-ID

Der Name des Benutzers.

Gruppenliste

Die Gruppen, denen der Benutzer angehört.

Die Tabelle "Admin-Benutzer" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen eines neuen Benutzers.

Löschen

Schaltfläche zum Löschen von mindestens einem ausgewählten Benutzer.

Verwandte Aufgaben

  • So wird ein Benutzer der Admin-Realm hinzugefügt

  • So werden Informationen für den Admin-Realm-Benutzer bearbeitet

  • So wird ein Benutzer aus der Admin-Realm gelöscht
