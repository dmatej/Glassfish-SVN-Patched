So wird eine bereitgestellte Anwendung erneut geladen

Durch erneutes Laden einer Anwendung wird die Anwendung lediglich neu gestartet und nicht erneut bereitgestellt, wobei die Einstellungen und anderen Optionen des ursprünglichen Anwendungs-Deployments beibehalten werden.

  1. Wählen Sie in der Navigationsbaumstruktur den Anwendungsknoten.

    Die Seite "Anwendungen" wird geöffnet.

  2. Klicken Sie in der Tabelle "Bereitgestellte Anwendungen" in der Zeile für die neu bereitzustellende Anwendung auf "Neu laden".

    Der Anwendungsname kann eine optionale Versions-ID enthalten, die dem Namen folgt und von diesem durch einen Doppelpunkt (:) getrennt ist. Weitere Informationen zu Modul- und Anwendungsversionen finden Sie unter Module and Application Versions im Oracle GlassFish Server Application Deployment Guide.

    Die Seite "Anwendungen" wird geöffnet und zeigt eine Meldung an, die angibt, ob das erneute Laden erfolgreich war.
