Neuer File Realm-Benutzer

Über die Seite "Neuer File Realm-Benutzer" können Sie einem neuen Benutzer Zugriff auf die aktuelle Realm gewähren.

Hinweis:

Wenn die sichere Administration aktiviert ist, können Sie keinen admin-realm-Benutzer mit einem leeren Kennwort erstellen.

Die Seite "Neuer File Realm-Benutzer" umfasst die folgenden Optionen.

Name der Konfiguration

Der Name der Konfiguration, für die die Einstellungen auf dieser Seite angewendet werden. Dieses Feld ist schreibgeschützt.

Realm-Name

Der Name der aktuell ausgewählten Realm. Dies ist ein schreibgeschütztes Feld.

Benutzer-ID

Die Benutzer-ID für den neuen Benutzeraccount.

Gruppenliste

Gruppen, zu denen der neue Benutzer gehören wird. Trennen Sie mehrere Gruppennamen durch Komma.

Für Benutzer von admin-realm ist dies ein schreibgeschütztes Feld, und die Gruppe ist asadmin.

Neues Kennwort

Kennwort für den neuen Benutzer.

Neues Kennwort bestätigen

Kennwort für den neuen Benutzer.

Verwandte Aufgaben

  • So wird ein Benutzer der File Realm hinzugefügt

  • So werden Informationen für den File Realm-Benutzer bearbeitet

  • So wird ein Benutzer aus der File Realm gelöscht

Zugehörige asadmin-Befehle
