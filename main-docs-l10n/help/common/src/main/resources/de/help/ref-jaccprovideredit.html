JACC-Provider bearbeiten

Verwenden Sie die Seite "JACC-Provider bearbeiten", um einen JACC-Provider für die ausgewählte GlassFish Server-Instanz oder das -Cluster zu konfigurieren.

Die Seite "JACC-Provider bearbeiten" enthält die folgenden Optionen.

Name der Konfiguration

Der Name der Konfiguration, für die die Einstellungen auf dieser Seite angewendet werden. Dieses Feld ist schreibgeschützt.

Name

Der Name des JACC-Providers. Dies ist ein schreibgeschütztes Feld. Sie können den Namen nur angeben, wenn Sie einen neuen JACC-Provider erstellen.

Policy-Konfiguration

Der vollständig angegebene Name für den javax.security.jacc.PolicyConfigurationFactory.provider, der die providerspezifische javax.security.jacc.PolicyConfigurationFactory implementiert.

Policy-Provider

Der vollständig angegebene Klassenname für den javax.security.jacc.policy.provider, der die java.security.Policy implementiert.

Weitere Eigenschaften

Zusätzliche Eigenschaften für den JACC-Provider. Eine Beschreibung der zusätzlichen Eigenschaften, die für JACC-Provider verfügbar sind, finden Sie unter Für JACC-Provider spezifische Eigenschaften

Verwandte Aufgaben

  • So wird ein JACC-Provider erstellt

  • So wird ein JACC-Provider bearbeitet

  • So wird ein JACC-Provider gelöscht
