Auditmodul bearbeiten

Auf der Seite "Auditmodul bearbeiten" können Sie ein Auditmodul ändern.

Die Seite "Auditmodul bearbeiten" umfasst die folgenden Optionen.

Name der Konfiguration

Der Name der Konfiguration, für die die Einstellungen auf dieser Seite angewendet werden. Dieses Feld ist schreibgeschützt.

Name

Der Name des Auditmoduls. Dies ist ein schreibgeschütztes Feld. Sie können den Namen nur festlegen, wenn Sie ein neues Auditmodul erstellen.

Klassenname

Der vollständig angegebene Name der Klasse, die dieses Modul implementiert. Der Klassenname für das Standardauditmodul lautet com.sun.enterprise.security.Audit.

Weitere Eigenschaften

Weitere Eigenschaften für das Auditmodul. Eine Beschreibung der zusätzlichen Eigenschaften, die für Auditmodule verfügbar sind, finden Sie unter Für Auditmodule spezifische Eigenschaften

Verwandte Aufgaben

  • So wird ein Auditmodul erstellt

  • So wird ein Auditmodul bearbeitet

  • So wird ein Auditmodul gelöscht

  • So wird das Audit-Logging aktiviert oder deaktiviert

  • So wird das aktive Auditmodul festgelegt

  • So wird das Standardauditmodul verwendet

Zugehörige asadmin-Befehle

  • create-audit-module(1)

  • list-audit-modules(1)

  • delete-audit-module(1)
