Verfügbare Updates

Die Seite "Verfügbare Updates" liefert Informationen zu neueren Versionen von Komponenten des Installationsimages, die installiert sind.

Halten Sie zum Installieren der verfügbaren Aktualisierungen zunächst die GlassFish Server-Domain an. Verwenden Sie dann einen der folgenden Befehle:

  • glassfish3/bin/updatetool, zum Starten des grafischen Updatetoolutilitys

  • glassfish3/bin/pkg, eine Befehlszeilenversion des Updatetools

Weitere Informationen zur Verwendung des Befehls pkg finden Sie in Chapter 11, Extending and Updating GlassFish Server, im Oracle GlassFish Server Administration Guide. Das Updatetool verfügt über eine eigene Onlinehilfe.

Starten Sie den Server neu, wenn die Installation der Aktualisierungen abgeschlossen ist.

Die Seite "Verfügbare Updates" umfasst die folgenden Optionen.

Installationspfad

Der Dateisystempfad des Root-Verzeichnisses oder des Verzeichnisses auf der obersten Ebene des Installationsimages.

Verfügbare Updates

Für jede Komponente, für die eine aktuellere Version verfügbar ist, stellt diese Tabelle die folgenden Informationen bereit.

Komponente

Der Name der Komponente.

Kategorie

Die Softwarekategorie, mit der die Komponente verbunden ist, z.B. Application Server, Systemtools und Datenbanken. Die Kategorie wird der Komponente vom Autor der Komponente bei deren Erstellung zugewiesen.

Installierte Version

Die Versionsnummer des Packages, in dem die aktuelle Komponente bereitgestellt wird.

Neue Version

Die Versionsnummer der verfügbaren Aktualisierung.

Ändern

Unterschied in der Größe zwischen der installierten Version und der verfügbaren Aktualisierung.

Quelle

Der Name des Netzwerk-Repository, aus dem die Komponenten abgerufen werden.

Verwandte Aufgaben
