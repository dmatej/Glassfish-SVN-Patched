GlassFish Server verfügt über drei vorkonfigurierte Realms: file, certificate und admin-realm. Darüber hinaus können die Realms ldap, JDBC, solaris, PAM sowie benutzerdefinierte Realms erstellt werden. Im Allgemeinen ist auf jedem Server eine Realm jeden Typs vorhanden, GlassFish Server verfügt jedoch über zwei "file" Realms: file und admin-realm. Dies sind zwei Realms desselben Typs, die für unterschiedliche Zwecke verwendet werden. Darüber hinaus kann für jeden virtuellen Server auf dem System eine separate Zertifikatdatenbank bereitgestellt werden.

  1. Blenden Sie unter dem Knoten "Konfigurationen" die Serverinstanz oder die Clusterkonfiguration ein, für die Sie Realms konfigurieren möchten.

  2. Blenden Sie den Knoten "Sicherheit" ein.

  3. Blenden Sie den Knoten "Realms" ein.

    Die Seite "Realms" wird geöffnet.

  4. Klicken Sie auf der Seite "Realms" auf "Neu".

    Die Seite "Neue Realm" wird geöffnet.

  5. Geben Sie im Feld "Name" einen Namen für die Realm ein.

  6. Sie haben folgende Möglichkeiten, den Klassennamen der Realm anzugeben:

    • Wählen Sie aus der Dropdown-Liste "Klassenname" den Klassennamen für die zu erstellende Realm.

      Zulässige Optionen sind in der folgenden Tabelle aufgeführt.

      Realm-Name Klassenname
      file com.sun.enterprise.security.auth.realm.file.FileRealm
      certificate com.sun.enterprise.security.auth.realm.certificate.CertificateRealm
      ldap com.sun.enterprise.security.auth.realm.ldap.LDAPRealm
      JDBC com.sun.enterprise.security.auth.realm.jdbc.JDBCRealm
      solaris com.sun.enterprise.security.auth.realm.solaris.SolarisRealm
      PAM com.sun.enterprise.security.auth.realm.solaris.PamRealm

    • Wählen Sie das untere Optionsfeld, und geben Sie den Klassennamen für die zu erstellende benutzerdefinierte Realm ein.

  7. Geben Sie im Abschnitt "Weitere Eigenschaften" weitere Eigenschaften an.

    Um eine Eigenschaft hinzuzufügen, klicken Sie auf die Schaltfläche "Eigenschaft hinzufügen". Geben Sie in die daraufhin angezeigte leere Zeile den Eigenschaftsnamen in das Feld "Name" und den Eigenschaftswert in das Feld "Wert" ein.

    Gültige Eigenschaften sind abhängig vom Typ der Realm.

    • Für die FileRealm-Klasse spezifische Eigenschaften

    • Für die CertificateRealm-Klasse spezifische Eigenschaften

    • Für die JDBCRealm-Klasse spezifische Eigenschaften

    • Für die LDAPRealm-Klasse spezifische Eigenschaften

    • Für die SolarisRealm-Klasse spezifische Eigenschaften

    • Für die PamRealm-Klasse spezifische Eigenschaften

  8. Klicken Sie auf "OK".
