So wird das Kennwort für einen Benutzer in der Admin-Realm geändert

Unter So wird das Administratorkennwort geändert wird beschrieben, wie das Kennwort für den aktuellen Administratoraccount geändert wird. Wenn admin-realm mehr als einen Benutzer hat, können Sie auch das Kennwort für andere Benutzer ändern.

Durch eine Verschlüsselung des Administratorkennworts wird die Sicherheit Ihrer Anwendung erhöht. Weitere Informationen über Sicherheitsprozeduren finden Sie in Chapter 1, Administering System Security, im Oracle GlassFish Server Security Guide.

  1. Blenden Sie unter dem Knoten "Konfigurationen" die Serverinstanz oder die Clusterkonfiguration ein, für die Sie ein Administrationskennwort verwalten möchten.

  2. Blenden Sie den Knoten "Sicherheit" ein.

  3. Blenden Sie den Knoten "Realms" ein.

  4. Wählen Sie den Knoten admin-realm.

    Die Seite "Realm bearbeiten" wird geöffnet.

  5. Klicken Sie auf der Seite "Realm bearbeiten" auf die Schaltfläche "Benutzer verwalten".

    Die Seite "Admin-Benutzer" wird geöffnet.

  6. Klicken Sie auf der Seite "Admin-Benutzer" auf den Benutzer, für den Sie ein Kennwort hinzufügen oder ändern möchten.

    Die Seite "File Realm-Benutzer bearbeiten" wird geöffnet.

  7. Geben Sie in die Felder "Neues Kennwort" und "Neues Kennwort bestätigen" das neue Kennwort ein.

  8. Klicken Sie auf "Speichern".
