도메인 로그

[도메인 로그] 페이지에서는 지정된 GlassFish Server 인스턴스 또는 클러스터에 대한 로그 파일을 수집 및 다운로드하고 이러한 로그를 하나의 ZIP 아카이브로 컴파일할 수 있습니다.

주:

클러스터가 지정되어 있으면 ZIP 아카이브에도 DAS(도메인 관리 서버)에 대한 로그 파일이 포함됩니다.

[도메인 로그] 페이지에는 다음 옵션이 포함되어 있습니다.

클러스터

클러스터에 대한 드롭다운 목록입니다(클러스터가 있는 경우).

인스턴스

서버 인스턴스에 대한 드롭다운 목록입니다.

로그 수집

로그를 log-files-name.zip 파일에 수집하는 단추입니다. 여기서 name은 클러스터 또는 인스턴스의 이름입니다.

브라우저의 다운로드 방식을 사용하여 log-files-name.zip 파일이 다운로드됩니다.

관련 작업

  • 로그를 수집 및 다운로드하는 방법

  • DAS에 대한 로그 파일을 보는 방법

동일한 asadmin 하위 명령

collect-log-files(1)
