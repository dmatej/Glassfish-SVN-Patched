보안 관리를 사용 안함으로 설정하는 방법

  1. 탐색 트리에서 서버(관리 서버) 노드를 선택합니다.

    [일반 정보] 페이지가 열립니다.

  2. [일반 정보] 페이지에서 [보안 관리] 단추를 누릅니다.

    [보안 관리] 페이지가 열립니다.

  3. [보안 관리 사용 안함]을 누릅니다.

    보안 관리를 사용 안함으로 설정하면 DAS가 즉시 재시작됨을 알리는 메시지가 나타납니다.

  4. [확인]을 누릅니다.

    보안 관리가 사용 안함으로 설정되고 서버 재시작 페이지가 열립니다.
