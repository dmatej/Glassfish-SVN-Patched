리소스 모니터링 데이터를 보는 방법

시작하기 전에

모니터링 데이터를 보려면 먼저 모니터링을 구성해야 합니다. 자세한 내용은 모니터링 서비스를 구성하는 방법을 참조하십시오.

  1. 탐색 트리에서 [모니터링 데이터] 노드를 선택합니다.

    [모니터링] 페이지가 열립니다.

  2. [모니터링 데이터 보기] 열에서 모니터링할 서버 인스턴스에 해당하는 [리소스] 링크를 누릅니다.

    [리소스 모니터링] 페이지가 열립니다.

  3. [접속 풀] 드롭다운 목록에서 모니터링 데이터를 표시할 접속 풀을 선택합니다.

  4. [새로 고침] 단추를 눌러 표시된 모니터링 데이터를 업데이트합니다.

대체 단계:

  1. 탐색 트리에서 서버(관리 서버) 노드를 선택합니다.

    [일반 정보] 페이지가 열립니다.

  2. [일반 정보] 페이지에서 [모니터] 탭을 누릅니다.

    [응용 프로그램 모니터링] 페이지가 열립니다.

  3. [리소스] 탭을 누릅니다.

    [리소스 모니터링] 페이지가 열립니다.

  4. [접속 풀] 드롭다운 목록에서 모니터링 데이터를 표시할 접속 풀을 선택합니다.

  5. [새로 고침] 단추를 눌러 표시된 모니터링 데이터를 업데이트합니다.

관련 항목

  • DAS에 대한 일반 정보를 보는 방법

  • DAS에 대한 로그 파일을 보는 방법

  • 표시된 로그 레코드를 필터링하는 방법

  • 응용 프로그램 구성을 설정하는 방법

  • 도메인 속성을 설정하는 방법

  • 관리자 비밀번호를 변경하는 방법

  • 시스템 속성을 관리하는 방법

  • 응용 프로그램 모니터링 데이터를 보는 방법

  • 서버 모니터링 데이터를 보는 방법
