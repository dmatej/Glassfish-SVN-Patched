영역을 생성하는 방법

GlassFish Server은(는) file, certificate 및 admin-realm의 세 가지 영역으로 미리 구성됩니다. ldap, JDBC, solaris, PAM 및 사용자 정의 영역을 생성할 수도 있습니다. 일반적으로 한 서버에 각 유형당 하나의 영역이 있지만 GlassFish Server에는 file 및 admin-realm의 두 파일 영역이 있습니다. 이 두 영역의 유형은 동일하지만 사용 목적은 다릅니다. 사용자 시스템의 각 가상 서버가 서로 다른 인증서 데이터베이스를 사용하도록 지정할 수도 있습니다.

  1. [구성] 노드에서 영역을 구성할 서버 인스턴스나 클러스터 구성을 확장합니다.

  2. [보안] 노드를 확장합니다.

  3. [영역] 노드를 확장합니다.

    [영역] 페이지가 열립니다.

  4. [영역] 페이지에서 [새로 만들기]를 누릅니다.

    [새 영역] 페이지가 열립니다.

  5. [이름] 필드에 영역의 이름을 입력합니다.

  6. 영역에 클래스 이름을 지정하려면 다음 중 하나를 수행합니다.

    • [클래스 이름] 드롭다운 목록에서 생성할 영역의 클래스 이름을 선택합니다.

      다음 표에 적합한 선택 항목이 나열되어 있습니다.

      영역 이름 클래스 이름
      file com.sun.enterprise.security.auth.realm.file.FileRealm
      certificate com.sun.enterprise.security.auth.realm.certificate.CertificateRealm
      ldap com.sun.enterprise.security.auth.realm.ldap.LDAPRealm
      JDBC com.sun.enterprise.security.auth.realm.jdbc.JDBCRealm
      solaris com.sun.enterprise.security.auth.realm.solaris.SolarisRealm
      PAM com.sun.enterprise.security.auth.realm.solaris.PamRealm

    • 아래쪽 라디오 단추를 선택하고 생성하려는 사용자 정의 영역의 클래스 이름을 입력합니다.

  7. [추가 속성] 섹션에서 추가 속성을 지정합니다.

    속성을 추가하려면 [속성 추가] 단추를 누릅니다. 빈 행이 표시되면 [이름] 필드에 속성 이름을 입력하고 [값] 필드에 속성값을 입력합니다.

    적합한 속성은 영역 유형에 따라 다릅니다.

    • FileRealm 클래스 관련 속성

    • CertificateRealm 클래스 관련 속성

    • JDBCRealm 클래스 관련 속성

    • LDAPRealm 클래스 관련 속성

    • SolarisRealm 클래스 관련 속성

    • PamRealm 클래스 관련 속성

  8. [확인]을 누릅니다.
