파일 영역에서 사용자를 삭제하는 방법

  1. [구성] 노드에서 관리 영역 사용자를 관리할 서버 인스턴스나 클러스터 구성을 확장합니다.

  2. [보안] 노드를 확장합니다.

  3. [영역] 노드를 확장합니다.

  4. file 노드를 선택합니다.

    [영역 편집] 페이지가 열립니다.

  5. [영역 편집] 페이지에서 [사용자 관리] 단추를 누릅니다.

    [파일 사용자] 페이지가 열립니다.

  6. [파일 사용자] 페이지에서 삭제할 사용자 이름의 왼쪽에 있는 체크 박스를 선택합니다.

  7. [삭제]를 누릅니다.

    사용자를 삭제해야 하는지 확인하는 대화상자가 표시됩니다.

  8. 대화상자에서 [확인]을 누릅니다.
