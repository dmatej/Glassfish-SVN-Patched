사전 정의된 시스템 속성

GlassFish Server에 대해 사전 정의된 시스템 속성은 다음과 같습니다.

com.sun.aas.installRoot

GlassFish Server가 설치되는 디렉토리입니다.

com.sun.aas.instanceRoot

GlassFish Server 도메인의 최상위 레벨 디렉토리입니다.

com.sun.aas.hostName

호스트(시스템)의 이름입니다.

com.sun.aas.javaRoot

Java 런타임의 설치 디렉토리입니다.

com.sun.aas.imqLib

Oracle Message Queue 소프트웨어의 라이브러리 디렉토리입니다.

com.sun.aas.domainName

도메인 이름입니다. 이 속성은 기본 구성에 사용되지는 않지만 구성을 사용자 정의하는 데 사용될 수 있습니다. 기본값은 domain1입니다.
