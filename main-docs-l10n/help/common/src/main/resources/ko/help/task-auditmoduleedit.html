감사 모듈을 편집하는 방법

기본적으로 감사 모듈은 사용으로 설정되어 있지 않습니다. 감사 모듈을 활성화하는 방법은 감사 로깅을 사용 또는 사용 안함으로 설정하는 방법을 참조하십시오.

  1. 탐색 트리에서 [구성] 노드를 확장합니다.

  2. 구성할 인스턴스를 선택합니다.

    • 특정 인스턴스를 구성하려면 해당 인스턴스의 구성 노드를 확장합니다. 예를 들어, 기본 인스턴스 서버에 대해 server-config 노드를 확장합니다.

    • 모든 인스턴스의 기본 설정을 구성하려면 default-config 노드를 확장합니다.

  3. [구성] 노드에서 [보안] 노드를 확장합니다.

  4. [보안] 노드 아래에서 [감사 모듈] 노드를 선택합니다.

    [감사 모듈] 페이지가 열립니다.

  5. [감사 모듈] 페이지에서 편집 중인 감사 모듈의 이름을 누릅니다.

    [감사 모듈 편집] 페이지가 열립니다.

  6. [감사 모듈 편집] 페이지에서 [클래스 이름] 필드에 새 값을 입력하여 클래스 이름을 수정합니다.

  7. [추가 속성] 섹션에서 추가 속성을 지정합니다.

    감사 모듈에 사용 가능한 추가 속성에 대한 설명은 감사 모듈 관련 속성을 참조하십시오.

    • 속성을 추가하려면 [속성 추가] 단추를 누릅니다. 빈 행이 표시되면 [이름] 필드에 속성 이름을 입력하고 [값] 필드에 속성값을 입력합니다.

    • 속성을 수정하려면 속성의 [값] 필드를 편집합니다.

    • 속성을 삭제하려면 삭제할 속성의 [이름] 필드 왼쪽에 있는 체크 박스를 선택한 다음 [속성 삭제] 단추를 누릅니다.

  8. [저장]을 누릅니다.

관련 항목

  • 감사 모듈을 생성하는 방법

  • 감사 모듈을 삭제하는 방법

  • 감사 로깅을 사용 또는 사용 안함으로 설정하는 방법

  • 활성 감사 모듈을 설정하는 방법

  • 기본 감사 모듈을 사용하는 방법
