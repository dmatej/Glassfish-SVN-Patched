구성 요소 세부 정보

[구성 요소 세부 정보] 페이지에서는 구성 요소에 대한 자세한 정보를 제공합니다.

[구성 요소 세부 정보] 페이지에는 다음 옵션이 포함되어 있습니다.

뒤로

이 단추를 누르면 세부 정보를 보고 있는 선택한 구성 요소의 페이지로 돌아갑니다.

패키지 이름

구성 요소를 제공하는 패키지의 이름입니다.

고유 식별자

저장소에 있는 경우 이 구성 요소의 전체 식별자입니다.

범주

예를 들어 Application Server, 시스템 도구 및 데이터베이스와 같이 구성 요소가 연관된 소프트웨어의 범주입니다. 구성 요소가 생성되면 범주가 구성 요소의 작성자에 의해 해당 구성 요소에 지정됩니다.

버전

구성 요소를 제공하는 패키지의 버전 번호입니다.

게시된 날짜

이 버전의 구성 요소를 저장소에 게시한 날짜 및 시간입니다. 이 날짜는 구성 요소가 설치 이미지 내에 설치된 날짜 및 시간이 아닙니다.

설치된 크기

구성 요소의 설치 시 사용하는 디스크 공간의 크기입니다.

소스

설치된 구성 요소의 경우 구성 요소를 가져온 네트워크 저장소의 이름입니다.

추가 기능 구성 요소 또는 업데이트의 경우 구성 요소를 가져온 네트워크 저장소의 이름입니다.

설명

구성 요소에 대한 개요입니다.

관련 작업

구성 요소에 대한 자세한 정보를 보는 방법
