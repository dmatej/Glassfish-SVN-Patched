응용 프로그램 대상을 보는 방법

  1. 탐색 트리에서 [응용 프로그램] 노드를 선택합니다.

    [응용 프로그램] 페이지가 열립니다.

  2. 대상을 표시할 응용 프로그램을 선택합니다.

    응용 프로그램 이름에는 선택적인 버전 식별자가 포함될 수 있으며 이 식별자는 이름 뒤에 오고 콜론(:)을 통해 이름과 구분됩니다. 모듈 및 응용 프로그램 버전에 대한 자세한 내용은 Oracle GlassFish Server Application Deployment Guide의 Module and Application Versions를 참조하십시오.

    응용 프로그램 페이지가 열립니다.

  3. [대상] 탭을 선택합니다.

    [응용 프로그램 대상] 페이지가 열리고 응용 프로그램에 대해 현재 지정된 대상이 표시됩니다.

  4. 특정 대상 인스턴스나 클러스터에서 응용 프로그램을 사용 또는 사용 안함으로 설정하려면 대상 옆에 있는 체크 박스를 누르고 [추가 작업] 드롭다운 메뉴에서 [사용] 또는 [사용 안함]을 선택합니다.

  5. 특정 대상 인스턴스나 클러스터에서 대상의 로드 균형을 조정하려면 대상 옆에 있는 체크 박스를 누르고 추가 작업 드롭다운 메뉴에서 [로드 밸런서 사용] 또는 [로드 밸런서 사용 안함]을 선택합니다.

  6. 응용 프로그램과 특정 가상 서버를 연결하려면 원하는 대상 인스턴스나 클러스터에 대해 [가상 서버 관리]를 선택합니다.

    [가상 서버 대상] 페이지가 열립니다.

    대상 가상 서버를 추가하려면 사용 가능한 대상 목록에서 해당 가상 서버를 선택하고 [추가] 단추를 사용하여 [선택한 대상] 목록으로 이동합니다.

    기본적으로 모듈이나 응용 프로그램은 server라는 기본 가상 서버와 연관됩니다. 대상을 모두 추가하려면 [모두 추가] 단추를 사용합니다. [선택한 대상] 목록에서 대상을 제거하려면 [제거] 및 [모두 제거] 단추를 사용합니다.

    [응용 프로그램 대상] 페이지로 돌아가려면 [저장]을 선택합니다.

  7. 응용 프로그램의 대상 인스턴스나 클러스터를 추가하거나 삭제하려면 [대상 관리]를 선택합니다.

    [대상 관리] 페이지가 열립니다.

    대상을 추가하려면 [사용 가능한 대상] 목록에서 해당 대상을 선택하고 [추가] 단추를 사용하여 [선택한 대상] 목록으로 이동합니다.

    대상을 모두 추가하려면 [모두 추가] 단추를 사용합니다. [선택한 대상] 목록에서 대상을 제거하려면 [제거] 및 [모두 제거] 단추를 사용합니다.

    [응용 프로그램 대상] 페이지로 돌아가려면 [저장]을 선택합니다.

  8. [저장]을 누릅니다.

    이제 응용 프로그램을 수정한 대상 목록에서 사용할 수 있습니다.
