관리자 비밀번호를 변경하는 방법

  1. 탐색 트리에서 [도메인] 노드를 선택합니다.

    [도메인 속성] 페이지가 열립니다.

  2. [관리자 비밀번호] 탭을 선택합니다.

    [새 관리자 비밀번호] 페이지가 열립니다.

    페이지에 현재 관리자 계정의 사용자 ID와 그룹 이름이 표시됩니다. 기본적으로 사용자 ID는 admin이며 그룹 이름은 asadmin입니다.

  3. [새 비밀번호] 필드에 현재 관리자 계정의 새 비밀번호를 입력합니다.

  4. [새 비밀번호 확인] 필드에 새 비밀번호를 다시 입력합니다.

  5. [저장]을 누릅니다.

  • 파일 영역에서 사용자를 삭제하는 방법

  • DAS에 대한 일반 정보를 보는 방법

  • DAS에 대한 로그 파일을 보는 방법

  • 표시된 로그 레코드를 필터링하는 방법

  • 응용 프로그램 구성을 설정하는 방법

  • 도메인 속성을 설정하는 방법

  • 시스템 속성을 관리하는 방법
