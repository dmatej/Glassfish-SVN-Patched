응용 프로그램 클라이언트 실행

[응용 프로그램 클라이언트 실행] 페이지에서는 배치된 응용 프로그램 클라이언트를 실행할 수 있습니다. 클라이언트는 Java Web Start로 실행할 수 있는 웹 클라이언트입니다.

[응용 프로그램 클라이언트 실행] 페이지에는 다음 옵션이 포함되어 있습니다.

응용 프로그램

응용 프로그램 클라이언트를 포함하는 응용 프로그램의 이름입니다.

이름에는 버전 식별자(옵션)가 포함될 수 있으며, 이 식별자는 이름 다음에 오고 콜론(:)으로 이름과 구분됩니다. 모듈 및 응용 프로그램 버전에 대한 자세한 내용은 Oracle GlassFish Server Application Deployment Guide의 Module and Application Versions를 참조하십시오.

모듈

응용 프로그램 클라이언트를 포함하는 모듈의 이름입니다.

링크

클라이언트가 배치된 URL입니다.

인수

응용 프로그램에 전달할 인수입니다.

관련 작업

  • 응용 프로그램 클라이언트를 실행하는 방법

  • 배치된 응용 프로그램 목록을 보는 방법

  • 웹 응용 프로그램을 배치하는 방법

  • 엔터프라이즈 응용 프로그램을 배치하는 방법

  • 응용 프로그램 클라이언트를 배치하는 방법

  • 커넥터 모듈을 배치하는 방법

  • EJB Jar를 배치하는 방법

  • 배치된 응용 프로그램을 편집하는 방법

  • 응용 프로그램 기술자를 보는 방법

  • 웹 서비스에 대한 끝점 정보를 보는 방법

  • 응용 프로그램을 실행하는 방법

  • 응용 프로그램 클라이언트 스텁을 다운로드하는 방법

  • 배치된 응용 프로그램을 다시 로드하는 방법

  • 응용 프로그램 범위 리소스를 확인하는 방법
