응용 프로그램을 실행하는 방법

시작하기 전에

실행할 웹 응용 프로그램이 사용으로 설정되었는지 확인합니다.

  1. 탐색 트리에서 [응용 프로그램] 노드를 선택합니다.

    [응용 프로그램] 페이지가 열립니다.

  2. [배치된 응용 프로그램] 테이블에서 실행할 웹 응용 프로그램 행에 있는 [실행]을 누릅니다.

    응용 프로그램 이름에는 선택적인 버전 식별자가 포함될 수 있으며 이 식별자는 이름 뒤에 오고 콜론(:)을 통해 이름과 구분됩니다. 모듈 및 응용 프로그램 버전에 대한 자세한 내용은 Oracle GlassFish Server Application Deployment Guide의 Module and Application Versions를 참조하십시오.

    응용 프로그램이 새 브라우저 창에서 열립니다.

관련 항목

  • 배치된 응용 프로그램 목록을 보는 방법

  • 웹 응용 프로그램을 배치하는 방법

  • 엔터프라이즈 응용 프로그램을 배치하는 방법

  • 응용 프로그램 클라이언트를 배치하는 방법

  • 커넥터 모듈을 배치하는 방법

  • EJB Jar를 배치하는 방법

  • 배치된 응용 프로그램을 편집하는 방법

  • 응용 프로그램 기술자를 보는 방법

  • 웹 서비스에 대한 끝점 정보를 보는 방법

  • 응용 프로그램 클라이언트를 실행하는 방법

  • 응용 프로그램 클라이언트 스텁을 다운로드하는 방법

  • 배치된 응용 프로그램을 재배치하는 방법

  • 배치된 응용 프로그램을 다시 로드하는 방법

  • 응용 프로그램 범위 리소스를 확인하는 방법

  • 응용 프로그램 범위 리소스를 수정하는 방법
