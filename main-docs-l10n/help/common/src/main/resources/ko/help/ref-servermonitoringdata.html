서버 모니터링

[서버 모니터링] 페이지에서는 서버 인스턴스에 대한 모니터링 데이터를 볼 수 있습니다.

[서버 모니터링] 페이지에는 다음 정보가 포함되어 있습니다.

인스턴스 이름

서버 인스턴스의 이름입니다.

보기

모니터링 통계가 표시되는 가상 서버, HTTP 리스너, 불필요한 정보 수집기 또는 스레드 시스템입니다.

모니터링되는 속성에 대한 설명은 Oracle GlassFish Server Administration Guide의 8장, Administering the Monitoring Service를 참조하십시오.

관련 작업

  • 모니터링 서비스를 구성하는 방법

  • 응용 프로그램 모니터링 데이터를 보는 방법

  • 서버 모니터링 데이터를 보는 방법

  • 리소스 모니터링 데이터를 보는 방법

관련 asadmin 명령

  • enable-monitoring(1)

  • disable-monitoring(1)

  • monitor(1)
