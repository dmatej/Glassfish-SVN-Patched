수명 주기 모듈 편집

[수명 주기 모듈 편집] 페이지에서는 수명 주기 모듈에 대한 설정을 수정할 수 있습니다.

[수명 주기 모듈 편집] 페이지에는 다음 옵션이 포함되어 있습니다.

이름

수명 주기 모듈의 이름입니다. 이름은 읽기 전용 필드입니다. 새 수명 주기 모듈을 생성할 때만 이름을 지정할 수 있습니다.

클래스 이름

수명 주기 모듈의 클래스 파일에 대한 전체 이름입니다. 클래스는 com.sun.appserv.server.LifecycleListener 인터페이스를 구현해야 합니다.

클래스 경로

수명 주기 모듈의 클래스 경로입니다. 수명 주기 모듈이 있는 위치를 지정하는 클래스 경로입니다. 기본값은 domain-dir/applications입니다. 모듈이 이미 이 디렉토리(즉, 서버 클래스 경로)에 있으면 이 필드를 비워둘 수 있습니다.

로드 순서

시작할 때 이 수명 주기 모듈을 로드할 순서입니다. 정수 로드 순서 값이 작은 모듈부터 로드됩니다. 값의 범위는 101부터 운영 체제의 MAXINT까지일 수 있습니다. 1부터 100까지의 값은 예약되어 있습니다.

설명

수명 주기 모듈에 대한 설명입니다.

상태

이 옵션을 선택하면 수명 주기 모듈이 사용으로 설정됩니다. 이 옵션은 기본적으로 사용으로 설정되어 있습니다.

로드 실패 시

이 옵션을 선택하면 수명 주기 모듈이 로드에 실패할 경우 서버가 종료됩니다. 이 옵션은 기본적으로 사용 안함으로 설정되어 있습니다.
