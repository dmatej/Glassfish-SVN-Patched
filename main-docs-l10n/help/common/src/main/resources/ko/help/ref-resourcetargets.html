리소스 대상

[리소스 대상] 페이지에서는 리소스를 사용할 수 있는 대상 클러스터와 독립형 서버 인스턴스를 볼 수 있습니다.

[리소스 대상] 페이지에는 각 대상에 대한 다음 정보가 포함되어 있습니다.

대상 이름

클러스터나 독립형 서버 인스턴스의 이름입니다.

사용

리소스가 대상에서 사용으로 설정된 경우 선택 표시가 나타나고 사용 안함으로 설정된 경우 X 표시가 나타납니다.

대상 테이블에는 다음 옵션도 포함되어 있습니다.

대상 관리

리소스 대상을 관리하는 단추입니다.

사용

선택한 대상에서 리소스를 사용으로 설정하는 단추입니다.

사용 안함

선택한 대상에서 리소스를 사용 안함으로 설정하는 작업입니다.
