XML 파일에 구성된 리소스를 추가하는 방법

  1. 탐색 트리에서 [리소스] 노드를 선택합니다.

    [리소스] 페이지가 열립니다.

  2. [리소스 추가] 단추를 누릅니다.

    [리소스 추가] 페이지가 열립니다.

  3. 다음 옵션 중 하나를 선택하여 glassfish-resources.xml 파일의 위치를 지정합니다.

    서버에 업로드할 XML 파일

    원격 시스템에 있는 파일의 경로를 입력합니다.

    GlassFish Server에서 액세스 가능한 로컬 XML 파일

    DAS(도메인 관리 서버) 시스템에 있는 파일의 경로를 입력합니다.

  4. [대상] 드롭다운 목록에서 리소스를 생성할 클러스터 또는 독립형 서버 인스턴스를 선택합니다.

  5. [확인]을 누릅니다.
