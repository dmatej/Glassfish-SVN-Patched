리소스 대상 관리

[리소스 대상 관리] 페이지에서는 리소스를 사용할 수 있는 대상 클러스터와 독립형 서버 인스턴스를 변경할 수 있습니다. 리소스는 선택한 대상 열에 있는 대상에서만 사용으로 설정할 수 있습니다.

[리소스 대상 관리] 페이지에는 다음 정보가 포함되어 있습니다.

사용 가능한 대상

리소스를 사용으로 설정할 수 없는 클러스터와 인스턴스입니다.

선택한 대상

리소스를 사용할 수 있는 클러스터와 인스턴스입니다.

추가

선택한 대상을 사용 가능한 대상에서 선택한 대상으로 이동하는 단추입니다.

모두 추가

사용 가능한 대상을 모두 선택한 대상으로 이동하는 단추입니다.

제거

선택한 대상을 선택한 대상에서 사용 가능한 대상으로 이동하는 단추입니다.

모두 제거

선택한 대상을 모두 사용 가능한 대상으로 이동하는 단추입니다.
