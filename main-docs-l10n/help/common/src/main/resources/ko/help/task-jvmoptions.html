JVM 옵션을 구성하는 방법

[JVM 옵션] 페이지에서는 GlassFish Server를 실행하는 Java 응용 프로그램 실행 프로그램(java)을 지정할 수 있습니다. -D 옵션은 GlassFish Server에만 해당하는 속성을 지정합니다.

  1. 탐색 트리에서 [구성] 노드를 선택합니다.

  2. 변경 중인 구성을 선택합니다.

  3. [구성] 노드에서 [JVM 설정] 노드를 선택합니다.

    [JVM 일반 설정] 페이지가 열립니다.

  4. [JVM 일반 설정] 페이지에서 [JVM 옵션] 탭을 누릅니다.

    [JVM 옵션] 페이지가 열립니다.

  5. 옵션을 수정하려면 해당 옵션의 [값] 필드를 편집합니다.

  6. 옵션을 추가하려면 다음을 수행합니다.

    1. [JVM 옵션 추가] 단추를 누릅니다.

    2. 빈 행이 나타나면 [값] 필드에 옵션 정보를 입력합니다.

      옵션 정보에 하나 이상의 공백이 포함되어 있으면 전체 문자열을 큰 따옴표로 묶습니다.

  7. 옵션을 삭제하려면 다음을 수행합니다.

    1. 삭제할 옵션의 [값] 필드 왼쪽에 있는 체크 박스를 선택합니다.

    2. [삭제]를 누릅니다.

  8. [저장]을 누릅니다.

  9. 서버를 재시작합니다.
