JACC 제공자를 삭제하는 방법

  1. 탐색 트리의 [구성] 노드에서 JACC 제공자를 생성할 GlassFish Server 인스턴스나 클러스터를 확장합니다.

    모든 인스턴스와 클러스터에서 사용할 수 있는 제공자를 생성하려면 default-config 노드를 선택합니다.

  2. 확장된 인스턴스나 클러스터 구성 노드에서 [보안] 노드를 확장합니다.

  3. [보안] 노드에서 [JACC 제공자] 노드를 선택합니다.

    [JACC 제공자] 페이지가 열립니다.

  4. 편집 중인 JACC 제공자 왼쪽의 체크 박스를 선택합니다.

  5. [삭제]를 누릅니다.
