시스템 속성

시스템 속성 페이지에는 선택한 명명된 구성에 정의된 Java 시스템 속성 목록이 표시됩니다. Java 시스템 속성은 GlassFish Server 시작 시 Java 응용 프로그램 실행기의 -D 옵션을 통해 Java 응용 프로그램 실행기로 전달됩니다.

이러한 속성은 구성을 참조하는 모든 GlassFish Server 인스턴스에 대한 기본 포트 설정 값을 정의합니다.

구성의 미리 정의된 포트 설정에 대한 자세한 내용은 사전 정의된 시스템 속성을 참조하십시오.

각 속성에 대해 표시되는 정보는 다음과 같습니다.

인스턴스 변수 이름

시스템 속성의 이름입니다.

기본값

인스턴스가 참조하는 명명된 구성의 속성에 대해 설정된 값입니다. 이 필드는 읽기 전용입니다.

인스턴스 값

속성의 인스턴스 값 페이지에 대한 링크입니다.

추가 속성 테이블에는 다음 옵션도 포함되어 있습니다.

속성 추가

속성을 추가하는 단추입니다. 이 단추를 누르면 추가 속성 테이블에 행이 추가됩니다.

속성 삭제

선택한 속성을 하나 이상 삭제하는 단추입니다. 삭제된 속성은 해당 기본값으로 되돌려지거나 기본값이 설정되어 있지 않은 경우 정의되지 않습니다.

[또한 시스템 속성] 페이지에는 다음 옵션도 포함되어 있습니다.

구성 이름

이 페이지의 설정이 적용되는 구성의 이름입니다. 이 필드는 읽기 전용입니다.

동적 재구성

이 옵션이 사용으로 설정된 경우 구성 변경 사항은 인스턴스를 재시작하지 않아도 구성을 참조하는 인스턴스에 적용됩니다.

해당 구성을 사용하는 클러스터 및/또는 인스턴스

각 클러스터나 인스턴스 실행 여부 표시로 구성을 참조하는 클러스터 및 인스턴스의 목록입니다.

  • 클러스터 또는 인스턴스가 실행 중인 경우 선택 표시 기호(✓)가 나타납니다.

  • 클러스터나 인스턴스가 정지된 경우 느낌표(!)가 표시됩니다.

관련 작업

구성 속성의 기본값을 설정하는 방법

동일한 asadmin 하위 명령

  • create-system-properties(1)

  • delete-system-property(1)

  • list-system-properties(1)
