네트워크 저장소에 액세스할 수 있도록 네트워크 프록시 설정을 구성하는 방법

네트워크 프록시 설정에 따라 GlassFish Server가 인터넷에 접속하는 방법이 결정됩니다. 인터넷을 통해 업데이트 도구가 네트워크 저장소에 액세스할 수 있도록 하려면 네트워크 프록시 설정을 구성합니다.

  1. 탐색 트리에서 업데이트 도구 노드를 선택합니다.

    [사용 가능한 추가 기능] 페이지가 열립니다.

  2. [사용 가능한 추가 기능] 페이지에서 [구성] 탭을 누릅니다.

    업데이트 도구 구성 페이지가 표시됩니다.

  3. 업데이트 도구 구성 페이지에서 GlassFish Server가 실행 중인 시스템의 인터넷 접속 방식을 지정합니다.

    • 시스템을 인터넷에 직접 접속한 경우 [직접 접속] 옵션을 선택합니다.

    • 시스템이 프록시를 통해 접속된 경우 [프록시를 통해] 옵션을 선택합니다.

  4. GlassFish Server가 실행 중인 시스템에서 프록시를 통해 인터넷에 접속하는 경우에는 프록시 호스트 이름과 프록시 포트 번호를 제공합니다.

    GlassFish Server가 실행 중인 시스템이 인터넷에 직접 접속된 경우 이 단계를 생략합니다.

    웹 브라우저와 GlassFish Server가 동일한 시스템에서 실행 중일 수 있습니다. 그럴 경우 웹 브라우저에서 프록시의 호스트 이름과 포트 번호를 가져올 수 있습니다. 그렇지 않으면 시스템 관리자에게 이 정보를 문의하십시오.

  5. [저장]을 누릅니다.

관련 항목

  • 구성 요소의 사용 가능한 업데이트를 확인하는 방법

  • 추가 기능 구성 요소를 설치하는 방법

  • 설치된 구성 요소를 확인하는 방법

  • 구성 요소에 대한 자세한 정보를 보는 방법
