영역을 편집하는 방법

  1. [구성] 노드에서 영역을 구성할 서버 인스턴스나 클러스터 구성을 확장합니다.

  2. [보안] 노드를 확장합니다.

  3. [영역] 노드를 확장합니다.

    [영역] 페이지가 열립니다.

  4. 기존 영역 이름을 선택합니다.

    [영역 편집] 페이지가 열립니다.

  5. 영역의 클래스 이름을 수정하려면 다음 중 하나를 수행합니다.

    • [클래스 이름] 드롭다운 목록에서 생성할 영역의 클래스 이름을 선택합니다.

    • 두번째 라디오 단추를 선택하고 생성하려는 사용자 정의 영역의 클래스 이름을 입력합니다.

  6. [추가 속성] 섹션에서 추가 속성을 지정합니다.

    적합한 속성은 영역 유형에 따라 다릅니다.

    • FileRealm 클래스 관련 속성

    • CertificateRealm 클래스 관련 속성

    • JDBCRealm 클래스 관련 속성

    • LDAPRealm 클래스 관련 속성

    • SolarisRealm 클래스 관련 속성

    • PamRealm 클래스 관련 속성

    • 속성을 추가하려면 [속성 추가] 단추를 누릅니다. 빈 행이 표시되면 [이름] 필드에 속성 이름을 입력하고 [값] 필드에 속성값을 입력합니다.

    • 속성을 수정하려면 속성의 [값] 필드를 편집합니다.

    • 속성을 삭제하려면 삭제할 속성의 [이름] 필드 왼쪽에 있는 체크 박스를 선택한 다음 [속성 삭제] 단추를 누릅니다.

  7. [저장]을 누릅니다.
