모니터링

[모니터링] 페이지에서는 모니터링을 구성하고 클러스터된/클러스터링되지 않은 서버 인스턴스에 대한 모니터링 데이터를 볼 수 있습니다.

각 서버 인스턴스에 대해 제공되는 정보는 다음과 같습니다.

인스턴스 이름

서버 인스턴스의 이름입니다. 인스턴스 이름을 누르면 해당 인스턴스에 대한 [일반 정보] 페이지로 이동합니다.

클러스터 이름

서버 인스턴스가 속해 있는 클러스터의 이름입니다(클러스터된 인스턴스인 경우). 클러스터 이름을 누르면 해당 클러스터에 대한 [일반 정보] 페이지로 이동합니다.

작업

서버 인스턴스에 대해 다음 작업을 수행할 수 있는 링크입니다.

모니터링 구성

클러스터 또는 독립형 인스턴스 구성에 대한 모니터링 서비스 페이지가 표시됩니다.

모니터링 데이터 보기

서버 인스턴스에 대한 다음 유형의 모니터링 데이터를 볼 수 있는 링크입니다.

응용 프로그램

서버 인스턴스에 대한 [응용 프로그램 모니터링] 페이지가 표시됩니다.

서버

서버 인스턴스에 대한 [서버 모니터링] 페이지가 표시됩니다.

리소스

서버 인스턴스에 대한 [리소스 모니터링] 페이지가 표시됩니다.

관련 작업

  • 모니터링 서비스를 구성하는 방법

  • 응용 프로그램 모니터링 데이터를 보는 방법

  • 서버 모니터링 데이터를 보는 방법

  • 리소스 모니터링 데이터를 보는 방법

관련 asadmin 명령

  • enable-monitoring(1)

  • disable-monitoring(1)

  • monitor(1)
