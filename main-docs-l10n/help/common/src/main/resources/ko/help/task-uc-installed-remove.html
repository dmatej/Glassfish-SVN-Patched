설치된 구성 요소를 확인하는 방법

[설치된 구성 요소] 페이지에서는 설치된 설치 이미지의 구성 요소에 대한 정보를 제공합니다.

구성 요소 설치 제거에는 다음 항목의 제거가 포함됩니다.

  • 구성 요소의 원래 설치에 의해 적용된 모든 파일 및 디렉토리

  • 원래 설치 후 구성 요소에 대한 업데이트에 의해 적용된 모든 파일 및 디렉토리

  1. 탐색 트리에서 업데이트 도구 노드를 선택합니다.

    [사용 가능한 추가 기능] 페이지가 열립니다.

  2. [사용 가능한 추가 기능] 페이지에서 [설치된 구성 요소] 탭을 누릅니다.

    [설치된 구성 요소] 페이지가 열립니다.

다음 단계

설치된 구성 요소를 제거하려면 먼저 GlassFish Server 도메인을 정지해야 합니다. 그런 후 다음 명령 중 하나를 사용하십시오.

  • as-install-parent/bin/updatetool - 업데이트 툴 그래픽 유틸리티를 시작합니다.

  • as-install-parent/bin/pkg - 업데이트 툴의 명령행 버전입니다.

pkg 명령 사용에 대한 자세한 내용은 Oracle GlassFish Server Administration Guide의 11 장, Extending and Updating GlassFish Server를 참조하십시오. 업데이트 툴 유틸리티에는 자체 온라인 도움말이 있습니다.

구성 요소 제거가 끝난 후 서버를 재시작하십시오.

관련 항목

  • 구성 요소의 사용 가능한 업데이트를 확인하는 방법

  • 추가 기능 구성 요소를 설치하는 방법

  • 구성 요소에 대한 자세한 정보를 보는 방법

  • 네트워크 저장소에 액세스할 수 있도록 네트워크 프록시 설정을 구성하는 방법
