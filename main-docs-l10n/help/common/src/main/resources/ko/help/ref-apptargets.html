응용 프로그램 대상

[응용 프로그램 대상] 페이지에서는 응용 프로그램을 사용할 수 있는 대상 클러스터와 독립형 서버 인스턴스를 볼 수 있습니다.

[응용 프로그램 대상] 페이지에는 다음 정보가 포함되어 있습니다.

대상 이름

클러스터나 독립형 서버 인스턴스의 이름입니다.

사용

응용 프로그램이 대상에서 사용으로 설정된 경우 true가 표시되고 사용 안함으로 설정된 경우 false가 표시됩니다.

LB 사용

대상에서 응용 프로그램이 로드 균형 조정에 대해 사용으로 설정된 경우 true가 표시되고 사용 안함으로 설정된 경우 false가 표시됩니다.

가상 서버

응용 프로그램을 대상의 특정 가상 서버와 연결하려면 해당 대상에 대해 [가상 서버 관리]를 선택합니다.

대상 테이블에는 다음 옵션도 포함되어 있습니다.

대상 관리

응용 프로그램 대상을 관리하는 단추입니다.

추가 작업

다음 작업에 대한 드롭다운 목록입니다.

사용

선택한 대상에서 응용 프로그램을 사용으로 설정하는 작업입니다.

사용 안함

선택한 대상에서 응용 프로그램을 사용 안함으로 설정하는 작업입니다.

로드 밸런서 사용

선택한 대상에서 로드 균형을 조정하기 위해 응용 프로그램을 사용으로 설정하는 작업입니다.

로드 밸런서 사용 안함

선택한 대상에서 로드 균형을 조정하기 위해 응용 프로그램을 사용 안함으로 설정하는 작업입니다.
