관리 영역 사용자 정보를 편집하는 방법

  1. [구성] 노드에서 관리 영역 사용자를 관리할 서버 인스턴스나 클러스터 구성을 확장합니다.

  2. [보안] 노드를 확장합니다.

  3. [영역] 노드를 확장합니다.

  4. admin-realm 노드를 선택합니다.

    [영역 편집] 페이지가 열립니다.

  5. [영역 편집] 페이지에서 [사용자 관리] 단추를 누릅니다.

    [관리 사용자] 페이지가 열립니다.

  6. [관리 사용자] 페이지에 있는 테이블의 [사용자 ID] 열에서 수정할 사용자의 이름을 누릅니다.

    [파일 영역 사용자 편집] 페이지가 열립니다.

  7. [그룹 목록] 필드를 편집하여 사용자가 속한 그룹을 수정합니다.

    콤마를 사용하여 그룹 이름을 구분합니다. 이 그룹은 다른 곳에서 정의할 필요가 없습니다.

  8. [새 비밀번호] 및 [새 비밀번호 확인] 필드에 새 비밀번호를 입력하여 사용자 비밀번호를 변경합니다.

  9. [저장]을 누릅니다.

관련 항목

  • 관리 영역에 사용자를 추가하는 방법

  • 관리 영역에서 사용자를 삭제하는 방법
