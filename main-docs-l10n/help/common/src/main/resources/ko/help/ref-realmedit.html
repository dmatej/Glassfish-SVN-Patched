영역 편집

[영역 편집] 페이지에서는 영역을 수정할 수 있습니다.

[영역 편집] 페이지에는 다음 옵션이 포함되어 있습니다.

구성 이름

이 페이지의 설정이 적용되는 구성의 이름입니다. 이 필드는 읽기 전용입니다.

영역 이름

영역의 이름입니다. 읽기 전용 필드입니다. 새 영역을 생성할 때만 이름을 지정할 수 있습니다.

클래스 이름

편집 중인 영역의 클래스 이름입니다.

추가 속성

영역에 대한 추가 속성입니다. 적합한 속성은 영역 유형에 따라 다릅니다.

  • FileRealm 클래스 관련 속성

  • CertificateRealm 클래스 관련 속성

  • JDBCRealm 클래스 관련 속성

  • LDAPRealm 클래스 관련 속성

  • SolarisRealm 클래스 관련 속성

  • PamRealm 클래스 관련 속성

관련 작업

  • 영역을 생성하는 방법

  • 영역을 편집하는 방법

  • 영역을 삭제하는 방법

  • 사용자 정의 영역을 생성하는 방법

관련 asadmin 명령

  • create-auth-realm(1)

  • delete-auth-realm(1)

  • list-auth-realms(1)
