메시지 보안 구성

[메시지 보안 구성] 페이지에서는 메시지 보안 제공자에 대해 사용 가능한 구성을 보고 수정할 수 있습니다.

메시지 보안을 구성하려면 보안 및 웹 서비스 개념을 기본적으로 이해해야 합니다. 자세한 내용은 Oracle GlassFish Server Security Guide의 3장, Administering Message Security, 및 Oracle GlassFish Server Administration Guide의 6장, Administering Web Applications,를 참조하십시오.

GlassFish Server은(는) SOAP 처리 계층에 통합된 메시지 보안 제공자를 사용하여 메시지 보안을 구현합니다. 메시지 보안 제공자는 GlassFish Server의 다른 보안 기능에 따라 달라집니다.

각 구성에 대해 제공되는 정보는 다음과 같습니다.

인증 계층

인증이 수행되는 메시지 계층입니다. 기본적으로 SOAP 구성이 제공됩니다. HttpServlet 구성을 제공할 수도 있습니다.

기본 제공자

이 구성에 대한 기본 서버 제공자입니다. 기본적으로 기본 서버 제공자는 구성되어 있지 않습니다.

기본 클라이언트 제공자

이 구성에 대한 기본 클라이언트 제공자입니다. 기본적으로 기본 클라이언트 제공자는 구성되어 있지 않습니다.

메시지 보안 구성 테이블에는 다음 옵션도 포함되어 있습니다.

삭제

선택한 구성을 하나 이상 삭제하는 단추입니다.
