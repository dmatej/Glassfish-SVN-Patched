웹 서비스 끝점 정보

[웹 서비스 끝점 정보] 페이지에서는 웹 서비스 끝점에 대한 정보를 볼 수 있습니다.

[웹 서비스 끝점 정보] 페이지에는 다음 정보가 포함되어 있습니다.

응용 프로그램 이름

배치된 응용 프로그램의 이름입니다. [응용 프로그램 편집] 페이지로 돌아가려면 이 링크 또는 [뒤로] 단추를 누릅니다.

이름에는 버전 식별자(옵션)가 포함될 수 있으며, 이 식별자는 이름 다음에 오고 콜론(:)으로 이름과 구분됩니다. 모듈 및 응용 프로그램 버전에 대한 자세한 내용은 Oracle GlassFish Server Application Deployment Guide의 Module and Application Versions를 참조하십시오.

테스터

[테스터] 링크는 응용 프로그램의 배치 유형이 RI가 아닐 경우(예: 109)에만 표시됩니다. 매개변수를 제공하여 웹 서비스 방식을 테스트할 수 있는 페이지를 보려면 이 링크를 누릅니다.

WSDL

끝점의 WSDL(웹 서비스 정의 언어)입니다. 새 브라우저 탭 또는 창에서 WSDL 파일을 보려면 이 링크를 누릅니다.

WSDL 링크를 누르면 일부 웹 브라우저가 WSDL 소스 해석을 시도하므로 빈 페이지가 표시됩니다. 그러한 브라우저에서는 빈 페이지에서 브라우저의 소스 보기 명령을 사용하여 WSDL 소스를 볼 수 있습니다.
