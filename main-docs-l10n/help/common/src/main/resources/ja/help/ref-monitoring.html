モニター

「モニター」ページを使用して、モニターを構成し、クラスタ化されたサーバー・インスタンスおよびクラスタ化されていないサーバー・インスタンスのモニター・データを表示します。

サーバー・インスタンスごとに、次の情報が表示されます。

インスタンス名

サーバー・インスタンスの名前。インスタンス名をクリックすると、インスタンスの「一般情報」ページに移動します。

クラスタ名

サーバー・インスタンスが属するクラスタの名前(クラスタ化されたインスタンスの場合)。クラスタ名をクリックすると、クラスタの「一般情報」ページに移動します。

アクション

サーバー・インスタンス上で次の操作を実行できるリンク。

モニターを構成

クラスタまたはスタンドアロン・インスタンス構成の「モニターを構成」ページが表示されます。

モニター・データを表示

サーバー・インスタンスの次のタイプのモニター・データを表示できるリンク。

アプリケーション

サーバー・インスタンスの「アプリケーションのモニター」ページが表示されます。

サーバー

サーバー・インスタンスの「サーバーのモニター」ページが表示されます。

リソース

サーバー・インスタンスの「リソースのモニター」ページが表示されます。
