管理者パスワードを変更するには

  1. ナビゲーション・ツリーで、「ドメイン」ノードを選択します。

    「ドメイン属性」ページが開きます。

  2. 「管理者パスワード」タブを選択します。

    「新規管理者パスワード」ページが開きます。

    このページには、現在の管理者アカウントのユーザーIDとグループ名が表示されます。デフォルトでは、ユーザーIDはadmin、グループ名はasadminです。

  3. 「新規パスワード」フィールドに、現在の管理者アカウントの新規パスワードを入力します。

  4. 「新規パスワードの確認」フィールドに、新規パスワードを再度入力します。

  5. 「保存」をクリックします。
