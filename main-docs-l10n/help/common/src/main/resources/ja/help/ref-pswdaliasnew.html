新規パスワード・エイリアス

「新規パスワード・エイリアス」ページを使用し、パスワード・ファイルまたはドメイン構成ファイルで使用する新規パスワード・エイリアスを作成します。

「新規パスワード・エイリアス」ページには、次のオプションが含まれます。

エイリアス

エイリアスの名前。この名前は、ドメイン内のすべてのパスワード・エイリアスの中で一意である必要があります。

パスワード

パスワード・エイリアスによって暗号化された形式で格納されるパスワード。

パスワードの確認

パスワード・エイリアスによって暗号化された形式で格納されるパスワード。「パスワード」と「パスワードの確認」の値が比較され、タイプミスのないパスワードが入力されたか確認されます。
