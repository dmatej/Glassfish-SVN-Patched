アプリケーションのモニター・データを表示するには

開始前

モニター・データを表示する前に、モニターを構成する必要があります。詳細は、モニター・サービスを構成するにはを参照してください。

  1. ナビゲーション・ツリーで、「モニター・データを表示」ノードを選択します。

    「モニター」ページが開きます。

  2. 「モニター・データを表示」列で、モニターしているサーバー・インスタンスに対応する「アプリケーション」リンクをクリックします。

    「アプリケーションのモニター」ページが開きます。

  3. 「アプリケーション」ドロップダウン・リストから、モニター・データを表示するアプリケーションを選択します。

  4. 「コンポーネント」ドロップダウン・リストから、モニター・データを表示するアプリケーション・コンポーネントを選択します。

  5. 表示されているモニター・データを更新するには、「リフレッシュ」ボタンをクリックします。

代替ステップ:

  1. ナビゲーション・ツリーで、サーバー(管理サーバー)ノードを選択します。

    「一般情報」ページが開きます。

  2. 「一般情報」ページで、「モニター」タブをクリックします。

    「アプリケーションのモニター」ページが開きます。

  3. 「アプリケーション」ドロップダウン・リストから、モニター・データを表示するアプリケーションを選択します。

  4. 「コンポーネント」ドロップダウン・リストから、モニター・データを表示するアプリケーション・コンポーネントを選択します。

  5. 表示されているモニター・データを更新するには、「リフレッシュ」ボタンをクリックします。
