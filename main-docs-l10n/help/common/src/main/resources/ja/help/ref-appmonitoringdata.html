アプリケーションのモニター

「アプリケーションのモニター」ページを使用して、選択したアプリケーションのモニター・データを表示します。

「アプリケーションのモニター」ページには、次の情報が表示されます。

インスタンス名

サーバー・インスタンスの名前。

アプリケーション

モニター統計が表示されるアプリケーション。

コンポーネント

モニター統計が表示されるアプリケーション・コンポーネント。

モニター対象のプロパティについては、Oracle GlassFish Server管理ガイドの第8章、Administering the Monitoring Serviceを参照してください。
