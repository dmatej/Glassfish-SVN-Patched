ライフサイクル・モジュールを削除するには

  1. ナビゲーション・ツリーで、「ライフサイクル・モジュール」ノードを選択します。

    「ライフサイクル・モジュール」ページが開きます。

  2. 「ライフサイクル・モジュール」ページで、削除するライフサイクル・モジュールの左側にあるチェック・ボックスを選択します。

  3. 「削除」をクリックします。
