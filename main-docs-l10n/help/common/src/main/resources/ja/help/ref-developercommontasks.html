共通タスク

「共通タスク」ページには、管理コンソールの一般的なタスクのためのショートカットが用意されています。

「共通タスク」ページを使用して次の操作を実行できます。

  • GlassFish Serverの「登録」ページを表示するには

  • GlassFish Serverのサブスクリプション・ページを表示するには

  • GlassFish Serverブログを表示するには

  • デプロイされたアプリケーションのリストを表示するには

  • Webアプリケーションをデプロイするには

  • エンタープライズ・アプリケーションをデプロイするには

  • アプリケーション・クライアントをデプロイするには

  • コネクタ・モジュールをデプロイするには

  • EJB JARをデプロイするには

  • モニター・サービスを構成するには

  • アプリケーションのモニター・データを表示するには

  • サーバーのモニター・データを表示するには

  • リソースのモニター・データを表示するには

  • インストール済コンポーネントを表示するには

  • アドオン・コンポーネントをインストールするには

  • コンポーネントの使用可能な更新を表示するには

  • JDBC接続プールを作成するには

「共通タスク」ページには、一部のGlassFish Serverドキュメントへのリンクも表示されます。
