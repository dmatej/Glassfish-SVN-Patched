ファイル・ユーザー

fileレルムのユーザーを管理するには、「ファイル・ユーザー」ページを使用します。fileレルムのユーザーおよびグループは鍵ファイルにリストされ、その場所はfileプロパティで指定されます。

fileレルムのユーザーは、共通の特性で分類されるユーザーのカテゴリである「Javaグループ」に属することができます。たとえば、Eコマース・アプリケーションの顧客はCUSTOMERグループに属しますが、お得意様はPREFERREDグループに属します。ユーザーをグループに分類すると、大量のユーザーによるアクセスを容易に制御できます。

GlassFish Serverのインストール後の当初は、ユーザーはインストールの際に入力した管理者のみです。このユーザーは、デフォルトでGlassFish Serverを変更する権限を付与される、admin-realmレルムのasadminグループに属します。このグループに割り当てられるすべてのユーザーは、管理者権限が付与されます。つまり、asadminツールおよび管理コンソールへのアクセス権があります。

「ファイル・ユーザー」ページには、選択したレルムのユーザーのリストが表示されます。ユーザーごとに、次の情報が表示されます。

ユーザーID

ユーザー名。

グループ・リスト

ユーザーが属しているグループ。

「ファイル・ユーザー」の表には、次のオプションも表示されます。

新規

新規ユーザーを作成するボタン。

削除

選択した1人または複数のユーザーを削除するボタン。
