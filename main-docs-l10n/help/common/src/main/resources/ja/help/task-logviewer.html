DASのログ・ファイルを表示するには

  1. ナビゲーション・ツリーで、サーバー(管理サーバー)ノードを選択します。

    「一般情報」ページが開きます。

  2. 「一般情報」ページで、「ログ・ファイルを表示」ボタンをクリックします。

    「ログ・ビューア」ウィンドウが開きます。
