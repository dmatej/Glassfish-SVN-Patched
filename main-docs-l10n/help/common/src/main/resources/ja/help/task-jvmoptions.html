JVMオプションを構成するには

「JVMオプション」ページを使用して、GlassFish Serverを実行するJavaアプリケーション・ランチャ(java)のオプションを指定します。-Dオプションは、GlassFish Serverに固有のプロパティを指定します。

  1. ナビゲーション・ツリーで、「構成」ノードを選択します。

  2. 変更する構成を選択します。

  3. 「構成」ノードの下で、「JVM設定」ノードを選択します。

    「JVMの全般的な設定」ページが開きます。

  4. 「JVMの全般的な設定」ページで、「JVMオプション」タブをクリックします。

    「JVMオプション」ページが開きます。

  5. オプションを変更するには、そのオプションの「値」フィールドを編集します。

  6. オプションを追加するには、次の手順を実行します。

    1. 「JVMオプションを追加」ボタンをクリックします。

    2. 表示される空白行に、「値」フィールドのオプション情報を入力します。

      オプション情報にスペースが1つ以上含まれている場合は、文字列全体を二重引用符で囲みます。

  7. オプションを削除するには、次の手順を実行します。

    1. 削除するオプションの「値」フィールドの左側にあるチェック・ボックスを選択します。

    2. 「削除」をクリックします。

  8. 「保存」をクリックします。

  9. サーバーを再起動します。
