新規ファイル・レルム・ユーザー

「新規ファイル・レルム・ユーザー」ページを使用して、新規ユーザーに現在のレルムに対するアクセス権を付与します。

注意:

セキュリティ保護された管理が有効な場合は、空のパスワードを指定してadmin-realmユーザーを作成できません。

「新規ファイル・レルム・ユーザー」ページには、次のオプションが含まれます。

構成名

このページの設定が適用される構成の名前。このフィールドは読取り専用です。

レルム名

現在選択しているレルムの名前。このフィールドは読取り専用です。

ユーザーID

新規ユーザー・アカウントのユーザーID。

グループ・リスト

新規ユーザーが属するグループ。複数のグループ名をカンマで区切ります。

admin-realmユーザーの場合、このフィールドは読取り専用です。グループはasadminになります。

新規パスワード

新規ユーザーのパスワード。

新規パスワードの確認

新規ユーザーのパスワード。
