システム・プロパティを管理するには

  1. ナビゲーション・ツリーで、GlassFish Serverノードを選択します。

    「一般情報」ページが開きます。

  2. 「一般情報」ページで、「システム・プロパティ」タブをクリックします。

    「システム・プロパティ」ページが開きます。

  3. 「追加プロパティ」セクションで、構成のシステム・プロパティを指定します。

    多数のシステム・プロパティが事前定義されていますが、これらはこのページに表示されません。詳細は、定義済のシステム・プロパティを参照してください。

    • プロパティを追加するには、「プロパティの追加」ボタンをクリックします。表示される空白行で、「名前」フィールドにプロパティ名を入力し、「値」フィールドにプロパティ値を入力します。

    • プロパティを変更するには、そのプロパティの「値」フィールドを編集します。

    • プロパティを削除するには、削除するプロパティの「名前」フィールドの左側にあるチェック・ボックスを選択し、「プロパティの削除」ボタンをクリックします。

  4. 「保存」をクリックします。
