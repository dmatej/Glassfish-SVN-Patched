インスタンス値

「インスタンス値」ページを使用し、選択した名前付き構成を参照するGlassFish Serverインスタンスの構成システム・プロパティの値を編集します。

「インスタンス値」ページには、プロパティが設定されている選択した構成を参照するインスタンスのリストが表示されます。

インスタンス名

インスタンスが作成されたときにインスタンスに割り当てられた名前。この名前をクリックすると、インスタンスの「一般情報」ページが開きます。

クラスタ名

インスタンスがクラスタのメンバーである場合、そのクラスタの名前。このフィールドは読取り専用です。

デフォルト値

インスタンスが参照する名前付き構成のプロパティに対して設定されている値。このフィールドは読取り専用です。

オーバーライド値

選択したインスタンスに対して設定されているプロパティの値。この値はデフォルト値をオーバーライドします。
