セキュリティ保護された管理を無効にするには

  1. ナビゲーション・ツリーで、サーバー(管理サーバー)ノードを選択します。

    「一般情報」ページが開きます。

  2. 「一般情報」ページで、「セキュリティ保護された管理」ボタンをクリックします。

    「セキュリティ保護された管理」ページが開きます。

  3. 「セキュリティ保護された管理の無効化」をクリックします。

    セキュリティ保護された管理を無効にすると、DASが即時に再起動されることを示すメッセージが表示されます。

  4. 「OK」をクリックします。

    セキュリティ保護された管理が無効になり、サーバーを再起動するページが開きます。
