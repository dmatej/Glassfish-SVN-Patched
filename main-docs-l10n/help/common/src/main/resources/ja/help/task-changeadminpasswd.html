管理レルムのユーザーのパスワードを変更するには

現在の管理者アカウントのパスワードを変更する方法は、管理者パスワードを変更するにはを参照してください。admin-realmに複数のユーザーがある場合は、他のユーザーのパスワードも変更できます。

管理者パスワードを暗号化すると、アプリケーションのセキュリティが向上します。セキュリティ設定手順の詳細は、Oracle GlassFish Serverセキュリティ・ガイドの第1章、Administering System Securityを参照してください。

  1. 「構成」ノードの下で、管理パスワードを変更するサーバー・インスタンスまたはクラスタの構成を展開します。

  2. 「セキュリティ」ノードを展開します。

  3. 「レルム」ノードを展開します。

  4. 「admin-realm」ノードを選択します。

    「レルムの編集」ページが開きます。

  5. 「レルムの編集」ページで、「ユーザーの管理」ボタンをクリックします。

    「管理ユーザー」ページが開きます。

  6. 「管理ユーザー」ページで、パスワードを追加または変更するユーザーをクリックします。

    「ファイル・レルム・ユーザーの編集」ページが開きます。

  7. 「新規パスワード」および「新規パスワードの確認」フィールドに、新規パスワードを入力します。

  8. 「保存」をクリックします。
