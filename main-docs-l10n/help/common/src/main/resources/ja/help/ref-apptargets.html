アプリケーション・ターゲット

「アプリケーション・ターゲット」ページを使用し、アプリケーションを有効にできるターゲット・クラスタとスタンドアロン・サーバー・インスタンスを表示します。

「アプリケーション・ターゲット」ページには、次の情報が表示されます。

ターゲット名

クラスタまたはスタンドアロン・サーバー・インスタンスの名前。

有効

ターゲットでアプリケーションが有効に設定されているとtrueが表示され、アプリケーションが無効に設定されているとfalseが表示されます。

LB有効

ターゲットでアプリケーションのロード・バランシングが有効に設定されているとtrueが表示され、アプリケーションのロード・バランシングが無効に設定されているとfalseが表示されます。

仮想サーバー

アプリケーションをターゲット上の特定の仮想サーバーと関連付けるには、そのターゲットに「仮想サーバーの管理」を選択します。

「ターゲット」の表には、次のオプションも表示されます。

ターゲットの管理

アプリケーション・ターゲットを管理するボタン。

その他の操作

次のアクションのドロップダウン・リスト。

有効

選択したターゲットでアプリケーションを有効にするアクション。

無効

選択したターゲットでアプリケーションを無効にするアクション。

ロード・バランサ有効

選択したターゲットでのアプリケーションのロード・バランシングを有効にするアクション。

ロード・バランサ無効

選択したターゲットでのアプリケーションのロード・バランシングを無効にするアクション。
