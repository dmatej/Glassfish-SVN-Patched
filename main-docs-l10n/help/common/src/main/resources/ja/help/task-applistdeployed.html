デプロイされたアプリケーションのリストを表示するには

  1. ナビゲーション・ツリーで、「アプリケーション」ノードを選択します。

    「アプリケーション」ページが開きます。表に、デプロイされたアプリケーションがリストされます。

  2. 「フィルタ」ドロップダウン・リストでモジュール・タイプを選択して、アプリケーションのリストをモジュール・タイプでフィルタリングします。
