モジュール記述子

アプリケーションのモジュール記述子を表示するには、「モジュール記述子」ページを使用します。

「モジュール記述子」ページには、次のオプションが含まれます。

デプロイメント記述子

アプリケーションの名前、サブコンポーネントの名前(ある場合)、記述子ファイルの名前を表示する表。
