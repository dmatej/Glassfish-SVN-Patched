監査モジュール

監査モジュールを構成するには、「監査モジュール」ページを使用します。

監査は、GlassFish Serverがエラーやセキュリティ侵入などの重大なイベントを記録する方法です。GlassFish Serverは、すべての認証イベントをログファイルに記録します。完全なアクセス・ログには、GlassFish Serverで行われるすべてのアクセス・イベントが連続して記録されます。

監査モジュールごとに、次の情報が表示されます。

名前

監査モジュールの名前。

クラス名

このモジュールを実装するクラスの完全修飾名。デフォルトの監査モジュールのクラス名は、com.sun.enterprise.security.Auditです。

「モジュール」の表には、次のオプションも表示されます。

新規

新規監査モジュールを作成するボタン。

削除

選択した1つまたは複数の監査モジュールを削除するボタン。

関連するasadminコマンド

  • create-audit-module(1)

  • list-audit-modules(1)

  • delete-audit-module(1)
