ネットワーク・リポジトリにアクセスするネットワーク・プロキシ設定を構成するには

ネットワーク・プロキシ設定により、GlassFish Serverのインターネットへの接続方法が指定されます。更新ツールがインターネット経由でネットワーク・リポジトリにアクセスできるようにするには、ネットワーク・プロキシを構成します。

  1. ナビゲーション・ツリーで、更新ツールノードを選択します。

    「使用可能なアドオン」ページが開きます。

  2. 「使用可能なアドオン」ページで、「構成」タブをクリックします。

    「更新ツールの構成」ページが表示されます。

  3. 「更新ツールの構成」ページで、GlassFish Serverが実行されているマシンがインターネットに接続する方法を指定します。

    • マシンがインターネットに直接接続する場合は、「直接接続」オプションを選択します。

    • マシンがプロキシ経由でインターネットに接続する場合は、「プロキシ経由」オプションを選択します。

  4. GlassFish Serverを実行しているマシンがプロキシを介してインターネットに接続している場合、プロキシのプロキシ・ホスト名とプロキシ・ポート番号を入力します。

    GlassFish Serverを実行しているマシンがインターネットに直接接続している場合は、このステップを省略してください。

    GlassFish Serverを実行しているマシンでWebブラウザを実行している場合があります。その場合は、プロキシのホスト名とポート番号をWebブラウザから取得できます。それ以外の場合は、システム管理者に問い合せてください。

  5. 「保存」をクリックします。
