アドオン・コンポーネントをインストールするには

アドオン・コンポーネントのインストールには次の操作が含まれます。

  • ネットワーク・リポジトリからコンポーネントとコンポーネントの依存性の内容を転送する

  • システム上のGlassFish Serverのインストール・イメージの下にコンポーネントをインストールする

コンポーネントにまだインストールされていない他のコンポーネントが必要な場合は、そのコンポーネントもインストールされます。

  1. ナビゲーション・ツリーで、更新ツールノードを選択します。

    「使用可能なアドオン」ページが開きます。

  2. 「使用可能なアドオン」ページの使用可能なアドオン・コンポーネントの表で、インストールするコンポーネントを選択します。

    使用可能なアドオン・コンポーネントがない場合、表は空になります。

    • 個々のコンポーネントを選択するには、そのコンポーネントの名前の隣にあるチェック・ボックスを選択します。

    • すべてのコンポーネントを選択するには、表ヘッダーの「すべてを選択」アイコンをクリックします。

    • すべてのコンポーネントを選択解除するには、表ヘッダーの「すべてを選択解除」アイコンをクリックします。

  3. 「インストール」をクリックします。

    選択したコンポーネントとそれが必要とするコンポーネントのインストールが必要であることを確認するダイアログ・ボックスが表示されます。

  4. ダイアログ・ボックスの「OK」をクリックします。

    インストールするコンポーネントのライセンス契約の受諾が求められます。

  5. インストールするコンポーネントのライセンス契約を受諾します。

    GlassFish Serverはインストールが完了したことを確認します。アドオン・コンポーネントの表からコンポーネントが除去され、「インストール済コンポーネント」ページのインストール済コンポーネントの表に追加されます。

  6. サーバーを再起動します。

    詳細な手順は、次のドキュメントを参照してください。

    • Oracle GlassFish Server管理ガイドのTo Stop a Domain

    • Oracle GlassFish Server管理ガイドのTo Start a Domain
