JVMプロファイラ設定

プロファイラ・ツールは、パフォーマンスを分析し、潜在的な障害を識別するための情報を生成します。

プロファイラ設定を指定します。指定する情報は、使用するプロファイラ製品によって異なります。例および手順については、Oracle GlassFish Serverアプリケーション開発ガイドの第3章、Debugging Applicationsを参照してください。
