構成

「構成」ページを使用して、構成を管理し、その構成を使用するスタンドアロン・サーバー・インスタンスおよびクラスタを表示します。

GlassFish Serverでは、default-config構成が事前定義されています。ドメインが作成されると、ドメイン管理サーバー(DAS)に対してserver-config構成が自動的に作成されます。詳細は、Oracle GlassFish Server高可用性管理ガイドのAbout Named Configurationsを参照してください。

構成ごとに、次の情報が表示されます。

名前

構成の名前。

インスタンス

構成を使用するスタンドアロン・インスタンスおよびクラスタのリスト。

「構成」の表には、次のオプションも表示されます。

新規

新規構成を作成するボタン。

削除

選択した1つまたは複数の構成を削除するボタン。
