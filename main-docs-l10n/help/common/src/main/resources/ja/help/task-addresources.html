XMLファイルで構成されているリソースを追加するには

  1. ナビゲーション・ツリーで、「リソース」ノードを選択します。

    「リソース」ページが開きます。

  2. 「リソースの追加」ボタンをクリックします。

    「リソースの追加」ページが開きます。

  3. 次のいずれかのオプションを選択して、glassfish-resources.xmlファイルの場所を指定します。

    サーバーにアップロードされるXMLファイル

    リモート・マシンに配置されている場合は、ファイルのパスを入力します。

    GlassFish Serverからアクセス可能なローカルXMLファイル

    ドメイン管理サーバー(DAS)マシンに配置されている場合は、ファイルのパスを入力します。

  4. 「ターゲット」ドロップダウン・リストから、リソースを作成するクラスタまたはスタンドアロン・サーバー・インスタンスを選択します。

  5. 「OK」をクリックします。
