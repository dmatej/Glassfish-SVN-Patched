ドメイン・ログ

「ドメイン・ログ」ページを使用して、指定したGlassFish Serverインスタンスまたはクラスタのログ・ファイルを収集およびダウンロードし、それらのログを1つのZIPアーカイブにまとめます。

注意:

クラスタを指定した場合、ZIPアーカイブにもドメイン管理サーバー(DAS)のログ・ファイルが含まれます。

「ドメイン・ログ」ページには、次のオプションが含まれます。

クラスタ

クラスタのドロップダウン・リスト(クラスタが存在する場合)。

インスタンス

サーバー・インスタンスのドロップダウン・リスト。

ログの収集

ログをlog-files-name.zipファイルに収集するボタン。nameはクラスタまたはインスタンスの名前です。

log-files-name.zipファイルは、使用しているブラウザのダウンロード・メカニズムを使用してダウンロードされます。
