ドメイン・ログを収集およびダウンロードするには

GlassFish Serverインスタンスまたはクラスタの使用可能なログ・ファイルをすべて収集し、それらのログを1つのZIPアーカイブにまとめることができます。

注意:

クラスタを指定した場合、ZIPアーカイブにもドメイン管理サーバー(DAS)のログ・ファイルが含まれます。

  1. ナビゲーション・ツリーで、「ドメイン」ノードを選択します。

    「ドメイン属性」ページが開きます。

  2. 「ドメイン・ログ」タブを選択します。

    「ドメイン・ログ」ページが開きます。

    クラスタが存在する場合は、そのクラスタおよびインスタンスのドロップダウン・リストが表示されます。クラスタが存在しない場合は、インスタンスのドロップダウン・リストのみが表示されます。

  3. クラスタまたはインスタンスを選択します。

  4. ログ・ファイルを収集してダウンロードするクラスタまたはインスタンスを選択します。

  5. 「ログの収集」をクリックします。

    ブラウザのダウンロード・メカニズムにより、log-files-name.zipファイル(nameはクラスタまたはインスタンスの名前)のダウンロードを求めるプロンプトが表示されます。
