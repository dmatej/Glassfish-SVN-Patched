リソースのモニター

選択したリソースのモニター・データを表示するには、「リソースのモニター」ページを使用します。

「リソースのモニター」ページには、次の情報が表示されます。

インスタンス名

サーバー・インスタンスの名前。

接続プール

モニター統計が表示される接続プール。

モニター対象のプロパティについては、Oracle GlassFish Server管理ガイドの第8章、Administering the Monitoring Serviceを参照してください。
