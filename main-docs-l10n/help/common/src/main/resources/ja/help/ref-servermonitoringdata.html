サーバーのモニター

「サーバーのモニター」ページを使用して、サーバー・インスタンスのモニター・データを表示します。

「サーバーのモニター」ページには、次の情報が表示されます。

インスタンス名

サーバー・インスタンスの名前。

表示

モニター統計が表示される仮想サーバー、HTTPリスナー、ガベージ・コレクタ、またはスレッド・システム。

モニター対象のプロパティについては、Oracle GlassFish Server管理ガイドの第8章、Administering the Monitoring Serviceを参照してください。
