安全管理

使用“安全管理”页可启用或禁用安全管理。

如果启用了安全管理：

  • DAS 和域中的实例将使用 SSL 证书来加密在它们之间发送的消息。

  • DAS 接受来自远程客户机 (例如 asadmin 实用程序和 IDE) 的管理消息。

“安全管理”页包含以下信息。

启用安全管理

用于启用安全管理的按钮。此按钮仅在未启用安全管理时显示。

注:

如果域中的任何管理用户具有空口令，启用安全管理的尝试将会失败。
禁用安全管理

用于禁用安全管理的按钮。此按钮仅在已启用安全管理时显示。

管理别名

引用 DAS 上的 SSL/TLS 证书的别名。DAS 使用该别名向实例标识自己。默认值为 s1as。此选项仅在未启用安全管理时显示。

实例别名

引用实例上的 SSL/TLS 证书的别名。实例使用该别名向 DAS 标识自己。默认值为 glassfish-instance。此选项仅在未启用安全管理时显示。
