管理系统属性

  1. 在导航树中，选择“GlassFish Server”节点。

    将打开“一般信息”页。

  2. 在“一般信息”页上，单击“系统属性”选项卡。

    此时将打开“系统属性”页。

  3. 在“其他属性”部分中，指定配置的系统属性。

    许多系统属性已预定义，但未显示在此页上。有关详细信息，请参阅预定义系统属性。

    • 要添加属性，请单击“添加属性”按钮。在显示的空白行中，在“名称”字段中键入属性名称，在“值”字段中键入属性值。

    • 要修改属性，请编辑该属性的“值”字段。

    • 要删除属性，请选中位于您要删除属性的“名称”字段左边的复选框，然后单击“删除属性”按钮。

  4. 单击“保存”。
