编辑口令别名

使用“编辑口令别名”页可更改与现有口令别名关联的口令。

“编辑口令别名”页包含以下选项：

别名名称

别名的名称。此选项不可编辑。

口令

口令别名以加密形式存储的口令。

确认口令

口令别名以加密形式存储的口令。系统将比较“口令”和“确认口令”值，确保输入的口令没有键入错误。

等效的 asadmin 子命令

  • update-password-alias(1)

  • create-password-alias(1)

  • delete-password-alias(1)

  • list-password-aliases(1)
