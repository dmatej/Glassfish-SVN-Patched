Web 服务端点信息

使用“Web 服务端点信息”页可以查看关于 Web 服务端点的信息。

“Web 服务端点信息”页包含以下信息。

应用程序名称

已部署的应用程序的名称。单击此链接或“上一页”按钮可以返回“编辑应用程序”页。

名称可包含可选的版本标识符，它跟在名称后面，并与名称以冒号 (:) 分隔。有关模块和应用程序版本的详细信息，请参阅 Oracle GlassFish Server Application Deployment Guide 中的 Module and Application Versions。

测试器

仅当应用程序的部署类型不是 RI (例如为 109) 时，才可以看到“测试器”链接。单击此链接可以查看一个页，通过该页，可以提供参数来测试 Web 服务的方法。

WSDL

端点的 Web 服务定义语言 (WSDL)。单击此链接可以在新的浏览器选项卡或窗口中查看 WSDL 文件。

请注意，单击 WSDL 链接时，某些 Web 浏览器会尝试解释 WSDL 源代码，导致显示空白页。对于这类浏览器，请在空白页上使用浏览器的“查看源代码”命令来查看 WSDL 源代码。

端点名称

端点的名称。

服务名称

Web 服务的名称。

端口名称

Web 服务端口的名称。

部署类型

Web 服务的部署类型 (例如 RI 或 109)。

实现类型

Web 服务的实现类型 (例如 SERVLET 或 EJB)。

实现类名称

Web 服务的实现类名称。

端点地址 URI

端点地址的 URI。

名称空间

与 Web 服务相关联的目标名称空间。

说明

端点说明 (如果存在)。
