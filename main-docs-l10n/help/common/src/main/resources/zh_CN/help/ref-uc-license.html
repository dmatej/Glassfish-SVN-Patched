许可证协议

“许可证协议”页显示了选择进行安装或更新的所有组件的许可证协议。组件的许可证协议由组件编写人员随此组件一起提供。网络资料档案库中的不同组件可能在不同的许可证下发布，例如：

  • 通用开发和发布许可证 (CDDL) (http://www.sun.com/cddl/)

  • GNU 通用公共许可证 (GPL) (http://www.gnu.org/copyleft/gpl.html)

  • Apache 许可证 (http://www.apache.org/licenses/)

  • wxWindows 库许可证 (http://www.opensource.org/licenses/wxwindows.php)

“许可证协议”页包含以下选项。

接受

此按钮接受所有所选组件的许可证条款，并继续进行安装或升级。

拒绝

此按钮拒绝所有所选组件的许可证条款，并使您返回到从中选择组件的页。
