特定于消息安全提供方配置的属性

以下属性可用于消息安全提供方配置。

server.config

包含服务器配置信息的 XML 文件的目录和文件名。默认值为 domain-dir/config/wss-server-config-1.0.xml。

debug

如果为真，则将服务器提供方调试消息转储到服务器日志。默认值为假。

dynamic.username.password

如果为真，则通知提供方运行时从每个请求的 CallbackHandler 收集用户名和口令。如果为假，则在模块初始化过程中收集一次 wsse:UsernameToken 的用户名和口令。此属性仅适用于 ClientAuthModule。默认值为假。

encryption.key.alias

指定提供方使用的加密密钥。可通过密钥的密钥库别名来指定密钥。默认值为 s1as。

signature.key.alias

指定提供方使用的签名密钥。可通过密钥的密钥库别名来指定密钥。默认值为 s1as。
