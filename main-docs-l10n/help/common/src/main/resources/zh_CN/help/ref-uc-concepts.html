UpdateTool 密钥概念

UpdateTool管理 GlassFish Server 软件组件 (包含在您系统上的单个文件系统目录中) 的安装。

通过UpdateTool的管理控制台版本，您可以执行以下操作：

  • 安装附加组件

  • 查看已安装组件的更新

  • 查看已安装组件

要更新或删除已安装组件，必须停止 GlassFish Server 域，然后使用以下命令之一：

  • as-install-parent/bin/updatetool，可用于启动 UpdateTool 图形实用程序

  • as-install-parent/bin/pkg，UpdateTool 的命令行版本

在下面的章节中描述了UpdateTool的密钥概念。

组件

组件是由UpdateTool管理的可安装单元。组件由安装映像中作为一个单元安装到您系统上的各个文件组成。

安装映像

安装映像是包含软件组件的目录，其中包含目录内容。作为下载并安装 GlassFish Server 软件的结果，安装映像将添加到UpdateTool。为了启用 GlassFish Server 软件组件的添加和更新，每个安装映像都要包含有关安装组件的信息。

网络资料档案库

网络资料档案库包含以下项：

  • 可以添加到安装映像的组件

  • 对安装映像中现有组件的更新

安装映像与一个或多个网络资料档案库相关联。每个网络资料档案库都由名称以及关联的 URL 表示。每个安装映像的资料档案库名称都是唯一的，并且代表管理资料档案库和已定义完备的内容范围的组织或项目。要了解为 GlassFish Server 预先配置的网络资料档案库，请参阅 Oracle GlassFish Server Administration Guide 中的 Preconfigured Repositories for GlassFish Server。

附加组件

附加组件是尚未安装的安装映像的组件。
