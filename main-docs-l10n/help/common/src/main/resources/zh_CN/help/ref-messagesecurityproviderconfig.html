提供方配置

使用“提供方配置”页可以为 SOAP 或 HttpServlet 验证层配置消息安全提供方配置。

默认情况下，已为 SOAP 验证层配置了四个提供方。

对于每个提供方配置，将提供以下信息。

提供方 ID

提供方配置名称。

提供方类型

提供方的类型，可以为以下任意一种：

客户机

指定提供方为客户机验证提供方。

服务器

指定提供方为服务器验证提供方。

客户机-服务器

指定提供方既为客户机验证提供方也为服务器验证提供方。

默认提供方

如果设置为真，则提供方将为默认提供方。

类名

提供方的 Java 实现类的名称。

“提供方配置”表还包含以下选项。

新建

用于创建新的提供方配置的按钮。

删除

用于删除一个或多个所选提供方配置的按钮。
