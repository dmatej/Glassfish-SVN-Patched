配置

使用“配置”页可管理配置并查看使用这些配置的独立服务器实例和集群。

default-config 配置由 GlassFish Server 预定义。创建域时，将自动为域管理服务器 (DAS) 创建 server-config 配置。有关详细信息，请参阅 Oracle GlassFish Server High Availability Administration Guide 中的 About Named Configurations。

对于每个配置，将提供以下信息。

名称

配置的名称。

实例

使用配置的独立实例和集群的列表。

“配置”表还包含以下选项。

新建

用于创建新配置的按钮。

删除

用于删除一个或多个所选配置的按钮。
