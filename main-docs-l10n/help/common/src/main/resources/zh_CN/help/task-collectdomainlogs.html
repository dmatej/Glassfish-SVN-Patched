收集和下载域日志

您可以收集指定 GlassFish Server 实例或集群的所有可用日志文件，并将日志编译到一个 ZIP 档案中。

注:

如果指定集群，则 ZIP 档案还将包含域管理服务器 (DAS) 的日志文件。

  1. 在导航树中，选择“域”节点。

    此时将打开“域属性”页。

  2. 选择“域日志”选项卡。

    此时将打开“域日志”页。

    如果存在集群，则显示集群和实例的下拉列表。如果没有集群存在，则仅显示实例的下拉列表。

  3. 选择“集群”或“实例”。

  4. 选择正为其收集和下载日志文件的集群或实例。

  5. 单击“收集日志”。

    浏览器的下载机制提示您下载 log-files-name.zip 文件，其中 name 为集群或实例的名称。
