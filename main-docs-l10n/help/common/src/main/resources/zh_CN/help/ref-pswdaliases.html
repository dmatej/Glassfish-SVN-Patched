口令别名

使用“口令别名”页可创建和管理口令别名。

“口令别名”页显示域中口令别名的列表。对于每个别名，都将显示以下信息：

名称

创建口令别名时分配给它的名称。单击该名称将打开别名的“编辑口令别名”页。

“口令别名”表还包含以下选项。

新建

创建口令别名的按钮。

删除

删除一个或多个所选口令别名的按钮。

等效的 asadmin 子命令

  • list-password-aliases(1)

  • create-password-alias(1)

  • delete-password-alias(1)

  • update-password-alias(1)
