配置用于访问网络资料档案库的网络代理设置

网络代理设置确定 GlassFish Server 连接到 Internet 的方式。要使UpdateTool可以通过 Internet 访问网络资料档案库，请配置您的网络代理设置。

  1. 在导航树中，选择“UpdateTool”节点。

    此时将打开“可用的附加软件”页。

  2. 在“可用的附加软件”页上，单击“配置”选项卡。

    此时将显示“UpdateTool配置”页。

  3. 在“UpdateTool配置”页，指定运行 GlassFish Server 的计算机连接到 Internet 的方式。

    • 如果此计算机直接连接到 Internet，请选择“直接连接”选项。

    • 如果此计算机通过代理连接到 Internet，请选择“通过代理服务器”选项。

  4. 如果运行 GlassFish Server 的计算机通过代理连接到 Internet，请提供代理的代理主机名和代理端口号。

    如果运行 GlassFish Server 的计算机直接连接到 Internet，则忽略此步骤。

    运行 GlassFish Server 的计算机上可能正在运行 Web 浏览器。在这种情况下，可以通过 Web 浏览器获取代理的主机名和端口号。否则，请您的系统管理员提供这些信息。

  5. 单击“保存”。
