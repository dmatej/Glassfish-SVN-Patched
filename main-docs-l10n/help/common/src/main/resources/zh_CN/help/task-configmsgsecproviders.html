将 GlassFish Server 工具配置为供消息安全提供方使用

  1. 如果使用了用户名标记，请在必要时配置用户数据库。

    使用用户名/口令标记时，请先配置适当的领域并为该领域配置适当的用户数据库。

    编辑领域中讨论了如何配置用户数据库。

  2. 管理证书和私有密钥 (如有必要)。

后续步骤

将 GlassFish Server 的工具配置为供消息安全提供方使用后，则可以启用随 GlassFish Server 一起安装的提供方，如编辑消息安全配置中所述。
