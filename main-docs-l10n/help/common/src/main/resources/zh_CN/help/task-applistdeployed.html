查看已部署应用程序列表

  1. 在导航树中，选择“应用程序”节点。

    此时将打开“应用程序”页。该表列出已部署的应用程序。

  2. 从“过滤器”下拉列表中选择模块类型，根据模块类型过滤应用程序列表。
