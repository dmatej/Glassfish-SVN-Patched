查看 DAS 的日志文件

  1. 在导航树中，选择服务器 (管理服务器) 节点。

    将打开“一般信息”页。

  2. 在“一般信息”页上，单击“查看日志文件”按钮。

    此时将打开“日志查看器”窗口。
