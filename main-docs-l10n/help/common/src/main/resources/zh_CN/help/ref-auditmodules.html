审计模块

使用“审计模块”页可以配置审计模块。

审计指的是 GlassFish Server 对错误或安全性破坏等重要事件进行记录。GlassFish Server 在日志文件中记录所有验证事件。完整的访问日志提供了 GlassFish Server 访问事件的顺序线索。

对于每个审计模块，将提供以下信息。

名称

审计模块名称。

类名

实现此模块的类的全限定名称。默认审计模块的类名为 com.sun.enterprise.security.Audit。

“模块”表还包含以下选项。

新建

用于创建新审计模块的按钮。

删除

用于删除一个或多个所选审计模块的按钮。

相关的 asadmin 命令

  • create-audit-module(1)

  • list-audit-modules(1)

  • delete-audit-module(1)
