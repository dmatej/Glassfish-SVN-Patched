监视服务

使用“监视服务”页可为各个服务器组件或服务配置监视选项。

您必须首先部署应用程序、组件或服务，然后才能为它们配置监视。

“监视服务”页包含以下选项。

配置名称

此页上的设置应用到的配置的名称。此字段为只读。

监视服务

如果选中“启用监视服务”复选框，则将为 GlassFish Server 启用监视。默认情况下启用此选项。

监视 MBean

如果选中“启用监视 MBean”复选框，则将在 GlassFish Server 上部署监视所需的所有 MBean。默认情况下启用此选项。

如果未选中该复选框，则无法在管理控制台中查看监视数据，即使选中“监视服务”复选框也是如此。此时，您必须改用命令行来查看监视数据。如果取消选择该复选框，则会减少 GlassFish Server 的内存占用。

DTrace

如果选中“启用 DTrace”复选框，并且存在 Oracle Enterprise Manager DTrace Monitoring，则可使用 Solaris DTrace 工具。默认情况下禁用此选项。

对于每个可用组件，将提供以下信息。

组件

GlassFish Server 组件名称。

监视级别

为组件设置的监视级别。对于每个组件，可用级别为“低”、“高”和“关闭”。默认级别为“关闭”。在此发行版中，“低”和“高”选项所提供的监视级别之间没有差别。

“组件级别设置”表还包含以下选项。

更改级别

用于更改一个或多个所选组件的监视级别的按钮。

级别

一个下拉列表，您可从该列表中选择要应用到所有所选组件的监视级别。
