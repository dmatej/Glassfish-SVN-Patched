常见任务

“常见任务”页提供日常管理控制台任务的快捷方式。

可使用“常见任务”页进行以下操作：

  • 查看 GlassFish Server 的“注册”页

  • 查看 GlassFish Server 的“订阅”页

  • 查看 GlassFish Server 博客

  • 查看已部署应用程序列表

  • 部署 Web 应用程序

  • 部署企业应用程序

  • 部署应用程序客户机

  • 部署连接器模块

  • 部署 EJB Jar

  • 配置监视服务

  • 查看应用程序监视数据

  • 查看服务器监视数据

  • 查看资源监视数据

  • 查看安装的组件

  • 安装附加组件

  • 查看组件的可用更新

  • 创建 JDBC 连接池

“常见任务”页还提供了某些 GlassFish Server 文档的链接。
