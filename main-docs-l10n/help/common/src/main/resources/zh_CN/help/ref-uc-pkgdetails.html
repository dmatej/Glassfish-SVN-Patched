组件详细信息

“组件详细信息”页提供关于组件的详细信息。

“组件详细信息”页包含以下选项。

上一页

此按钮可以使您返回到从中选择要查看详细信息的组件的页。

程序包名称

提供组件的程序包的名称。

唯一标识符

此组件在资料档案库中时的全限定标识符。

类别

与组件相关联的软件类别，例如“应用程序服务器”、“系统工具”和“数据库”。类别是在创建组件时由组件作者分配到组件中的。

版本

提供组件的程序包的版本号。

发布日期

此版本的组件发布到资料档案库的日期和时间。该日期不是此组件在安装映像中安装的日期和时间。

安装大小

安装组件所消耗的磁盘空间大小。

源

对于已安装的组件，指从中获取该组件的网络资料档案库名称。

对于附加组件或更新，指从中获取该组件的网络资料档案库的名称。

说明

组件概览。
