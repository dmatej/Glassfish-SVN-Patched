启用安全管理

  1. 在导航树中，选择服务器 (管理服务器) 节点。

    将打开“一般信息”页。

  2. 在“一般信息”页上，单击“安全管理”按钮。

    此时将打开“安全管理”页。

  3. 如果 GlassFish Server 安装使用定制管理或实例别名值，则根据需要更新“管理别名”和“实例别名”字段中显示的默认值。

  4. 单击“启用安全管理”。

    此时会显示一条消息，指示启用安全管理将立即重新启动 DAS。

  5. 单击“确定”。

    此时将启用安全管理并打开“服务器重新启动”页。
