监视

使用“监视”页可为集群和非集群服务器实例配置监视并查看这些实例的监视数据。

对于每个服务器实例，将提供以下信息。

实例名称

服务器实例的名称。单击实例名称可转至实例的“一般信息”页。

集群名称

服务器实例 (如果是集群实例) 所属的集群的名称。单击集群名称可转至集群的“一般信息”页。

操作

用于在服务器实例上执行以下操作的链接。

配置监视

显示集群或独立实例配置的“监视服务”页。

查看监视数据

用于查看服务器实例的以下类型的监视数据的链接。

应用程序

显示服务器实例的“应用程序监视”页。

服务器

显示服务器实例的“服务器监视”页。

资源

显示服务器实例的“资源监视”页。
