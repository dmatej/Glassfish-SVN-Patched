1. 在“配置”节点下，展开您要为其管理“管理领域”用户的服务器实例或集群配置。

  2. 展开“安全性”节点。

  3. 展开“领域”节点。

  4. 选择文件节点。

    将打开“编辑领域”页。

  5. 在“编辑领域”页上，单击“管理用户”按钮。

    将打开“文件用户”页。

  6. 在“文件用户”页上，单击“新建”。

    将打开“新建文件领域用户”页。

  7. 在“用户 ID”字段中，键入用户的名称。

  8. 在“组列表”字段中，键入用户所属的组列表。

    使用逗号分隔组名。无需在其他位置定义这些组。

  9. 在“口令”字段中，键入一个唯一的口令。

  10. 在“确认新口令”字段中，再次键入用户口令。

  11. 单击“确定”。
