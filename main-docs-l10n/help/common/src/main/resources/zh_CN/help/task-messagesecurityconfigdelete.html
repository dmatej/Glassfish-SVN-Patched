删除消息安全配置

  1. 在导航树中，展开“配置”节点。

  2. 展开“安全性”节点。

  3. 选择“消息安全”节点。

    此时将打开“消息安全配置”页。

  4. 在“消息安全配置”页上，选中要删除的消息安全配置左侧的复选框。

    注:

    无法创建或删除默认 server-config 配置的消息安全提供方。只有 SOAP 和 HttpServlet 这两个消息安全提供方支持默认的 server-config。因为默认情况下安装 GlassFish Server 时已经创建了这两个消息提供方，所以在默认 server-config 的“消息安全配置”页上不显示“删除”按钮。

  5. 单击“删除”。
