查看 DAS 的一般信息

  1. 在导航树中，选择服务器 (管理服务器) 节点。

    将打开“一般信息”页。

  2. 在“一般信息”页上，执行以下任一步骤。

    • 查看只读“状态”显示可了解服务器是否正在运行。

    • 查看只读“正常运行时间”显示可了解服务器连续运行的小时和分钟数。

    • 单击“停止”按钮以停止 GlassFish Server。

    • 单击“重新启动”按钮以停止并重新启动 GlassFish Server。

    • 单击“查看日志文件”按钮可查看 GlassFish Server 实例或集群的日志文件。

      有关详细信息，请参阅查看 DAS 的日志文件。

    • 单击“JVM 报告”可打开一个单独的窗口，通过此窗口可查看有关 Java 虚拟机的报告，包括概要报告、内存管理报告、类加载报告和当前线程转储。
