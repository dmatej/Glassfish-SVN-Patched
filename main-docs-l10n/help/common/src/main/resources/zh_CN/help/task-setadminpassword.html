更改管理员口令

  1. 在导航树中，选择“域”节点。

    此时将打开“域属性”页。

  2. 选择“管理员口令”选项卡。

    此时将打开“新建管理员口令”页。

    此页显示当前管理员帐户的用户 ID 和组名。默认情况下，用户 ID 为 admin，组名为 asadmin。

  3. 在“新口令”字段中，键入当前管理员帐户的新口令。

  4. 在“确认新口令”字段中，再次键入新口令。

  5. 单击“保存”。
