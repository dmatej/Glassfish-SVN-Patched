查看安装的组件

“安装的组件”页提供了有关已安装的安装映像组件的信息。

卸载组件涉及到删除以下项：

  • 在组件的初始安装时引入的所有文件和目录

  • 在初始安装后更新组件时引入的所有文件和目录

  1. 在导航树中，选择“UpdateTool”节点。

    此时将打开“可用的附加软件”页。

  2. 在“可用的附加软件”页上，单击“安装的组件”选项卡。

    此时将打开“安装的组件”页。

后续步骤

要卸载任何已安装组件，请首先停止 GlassFish Server 域。然后使用以下命令之一：

  • as-install-parent/bin/updatetool，可用于启动 UpdateTool 图形实用程序

  • as-install-parent/bin/pkg，UpdateTool 的命令行版本

有关使用 pkg 命令的信息，请参阅 Oracle GlassFish Server Administration Guide 中的第 11 章：Extending and Updating GlassFish Server。UpdateTool 实用程序有自己的联机帮助。

在卸载完组件后，请重新启动服务器。
