服务器监视

使用“服务器监视”页可查看服务器实例的监视数据。

“服务器监视”页包含以下信息。

实例名称

服务器实例的名称。

查看

为其显示监视统计信息的虚拟服务器、HTTP 监听程序、垃圾收集器或线程系统。

有关受监视属性的说明，请参阅 Oracle GlassFish Server Administration Guide 中的第 8 章：Administering the Monitoring Service。
