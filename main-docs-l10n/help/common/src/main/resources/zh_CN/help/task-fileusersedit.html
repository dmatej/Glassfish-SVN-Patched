编辑文件领域用户信息

  1. 在“配置”节点下，展开您要为其管理“管理领域”用户的服务器实例或集群配置。

  2. 展开“安全性”节点。

  3. 展开“领域”节点。

  4. 选择文件节点。

    将打开“编辑领域”页。

  5. 在“编辑领域”页上，单击“管理用户”按钮。

    将打开“文件用户”页。

  6. 在“文件用户”页上表的“用户 ID”列中，单击要修改的用户名称。

    将打开“编辑文件领域用户”页。

  7. 编辑“组列表”字段，以修改用户所属的组。

    使用逗号分隔组名。无需在其他位置定义这些组。

  8. 在“新口令”和“确认新口令”字段中，键入一个新口令，以更改用户的口令。

  9. 单击“保存”。
