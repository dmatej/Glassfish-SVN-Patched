系统属性

使用“系统属性”页可以查看和管理 GlassFish Server 属性。

系统属性定义服务器级别设置的公值。通过将系统属性括在美元符号和花括号中，可以在“管理控制台”文本字段中引用系统属性。例如，要引用名为 prop-name 的属性，您将使用语法 ${prop-name}。

对于每个属性，“系统属性”页包含以下信息。

名称

属性名称。

值

属性值。

说明

属性的可选说明。

许多系统属性已预定义，但未显示在此页上。有关详细信息，请参阅预定义系统属性。
