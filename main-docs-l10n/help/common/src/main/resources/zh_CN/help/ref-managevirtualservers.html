虚拟服务器目标

使用“虚拟服务器目标”页可更改应用程序或模块所关联的目标虚拟服务器。

“虚拟服务器目标”页包含以下信息。

可用目标

未关联应用程序或模块的虚拟服务器。

所选目标

已关联应用程序或模块的虚拟服务器。

添加

将所选目标从“可用目标”移动到“所选目标”的按钮。

全部添加

将所有“可用目标”移动到“所选目标”的按钮。

删除

将所选目标从“所选目标”移动到“可用目标”的按钮。

全部删除

将所有“所选目标”移动到“可用目标”的按钮。
