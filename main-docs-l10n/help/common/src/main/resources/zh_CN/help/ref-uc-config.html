UpdateTool 配置

“配置”页提供有关UpdateTool的配置信息，可用于更改UpdateTool连接到 Internet 的方式。要更改 UpdateTool 配置的其他方面，请使用位于as-install-parent/bin 中的独立 updatetool 应用程序。

“UpdateTool配置”页包含以下选项。

安装路径

安装映像的根目录或顶层目录的文件系统路径。

首选资料档案库

与此安装映像关联的首选网络资料档案库的名称，例如 glassfish.org。

Internet 连接

GlassFish Server 连接到 Internet 的方式：

直接连接

此选项指定 GlassFish Server 直接连接到 Internet。

通过代理服务器

此选项指定 GlassFish Server 通过代理服务器连接到 Internet。选择此选项后，“代理主机”字段和“代理端口”字段都将添加到此页中。

代理主机

运行 GlassFish Server 的计算机的代理服务器的主机名。

仅当运行 GlassFish Server 的计算机通过代理服务器连接到 Internet 时，此字段才适用。

代理端口

运行 GlassFish Server 的计算机的代理服务器的端口号。

仅当运行 GlassFish Server 的计算机通过代理服务器连接到 Internet 时，此字段才适用。

资料档案库

此表提供了与此安装映像关联的每个网络资料档案库的以下信息：

资料档案库

资料档案库的名称，例如 dev.glassfish.org。

相关任务

配置用于访问网络资料档案库的网络代理设置

相关的 asadmin 命令
