新建审计模块

使用“新建审计模块”页可以定义新的审计模块。

“新建审计模块”页包含以下选项。

配置名称

此页上的设置应用到的配置的名称。此字段为只读。

名称

新审计模块的名称。

类名

实现此模块的类的全限定名称。默认审计模块的类名为 com.sun.enterprise.security.Audit。

其他属性

审计模块的其他属性。有关可用于审计模块的其他属性的说明，请参阅特定于审计模块的属性。
