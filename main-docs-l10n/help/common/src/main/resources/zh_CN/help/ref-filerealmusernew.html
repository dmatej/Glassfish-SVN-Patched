新建文件领域用户

使用“新建文件领域用户”页可以授予新用户对当前领域的访问权限。

注:

如果启用了安全管理，则无法创建具有空口令的 admin-realm 用户。

“新建文件领域用户”页包含以下选项。

配置名称

此页上的设置应用到的配置的名称。此字段为只读。

领域名称

当前选中领域的名称。这是只读字段。

用户 ID

新用户帐户的用户 ID。

组列表

新用户所属的组。使用逗号分隔多个组名称。

对于 admin-realm 用户，这是只读字段，并且组为 asadmin。

新口令

新用户的口令。

确认新口令

新用户的口令。
