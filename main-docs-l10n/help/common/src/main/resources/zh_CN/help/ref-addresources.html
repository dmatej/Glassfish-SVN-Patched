添加资源

使用“添加资源”页可从指定目标上的 glassfish-resources.xml 文件创建一个或多个资源。

“添加资源”页包含以下信息。

位置

glassfish-resources.xml 文件的位置：

要上载到服务器的 XML 文件

该文件在远程计算机上的路径。

可以从 GlassFish Server 访问的本地 XML 文件

该文件在域管理服务器 (DAS) 计算机上的路径。

目标

集群和独立服务器实例的下拉列表。

相关任务

添加在 XML 文件中配置的资源

相关的 asadmin 命令

add-resources(1)
