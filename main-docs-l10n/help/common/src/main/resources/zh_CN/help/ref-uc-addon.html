可用的附加软件

“可用的附加软件”页提供了有关尚未安装的安装映像组件的信息。

“可用的附加软件”页包含以下选项。

安装路径

安装映像的根目录或顶层目录的文件系统路径。

安装

用于安装一个或多个所选附加组件的按钮。

可用的附加软件

对于尚未安装的每个安装映像组件，此表提供以下信息。

组件

组件名称。

类别

与组件相关联的软件类别，例如“应用程序服务器”、“系统工具”和“数据库”。类别是在创建组件时由组件作者分配到组件中的。

版本

提供组件的程序包的版本号。

安装大小

安装组件所消耗的磁盘空间大小。

源

将从其获取组件的网络资料档案库名称。

相关任务

安装附加组件

相关的 asadmin 命令

无。
