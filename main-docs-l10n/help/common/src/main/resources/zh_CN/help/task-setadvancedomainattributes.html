设置域属性

  1. 在导航树中，选择“域”节点。

    此时将打开“域属性”页。

  2. 在“启动 DAS 之后加载控制台”字段中，从下拉列表中选择以下选项之一：

    基于使用情况

    如果控制台已在最近 24 小时内使用，或者除 DAS 外还存在其他服务器实例，则在启动域管理服务器 (DAS) 后加载管理控制台。这是默认选项。

    始终

    始终在启动 DAS 后加载管理控制台。在主 GlassFish Server 启动序列完成后加载控制台。

    从不

    从不在启动 DAS 后加载管理控制台。在请求访问控制台时加载控制台。

  3. 在“应用程序根目录”字段中，键入已部署的应用程序所在目录的完整路径。

    默认值为 domain-dir/applications。

  4. 在“日志根目录”字段中，键入保存服务器日志文件目录的完整路径。

    默认值为 domain-dir/logs。

  5. 将“区域设置”字段保留为空，以便使用主机的默认区域设置。

    区域设置是一种标识符，用于指定特定的语言和区域组合。例如，美国英语的区域设置是 en_US，日语的区域设置是 ja_JP。要使用非英语区域设置，GlassFish Server 必须经过本地化，即必须将英语翻译成其他语言。

  6. 单击“保存”。

  7. 重新启动服务器。
