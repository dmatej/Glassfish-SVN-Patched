Editar usuario de dominio de archivo

Utilice la página Editar usuario de dominio de archivo para modificar la contraseña del dominio actual.

Nota:

Si se activa la administración segura, no se puede cambiar la contraseña de un usuario admin-realm por un valor en blanco.

La página Editar usuario de dominio de archivo contiene las siguientes opciones.

Nombre de Configuración

Nombre de la configuración a la que se aplican los valores de esta página. Este campo es de sólo lectura.

Nombre de dominio

Nombre del dominio actualmente seleccionado. Éste es un campo de sólo lectura.

ID de Usuario

Nombre de la cuenta que dese editar. Éste es un campo de sólo lectura.

Lista de Grupos

Grupos a los que pertenece el usuario. Separe los nombres de los grupos con comas.

Para usuarios de admin-realm, éste es un campo de sólo lectura y el grupo es asadmin.

Contraseña Nueva

Contraseña nueva para este usuario.

Confirmar Contraseña Nueva

Contraseña nueva para este usuario.
