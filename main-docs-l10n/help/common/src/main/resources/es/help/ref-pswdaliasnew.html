Nuevo alias de contraseña

Utilice la página Nuevo alias de contraseña para crear un nuevo alias de contraseña para usarlo en un archivo de contraseñas o en el archivo de configuración del dominio.

La página Nuevo Alias de Contraseña contiene las siguientes opciones:

Nombre de Alias

Nombre del alias. Este nombre debe ser único entre todos los alias de contraseña del dominio.

Contraseña

Contraseña que el alias almacena en un formato cifrado.

Confirmar Contraseña

Contraseña que el alias almacena en un formato cifrado. Los valores Contraseña y Confirmar Contraseña se comparan entre sí para garantizar que la contraseña se ha escrito sin errores.
