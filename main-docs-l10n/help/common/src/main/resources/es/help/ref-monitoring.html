Supervisión

Utilice la página Supervisión para configurar la supervisión y ver los datos de supervisión para las instancias de servidor en cluster y no en cluster.

Para cada instancia de servidor se proporciona la información siguiente.

Nombre de instancia

Nombre de la instancia del servidor. Haga clic en el nombre de instancia para ir a la página Información General de la instancia.

Nombre de cluster

Nombre del cluster al que pertenece la instancia de servidor, si es una instancia en cluster. Haga clic en el nombre de cluster para ir a la página Información General del cluster.

Acción

Enlaces que permiten realizar las operaciones siguientes en las instancias de servidor.

Configurar Supervisión

Muestra la página Servicio de Supervisión para la configuración de clusters o de instancias independientes.

Ver Datos de Supervisión

Enlaces que permiten ver los siguientes tipos de datos de supervisión para las instancias de servidor.

Aplicación

Muestra la página Supervisión de Aplicación para la instancia de servidor.

Servidor

Muestra la página Supervisión de Servidor para la instancia de servidor.

Recursos

Muestra la página Supervisión de Recursos para la instancia de servidor.
