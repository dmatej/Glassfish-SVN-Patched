Configurar los valores de ruta de acceso de JVM

  1. En el árbol de navegación, seleccione el nodo Configuración.

  2. Seleccione la configuración que desea cambiar.

  3. En el nodo Configuración, seleccione el nodo Configuración de JVM.

    Se abrirá la página Configuración General de JVM.

  4. En la página Configuración General de JVM, haga clic en el separador Configuración de Ruta de Acceso.

    Se abrirá la página Configuración de Ruta de Acceso de JVM.

  5. En el campo Prefijo de Ruta de Acceso de Biblioteca Nativa, introduzca las rutas de acceso completas de los archivos JAR que se van a anteponer a la ruta de acceso de biblioteca nativa.

    Para anteponer varios archivos, escriba una ruta de acceso por línea.

  6. En el campo Sufijo de Ruta de Acceso de Biblioteca Nativa, introduzca las rutas de acceso completas de los archivos JAR que se van a agregar a la ruta de acceso de biblioteca nativa.

    Para agregar varios archivos, escriba una ruta de acceso por línea.

  7. Haga clic en Guardar.

  8. Reinicie el servidor.
