Gestionar Destinos

Utilice la página Gestionar Destinos de Recurso para cambiar los clusters de destino y las instancias de servidor independientes en los que se puede activar un módulo de ciclo de vida. El módulo de ciclo de vida sólo se puede activar en destinos que aparezcan en la columna Destinos Seleccionados.

La página Gestionar Destinos de Recurso contiene la siguiente información.

Destinos Disponibles

Clusters e instancias en los que no se puede activar el módulo de ciclo de vida.

Destinos Seleccionados

Clusters e instancias en los que se puede activar el módulo de ciclo de vida.

Agregar

Botón para mover el destino seleccionado de Destinos Disponibles a Destinos Seleccionados.

Agregar Todos

Botón para mover todos los destinos disponibles a Destinos Seleccionados.

Eliminar

Botón para mover el destino seleccionado de Destinos Seleccionados a Destinos Disponibles.

Eliminar Todos

Botón para mover todos los destinos seleccionados a Destinos disponibles.
