Cambiar la contraseña de un usuario en el dominio de administración

Para cambiar la contraseña de la cuenta de administrador actual, consulte Cambiar la contraseña de administrador. Si admin-realm tiene más de un usuario, puede cambiar también la contraseña de los demás usuarios.

El cifrado de la contraseña del administrador mejora la seguridad de su aplicación. Consulte el Capítulo 1, sobre la administración de la seguridad del sistema, de Oracle GlassFish Server Security Guide, para obtener más información sobre procedimientos de seguridad.

  1. En el nodo Configuraciones, amplíe la configuración de instancia o cluster del servidor para el que desee modificar una contraseña de administración.

  2. Amplíe el nodo Seguridad.

  3. Amplíe el nodo Dominios.

  4. Seleccione el nodo admin-realm.

    Se abrirá la página Editar Dominio.

  5. En la página Editar Dominio, haga clic en el botón Gestionar Usuarios.

    Se abrirá la página Usuarios administradores.

  6. En la página Usuarios administradores, haga clic en el usuario cuya contraseña desee agregar o cambiar.

    Se abrirá la página Editar Usuario de Dominio de Archivo.

  7. En los campos Contraseña Nueva y Confirmar Contraseña Nueva, escriba la nueva contraseña.

  8. Haga clic en Guardar.
