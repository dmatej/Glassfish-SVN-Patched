Gestionar propiedades del sistema

  1. En el árbol de navegación, seleccione el nodo GlassFish Server.

    Se abrirá la página Información General.

  2. En la página Información General, haga clic en el separador Propiedades del Sistema.

    Se abrirá la página Propiedades del Sistema.

  3. En la sección Propiedades Adicionales, especifique propiedades del sistema para la configuración.

    Existe un número de propiedades del sistema predefinidas pero no se muestran en esta página. Para obtener más información, consulte Propiedades del sistema predefinidas.

    • Para agregar una propiedad, haga clic en el botón Agregar Propiedad. En la fila en blanco que aparece, introduzca el nombre de propiedad en el campo Nombre y escriba el valor de propiedad en el campo Valor.

    • Para modificar una propiedad, edite el campo Valor.

    • Para suprimir una propiedad, active la casilla de control a la izquierda del campo Nombre de la propiedad que está suprimiendo y, a continuación, haga clic en el botón Suprimir Propiedades.

  4. Haga clic en Guardar.
