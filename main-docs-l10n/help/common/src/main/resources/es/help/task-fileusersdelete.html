Suprimir un usuario del dominio de archivo

  1. En el nodo Configuraciones, amplíe la configuración de instancia o cluster del servidor para el que desee gestionar los usuarios del dominio de administración.

  2. Amplíe el nodo Seguridad.

  3. Amplíe el nodo Dominios.

  4. Seleccione el nodo file.

    Se abrirá la página Editar Dominio.

  5. En la página Editar Dominio, haga clic en el botón Gestionar Usuarios.

    Se abrirá la página Usuarios de Archivo.

  6. En la página Usuarios de Archivo, active la casilla de control situada a la izquierda del nombre del usuario que desee suprimir.

  7. Haga clic en Suprimir.

    Aparecerá un cuadro de diálogo para que confirme que debe suprimirse el usuario.

  8. En el cuadro de diálogo, haga clic en Aceptar.
