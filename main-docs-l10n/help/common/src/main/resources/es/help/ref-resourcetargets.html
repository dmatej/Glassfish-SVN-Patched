Destinos de Recursos

Utilice la página Destinos de Recursos para ver los clusters de destino y las instancias de servidor independientes en los que se puede activar el recurso.

La página Destinos de Recursos contiene la siguiente información para cada destino.

Nombre de Destino

Nombre del cluster o instancia de servidor independiente.

Activado

Muestra una marca de control si el recurso está activado en el destino, o una equis (X) si está desactivado.

La tabla Destinos también contiene las siguientes opciones.

Gestionar Destinos

Botón para gestionar los destinos de recursos.

Activar

Acción para activar el recurso en el destino seleccionado.

Desactivar

Acción para desactivar el recurso en el destino seleccionado.
