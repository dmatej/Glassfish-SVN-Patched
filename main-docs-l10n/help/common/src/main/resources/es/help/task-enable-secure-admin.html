Activar la Administración Segura

  1. En el árbol de navegación, seleccione el nodo del servidor (servidor de administración).

    Se abrirá la página Información General.

  2. En la página Información General, haga clic en el botón Administración Segura.

    Se abrirá la página Administración segura.

  3. Si la instalación de GlassFish Server utiliza valores de alias de administración o instancia personalizados, actualice los valores por defecto de los campos Alias de Administración y Alias de Instancia como sea necesario.

  4. Haga clic en Activar Administración Segura.

    Aparece un mensaje que indica que activar la administración segura reiniciará DAS de inmediato.

  5. Haga clic en Aceptar.

    Se activa la administración segura y se abre la página de reinicio del servidor.
