Configurar el Servicio de Supervisión

  1. Para abrir la página Servicio de Supervisión, haga una de estas acciones:

    • Seleccione el nodo Datos de Supervisión y haga clic en el enlace Configurar Supervisión para la instancia de servidor para la que desee configurar la supervisión. Para configurar la supervisión para un cluster, elija cualquier instancia del cluster.

    • En el nodo Configuraciones, amplíe la configuración de la instancia de servidor o del cluster para la que desee configurar la supervisión y amplíe el nodo Supervisión.

    Se abrirá la página Servicio de Supervisión.

  2. Active la casilla de control de activación de servicio de supervisión activado para activar la supervisión para GlassFish Server o desactive la casilla de control para desactivar la supervisión.

    Esta opción se encuentra activada por defecto.

  3. Active la casilla de control de activación de MBeans de supervisión para desplegar todos los MBean necesarios para la supervisión o desactive la casilla de control para anular el despliegue de estos MBeans.

    Esta opción se encuentra activada por defecto.

    Si la casilla de control no está activada, no podrá ver los datos de supervisión en la Consola de administración incluso si la casilla de control Servicio de supervisión sí está activada. En su lugar, debe utilizar la línea de comandos para ver los datos de supervisión. Si desactiva la casilla de control, la huella de memoria de GlassFish Server se reduce.

  4. Active la casilla de control DTrace Activado para activar el uso de la utilidad DTrace de Solaris.

    Para poder utilizar la utilidad DTrace de Solaris también debe tener la supervisión DTrace Manager Enterprise de Oracle. Esta opción se activada por defecto.

  5. En la tabla, realice una de las siguientes acciones.

    • En la columna Nivel de Supervisión, seleccione BAJO o ALTO en la lista desplegable para uno o más componentes de la columna Componente para activar la supervisión para esos componentes.

    • Active las casillas de control junto a uno o más componentes y, a continuación, seleccione LOW o HIGH en la lista desplegable Nivel en la parte superior de la tabla y haga clic en el botón Cambiar Nivel para definir el nivel de supervisión en ese nivel para todos esos componentes.

    No existe diferencia alguna entre los niveles de supervisión que ofrecen las opciones BAJO y ALTO de esta versión. Para desactivar la supervisión de uno o más componentes, seleccione OFF. El valor por defecto es OFF.

    Debe desplegar una aplicación, componente o servicio antes de configurar su supervisión.

  6. Haga clic en Guardar.
