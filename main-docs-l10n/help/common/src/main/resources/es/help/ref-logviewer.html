Visor de logs

Utilice la página Visor de Logs para ver los archivos log para las instancias y clusters seleccionados de GlassFish Server y para personalizar la visualización de logs.

La página Visor de Logs contiene cuatro áreas:

  • Criterios de Búsqueda

  • Búsqueda Avanzada

  • Modificar Búsqueda

  • Resultados de Búsqueda

El área Criterios de Búsqueda contiene las siguientes opciones.

Búsqueda Avanzada

Al hacer clic en este enlace aparecerá un área que le permitirá realizar acotaciones adicionales al visor de logs.

Registro de Hora

El botón de radio Más Reciente muestra los mensajes más recientes. El botón de radio Rango Específico activa los campos De y A que le permiten ver mensajes de un período concreto.

Para el valor Hora, la sintaxis debe tener la forma siguiente (SSS significa milisegundos):

hh:mm:ss.SSS

Por ejemplo:

17:10:00.000

Si el valor introducido en el campo De es posterior al valor introducido en el campo A, aparece un mensaje de error.

Nivel de Log

Nivel de registro seleccionado para el filtrado. Por defecto, la pantalla incluye todos los mensajes que aparecen en el log del servidor en el nivel de log seleccionado y niveles más graves. Active la casilla de control "No incluir más mensajes graves" para mostrar mensajes sólo en el nivel seleccionado.

Para garantizar que los mensajes que desea visualizar aparecen en el log del servidor, primero defina los niveles de log adecuados en la página Niveles de log del módulo. Consulte Configurar niveles de log.

Si decide filtrar mensajes de log según el nivel de log, sólo se muestran los mensajes que se ajustan a los criterios de filtro especificados. Sin embargo, este filtro no afecta a qué mensajes se registran en el log del servidor.

El área Búsqueda Avanzada contiene las siguientes opciones.

Registrador

Espacios de nombres del registrador para filtrar por módulos. Utilice la tecla Mayús o la tecla Ctrl y haga clic para seleccionar varios espacios de nombres.

La selección de un espacio de nombres en un nivel superior selecciona todos los espacios de nombres existentes por debajo. Por ejemplo, al seleccionar javax.enterprise.system también se seleccionan los registradores de todos los módulos existentes por debajo de ese espacio de nombres: javax.enterprise.system.core, javax.enterprise.system.tools.admin, etc.

Registrador Personalizado

Registradores específicos de una aplicación determinada, uno por línea.

Si la aplicación tiene varios módulos, puede visualizar uno o todos ellos. Por ejemplo, suponga que la aplicación tiene registradores con los nombres siguientes:

com.mycompany.myapp.module1
com.mycompany.myapp.module2
com.mycompany.myapp.module3

Para ver los mensajes de todos los módulos de la aplicación, introduzca com.mycompany.myapp. Para ver los mensajes sólo de module2, introduzca com.mycompany.myapp.module2.

Cuando especifica uno o varios registradores personalizados, los mensajes de los módulos de GlassFish Server aparecen sólo si los indica explícitamente en el área Registrador.

Mostrar

Número de mensajes que se ven a la vez. El número por defecto de mensajes mostrados es 40. También puede seleccionar 100, 250 y 1000.

Para ver rastreos de pila, desactive la casilla de control "Limitar mensajes demasiado largos". Por defecto, los seguimientos de pila no aparecen en el visor; para verlos, haga clic en el enlace (details) para un mensaje.

El área Modificar Búsqueda permite especificar la instancia de servidor o el cluster y el nombre de un archivo log asociado para visualizar.

Instancia

La instancia de servidor o el cluster para el que se mostrarán los registros de log. El valor por defecto es server. Otras opciones dependen de los clusters e instancias que se hayan definido o activado en el dominio.

Archivo Log

Nombre del archivo log que desea visualizar. El valor por defecto es server.log. Otras opciones dependen de la configuración de log para el destino seleccionado.

La tabla Resultados del Visor de Logs muestra los mensajes de log del servidor. Utilice los botones Registros anteriores a y Registros posteriores a de la parte superior de la tabla para elegir el rango de registros que desee ver.

La tabla Resultados del Visor de Logs contiene la siguiente información para cada mensaje.

Número de Registro

Número del mensaje en el log del servidor.

Nivel de Log

Nivel de log del mensaje.

Mensaje

Breve texto del mensaje de log. Haga clic en el enlace (details) para ver una versión con formato de un mensaje.

Registrador

Módulo de registrador que generó el mensaje de log.

Registro de Hora

Fecha y hora en que se emitió el mensaje de log.

Pares de Nombre/Valor

Nombre de thread e ID del mensaje de log.
