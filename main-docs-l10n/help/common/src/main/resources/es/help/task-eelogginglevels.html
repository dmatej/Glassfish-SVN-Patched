Configurar niveles de log

  1. En el árbol de navegación, amplíe el nodo Configuraciones.

  2. En el nodo Configuración, haga clic en la configuración de instancia de servidor o cluster para el que desee definir la configuración de registrador.

    Se abrirá la página Configuración.

  3. En la página Configuración, haga clic en Configuración de Registrador.

    Se abrirá la página Configuración de Registrador del destino de configuración seleccionado.

  4. En la página Configuración de Registrador, haga clic en el separador Niveles de Registro.

    Se abrirá la página Niveles de Log del Módulo.

  5. En la página Niveles de Log del Módulo, cambie los niveles de log para uno o más módulos de la siguiente manera.

    • Para cambiar el nivel de log para un módulo individual, seleccione un nuevo valor de la lista desplegable frente al módulo cuyo nivel de log desea cambiar.

    • Para cambiar el nivel de log para un grupo de módulos:

      1. Active las casillas de control junto a los nombres de los módulos cuyos niveles de log desea cambiar.

      2. Seleccione un valor en la lista desplegable en la parte superior de la tabla.

      3. Haga clic en el botón Cambiar Nivel.

    Los mensajes en el nivel seleccionado o superior aparecen en el log. Seleccione alguno de los valores siguientes (listados de mayor a menor):

    • EMERGENCY

    • ALERT

    • SEVERE

    • WARNING

    • INFO

    • CONFIG

    • FINE

    • FINER

    • FINEST

    • OFF

    Para obtener más información acerca del significado de los niveles de log, consulte Niveles de Log del módulo. El nivel por defecto es INFO, por tanto el log contendrá mensajes de niveles INFO, WARNING o SEVERE.

  6. Haga clic en Guardar.

    Los cambios en la configuración del nivel de log surten efecto de forma inmediata. Asimismo, se guardan para su posterior uso cuando el servidor se reinicia.
