Supervisión de servidor

Utilice la página Supervisión de Servidor para ver los datos de supervisión para la instancia de servidor.

La página Supervisión de Servidor contiene la información siguiente.

Nombre de instancia

Nombre de la instancia del servidor.

Ver

Servidor virtual, listener HTTP, recolector de basura o sistema de threads para el que se muestran estadísticas de supervisión.

Para obtener una descripción de las propiedades supervisadas, consulte el capítulo 8, sobre la administración del servicio de supervisión, en Oracle GlassFish Server Administration Guide.
