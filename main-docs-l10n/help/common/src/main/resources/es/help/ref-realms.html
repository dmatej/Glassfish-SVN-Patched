Dominios

GlassFish Server viene preconfigurado con tres dominios: file, certificate y admin-realm. También se pueden crear los dominios ldap, JDBC, solaris y personalizados. Normalmente se tiene un dominio de cada tipo en un servidor, pero en GlassFish Server existen dos dominios de archivo: file y admin-realm. Se trata de dos dominios del mismo tipo utilizados para dos fines diferentes. También se puede tener una base de datos de certificado distinta para cada servidor virtual en el sistema.

Utilice la página Dominios para configurar los dominios.

Para cada dominio, se proporciona la siguiente información.

Nombre

Nombre del dominio.

Nombre de clase

Nombre de clase del dominio.

La tabla Dominios también contiene las siguientes opciones.

Nuevo

Botón para crear un nuevo dominio.

Suprimir

Botón para suprimir uno o varios dominios seleccionados.
