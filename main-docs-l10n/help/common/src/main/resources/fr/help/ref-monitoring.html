Surveillance

La page Surveillance permet de configurer la surveillance et de visualiser les données de surveillances relatives aux instances de serveur incluses ou non dans un cluster.

Pour chaque instance de serveur, les informations ci-après sont fournies.

Nom d'instance

Nom de l'instance de serveur. Cliquez sur le nom de l'instance pour accéder à la page Informations générales correspondante.

Nom du cluster

Nom du cluster auquel appartient l'instance de serveur, s'il s'agit d'une instance incluse dans un cluster. Cliquez sur le nom du cluster pour accéder à la page Informations générales correspondante.

Action

Liens permettant d'effectuer les opérations suivantes sur les instances de serveur.

Configurer la surveillance

Affiche la page Service de surveillance pour la configuration du cluster ou de l'instance autonome.

Afficher les données de surveillance

Liens permettant de visualiser les types suivants de données de surveillance pour les instances de serveur.

Application

Affiche la page Surveillance d'application pour l'instance de serveur.

Serveur

Affiche la page Surveillance de serveur pour l'instance de serveur.

Ressources

Affiche la page Surveillance de ressource pour l'instance de serveur.

  • Affichage des données de surveillance d'application

  • Affichage des données de surveillance de serveur

  • Affichage des données de surveillance de ressource
