Niveaux de journalisation de module

La page Niveaux de journalisation de module permet de configurer les niveaux de journalisation des modules individuels.

Pour chaque module, les informations ci-après sont fournies.

Nom du journaliseur

Nom du journaliseur pour le module (par exemple, javax.enterprise.system.tools.admin).

Niveau du journaliseur

Niveau de journalisation en cours du module.

La table Paramètres du journaliseur contient également les options ci-après.

Ajouter un journaliseur

Bouton permettant d'ajouter un module de journalisation. Un clic sur ce bouton permet d'ajouter une ligne à la table Paramètres du journaliseur. Entrez les informations relatives au module de journalisation dans la nouvelle ligne.

Supprimer le journaliseur

Bouton permettant de supprimer les modules sélectionnés.

Niveau

Liste déroulante dans laquelle vous pouvez sélectionner un niveau de journalisation à appliquer à tous les modules sélectionnés.

Modifier le niveau

Bouton permettant de modifier le niveau de journalisation des modules sélectionnés.

Les niveaux de journalisation suivants sont disponibles. Ils sont répertoriés par ordre décroissant.

EMERGENCY

Le serveur présente un état inutilisable. Une panne ou une urgence système grave s'est produite.

ALERT

Un service spécifique présente un état inutilisable. La récupération automatique n'est pas possible.

SEVERE

Evénements qui interfèrent avec l'exécution normale des programmes.

WARNING

Avertissements, y compris les exceptions.

INFO

Messages liés à la configuration du serveur ou au statut du serveur, à l'exclusion des erreurs.

CONFIG

Messages liés à la configuration du serveur.

FINE

Verbosité minimale.

FINER

Verbosité modérée.

FINEST

Verbosité maximale.

OFF

Aucun message de journalisation.
