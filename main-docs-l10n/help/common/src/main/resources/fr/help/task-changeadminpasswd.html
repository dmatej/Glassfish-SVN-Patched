Modification du mot de passe d'un utilisateur dans le domaine d'administration

Pour modifier le mot de passe du compte administrateur en cours, reportez-vous à Modification du mot de passe administrateur. Si le domaine admin-realm comporte plusieurs utilisateurs, vous pouvez également modifier le mot de passe d'autres utilisateurs.

Le cryptage du mot de passe administrateur renforce la sécurité de votre application. Pour plus d'informations sur les procédures de sécurité, reportez-vous au chapitre 1, Administering System Security, du manuel Oracle GlassFish Server Security Guide.

  1. Sous le noeud Configurations, développez la configuration d'instance de serveur ou de cluster pour laquelle vous souhaitez modifier un mot de passe d'administration.

  2. Développez le noeud Sécurité.

  3. Développez le noeud Domaines.

  4. Sélectionnez le noeud admin-realm.

    La page Modifier le domaine apparaît.

  5. Sur la page Modifier le domaine, cliquez sur le bouton Gérer les utilisateurs.

    La page Administrateurs apparaît.

  6. Sur la page Administrateurs, cliquez sur l'utilisateur pour lequel vous voulez ajouter ou modifier le mot de passe.

    La page Modifier l'utilisateur du domaine de fichiers apparaît.

  7. Dans les champs Nouveau mot de passe et Confirmer le nouveau mot de passe, saisissez le nouveau mot de passe.

  8. Cliquez sur Enregistrer.
