Extensions disponibles

La page Extensions disponibles contient des informations sur les composants de l'image d'installation qui n'ont pas encore été installés.

Elle contient les options ci-après.

Chemin d'installation

Chemin du système de fichiers du répertoire racine ou du répertoire de niveau supérieur de l'image d'installation.

Installer

Bouton permettant d'installer les composants d'extension sélectionnés.

Extensions disponibles

Pour chaque composant de l'image d'installation qui n'est pas encore installé, cette table fournit les informations ci-après.

Composant

Nom du composant.

Catégorie

Catégorie du logiciel auquel le composant est associé (par exemple, serveurs d'applications, outils système et bases de données). La catégorie est affectée au composant par le fabricant au moment de la création du composant.

Version

Numéro de version du package comprenant le composant.

Taille de l'installation

Espace disque utilisé pour l'installation du composant.

Source

Nom du Référentiel réseau à partir duquel le composant peut être obtenu.

Tâches connexes

Installation de composants d'extension

Commandes asadmin connexes

Aucune.
