Modification d'un fournisseur JACC

  1. Sous le noeud Configurations dans l'arborescence de navigation, développez le cluster ou l'instance GlassFish Server pour lequel vous souhaitez créer un fournisseur JACC.

    Pour créer un fournisseur qui sera disponible pour l'ensemble des instances et clusters, choisissez le noeud default-config.

  2. Sous le noeud développé de configuration du cluster ou de l'instance, développez le noeud Sécurité.

  3. Sous le noeud Sécurité, sélectionnez le noeud Fournisseurs JACC.

    La page Fournisseurs JACC apparaît.

  4. Sur la page Fournisseurs JACC, cliquez sur le nom du fournisseur JACC à modifier.

    La page Modifier un fournisseur JACC apparaît.

  5. Sur la page Modifier un fournisseur JACC, dans le champ Configuration de règles, saisissez le nom de la classe qui implémente la fabrique de configuration des règles.

  6. Dans le champ Fournisseur de règles, saisissez le nom de la classe qui implémente la fabrique de règles.

  7. Dans la section Autres propriétés, spécifiez les propriétés supplémentaires.

    Pour obtenir une description de la propriété supplémentaire disponible pour les fournisseurs JACC, reportez-vous à Propriétés propres aux fournisseurs JACC.

    • Pour ajouter une propriété, cliquez sur le bouton Ajouter une propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur.

    • Pour modifier une propriété, modifiez le champ Valeur de cette propriété.

    • Pour supprimer une propriété, sélectionnez la case située à gauche du champ Nom de la propriété à supprimer, puis cliquez sur le bouton Supprimer des propriétés.

  8. Cliquez sur Enregistrer.
