Configurations

La page Configurations permet de gérer les configurations et de visualiser les clusters et les instances de serveur autonomes utilisant ces configurations.

La configuration default-config est prédéfinie par GlassFish Server. La configuration server-config est automatiquement créée pour le serveur d'administration de domaine (DAS) lors de la création du domaine. Pour plus d'informations, reportez-vous à About Named Configurations dans le manuel Oracle GlassFish Server High Availability Administration Guide.

Pour chaque configuration, les informations ci-après sont fournies.

Nom

Nom de la configuration.

Instance

Liste des clusters et instances autonomes utilisant la configuration.

La table Configurations contient également les options ci-après.

Nouveau

Bouton permettant de créer une configuration.

Supprimer

Bouton permettant de supprimer les configurations sélectionnées.
