Suppression d'un utilisateur du domaine de fichiers

  1. Sous le noeud Configurations, développez la configuration d'instance de serveur ou de cluster pour laquelle vous souhaitez gérer les utilisateurs du domaine d'administration.

  2. Développez le noeud Sécurité.

  3. Développez le noeud Domaines.

  4. Sélectionnez le noeud file.

    La page Modifier le domaine apparaît.

  5. Sur la page Modifier le domaine, cliquez sur le bouton Gérer les utilisateurs.

    La page Utilisateurs de fichiers apparaît.

  6. Sur la page Utilisateurs de fichiers, cochez la case située à gauche du nom de l'utilisateur à supprimer.

  7. Cliquez sur Supprimer.

    Une boîte de dialogue vous demande de confirmer que l'utilisateur doit être supprimé.

  8. Dans la boîte de dialogue, cliquez sur OK.
