Désactivation de l'administration sécurisée

  1. Dans l'arborescence de navigation, sélectionnez le noeud du serveur (serveur d'administration).

    La page Informations générales apparaît.

  2. Sur la page Informations générales, cliquez sur le bouton Administration sécurisée.

    La page Administration sécurisée apparaît.

  3. Cliquez sur Désactiver l'administration sécurisée.

    Un message s'affiche et indique que la désactivation de l'administration sécurisée entraînera le redémarrage immédiat du serveur DAS.

  4. Cliquez sur OK.

    L'administration sécurisée est désactivée et la page de redémarrage du serveur apparaît.
