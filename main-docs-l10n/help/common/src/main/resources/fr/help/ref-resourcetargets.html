Cibles de ressources

La page Cibles de ressources permet d'afficher les clusters et les instances de serveur autonomes cible sur lesquels la ressource peut être activée.

Elle contient les informations ci-après pour chaque cible.

Nom de cible

Nom du cluster ou de l'instance de serveur autonome.

Activé

Affiche une coche si la ressource est activée sur la cible ou un X si la ressource est désactivée.

La table Cibles contient également les options ci-après.

Gérer les cibles

Bouton permettant de gérer les cibles de ressources.

Activer

Bouton permettant d'activer la ressource sur la cible sélectionnée.

Désactiver

Action permettant de désactiver la ressource sur la cible sélectionnée.
