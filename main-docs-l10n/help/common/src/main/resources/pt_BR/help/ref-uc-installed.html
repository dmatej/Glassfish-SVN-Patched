Componentes Instalados

A página de Componentes Instalados fornece informações sobre componentes da imagem de instalação que ainda não foi instalada.

Para desinstalar quaisquer componentes instalados, interrompa primeiro o domínio do GlassFish Server. A seguir, use um destes comandos:

  • glassfish3/bin/updatetool, que começa com o recurso gráfico da Ferramenta de Atualização

  • glassfish3/bin/pkg, uma versão da linha de comandos da Ferramenta de Atualização

Para obter mais informações sobre o uso do comando pkg, consulte o Capítulo 11, Extending and Updating GlassFish Server, no Oracle GlassFish Server Administration Guide. O utilitário Ferramenta de Atualização tem sua própria ajuda on-line.

Reinicie o servidor após ter finalizado a desinstalação dos componentes.

A página Componentes Instalados contém estas opções:

Caminho da Instalação

O caminho do sistema de arquivos do diretório-raiz ou de nível superior da imagem de instalação.

Componentes Instalados

Para cada componente instalado, esta tabela fornece as seguintes informações.

Componente

O nome do componente.

Categoria

A categoria de software que é associada ao componente; por exemplo, Servidores de Aplicações, Ferramentas do Sistema e Bancos de Dados. A categoria é designada pelo autor do componente quando o componente é criado.

Versão

O número de versão do pacote em que o componente foi fornecido.

Tamanho Instalado

A quantidade de espaço em disco que a instalação do componente consome.

Origem

O nome do Repositório de Rede de onde o componente pode ser obtido.
