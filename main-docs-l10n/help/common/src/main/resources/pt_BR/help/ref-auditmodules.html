Módulos de Auditoria

Use a página Módulos de Auditoria para configurar os módulos de auditoria.

A auditoria é como o GlassFish Server grava eventos significativos, como erros ou vulnerabilidades de segurança. O GlassFish Server registra todos os eventos de autenticação em arquivos de log. Um log de acesso completo fornece uma pista sequencial dos eventos de acesso do GlassFish Server.

Para cada módulo de auditoria, as seguintes informações são fornecidas

Nome

O nome do módulo de auditoria.

Nome da Classe

O nome totalmente qualificado da classe que implementa o módulo. O nome da classe do módulo de auditoria default é com.sun.enterprise.security.Audit .

A tabela Módulos também contém as seguintes opções.

Novo

Botão para criar um novo módulo de auditoria

Deletar

Botão para deletar um ou mais módulos de auditoria selecionados.
