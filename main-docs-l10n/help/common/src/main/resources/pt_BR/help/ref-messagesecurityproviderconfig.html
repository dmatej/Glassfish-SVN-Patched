Configurações do Provedor

Use a página Configurações do Provedor para definir as configurações do provedor de segurança de mensagens para a camada e autenticação do SOAP ou do HttpServlet.

Por default, quatro provedores são configurados para a camada de autenticação SOAP.

Para cada configuração de provedor, as seguintes informações são fornecidas.

ID do Provedor

O nome da configuração do provedor.

Tipo de Provedor

O tipo do provedor, que pode ser qualquer um dos seguintes:

client

Especifica que o provedor é o provedor de autenticação do cliente.

server

Especifica que o provedor é o provedor de autenticação do servidor.

client-server

Especifica que o provedor é um provedor de autenticação do cliente e do servidor.

Provedor Default

Se definido como true, o provedor será o provedor default.

Nome da Classe

O nome da classe de implementação Java do provedor.

A tabela Configurações do Provedor também contém as seguintes opções.

Novo

Botão para criar uma nova configuração de provedor.

Deletar

Botão para deletar uma ou mais configurações do provedor selecionadas.
