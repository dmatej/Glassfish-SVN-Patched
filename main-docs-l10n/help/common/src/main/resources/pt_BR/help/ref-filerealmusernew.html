Novo Usuário de Realm de Arquivos

A página Novo Usuário de Realm de Arquivos concede acesso ao realm atual para um novo usuário.

Observação:

Se a administração segura for ativada, não será possível criar um usuário admin-realm com uma senha vazia.

A página Novo Usuário de Realm de Arquivos contém as opções a seguir.

Nome da Configuração

O nome da configuração à qual as definições nesta página se aplicam. Este campo é somente para leitura.

Nome do Realm

O nome do realm selecionado atualmente. Este é um campo somente para leitura.

ID do Usuário

O ID de usuário da nova conta de usuário.

Lista de Grupos

Os grupos aos quais o novo usuário pertence. Os nomes dos vários grupos são separados por vírgulas.

Para usuários do admin-realm, esse é um campo somente para leitura, e o grupo é asadmin.

Nova Senha

Senha do novo usuário.

Confirmar Nova Senha

Senha do novo usuário.
