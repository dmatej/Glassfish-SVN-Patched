Propriedades do Sistema

Usar a página Propriedades do Sistema para exibir e gerenciar propriedades do GlassFish Server.

Uma propriedade do sistema define um valor comum de uma definição no nível do servidor. Você pode se referir a uma propriedade do sistema em um campo de texto da Console de Administração incluindo um cifrão e chaves. Por exemplo, para se referir a uma propriedade denominadaprop-name, use a sintaxe${prop-name}.

Para cada propriedade, a página Propriedades do Sistema contém as seguintes informações.

Nome

O nome da propriedade.

Valor

O valor da propriedade.

Descrição

Uma descrição opcional da propriedade.

Uma série de propriedades do sistema estão predefinidas, mas não são mostradas nesta página. Para obter mais detalhes, consultePropriedades Predefinidas do Sistema.
