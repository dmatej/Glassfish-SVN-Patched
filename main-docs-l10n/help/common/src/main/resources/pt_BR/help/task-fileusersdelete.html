Para Deletar um Usuário do Realm do Arquivo

  1. No nó Configuração, expanda a configuração de cluster ou instância de servidor na qual você deseja gerenciar os usuários de realm administrativos.

  2. Expanda o nó Segurança.

  3. Expanda o nó Realms.

  4. Selecione o nó file.

    A página Editar Realm será aberta.

  5. Na página Editar Realm, clique no botão Gerenciar Usuários.

    A página Usuários do Arquivo será aberta.

  6. Na página Usuários do Arquivo, marque a caixa de seleção à esquerda do nome do usuário a ser deletado.

  7. Clique em Deletar.

    Uma caixa de diálogo pede que você confirme a deleção do usuário.

  8. Na caixa de diálogo, clique em OK.
