Serviço de Monitoramento

Use a página Serviço de Monitoramento para configurar as opções de monitoramento para componentes ou serviços individuais.

É preciso implantar uma aplicação, componente ou serviço, antes de configurar o monitoramento para o mesmo.

A página Serviço de Monitoramento contém estas opções:

Nome da Configuração

O nome da configuração à qual as definições nesta página se aplicam. Este campo é somente para leitura.

Serviço de Monitoramento

Se a caixa de seleção Serviço de Monitoramento Ativado estiver selecionada, o monitoramento será ativado para o GlassFish Server. Por default, a opção está ativada.

MBeans de Monitoramento

Se a caixa de seleção MBeans de Monitoramento Ativados estiver selecionada, todos os MBeans necessários para o monitoramento serão implantados no GlassFish Server. Por default, a opção está ativada.

Se a caixa de seleção não estiver marcada, não será possível exibir os dados de monitoramento no Console de Administração, mesmo se a caixa de seleção Serviço de Monitoramento estiver selecionada. Em vez disso, use a linha de comando para exibir os dados de monitoramento. Se você desmarcar a caixa de seleção, a memória do GlassFish Server será reduzida.

DTrace

Se a caixa de seleção DTrace Ativado for selecionada e você tiver Monitoramento de DTrace do Enterprise Manager do Oracle, você pode usar o recurso Solaris DTrace. Por default, essa opção está desativada.

Para cada componente disponível, as seguintes informações são fornecidas.

Componente

O nome do componente do GlassFish Server.

Nível de Monitoramento

O nível de monitoramento definido pelo componente. Para componente, os níveis disponíveis são LOW, HIGH e OFF. O nível default é OFF. Não há diferenças entre os níveis de monitoramento oferecidos pelas opções LOW e HIGH, nesta versão.

A tabela Definições no Nível de Componente também contém as seguintes opções.

Alterar Nível

Botão para alterar o nível de monitoramento para um ou mais componentes selecionados.

Nível

Lista drop-down a partir da qual é possível selecionar um nível de monitoramento a ser aplicado em todos os componentes selecionados.
