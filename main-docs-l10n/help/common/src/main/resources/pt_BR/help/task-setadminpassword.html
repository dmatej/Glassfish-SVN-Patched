Para Alterar a Senha do Administrador

  1. Na árvore de navegação, selecione o nó Domínio.

    A página Atributos do Domínio será aberta.

  2. Selecione a guia Senha do Administrador.

    A página Nova Senha do Administrador será aberta.

    A página exibe o ID do usuário e o nome do grupo da conta do administrador atual. Por default, o ID do usuário é admin e o nome do grupo é asadmin.

  3. No campo Nova Senha, digite a nova senha da conta do administrador atual.

  4. No campo Confirmar Nova Senha, torne a digitar a nova senha.

  5. Clique em Salvar.
