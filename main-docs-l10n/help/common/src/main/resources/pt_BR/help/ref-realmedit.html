Editar Realm

Use a página Editar Realm para modificar um realm.

A página Editar Realm contém as opções a seguir.

Nome da Configuração

O nome da configuração à qual as definições nesta página se aplicam. Este campo é somente para leitura.

Nome do Realm

O nome do realm. Este é um campo somente para leitura. Você só pode especificar o nome quando cria um novo realm.

Nome da Classe

O nome da classe do realm que está sendo editado.

Propriedades Adicionais

Propriedades adicionais do realm. As propriedades válidas dependem do tipo de realm.

  • Propriedades Específicas para a Classe FileRealm

  • Propriedades Específicas para a Classe CertificateRealm

  • Propriedades Específicas para a Classe JDBCRealm

  • Propriedades Específicas para a Classe LDAPRealm

  • Propriedades Específicas para a Classe SolarisRealm

  • Propriedades Específicas para a Classe PamRealm
