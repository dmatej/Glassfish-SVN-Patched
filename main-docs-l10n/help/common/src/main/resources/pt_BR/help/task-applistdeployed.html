Para Exibir uma Lista de Aplicações Implantadas

  1. Na árvore de navegação, selecione o nó Aplicações.

    A página Aplicações será aberta. A tabela lista as aplicações implantadas.

  2. Selecione um tipo de módulo na lista drop-down Filtrar para filtrar a lista de aplicações por tipo de módulo.
