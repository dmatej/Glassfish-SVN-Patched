Novo Módulo de Auditoria

Use a página Novo Módulo de Auditoria para defini-lo.

A página Novo Módulo de Auditoria contém as seguintes opções.

Nome da Configuração

O nome da configuração à qual as definições nesta página se aplicam. Este campo é somente para leitura.

Nome

Nome do novo módulo de auditoria.

Nome da Classe

O nome totalmente qualificado da classe que implementa o módulo. O nome da classe do módulo de auditoria default é com.sun.enterprise.security.Audit .

Propriedades Adicionais

Propriedades adicionais do módulo de auditoria. Para obter uma descrição da propriedade adicional que está disponível para módulos de auditoria, consulte Propriedades Específicas para os Módulos de Auditoria.
