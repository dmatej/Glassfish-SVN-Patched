Configurações

Use a página Configurações para gerenciar configurações e exibir as instâncias do servidor standalone e clusters usando as configurações.

A configuração default-config é predefinida por GlassFish Server A configuração server-config é criada automaticamente para o servidor de administração de domínios (DAS) quando o domínio é criado. Para obter mais informações, consulte About Named Configurations em Oracle GlassFish Server High Availability Administration Guide.

Para cada configuração, as seguintes informações são fornecidas.

Nome

O nome da configuração.

Instância

A lista de instâncias standalone e clusters que usam a configuração.

A tabela de Configurações também contém as seguintes opções.

Novo

Botão para criar uma nova configuração.

Deletar

Botão para deletar uma ou mais configurações selecionadas.
