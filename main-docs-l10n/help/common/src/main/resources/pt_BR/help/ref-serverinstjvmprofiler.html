Definições do Gerador de Perfil JVM

Uma ferramenta geradora de perfis que gera dados, os quais são utilizados para analisar o desempenho e identificar o potencial de afunilamento.

Especifique as definições do gerador de perfil. As informações especificadas dependem do produto gerador de perfis em uso. Para obter exemplos e instruções, consulte Capítulo 3, Debugging Applications, em Oracle GlassFish Server Application Development Guide.
