Realms

O GlassFish Server vem pré-configurado com estes três realms: file, certificate e admin-realm. Também é possível criar ldap, JDBC, solaris e realms personalizados. Geralmente, você tem um realm de cada tipo em um servidor, mas, no GlassFish Server há dois realms de arquivo: file e admin-realm. Há dois realms do mesmo tipo, mas usados para finalidades diferentes: É possível também ter um banco de dados de certificado para cada servidor virtual do sistema.

Use a página Realms para configurar os realms.

Para cada realm, as informações a seguir são apresentadas.

Nome

O nome do realm.

Nome da Classe

O nome da classe para o realm.

A tabela Realms também contém as opções a seguir.

Novo

Botão para criar um novo realm.

Deletar

Botão para deletar um ou mais realms selecionados.
