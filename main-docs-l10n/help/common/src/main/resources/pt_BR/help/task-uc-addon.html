Para Instalar Componentes Add-On

Instalar um componente add-on envolve as operações a seguir:

  • Transferir o conteúdo do componente e as respectivas dependências de um repositório da rede

  • Instalar o componente sob a imagem de instalação do GlassFish Server no seu sistema

Se um componente precisar de qualquer outro componente ainda não instalado, os componentes necessários também serão instalados.

  1. Na árvore de navegação, selecione o nó Ferramenta de Atualização.

    É aberta a página Add-ons Disponíveis.

  2. Na página Add-ons Disponíveis, na tabela de componentes add-on disponíveis, selecione os componentes que estão sendo instalados.

    Se não houver componentes add-on, a tabela estará vazia.

    • Para selecionar um componente individual, marque a caixa de seleção adjacente ao nome do componente.

    • Para selecionar todos os componentes, clique no ícone Selecionar Todos, no cabeçalho da tabela.

    • Para desmarcar todos os componentes, clique no ícone Selecionar Todos, no cabeçalho da tabela Desmarcar Tudo.

  3. Clique em Instalar.

    Uma caixa de diálogo pede que você confirme se devem ser instalados os componentes selecionados e qualquer outro componente necessário.

  4. Na caixa de diálogo, clique em OK.

    Será solicitado que você aceite os contratos de licença dos componentes que estiver instalando.

  5. Aceite os contratos de licença dos componentes que está instalando.

    O GlassFish Server confirma que a instalação está concluída. Os componentes são removidos da tabela de componentes add-on e adicionados à tabela de componentes instalados na página Componentes Instalados.

  6. Reinicie o servidor.

    Para obter instruções detalhadas, consulte a seguinte documentação:

    • To Stop a Domain no Oracle GlassFish Server Administration Guide

    • To Start a Domain no Oracle GlassFish Server Administration Guide
