Para Configurar as Definições de Segurança

O Console de Administração permite que você defina uma variedade de definições de segurança em todo o sistema.

  1. Na árvore de navegação, expanda o nó Configurações.

  2. No nó Configurações, clique na configuração de cluster ou instância de servidor na qual deseja configurar as definições de Segurança.

    A página Configuração será aberta.

  3. Na página Configuração, clique em Segurança.

    A página Definições de segurança do alvo da configuração selecionada será aberta.

  4. Marque a caixa de seleção Gerenciador de Segurança Ativado para ativar o gerenciador de segurança.

    Se essa opção estiver ativada, uma opção da JVM, -Djava.security.manager, será adicionada às definições da JVM do GlassFish Server. Consulte Para Configurar as Opções JVM. É preciso reiniciar o servidor para ativar a alteração.

    Verifique se você tem as permissões corretas para todas as aplicações. Para melhorar o desempenho, desative o gerenciador de segurança. Por default, essa opção está desativada.

  5. Marque a caixa de seleção Log de Auditoria Ativado para ativar o log de auditoria.

    Se essa opção for ativada, o servidor carregará e executará todos os módulos de auditoria selecionados na lista Módulos de Auditoria. Se essa opção for desativada, o servidor não acessará os módulos de auditoria. Por default, essa opção está desativada.

  6. Na lista drop-down Realm Default, selecione um realm ativo que o servidor usará para a autenticação.

    As aplicações irão usar esse realm, a menos que o descritor de implantação especifique outro realm. Todos os realms configurados são exibidos na lista. O valor default é file.

  7. Especifique a Informação do Principal Default.

    1. No campo Principal Default, digite o nome de usuário default.

      O servidor usa esse nome de usuário quando nenhum outro principal é fornecido.

    2. No campo Senha do Principal Default, digite a senha do principal default especificado no campo Principal Default.

    Um principal default não é exigido para operação normal do servidor.

  8. Na lista drop-down do JACC, selecione o nome de um provedor de JACC configurado.

    As opções default são default e simple. A opção default é default. Para obter informações sobre a criação de novos provedores de JACC, consulte ***.

  9. Na lista drop-down Módulos de Auditoria, selecione os módulos do provedor de auditoria que serão usados pelo subsistema de auditoria, se o log de auditoria estiver ativado.

    Por default, o servidor usa um módulo de auditoria chamado default. Para obter informações sobre como criar novos módulos de auditoria, consulte Para Criar um Módulo de Auditoria.

  10. Marque a caixa de seleção Mapeamento Ativado de Principal Default para Atribuição para aplicar um mapeamento de principal default para atribuição nas aplicações que não têm um mapeamento específico de aplicação.

  11. Se você tiver ativado o mapeamento principal para atribuição default, no campo Classe Principal Mapeada, digite o nome de uma classe de implementação java.security.Principal personalizada a ser usada no mapeamento principal para atribuição default.

  12. Na seção Propriedades Adicionais, especifique as propriedades adicionais.

    As propriedades válidas são dependentes do tipo de realm selecionado no campo Realm Default. Consulte a entrada apropriada na lista seguinte para obter o tipo de realm que deseja configurar.

    • Propriedades Específicas para a Classe FileRealm

    • Propriedades Específicas para a Classe CertificateRealm

    • Propriedades Específicas para a Classe JDBCRealm

    • Propriedades Específicas para a Classe LDAPRealm

    • Propriedades Específicas para a Classe SolarisRealm

    • Propriedades Específicas para a Classe PamRealm

    • Para adicionar uma propriedade, clique no botão Adicionar Propriedade. Na linha em branco que for exibida, digite o nome da propriedade no campo Nome e digite o valor da propriedade no campo Valor.

    • Para modificar uma propriedade, edite o campo Valor da propriedade.

    • Para deletar uma propriedade, marque a caixa de seleção à esquerda do campo Nome da Propriedade a ser deletada. Clique no botão Deletar Propriedades.

  13. Clique em Salvar.
