Adicionar Recursos

Use a página Adicionar Recursos para criar um ou mais recursos de um arquivo glassfish-resources.xml no alvo especificado.

A página Adicionar Recursos contém as seguintes informações.

Local

A localização do arquivo glassfih-resources.xml:

Arquivo XML a Ser Submetido a Upload para o Servidor

Caminho para o arquivo em uma máquina remota.

Arquivo de XML Local que está Acessível de GlassFish Server

Caminho para o arquivo na máquina do servidor de administração de domínios (DAS).

Alvo

Uma lista drop-down de clusters e instâncias do servidor standalone.
