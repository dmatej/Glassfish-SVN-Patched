Ferramenta de Atualização Conceitos-Chave

O Ferramenta de Atualização gerencia as instalações de componentes de software do GlassFish Server que estão contidos em um único diretório do sistema de arquivos do seu sistema.

A versão Console de Administração do Ferramenta de Atualização permite efetuar o seguinte:

  • Instalar componentes add-on

  • Exibir as atualizações para os componentes instalados

  • Exibir os componentes instalados

Para atualizar ou remover os componentes instalados, é preciso parar o domínio do GlassFish Server, e a seguir usar um dos seguintes comandos:

  • as-install-parent/bin/updatetool, que inicia o utilitário gráfico da Ferramenta de Atualização

  • as-install-parent/bin/pkg, uma versão da linha de comandos da Ferramenta de Atualização

Os principais conceitos do Ferramenta de Atualização estão descritos nas seções a seguir.

Componente

Um componente é a unidade que pode ser instalada que Ferramenta de Atualização gerencia. Um componente consiste em arquivos que são instalados como uma unidade no sistema, em uma imagem de instalação.

Imagem de Instalação

Uma imagem da instalação é o diretório que contém os componentes do software, incluindo o conteúdo do diretório. A imagem de instalação é adicionada ao Ferramenta de Atualização como resultado do download e da instalação do software GlassFish Server. Para ativar a adição e a atualização dos componentes de software do GlassFish Server, cada imagem de instalação contém informações sobre a instalação de componentes.

Repositório de Rede

Um repositório da rede contém os seguintes itens:

  • Componentes que podem ser adicionados a uma imagem de instalação

  • Atualizações para componentes existentes em uma imagem de instalação

Uma imagem de instalação é associada a um ou mais repositórios de rede. Cada repositório de rede é representado por um nome e um URL associado. Um nome de repositório é exclusivo para cada imagem de instalação e representa a organização ou o projeto que gerencia o repositório e um escopo bem definido de conteúdo. Para saber mais sobre os repositórios de rede preconfigurados para GlassFish Server, consultePreconfigured Repositories for GlassFish Server no &DSADG;.

Componente Add-on

Um componente add-on é um componente da imagem de instalação que ainda não foi instalada.
