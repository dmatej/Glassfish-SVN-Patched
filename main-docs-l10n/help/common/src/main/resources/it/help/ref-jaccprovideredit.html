Modifica provider JACC

Utilizzare la pagina Modifica provider JACC per modificare un provider JACC per l'istanza o il cluster GlassFish Server selezionato.

La pagina Modifica provider JACC contiene le opzioni riportate di seguito.

Nome configurazione

Il nome della configurazione a cui si applicano le impostazioni presenti in questa pagina. Questo campo è di sola lettura.

Nome

Il nome del provider JACC. Questo campo è di sola lettura. È possibile specificare il nome solo quando si crea un nuovo provider JACC.

Configurazione criteri

Il nome di classe completamente qualificato per javax.security.jacc.PolicyConfigurationFactory.provider che implementa la classe javax.security.jacc.PolicyConfigurationFactory specifica del provider.

Provider criteri

Il nome di classe completamente qualificato per javax.security.jacc.policy.provider che implementa java.security.Policy.

Proprietà aggiuntive

Proprietà aggiuntive per il provider JACC. Per la descrizione delle proprietà aggiuntive disponibili per i provider JACC, vedere Proprietà specifiche dei provider JACC.
