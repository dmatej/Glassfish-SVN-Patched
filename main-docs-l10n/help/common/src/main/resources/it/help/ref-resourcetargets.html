Destinazioni risorse

Utilizzare la pagina Destinazioni risorse per visualizzare i cluster e le istanze server standalone di destinazione in cui può essere abilitata la risorsa.

Per ogni destinazione la pagina Destinazioni risorse contiene le informazioni riportate di seguito.

Nome destinazione

Il nome del cluster o dell'istanza server standalone.

Abilitato

Visualizza un segno di spunta se la risorsa è abilitata nella destinazione oppure una X se la risorsa è disabilitata.

La tabella Destinazioni contiene inoltre le opzioni riportate di seguito.

Gestisci destinazioni

Pulsante che consente di gestire le destinazioni risorse.

Abilita

Pulsante che consente di abilitare la risorsa nella destinazione selezionata.

Disabilita

Azione che consente di disabilitare la risorsa nella destinazione selezionata.
