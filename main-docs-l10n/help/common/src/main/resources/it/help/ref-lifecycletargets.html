Destinazioni moduli del ciclo di vita

Utilizzare la pagina Destinazioni moduli del ciclo di vita per visualizzare i cluster e le istanze server standalone di destinazione in cui può essere abilitato il modulo del ciclo di vita.

La pagina Destinazioni moduli del ciclo di vita contiene le informazioni riportate di seguito.

Nome destinazione

Il nome del cluster o dell'istanza server standalone.

Abilitato

Visualizza true se il modulo del ciclo di vita è abilitato nella destinazione oppure false se il modulo del ciclo di vita è disabilitato.

La tabella Destinazioni contiene inoltre le opzioni riportate di seguito.

Gestisci destinazioni

Pulsante che consente di gestire le destinazioni dei moduli del ciclo di vita.

Altre azioni

Elenco a discesa che contiene le azioni riportate di seguito.

Abilita

Azione che consente di abilitare il modulo del ciclo di vita nella destinazione selezionata.

Disabilita

Azione che consente di disabilitare il modulo del ciclo di vita nella destinazione selezionata.
