Utenti amministratori

Utilizzare la pagina Utenti amministratori per gestire gli utenti di admin-realm. Il realm admin-realm è un realm file. Gli utenti e i gruppi nel realm file vengono elencati nel file di chiave, la cui posizione è specificata dalla proprietà file.

Inizialmente, dopo l'installazione di GlassFish Server, l'unico utente è l'amministratore immesso durante l'installazione. Per impostazione predefinita, questo utente appartiene al gruppo asadmin, nel realm admin-realm, che dispone dei diritti per la modifica di GlassFish Server. Qualsiasi utente assegnato a questo gruppo disporrà dei privilegi di amministratore, ovvero avrà accesso allo strumento asadmin e alla Console di amministrazione.

Nella pagina Utenti amministratori viene visualizzata la lista degli utenti per il realm selezionato. Per ogni utente vengono fornite le informazioni riportate di seguito.

ID utente

Il nome dell'utente.

Lista gruppi

I gruppi di appartenenza dell'utente.

La tabella Utenti amministratori contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare un nuovo utente.

Elimina

Pulsante che consente di eliminare uno o più utenti selezionati.
