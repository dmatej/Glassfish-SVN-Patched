Per aggiungere un utente al realm file

  1. Nel nodo Configurazioni espandere la configurazione dell'istanza server o del cluster per cui si desidera gestire gli utenti del realm admin.

  2. Espandere il nodo Sicurezza.

  3. Espandere il nodo Realm.

  4. Selezionare il nodo file.

    Viene visualizzata la pagina Modifica realm.

  5. Nella pagina Modifica realm fare clic sul pulsante Gestisci utenti.

    Viene visualizzata la pagina Utenti file.

  6. Nella pagina Utenti file fare clic su Nuovo.

    Viene visualizzata la pagina Nuovo utente realm.

  7. Nel campo ID utente digitare il nome dell'utente.

  8. Nel campo Lista gruppi digitare la lista dei gruppi di appartenenza dell'utente.

    Utilizzare le virgole per separare i nomi dei gruppi. Non è necessario definire altrove tali gruppi.

  9. Nel campo Password digitare una password univoca.

  10. Nel campo Conferma nuova password digitare di nuovo la password dell'utente.

  11. Fare clic su OK.
