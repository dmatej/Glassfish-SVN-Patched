Utenti file

Utilizzare la pagina Utenti file per gestire gli utenti del realm file. Gli utenti e i gruppi nel realm file vengono elencati nel file di chiave, la cui posizione è specificata dalla proprietà file.

Un utente nel realm file può appartenere a un gruppo Java, una categoria di utenti con caratteristiche comuni. Ad esempio, gli utenti di un'applicazione di e-commerce potrebbero far parte del gruppo CUSTOMER, ma quelli che spendono di più verrebbero inclusi nel gruppo PREFERRED. La suddivisione degli utenti in categorie di gruppi semplifica il controllo dell'accesso di numerosi utenti.

Inizialmente, dopo l'installazione di GlassFish Server, l'unico utente è l'amministratore immesso durante l'installazione. Per impostazione predefinita, questo utente appartiene al gruppo asadmin, nel realm admin-realm, che dispone dei diritti per la modifica di GlassFish Server. Qualsiasi utente assegnato a questo gruppo disporrà dei privilegi di amministratore, ovvero avrà accesso allo strumento asadmin e alla Console di amministrazione.

Nella pagina Utenti file viene visualizzata la lista degli utenti per il realm selezionato. Per ogni utente vengono fornite le informazioni riportate di seguito.

ID utente

Il nome dell'utente.

Lista gruppi

I gruppi di appartenenza dell'utente.

La tabella Utenti file contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare un nuovo utente.

Elimina

Pulsante che consente di eliminare uno o più utenti selezionati.
