Contratto di licenza d'uso

Nella pagina Contratto di licenza d'uso vengono visualizzati i contratti di licenza di tutti i componenti selezionati per l'installazione o l'aggiornamento. Il contratto di licenza di un componente viene fornito dall'autore insieme al componente. Nel Repository di rete è possibile pubblicare componenti diversi con licenze differenti, ad esempio:

  • Licenza comune di distribuzione e sviluppo (CDDL) (http://www.sun.com/cddl/)

  • Licenza pubblica generale (GPL) GNU (http://www.gnu.org/copyleft/gpl.html)

  • Licenza Apache (http://www.apache.org/licenses/)

  • Licenza della libreria wxWindows (http://www.opensource.org/licenses/wxwindows.php)

La pagina Contratto di licenza d'uso contiene le opzioni riportate di seguito.

Accetta

Questo pulsante consente di accettare le condizioni delle licenze di tutti i componenti selezionati e di procedere all'installazione o all'aggiornamento.

Rifiuta

Questo pulsante consente di rifiutare le condizioni delle licenze di tutti i componenti selezionati e di tornare alla pagina di selezione dei componenti.

Task correlati

  • Per visualizzare gli aggiornamenti disponibili per i componenti

  • Per installare i componenti aggiuntivi
