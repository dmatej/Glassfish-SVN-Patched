Host JMS

Utilizzare la pagina Host JMS per configurare gli host di servizio JMS.

Per ogni host vengono fornite le informazioni riportate di seguito.

Nome

Il nome dell'host JMS.

Host

Il nome o l'indirizzo IP (Internet Protocol) del sistema su cui è in esecuzione l'host JMS.

Porta

Il numero di porta del servizio JMS sull'host.

La tabella Host contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare un nuovo host JMS.

Elimina

Pulsante che consente di eliminare uno o più host JMS selezionati.
