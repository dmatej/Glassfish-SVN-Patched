Propriedades Específicas para o Serviço JMS

As propriedades de configuração do broker do Message Queue do Oracle a seguir estão disponíveis para o serviço JMS.

instance-name

Especifica o nome da instância do broker do Message Queue. O default é imqbroker.

instance-name-suffix

Especifica um sufixo a ser adicionado ao nome completo da instância do broker do Message Queue. O sufixo é separado do nome da instância por um caractere sublinhado (_). Por exemplo, se o nome da instância for imqbroker, acrescentar o sufixo xyz altera o nome da instância para imqbroker_xyz.

append-version

Se for verdadeiro, acrescenta os números da versão menor e da maior, precedidos de caracteres sublinhados (_), até o nome da instância do broker do Message Queue. Por exemplo, se o nome da instância for imqbroker, acrescentar os números da versão altera o nome da instância para imqbroker_8_0. O default é false.

user-name

O nome de usuário para a criação da conexão JMS. É necessário somente se nome de usuário/senha do convidado/convidado não estiver disponível no controlador. O valor default é guest.

password

A senha para a criação da conexão JMS. É necessário somente se nome de usuário/senha do convidado/convidado não estiver disponível no controlador. O valor default é guest.
