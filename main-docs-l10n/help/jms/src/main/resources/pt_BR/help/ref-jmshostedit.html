Editar Host JMS

Utilize a página Editar Host JMS para modificar as definições de um host JMS.

A página Editar Host JMS contém as opções a seguir.

Carregar Defaults

Botão para restaurar definições para seus valores default. Definições que não tiveram seus valores default alterados.

Nome da Configuração

O nome da configuração à qual as definições nesta página se aplicam. Este campo é somente para leitura.

Nome

O nome do host. O campo Nome é somente para leitura. Você só pode especificar o campo Nome ao criar um novo host JMS.

Host

O nome ou o endereço do Protocolo Internet (IP) do sistema em que o host JMS executará (localhost ou o nome do sistema local ou remoto).

Porta

O número da porta do serviço JMS. Altere esse campo somente se o serviço JMS a ser utilizado estiver sendo executado em uma porta não default. A porta default é 7676.

Nome de Usuário e Senha Admin

O nome de usuário e a senha do broker do Message Queue. São diferentes do nome de usuário e da senha do GlassFish Server. Edite esses campos somente se os valores do broker do Message Queue tiverem sido alterados com o comando as-install-parent/mq/bin/imqusermgr. Os valores default são admin e admin.
