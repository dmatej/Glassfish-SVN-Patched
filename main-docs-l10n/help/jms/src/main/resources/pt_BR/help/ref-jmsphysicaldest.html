Destinos Físicos do JMS

Os objetos de destino físico JMS (Java Message Service) são mantidos pelos brokers do Message Queue. A fila nomeada mq.sys.dmq é o destino do sistema para o qual mensagens expiradas ou não entregues são redirecionadas.

Utilize a página Destinos Físicos JMS para configurar destinos físicos do Message Queue.

Para cada destino físico, as informações a seguir são fornecidas.

Nome

O nome do destino físico.

Tipo

O tipo do destino físico.

Estatística

Estatística do destino físico.

O quadro Destinos também contêm as opções a seguir.

Novo

Botão utilizado para criar um novo destino físico.

Deletar

Botão para deletar um ou mais destinos físicos selecionados.

Deletar Todas as Mensagens

Botão para remover mensagens de um ou mais destinos físicos selecionados.
