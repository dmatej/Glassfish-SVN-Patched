JMS物理宛先の統計を表示するには

  1. ナビゲーション・ツリーで、「構成」ノードを展開します。

  2. 「構成」ノードで、「Javaメッセージ・サービス」ノードを展開します。

  3. 「Javaメッセージ・サービス」ノードで、「物理宛先」ノードを選択します。

    「物理宛先」ページが開きます。

  4. 「物理宛先」ページで、統計を表示する宛先の「表示」のリンクをクリックします。

    「JMS物理宛先の統計の表示」ページが開きます。

  5. 統計の確認が終了したら、「戻る」ボタンをクリックします。
