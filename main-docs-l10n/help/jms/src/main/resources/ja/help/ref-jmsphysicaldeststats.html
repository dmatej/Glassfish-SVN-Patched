JMS物理宛先の統計の表示

「JMS物理宛先の統計の表示」ページにはJMS物理宛先の使用状況に関する情報が表示されます。

宛先の名前とタイプの他に、次の詳細な情報が表示されます。

  • プロデューサとコンシューマの数

  • 宛先で処理されたメッセージの数とサイズ

  • ディスクの利用率
