JMS接続ファクトリ

JMS接続ファクトリ・リソースを構成するには、「JMS接続ファクトリ」ページを使用します。

リソースごとに、次の情報が表示されます。

プール名

接続ファクトリを識別する一意の名前。

JNDI名

接続ファクトリを識別するJNDI名。

有効

接続ファクトリが有効な場合はtrue、接続ファクトリが無効の場合はfalseです。

リソース・タイプ

接続ファクトリのリソース・タイプ。

説明

接続ファクトリの説明。

「接続ファクトリ」の表には、次のオプションも表示されます。

新規

新規接続ファクトリを作成するボタン。

削除

選択した1つまたは複数の接続ファクトリを削除するボタン。

有効

選択した1つまたは複数の接続ファクトリを有効にするボタン。

無効

選択した1つまたは複数の接続ファクトリを無効にするボタン。
