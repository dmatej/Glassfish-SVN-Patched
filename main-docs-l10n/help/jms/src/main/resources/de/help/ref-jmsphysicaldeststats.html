Statistiken für physische JMS-Ziele anzeigen

Auf der Seite "Statistiken für physische JMS-Ziele anzeigen" werden Informationen über die Verwendung eines physischen JMS-Ziels angezeigt.

Neben Name und Typ des Ziels enthält die Seite detaillierte Informationen zu folgenden Aspekten:

  • Anzahl Producer und Consumer

  • Anzahl und Größe der vom Ziel verwalteten Nachrichten

  • Datenträgernutzung

Verwandte Aufgaben

  • So wird der JMS-Service konfiguriert

  • So wird ein physisches JMS-Ziel erstellt

  • So wird ein physisches JMS-Ziel bearbeitet

  • So wird ein physisches JMS-Ziel gelöscht

  • So werden Nachrichten aus einem physischen JMS-Ziel entfernt

  • So werden Statistiken für ein physisches JMS-Ziel angezeigt

Zugehörige asadmin-Befehle

  • create-jmsdest(1)

  • list-jmsdest(1)

  • delete-jmsdest(1)

  • flush-jmsdest(1)

  • jms-ping(1)
