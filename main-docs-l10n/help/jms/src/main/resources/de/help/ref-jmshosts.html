JMS-Hosts

Auf der Seite "JMS-Hosts" können Sie JMS-Servicehosts konfigurieren.

Für jeden Host werden die folgenden Informationen zur Verfügung gestellt.

Name

Der Name des JMS-Hosts.

Host

Der Name oder die IP-Adresse (Internet Protocol) des Systems, auf dem der JMS-Host ausgeführt wird.

Port

Die Portnummer des JMS-Service auf dem Host.

Die Tabelle "Hosts" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen eines neuen JMS-Hosts.

Löschen

Schaltfläche zum Löschen von mindestens einem ausgewählten JMS-Host.

Verwandte Aufgaben

  • So wird der JMS-Service konfiguriert

  • So wird ein JMS-Host erstellt

  • So wird ein JMS-Host bearbeitet

  • So wird ein JMS-Host gelöscht

Zugehörige asadmin-Befehle

  • create-jms-host(1)

  • list-jms-hosts(1)

  • delete-jms-host(1)
