Propriétés propres au service JMS

Les propriétés suivantes de configuration du broker Message Queue de Oracle sont disponibles pour le service JMS.

instance-name

Indique le nom complet de l'instance de broker Message Queue. La valeur par défaut est imqbroker.

instance-name-suffix

Spécifie un suffixe à ajouter au nom complet de l'instance de broker Message Queue. Le suffixe est séparé du nom de l'instance par un trait de soulignement (_). Par exemple, si le nom de l'instance est imqbroker, l'ajout du suffixe xyz donne le nom imqbroker_xyz.

append-version

Si cette propriété a la valeur true, ajoute au nom d'instance de broker Message Queue les numéros de version principale et secondaire, précédés par des traits de soulignement (_). Par exemple, si le nom de l'instance est imqbroker, l'ajout des numéros de version donne le nom imqbroker_8_0. La valeur par défaut est false.

user-name

Nom utilisateur employé pour la création de la connexion JMS. Nécessaire uniquement si le nom utilisateur ou le mot de passe par défaut (guest/guest) n'est pas disponible dans le broker. La valeur par défaut est guest.

password

Mot de passe employé pour la création de la connexion JMS. Nécessaire uniquement si le nom utilisateur ou le mot de passe par défaut (guest/guest) n'est pas disponible dans le broker. La valeur par défaut est guest.
