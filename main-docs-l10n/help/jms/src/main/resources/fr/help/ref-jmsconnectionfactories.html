Fabriques de connexions JMS

La page Fabriques de connexions JMS permet de configurer les ressources de fabrique de connexions JMS.

Pour chaque ressource, les informations suivantes sont fournies.

Nom du pool

Nom unique identifiant la fabrique de connexions.

Nom JNDI

Nom JNDI identifiant la fabrique de connexions.

Activé

True si la fabrique de connexions est activée ou False si la fabrique de connexions est désactivée.

Type de ressource

Type de ressource de la fabrique de connexions.

Description

Description de la fabrique de connexions.

La table Fabriques de connexions contient également les options ci-après.

Nouveau

Bouton permettant de créer une fabrique de connexions.

Supprimer

Bouton permettant de supprimer les fabriques de connexions sélectionnées.

Activer

Bouton permettant d'activer les fabriques de connexions sélectionnées.

Désactiver

Bouton permettant de désactiver les fabriques de connexions sélectionnées.
