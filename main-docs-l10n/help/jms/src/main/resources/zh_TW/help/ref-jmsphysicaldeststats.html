檢視 JMS 實體目的地統計

「檢視 JMS 實體目的地統計」頁面會顯示有關使用 JMS 實體目的地的資訊。

除了目的地的名稱和類型之外，頁面會提供有關以下方面的詳細資訊：

  • 產生器和用戶數目

  • 目的地所處理的訊息數目和大小

  • 磁碟使用率
