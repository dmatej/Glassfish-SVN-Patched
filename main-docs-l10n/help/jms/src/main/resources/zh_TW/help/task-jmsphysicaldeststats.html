檢視 JMS 實體目的地的統計

  1. 在瀏覽樹狀結構中，展開「組態」節點。

  2. 在「組態」節點下，展開「Java 訊息服務」節點。

  3. 在「Java 訊息服務」節點下，選取「實體目的地」節點。

    將開啟「實體目的地」頁面。

  4. 在「實體目的地」頁面上，按一下您要檢視統計之目的地的「檢視」連結。

    將開啟「檢視 JMS 實體目的地統計」頁面。

  5. 當您檢視完統計後，請按一下「返回」按鈕。
