JMS 目的地資源

使用「JMS 目的地資源」頁面可組態 JMS 目的地資源。

對於每個目的地資源，都將提供以下資訊。

JNDI 名稱

可識別目的地資源的唯一名稱。

已啟用

如果資源已啟用，則為 True；如果資源已停用，則為 False。

資源類型

目的地資源的資源類型。

描述

目的地資源的描述。

目的地資源表也包含下列選項。

新建

建立新目的地資源的按鈕。

刪除

刪除一或多個所選取之目的地資源的按鈕。

啟用

啟用一或多個所選取之目的地資源的按鈕。

停用

停用一或多個所選取之目的地資源的按鈕。
