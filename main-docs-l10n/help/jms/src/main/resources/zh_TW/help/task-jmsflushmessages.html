從 JMS 實體目的地移除訊息

  1. 在瀏覽樹狀結構中，展開「組態」節點。

  2. 在「組態」節點下，展開「Java 訊息服務」節點。

  3. 在「Java 訊息服務」節點下，選取「實體目的地」節點。

    將開啟「實體目的地」頁面。

  4. 在「實體目的地」頁面上，選取要移除訊息之目的地名稱旁邊的核取方塊。

  5. 按一下「刪除所有訊息」按鈕。
