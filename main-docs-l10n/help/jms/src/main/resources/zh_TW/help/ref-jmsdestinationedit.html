編輯 JMS 目的地資源

使用「編輯 JMS 目的地資源」頁面可修改 JMS 目的地資源的設定。

「編輯 JMS 目的地資源」頁面包含下列選項。

載入預設值

用於將具有預設值的設定復原為設定值的按鈕。沒有預設值的設定則保持不變。

JNDI 名稱

可識別目的地資源的唯一名稱。「JNDI 名稱」是唯讀欄位。您僅能在建立新的 JMS 目的地資源時指定 JNDI 名稱。

實體目的地名稱

與此資源關聯的實體目的地名稱。

GlassFish Server 會在需要時自動建立實體目的地，並在您刪除目的地資源時將其刪除。

資源類型

目的地資源的類型。可用的選項包括 javax.jms.Topic 和 javax.jms.Queue。

描述

目的地資源的描述。

狀態

目的地資源的狀態。您可啟用或停用目的地資源。此選項預設為啟用。

其他特性

目的地資源的其他特性。GlassFish Server 未定義 JMS 目的地資源的任何其他特性。
