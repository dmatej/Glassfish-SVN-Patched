JMS 目标资源

使用“JMS 目标资源”页可以配置 JMS 目标资源。

对于每个目标资源，将提供以下信息。

JNDI 名称

标识目标资源的唯一名称。

已启用

如果资源已启用，则为“真”；如果资源已禁用，则为“假”。

资源类型

目标资源的资源类型。

说明

目标资源的说明。

“目标资源”表还包含以下选项。

新建

用于创建新目标资源的按钮。

删除

用于删除一个或多个所选目标资源的按钮。

启用

用于启用一个或多个所选目标资源的按钮。

禁用

用于禁用一个或多个所选目标资源的按钮。
