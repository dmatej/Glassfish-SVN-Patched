创建 JMS 主机

  1. 在导航树中，展开“配置”节点。

  2. 选择要配置的实例：

    • 要配置特定的实例，请选择该实例的配置节点。例如，对于默认实例 server，请选择 server-config 节点。

    • 要为以后使用 default-config 副本的实例配置默认设置，请选择 default-config 节点。

  3. 在“配置”节点下，展开“Java 消息服务”节点。

  4. 在“Java 消息服务”节点下，选择“JMS 主机”节点。

    此时将打开“JMS 主机”页。

  5. 在“JMS 主机”页中，单击“新建”。

    此时将显示“新建 JMS 主机”页。

  6. 在“名称”字段中，键入主机的名称。

    例如：

    NewJmsHost

  7. 在“主机”字段中，键入将运行 JMS 主机的系统的名称或 Internet 协议 (IP) 地址 (localhost 或者本地或远程系统的名称)。

  8. 在“端口”字段中，键入 JMS 服务的端口号。

    仅当使用的 JMS 服务在非默认端口上运行时，才需要更改此字段。默认端口为 7676。

  9. 在“管理员用户名”和“管理员口令”字段中，键入消息队列中介用户名和口令。

    该用户名和口令与 GlassFish Server 的用户名和口令不同。仅当已使用as-install-parent/mq/bin/imqusermgr 命令更改了消息队列中介值时，才需要编辑这些字段。默认值为 admin 和 admin。

  10. 单击“确定”。
