特定于 JMS 服务的属性

以下 Oracle消息队列中介配置属性可用于 JMS 服务。

instance-name

指定完整的消息队列中介实例名称。默认值为 imqbroker。

instance-name-suffix

指定要添加到完整的消息队列中介实例名称中的后缀。该后缀与实例名称之间以下划线字符 (_) 分隔。例如，如果实例名称为 imqbroker，则在附加后缀 xyz 之后，实例名称将更改为 imqbroker_xyz。

append-version

如果为真，则应在完整的消息队列中介实例名称后附加主要和次要版本号，这两个版本号之前都有一个下划线字符 (_)。例如，如果实例名为 imqbroker，则在附加版本号之后，实例名称将更改为 imqbroker_8_0。默认值为假。

user-name

用于创建 JMS 连接的用户名。仅当默认用户名/口令 (guest/guest) 在中介中不可用时，才需要使用此属性。默认值为 guest。

password

用于创建 JMS 连接的口令。仅当默认用户名/口令 (guest/guest) 在中介中不可用时，才需要使用此属性。默认值为 guest。
