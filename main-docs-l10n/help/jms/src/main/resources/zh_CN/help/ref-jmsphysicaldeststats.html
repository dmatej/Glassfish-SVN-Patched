查看 JMS 物理目标统计信息

“查看 JMS 物理目标统计信息”页显示有关 JMS 物理目标使用情况的信息。

除了目标的名称和类型以外，该页还提供了有关以下内容的详细信息：

  • 生成器和使用者数量

  • 目标处理的消息的数量和长度

  • 磁盘利用率
