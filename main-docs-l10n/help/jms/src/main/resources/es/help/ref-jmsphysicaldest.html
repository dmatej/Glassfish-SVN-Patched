Destinos físicos JMS

Los brokers de Message Queue mantienen los objetos de destino físico de JMS (Java Message Service). La cola llamada mq.sys.dmq es el destino del sistema al que se redireccionan los mensajes caducados y no enviados.

Utilice la página Destinos físicos JMS para configurar los destinos físicos de Message Queue.

Para cada destino físico, se muestra la siguiente información.

Nombre

Nombre del destino físico.

Tipo

Tipo del destino físico.

Estadísticas

Estadísticas del destino físico.

La tabla Destinos también contiene las siguientes opciones.

Nuevo

Botón para crear un nuevo destino físico.

Suprimir

Botón para suprimir uno o más destinos físicos seleccionados.

Suprimir Todos los Mensajes

Botón para eliminar mensajes de uno o más destinos físicos seleccionados.
